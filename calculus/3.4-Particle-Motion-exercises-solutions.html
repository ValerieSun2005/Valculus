3.4 Exercise Solutions: Particle Motion

EXERCISE 1
The distance from your apartment to the store is \(2\) miles. If you walk there and back, then calculate your displacement and distance traveled.
SOLUTION The displacement is \(\boxed{\Delta s = 0 \un{mi}}\) because your position undergoes no net change (that is, you end at the same place you start). But the distance is \(\boxed{d = 4 \un{mi}}.\)
EXERCISE 2
A ball rolls along the \(x\)-axis with time \(t;\) the positive direction is to the right. If \(v(3) \gt 0\) and \(a(3) \lt 0,\) then fully describe the ball's motion at \(t = 3.\)
SOLUTION Since \(v(3) \gt 0,\) the ball is moving to the right. But since \(a(3) \lt 0,\) the ball experiences a push to the left, meaning it is slowing down at \(t = 3.\)
EXERCISE 3
For
\[\ds s(t) = 6t^3 + 8t^2 - 6t \scoll t = 1\]
given the position function, find the position, velocity, and acceleration at the specified time.
SOLUTION The position at \(t = 1\) is \[s(1) = 6(1)^3 + 8(1)^2 - 6(1) = \boxed 8\] Differentiating \(s(t)\) gives the velocity function \[v(t) = 18t^2 + 16t - 6 \pd\] Then the velocity at \(t = 1\) is \[v(1) = 18(1)^2 + 16(1) - 6 = \boxed{28}\] Differentiating \(v(t)\) yields the acceleration function \[a(t) = 36t + 16 \pd\] Substituting \(t = 1\) gives \[a(1) = 36(1) + 16 = \boxed{52}\]
EXERCISE 4
For
\[\ds s(t) = \tfrac{1}{8} e^{2t} - e \scoll t = 3\]
given the position function, find the position, velocity, and acceleration at the specified time.
SOLUTION The position at \(t = 3\) is \[s(3) = \tfrac{1}{8} e^{2(3)} - e = \boxed{\tfrac{1}{8} e^{6} - e}\] Differentiating \(s(t)\) gives the velocity function \[v(t) = \tfrac{1}{4} e^{2t} \pd\] Then the velocity at \(t = 3\) is \[v(3) = \tfrac{1}{4} e^{2(3)} = \boxed{\tfrac{1}{4} e^6}\] Differentiating \(v(t)\) yields the acceleration function \[a(t) = \tfrac{1}{2} e^{2t} \pd\] Substituting \(t = 3\) yields \[a(3) = \tfrac{1}{2} e^{2(3)} = \boxed{\tfrac{1}{2} e^6}\]
EXERCISE 5
For
\[\ds x(t) = t^3 - 3t^2 - 24t + 16 \cmaa 0 \leq t \leq 6\]
given the position function and time interval, calculate the distance traveled.
SOLUTION First let's find the turning points. Differentiating the position function yields the velocity function: \[ \ba v(t) &= 3t^2 - 6t - 24 \nl &= 3(t^2 - 2t - 8) \nl &= 3(t - 4)(t + 2) \pd \ea \] Note that \(v(t) = 0\) for \(t = 4.\) (The other solution \(t = -2\) is outside the interval \([0, 6].\)) We construct the following table; indeed, \(t = 4\) is a turning point because \(v\) changes sign there.
Interval of \(t\) \(v(t) = 3(t - 4)(t + 2)\) Displacement
\((0, 4)\) \(-\) \(x(4) - x(0) = -80\)
\((4, 6)\) \(+\) \(x(6) - x(4) = 44\)
So the total distance traveled is \[d = \abs{-80} + \abs{44} = \boxed{124}\]
EXERCISE 6
For
\[\ds s(t) = \tfrac{1}{2} t^4 - 2t^3 + 9 \cmaa 1 \leq t \leq 4\]
given the position function and time interval, calculate the distance traveled.
SOLUTION First let's find the turning points. Differentiating the position function yields the velocity function: \[ \ba v(t) &= 2t^3 - 6t^2 \nl &= 2t^2 (t - 3) \pd \ea \] Note that \(v(t) = 0\) for \(t = 3.\) (The other solution \(t = 0\) is outside the interval \([1, 4].\)) We construct the following table; indeed, \(t = 3\) is a turning point because \(v\) changes sign there.
Interval of \(t\) \(v(t) = 2t^2 (t - 3)\) Displacement
\((1, 3)\) \(-\) \(s(3) - s(1) = -12\)
\((3, 4)\) \(+\) \(s(4) - s(3) = \tfrac{27}{2}\)
So the total distance traveled is \[d = \abs{-12} + \abs{\tfrac{27}{2}} = \boxed{\tfrac{51}{2}}\]
EXERCISE 7
The height, in feet, of an object in free fall is modeled by the function \[h(t) = -16t^2 - 16t + 96 \cma\] where \(t \geq 0\) is time in seconds.
  1. Find the object's vertical displacement from \(t = 0\) to \(t = 2.\)
  2. Calculate the object's initial speed.
  3. Show that the magnitude of the acceleration due to gravity is \(32\) feet per second per second.
  4. When the object strikes the ground, calculate its speed.
SOLUTION
  1. The vertical displacement from \(t = 0\) to \(t = 2\) is \[h(2) - h(0) = \boxed{-96 \un{ft}}\] The negative sign indicates a decrease in the object's distance from the ground (the position function's point of reference).
  2. Differentiating \(h(t),\) we find the velocity function to be \[v(t) = -32t - 16 \pd\] The word initial means at \(t = 0,\) which we substitute to get \[v(0) = -16 \undiv{ft}{sec} \pd\] The initial speed is the absolute value of this result—namely, \(\boxed{16 \undiv{ft}{sec}}.\)
  3. Differentiating the velocity function from part (b), we get the acceleration function \[a(t) = -32 \pd\] Thus, as requested, the magnitude of the acceleration due to gravity is \(32 \undiv{ft}{sec}^2.\)
  4. The object strikes the ground when \(h(t) = 0.\) Factoring shows \[ \ba h(t) &= -16 \par{t^2 + t - 6} \nl &= -16 (t - 2)(t + 3) \pd \ea \] The positive zero is \(t = 2.\) At this time, the velocity is \[v(2) = -32(2) - 16 = -80 \undiv{ft}{sec} \pd\] The speed is therefore \(\boxed{80 \undiv{ft}{sec}}.\)
EXERCISE 8
Let \(s(t) = e^{-5t} + 2t,\) \(t \geq 0,\) be a position function with time. If the positive direction is to the right, then for what values of \(t\) does a particle move to the left?
SOLUTION A particle moves to the left if its velocity is negative. Differentiating \(s\) gives the velocity function \[v(t) = -5e^{-5t} + 2 \pd\] Solving \(v(t) \lt 0,\) we get \[ \ba -5e^{-5t} + 2 &\lt 0 \nl e^{-5t} &\gt \tfrac{2}{5} \nl -5t &\gt \ln \tfrac{2}{5} \nl t &\lt -\tfrac{1}{5} \ln \tfrac{2}{5} = \tfrac{1}{5} \ln \tfrac{5}{2} \pd \ea \] Since \(t \geq 0\) is given, the particle moves to the left for \(t\) in \[\boxed{\par{0, \tfrac{1}{5} \ln \tfrac{5}{2}}}\]
EXERCISE 9
A motor pushes an object along the \(y\)-axis with time \(t \geq 0.\) The object's position function is \(y(t) = \tfrac{4}{3} t^4 + \ln t,\) and the positive direction is upward. For what values of \(t\) does the motor exert an upward force?
SOLUTION The motor exerts an upward when the acceleration is positive. Differentiating \(y(t)\) gives \[y'(t) = \tfrac{16}{3} t^3 + \frac{1}{t} \pd\] Then differentiating again yields the acceleration function: \[a(t) = y''(t) = 16t^2 - \frac{1}{t^2} \pd\] We solve \(a(t) \gt 0\) as follows: \[ \ba 16t^2 - \frac{1}{t^2} &\gt 0 \nl 16t^2 &\gt \frac{1}{t^2} \nl t^4 &\gt \tfrac{1}{16} \nl t &\gt \sqrt[4]{\tfrac{1}{16}} = \tfrac{1}{2} \pd \ea \] Hence, the motor applies an upward force over \[\boxed{\par{\tfrac{1}{2}, \infty}}\]
EXERCISE 10
A box slides across a rough surface, and its movement is impeded by friction. The box's position, in feet, from an arbitrary reference point as a function of time \(t \geq 0\) (in seconds) is given by \[s(t) = 10 + 20t - 2t^2 \pd\] How far does the box travel before stopping?
SOLUTION Differentiating gives \[v(t) = 20 - 4t \pd\] The object stops when \(v(t) = 0,\) which occurs when \(t = 5.\) So the box's displacement on \([0, 5]\) is \[ \ba \Delta s &= s(5) - s(0)\nl &= \boxed{50 \un{ft}} \ea \] Although \(v(t) \lt 0\) for \(t \gt 5,\) friction doesn't make the box suddenly turn around!
EXERCISE 11
The graph of a particle's velocity against time \(t \geq 0\) is shown in Figure 4. When is the particle
  1. stationary?
  2. changing direction?
  3. moving with a constant speed?
  4. speeding up?
  5. slowing down?
SOLUTION
  1. The particle is stationary when \(v(t) = 0,\) that is, when \(\boxed{t = 1}\) and \(\boxed{t = 5}.\)
  2. When \(v(t)\) changes sign, the particle changes direction. A sign change occurs at \(\boxed{t = 1}.\)
  3. The particle moves with a constant speed if \(v(t)\) is constant—namely, over \(\boxed{6 \leq t \leq 8}.\)
  4. The particle is speeding up when its velocity and acceleration have the same sign. Since acceleration is the derivative of velocity, the particle's acceleration equals the slope of \(y = v(t).\) On \((1, 3)\) and \((5, 6),\) \(v(t) \gt 0\) and \(v'(t) \gt 0,\) so the particle speeds up over these intervals. The union of these intervals is \[\boxed{(1, 3) \cup (5, 6)}\]
  5. The particle is slowing down when its velocity and acceleration have opposite signs. On \((0, 1),\) \(v(t) \lt 0\) but \(v'(t) \gt 0,\) so the particle is slowing down on this interval. Likewise, on \((3, 5)\) we have \(v(t) \gt 0\) but \(v'(t) \lt 0.\) The union of these intervals is \[\boxed{(0, 1) \cup (3, 5)}\]
EXERCISE 12
A particle travels along the \(y\)-axis with time \(t \geq 0\) as modeled by the position function \[y(t) = (\ln t)^2 - \frac{2t}{e} \pd\] Is the particle speeding up, slowing down, or stationary at
  1. \(t = 1 \ques\)
  2. \(t = e \ques\)
SOLUTION The velocity function is \(y'(t),\) or \[v(t) = \frac{2 \ln t}{t} - \frac{2}{e} \pd\] Differentiating again yields the acceleration function \[a(t) = \frac{2 - 2 \ln t}{t^2} \pd\]
  1. We see \[ \baf &&v(1) &= \frac{2 \ln 1}{1} - \frac{2}{e} = -\frac{2}{e} &\nl \laWord{and} &&a(1)&= \frac{2 - 2 \ln 1}{1^2} = 2 \pd \eaf \] Since the velocity and acceleration have opposite signs, the particle is slowing down.
  2. We see \[v(e) = \frac{2 \ln e}{e} - \frac{2}{e} = 0 \pd\] Hence, the particle is stationary.
EXERCISE 13
An object moves back and forth along the \(x\)-axis with time \(t \geq 0,\) as modeled by the position function \[x(t) = e^{-2t} \cos 2t \pd\]
  1. Over \(0 \leq t \leq \pi,\) when does the object have turning points?
  2. Over \(0 \leq t \leq \pi,\) find the farthest-left position and farthest-right position the particle reaches.
  3. Find and interpret \(\lim_{x \to \infty} x(t).\)
SOLUTION By differentiating the position function, we find the velocity function to be \[ \ba v(t) &= -2 e^{-2t} \cos 2t - 2 e^{-2t} \sin 2t \nl &= -2 e^{-2t} (\cos 2t + \sin 2t) \pd \ea \]
  1. The object has turning points when the sign of \(v(t)\) changes. We solve \(v(t) = 0\) as follows: \[ \ba -2 e^{-2t} (\cos 2t + \sin 2t) &= 0 \nl \cos 2t + \sin 2t &= 0 \nl \tan 2t &= -1 \nl \implies t &= \frac{3 \pi}{8} \cma \frac{7 \pi}{8} \pd \ea \] The following table shows the signs of \(v\) over intervals bounded by these values.
    Interval of \(t\) \(v(t) = -2 e^{-2t} (\cos 2t + \sin 2t)\)
    \(\ds \par{0, \frac{3 \pi}{8}}\) \(-\)
    \(\ds \par{\frac{3 \pi}{8}, \frac{7 \pi}{8}}\) \(+\)
    \(\ds \par{\frac{7 \pi}{8}, \pi}\) \(-\)
    Hence, the turning points are at \(\boxed{t = 3 \pi/8}\) and \(\boxed{t = 7 \pi/8}.\)
  2. On \(0 \leq t \leq \pi,\) the farthest-left position is the absolute minimum of \(x(t)\) and the farthest-right position is the absolute maximum of \(x(t).\) The critical points of \(x(t)\) are at \(t = 3 \pi/8\) and \(t = 7 \pi/8.\) By the Closed Interval Method (see Section 3.1), we sample \(x(t)\) at both critical points and the endpoints, \(t = 0\) and \(t = \pi,\) as shown by the following table.
    \(t\) \(x(t) = e^{-2t} \cos 2t\)
    \(0\) \(1\)
    \(\ds \frac{3 \pi}{8}\) \(\ds \frac{-e^{-3 \pi/4} \sqrt 2}{2}\)
    \(\ds \frac{7 \pi}{8}\) \(\ds \frac{e^{-7 \pi/4} \sqrt 2}{2}\)
    \(\pi\) \(e^{-2 \pi}\)
    Hence, the farthest-left position is \(\boxed{x = -e^{3 \pi/4} \sqrt 2/2}\) and the farthest-right position is \(\boxed{x = 1}.\)
  3. Note that \(-1 \leq \cos 2t \leq 1,\) so \[- e^{-2t} \leq e^{-2t} \cos 2t \leq e^{-2t} \pd\] Both \(-e^{-2t}\) and \(e^{2t}\) approach \(0\) as \(t \to \infty,\) so by the Squeeze Theorem (Section 1.3) \(e^{-2t} \cos 2t \to \boxed 0\) as \(t \to \infty.\) Thus, the particle's eventually approaches the origin, \(x = 0.\)
EXERCISE 14
A block is attached to a spring, and the spring is stretched a distance \(A\) past its natural length (equilibrium position), as in Figure 5. The object's vibration with time \(t\) is modeled by the position function \[s(t) = A \cos t \pd\] The block completes one cycle (returning to its initial position) over \(0 \leq t \leq 2 \pi.\) On this interval, when is the block
  1. stationary?
  2. speeding up?
  3. slowing down?
SOLUTION By differentiating the position function, we attain velocity and acceleration functions, respectively, of \[v(t) = -A \sin t \and a(t) = -A \cos t \pd\]
  1. The block is stationary when \(v(t) = -A \sin t\) \(= 0.\) The solutions in \([0, 2 \pi]\) are \[\boxed{\{0, \pi, 2 \pi\}}\]
  2. The block is speeding up when its velocity and acceleration have the same sign. Note that \(a(t) = -A \cos t = 0\) for \(t = \pi/2\) and \(t = 3 \pi/2.\) The following table shows the signs of \(v\) and \(a\) over intervals bounded by their zeros. (Recall that \(A\) is positive.)
    Interval of \(t\) \(v(t) = -A \sin t\) \(a(t) = -A \cos t\)
    \(\ds \par{0, \frac{\pi}{2}}\) \(-\) \(-\)
    \(\ds \par{\frac{\pi}{2}, \pi}\) \(-\) \(+\)
    \(\ds \par{\pi, \frac{3\pi}{2}}\) \(+\) \(+\)
    \(\ds \par{\frac{3\pi}{2}, 2 \pi}\) \(+\) \(-\)
    Hence, the object is speeding up for \(t\) in \[\boxed{\par{0, \frac{\pi}{2}} \cup \par{\pi, \frac{3 \pi}{2}}}\]
  3. The block is slowing down when its velocity and acceleration have opposite signs. From the table in part (b), the block is slowing down on \[\boxed{\par{\frac{\pi}{2}, \pi} \cup \par{\frac{3 \pi}{2}, 2\pi}}\]
EXERCISE 15

EXERCISE 17
If a particle travels along a straight line according to the position function \(s(t) = \sqrt[3]{2t + 4},\) \(t \geq 0,\) then show that the particle is always slowing down.
SOLUTION The velocity and acceleration functions are the first and second derivatives, respectively, of \(s(t).\) So we have \[v(t) = \tfrac{2}{3} (2t + 4)^{-2/3} \and a(t) = -\tfrac{8}{9} (2t + 4)^{-5/3} \pd\] For all \(t \geq 0,\) \(v(t) \gt 0\) and \(a(t) \lt 0.\) Since these signs are opposite, the object is always slowing down.
EXERCISE 18
A car travels with a speed of \(25\) meters per second before the driver suddenly slams the brakes. The car's initial deceleration is recorded to be \(10\) meters per second per second. Using linearization, approximate the time the car takes to stop completely.
SOLUTION Centering at \(t = 0,\) we approximate \(v(t)\) using the formula \[v(t) \approx v(0) + v'(0) (t - 0) \or v(t) \approx v(0) + a(0)t \pd\] (See Section 2.8.) With \(v(0) = 25 \undiv{m}{sec}\) and \(a(0) = -10 \undiv{m}{sec}^2\) (negative because the car is slowing down), we have \[v(t) \approx 25 - 10t \pd\] The car fully stops when \(v(t) = 0;\) the solution to \(25 - 10t = 0\) is \(\boxed{t = 2.5 \un{sec}}.\)
EXERCISE 19
Shortly after being thrown, a bowling ball's speed is measured to be \(8\) meters per second. Use differentials to estimate the distance the ball travels \(0.2\) second after its speed is measured.
SOLUTION Since \(\textderiv{s}{t} = v(t),\) it follows that \(\dd s = v(t) \di t.\) With \(\dd t \approx \Delta t\) and \(\dd s \approx \Delta s,\) we attain \[\Delta s = v(t) \Delta t \pd\] Taking \(v(t) = 8\) and \(\Delta t = 0.2,\) we have \[\Delta s = 8(0.2) = \boxed{0.4 \un{m}}\]
