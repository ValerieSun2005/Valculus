2.5 Exercise Solutions: Implicit Differentiation and Differentiating Inverse Functions

EXERCISE 1
What is implicit differentiation?
SOLUTION Implicit differentiation is the process by which we find \(\textderiv{y}{x}\) in an implicit equationâ€”an equation in which \(y\) isn't expressed as a function of \(x.\) To accomplish this goal we treat \(y\) as \(y(x),\) a function of \(x,\) and differentiate using the Chain Rule. Finally, we solve for \(\textderiv{y}{x},\) an expression that typically contains both \(x\) and \(y.\)
EXERCISE 2
Why do we use the number \(e\) in calculus?
SOLUTION The number \(e\) enables the derivatives of exponential and logarithmic functions to have very convenient derivatives; that is, \[\deriv{}{x} \par{e^x} = e^x \and \deriv{}{x} \par{\ln x} = \frac{1}{x} \pd\]
EXERCISE 3
For
\[\ds x^2 + xy - x = 2\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION It is not feasible to solve for \(y;\) thus, we apply implicit differentiation. We treat \(y\) as a function of \(x.\) Using the Chain Rule, we differentiate both sides of the equation with respect to \(x.\) Doing so gives \[ \ba \deriv{}{x}(x^2 + xy - x) &= \deriv{}{x}(2) \nl 2x + \par{x \deriv{y}{x} + y} - 1 &= 0 \nl x \deriv{y}{x} &= 1 - 2x - y \nl \implies \deriv{y}{x} &= \boxed{\frac{1 - 2x - y}{x}} \ea \]
EXERCISE 4
For
\[\ds x^7 - 3x = \sin y\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION To use implicit differentiation, we treat \(y\) as a function of \(x.\) Using the Chain Rule, we differentiate both sides of the equation with respect to \(x.\) Doing so gives \[ \ba \deriv{}{x}(x^7 - 3x) &= \deriv{}{x}(\sin y) \nl 7x^6 - 3 &= \cos y \deriv{y}{x} \nl \implies \deriv{y}{x} &= \boxed{\frac{7x^6 - 3}{\cos y}} \ea \]
EXERCISE 5
For
\[\ds y^3 + xy - 4x = 3\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Implicit differentiation requires us to treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides of the equation with respect to \(x \col\) \[ \ba \deriv{}{x}(y^3 + xy - 4x) &= \deriv{}{x}(3) \nl 3y^2 \deriv{y}{x} + y + x \deriv{y}{x} - 4 &= 0 \nl \deriv{y}{x} \par{3y^2 + x} &= 4 - y \nl \implies \deriv{y}{x} &= \boxed{\frac{4 - y}{3y^2 + x}} \ea \]
EXERCISE 6
For
\[\ds \tan y = x^2 - y\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION When we perform implicit differentiation, we treat \(y\) as a function of \(x.\) Using the Chain Rule to differentiate both sides, we find \[ \ba \deriv{}{x}(\tan y) &= \deriv{}{x}(x^2 - y) \nl \sec^2 y \deriv{y}{x} &= 2x - \deriv{y}{x} \nl \deriv{y}{x} \par{\sec^2 y + 1} &= 2x \nl \implies \deriv{y}{x} &= \boxed{\frac{2x}{\sec^2 y + 1}} \ea \]
EXERCISE 7
For
\[\ds x^2 - xy - y^3 = x + 8\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Implicit differentiation requires us to treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides of the equation with respect to \(x \col\) \[ \ba \deriv{}{x} \par{x^2 - xy - y^3} &= \deriv{}{x}(x + 8) \nl 2x - y - x \deriv{y}{x} - 3y^2 \deriv{y}{x} &= 1 \nl \deriv{y}{x} \par{-x - 3y^2} &= 1 + y - 2x \nl \implies \deriv{y}{x} &= \frac{1 + y - 2x}{-x - 3y^2} = \boxed{\frac{2x - y - 1}{x + 3y^2}} \ea \]
EXERCISE 8
For
\[\ds 4y \sin x = e^y\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Recall that the derivative of \(\sin x\) is \(\cos x\) and the derivative of \(e^x\) is \(e^x\) (see Section 2.2). Differentiating both sides of the implicit equation with respect to \(x\) gives \[4 \sin x \deriv{y}{x} + 4y \cos x = e^y \deriv{y}{x} \pd\] Moving all terms with \(\textderiv{y}{x}\) to the left side, we attain \[ \ba 4 \sin x \deriv{y}{x} - e^y \deriv{y}{x} &= 4y \cos x \nl \deriv{y}{x} \par{4 \sin x - e^y} &= 4 y \cos x \nl \implies \deriv{y}{x} &= \boxed{\frac{4 y \cos x}{4 \sin x - e^y}} \ea \]
EXERCISE 9
For
\[\ds x^2 y^3 + xy - 4 = x^6\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION When we perform implicit differentiation, we treat \(y\) as a function of \(x.\) Using the Chain Rule to differentiate both sides, we find \[ \ba \deriv{}{x} \par{x^2 y^3 + xy - 4} &= \deriv{}{x} \par{x^6} \nl 2x y^3 + 3x^2 y^2 \deriv{y}{x} + y + x \deriv{y}{x} - 0 &= 6x^5 \nl \deriv{y}{x} \par{3x^2 y^2 + x} &= 6x^5 - 2x y^3 - y \nl \implies \deriv{y}{x} &= \boxed{\frac{6x^5 - 2x y^3 - y}{3x^2 y^2 + x}} \ea \]
EXERCISE 10
For
\[\ds xe^y = y^2 - 4x\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION When we perform implicit differentiation, we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{xe^y} &= \deriv{}{x} \par{y^2 - 4x} \nl e^y + xe^y \deriv{y}{x} &= 2y \deriv{y}{x} - 4 \nl \deriv{y}{x} \par{xe^y - 2y} &= -4 - e^y \nl \implies \deriv{y}{x} &= \frac{-4 - e^y}{xe^y - 2y} = \boxed{\frac{4 + e^y}{2y - xe^y}} \ea \]

EXERCISE 12
For
\[\ds xe^{3y} = \tan y\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Recall that the derivative of \(\tan x\) is \(\sec^2 x.\) To perform implicit differentiation, we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{xe^{3y}} &= \deriv{}{x} \par{\tan y} \nl e^{3y} + 3xe^{3y} \deriv{y}{x} &= \sec^2 y \deriv{y}{x} \nl \deriv{y}{x} \par{\sec^2 y - 3xe^{3y}} &= e^{3y} \nl \implies \deriv{y}{x} &= \boxed{\frac{e^{3y}}{\sec^2 y - 3xe^{3y}}} \ea \]
EXERCISE 13
For
\[\ds x^2 \sqrt y - \sec y = x\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Note that the derivative of \(\sec x\) is \(\sec x \tan x.\) When we apply implicit differentiation, we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{x^2 \sqrt y - \sec y} &= \deriv{}{x} \par{x} \nl 2x \sqrt y + \frac{x^2}{2 \sqrt y} \deriv{y}{x} - \sec y \tan y \deriv{y}{x} &= 1 \nl \deriv{y}{x} \par{\frac{x^2}{2 \sqrt y} - \sec y \tan y} &= 1 - 2x \sqrt y \nl \implies \deriv{y}{x} &= \boxed{\frac{1 - 2x \sqrt y}{\dfrac{x^2}{2 \sqrt y} - \sec y \tan y}} \ea \]
EXERCISE 14
For
\[\ds xy^3 - \ln y = \frac{2}{x^2}\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION The derivative of \(\ln x\) is \(1/x.\) When we apply implicit differentiation, we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{xy^3 - \ln y} &= \deriv{}{x} \par{\frac{2}{x^2}} \nl y^3 + 3xy^2 \deriv{y}{x} - \frac{1}{y} \deriv{y}{x} &= -\frac{4}{x^3} \nl \deriv{y}{x} \par{3xy^2 - \frac{1}{y}} &= -y^3 - \frac{4}{x^3} \nl \implies \deriv{y}{x} &= \boxed{\frac{y^3 + \dfrac{4}{x^3}}{\dfrac{1}{y} - 3xy^2}} \ea \]

EXERCISE 17
For
\[\ds \atan y = y + x^8\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Note that the derivative of \(\atan x\) is \(1/(x^2 + 1).\) Using implicit differentiation requires that we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{\atan y} &= \deriv{}{x} \par{y + x^8} \nl \frac{1}{y^2 + 1} \cdot \deriv{y}{x} &= \deriv{y}{x} + 8x^7 \nl \deriv{y}{x} \par{\frac{1}{y^2 + 1} - 1} &= 8x^7 \nl \deriv{y}{x} \par{\frac{-y^2}{y^2 + 1}} &= 8x^7 \nl \implies \deriv{y}{x} &= \frac{8x^7}{\dfrac{-y^2}{y^2 + 1}} = \boxed{-\frac{8x^7 \par{y^2 + 1}}{y^2}} \ea \]
EXERCISE 18
For
\[\ds x^3 y^3 + \asin y = x^2 - 4\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Note that the derivative of \(\asin x\) is \(1/\sqrt{1 - x^2}.\) When we apply implicit differentiation, we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{x^3 y^3 + \asin y} &= \deriv{}{x} \par{x^2 - 4} \nl 3x^2 y^3 + 3x^3 y^2 \deriv{y}{x} + \frac{1}{\sqrt{1 - y^2}} \cdot \deriv{y}{x} &= 2x \nl \deriv{y}{x} \par{3x^3 y^2 + \frac{1}{\sqrt{1 - y^2}}} &= 2x - 3x^2 y^3 \nl \implies \deriv{y}{x} &= \boxed{\frac{2x - 3x^2 y^3}{\ds 3x^3 y^2 + \frac{1}{\sqrt{1 - y^2}}}} \ea \]
EXERCISE 19
For
\[\ds \acsc y = x^2 y - y^2 + 10\]
find \(\textderiv{y}{x}\) using implicit differentiation.
SOLUTION Note that the derivative of \(\acsc x\) is \(-1/\par{\abs{x} \sqrt{x^2 - 1}}.\) When we apply implicit differentiation, we treat \(y\) as a function of \(x.\) We use the Chain Rule to differentiate both sides with respect to \(x\) and then solve for \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{\acsc y} &= \deriv{}{x} \par{x^2 y - y^2 + 10} \nl -\frac{1}{\abs y \sqrt{y^2 - 1}} \deriv{y}{x} &= 2xy + x^2 \deriv{y}{x} - 2y \deriv{y}{x} \nl \deriv{y}{x} \par{x^2 - 2y + \frac{1}{\abs y \sqrt{y^2 - 1}}} &= -2xy \nl \implies \deriv{y}{x} &= \frac{-2xy}{\ds x^2 - 2y + \frac{1}{\abs y \sqrt{y^2 - 1}}} = \boxed{\frac{2xy}{\ds 2y - x^2 - \frac{1}{\abs y \sqrt{y^2 - 1}}}} \ea \]
EXERCISE 20
For
\[\ds x^2 + y^2 = 36\]
use implicit differentiation to find \(\textderivOrder{y}{x}{2}\).
SOLUTION To find \(\textderivOrder{y}{x}{2},\) we first attain \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{x^2 + y^2} &= \deriv{}{x} \par{36} \nl 2x + 2y \deriv{y}{x} &= 0 \nl \implies \deriv{y}{x} &= -\frac{2x}{2y} = -\frac{x}{y} \pd \ea \] We use the Quotient Rule to differentiate \(\textderiv{y}{x};\) in doing so, we apply the Chain Rule to differentiate all instances of \(y.\) Whenever we see the expression \(\textderiv{y}{x},\) we substitute the expression \(-x/y.\) We see \[ \ba \derivOrder{y}{x}{2} &= \deriv{}{x} \deriv{y}{x} \nl &= \deriv{}{x} \par{-\frac{x}{y}} \nl &= -\frac{\ds y - x \deriv{y}{x}}{y^2} \nl &= \frac{\ds -y + x \par{-\frac{x}{y}}}{y^2} \nl &= \boxed{-\frac{x^2 + y^2}{y^3}} \ea \]

use implicit differentiation to find \(\textderivOrder{y}{x}{2}\).

EXERCISE 22
For
\[\ds \ln y = x^3 - e^x\]
use implicit differentiation to find \(\textderivOrder{y}{x}{2}\).
SOLUTION Recall that the derivative of \(\ln x\) is \(1/x.\) To find \(\textderivOrder{y}{x}{2},\) we first attain \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{\ln y} &= \deriv{}{x} \par{x^3 - e^x} \nl \frac{1}{y} \deriv{y}{x} &= 3x^2 - e^x \nl \implies \deriv{y}{x} &= \par{3x^2 - e^x} y \pd \ea \] We use the Product Rule to differentiate \(\textderiv{y}{x};\) as we differentiate, we apply the Chain Rule to differentiate all instances of \(y.\) Whenever we see the expression \(\textderiv{y}{x},\) we substitute the expression we attained for it. Doing so shows \[ \ba \derivOrder{y}{x}{2} &= \deriv{}{x} \deriv{y}{x} \nl &= \deriv{}{x} \parbr{\par{3x^2 - e^x} y} \nl &= \par{6x - e^x} y + \par{3x^2 - e^x} \deriv{y}{x} \nl &= \par{6x - e^x} y + \par{3x^2 - e^x} \cdot \par{3x^2 - e^x} y \nl &= \boxed{\par{6x - e^x} y + \par{3x^2 - e^x}^2 y} \ea \]
EXERCISE 23

\[\ds 4e^y + y^2 = 2x^2 - 4\]
use implicit differentiation to find \(\textderivOrder{y}{x}{2}\).
SOLUTION To find \(\textderivOrder{y}{x}{2},\) we first attain \(\textderiv{y}{x},\) as follows: \[ \ba \deriv{}{x} \par{4e^y + y^2} &= \deriv{}{x} \par{2x^2 - 4} \nl 4e^y \deriv{y}{x} + 2y \deriv{y}{x} &= 4x \nl \implies \deriv{y}{x} &= \frac{4x}{4e^y + 2y} = \frac{2x}{2e^y + y} \pd \ea \] We use the Quotient Rule to differentiate \(\textderiv{y}{x};\) in doing so, we apply the Chain Rule to differentiate all instances of \(y.\) Whenever we have \(\textderiv{y}{x},\) we substitute the expression we attained for it. We find \[ \ba \derivOrder{y}{x}{2} &= \deriv{}{x} \deriv{y}{x} \nl &= \deriv{}{x} \par{\frac{2x}{2e^y + y}} \nl &= \frac{\ds 2 \par{2e^y + y} - 2x \par{2e^y + 1} \deriv{y}{x}}{\par{2e^y + y}^2} \nl &= \frac{\ds 2 \par{2e^y + y} - 2x \par{2e^y + 1} \par{\frac{2x}{2e^y + y}}}{\par{2e^y + y}^2} \nl &= \boxed{\frac{2 \par{2e^y + y}^2 - 4x^2 \par{2e^y + 1}}{\par{2e^y + y}^3}} \ea \]

EXERCISE 26
The function \(f\) satisfies the implicit equation \(2x^2 + [f(x)]^3 = 16.\) If \(f(2) = 2,\) then write an equation of the line tangent to the curve at \(x = 2.\)
SOLUTION Recall from Section 2.2 that an equation of a tangent line requires both the point of tangency and the slope at that point. The slope at \(x = 2\) is given by \(f'(2),\) which we find through implicit differentiation. Differentiating both sides of the implicit equation with respect to \(x,\) we see \[ \ba 4x + 3[f(x)]^2 f'(x) &= 0 \nl f'(x) &= -\frac{4x}{3[f(x)]^2} \nl \implies f'(2) &= -\frac{4(2)}{3[f(2)]^2} = -\frac{8}{3(2)^2} = -\frac{2}{3} \pd \ea \] The point of tangency is \((2, f(2))\) \(= (2, 2);\) thus, an equation of the tangent line is \[\boxed{y - 2 = -\tfrac{2}{3} (x - 2)}\]
EXERCISE 27
Consider the ellipse represented by the implicit equation \(x^2 - xy + y^2 = 3.\)
  1. Show that \[\deriv{y}{x} = \frac{y - 2x}{2y - x} \pd\]
  2. Find the points on the ellipse to which the tangent is horizontal.
  3. Locate the coordinates at which the ellipse has vertical tangents.
SOLUTION
  1. Differentiating both sides of the implicit equation with respect to \(x\) gives \[ \ba 2x - y - x \deriv{y}{x} + 2y \deriv{y}{x} &= 0 \nl \deriv{y}{x} \par{2y - x} &= y - 2x \nl \implies \deriv{y}{x} &= \frac{y - 2x}{2y - x} \cma \ea \] as requested.
  2. The tangent is horizontal to the curve when \(\textderiv{y}{x} = 0\)â€”that is, when its numerator is \(0\) and the denominator is nonzero. We see \[y - 2x = 0 \implies y = 2x \pd\] (For \(y = 2x,\) the denominator of \(\textderiv{y}{x}\) is nonzero.) Substituting this expression into the implicit equation reveals \[ \ba x^2 - x(2x) + (2x)^2 &= 3 \nl 3x^2 &= 3 \nl \implies x &= -1 \cma x = 1 \pd \ea \] These values correspond to \(y = -2\) and \(y = 2,\) respectively. So the curve has horizontal tangents at \(\boxed{(-1, -2)}\) and \(\boxed{(1, 2)}.\)
  3. The curve has vertical tangents when the denominator of \(\textderiv{y}{x}\) is \(0\) and the numerator is nonzero. Equating the denominator to \(0\) shows \[2y - x = 0 \implies x = 2y \pd\] Substituting this expression into the implicit equation reveals \[ \ba (2y)^2 - (2y)y + y^2 &= 3 \nl 3y^2 &= 3 \nl \implies y &= -1 \cma y = 1 \pd \ea \] Then \(x = -2\) and \(x = 2,\) respectively. Accordingly, the curve has vertical tangents at \(\boxed{(-2, -1)}\) and \(\boxed{(2, 1)}.\)
EXERCISE 28
Locate all the horizontal tangents to the curve \(x^3 - x^2y = 2.\)
SOLUTION We first find \(\textDeriv{y}{x} \col\) \[ \ba \deriv{}{x} \par{x^3 - x^2y} &= \deriv{}{x} (2) \nl 3x^2 - \par{2xy + x^2 \deriv{y}{x}} &= 0 \nl -x^2 \deriv{y}{x} &= 2xy - 3x^2 \nl \implies \deriv{y}{x} &= \frac{3x - 2y}{x} \pd \ea \] A horizontal tangent occurs when the slope of the tangent is \(0.\) Thus, we solve \(\textderiv{y}{x} = 0,\) which is satisfied when the numerator is \(0\) and the denominator is nonzero: \[3x - 2y = 0 \implies y = \tfrac{3}{2} x \pd\] Substituting this expression into the implicit equation \(x^3 - x^2y = 2\) shows \[ \ba x^3 - x^2 \par{\tfrac{3}{2} x} &= 2 \nl -\tfrac{1}{2} x^3 &= 2 \nl \implies x &= \sqrt[3]{-4} \pd \ea \] Then \(y = \tfrac{3}{2} \sqrt[3]{-4}.\) Hence, the curve has a horizontal tangent at \(\boxed{\par{\sqrt[3]{-4}, \tfrac{3}{2} \sqrt[3]{-4}}}.\)
EXERCISE 29
Find all the vertical tangents to the curve \(xy - y^2 = 4.\)
SOLUTION We differentiate both sides of the equation with respect to \(x\) to solve for \(\textDeriv{y}{x} \col\) \[ \ba \deriv{}{x} \par{xy - y^2} &= \deriv{}{x}(4) \nl \par{y + x \deriv{y}{x}} - 2y \deriv{y}{x} &= 0 \nl \deriv{y}{x} \par{x - 2y} &= -y \nl \implies \deriv{y}{x} &= \frac{y}{2y - x} \pd \ea \] To find the vertical tangents to the curve, we equate the denominator of \(\textderiv{y}{x}\) to \(0.\) We see \[2y - x = 0 \implies x = 2y \pd\] Substituting this expression into the implicit equation \(xy - y^2 = 4\) shows \[(2y) y - y^2 = 4 \implies y = \pm 2 \pd\] Since \(x = 2y,\) we have \[x = 2(-2) = -4 \and x = 2(2) = 4 \pd\] Therefore, the curve has vertical tangents at the points \(\boxed{(-4, -2)}\) and \(\boxed{(4, 2)}.\)
EXERCISE 30
A curve is defined by the implicit equation \(4xy - x^3 = y^2.\) Find the points on the curve at which the tangent to the curve is vertical.
SOLUTION We differentiate both sides of the equation with respect to \(x\) to solve for \(\textDeriv{y}{x} \col\) \[ \ba \deriv{}{x} \par{4xy - x^3} &= \deriv{}{x} \par{y^2} \nl 4y + 4x \deriv{y}{x} - 3x^2 &= 2y \deriv{y}{x} \nl \deriv{y}{x} \par{4x - 2y} &= 3x^2 - 4y \nl \implies \deriv{y}{x} &= \frac{3x^2 - 4y}{4x - 2y} \pd \ea \] The curve has a vertical tangent when \(\textderiv{y}{x}\) is undefined. So we equate the denominator of \(\textderiv{y}{x}\) to \(0,\) seeing \[4x - 2y = 0 \implies y = 2x \pd\] Substituting this expression into the implicit equation \(4xy - x^3 = y^2\) gives \[ \ba 4x(2x) - x^3 &= (2x)^2 \nl x^3 &= 4x^2 \nl \implies x &= 0 \cma x = 4 \pd \ea \] Observe that when \(x = 0,\) the implicit equation becomes \(0 - 0 = y^2,\) meaning \(y\) must be \(0.\) But when \(x = 0\) and \(y = 0,\) both the numerator and denominator of \(\textderiv{y}{x}\) are \(0,\) meaning we ignore the point \((0, 0).\) Instead, for our other solution \(x = 4,\) we have \(y = 2(4) = 8;\) at the point \((4, 8)\) only the denominator is \(0.\) Hence, the curve has a vertical tangent at the point \(\boxed{(4, 8)}.\)
EXERCISE 31
Find the points on the curve \(x^4 - 3y^2 = y^3\) to which the tangent is vertical.
SOLUTION We differentiate both sides of the equation with respect to \(x\) to solve for \(\textDeriv{y}{x} \col\) \[ \ba \deriv{}{x} \par{x^4 - 3y^2} &= \deriv{}{x} \par{y^3} \nl 4x^3 - 6y \deriv{y}{x} &= 3y^2 \deriv{y}{x} \nl \deriv{y}{x} \par{3y^2 + 6y} &= 4x^3 \nl \implies \deriv{y}{x} &= \frac{4x^3}{3y^2 + 6y} \pd \ea \] The curve has a vertical tangent when \(\textderiv{y}{x}\) is undefined. So we equate the denominator of \(\textderiv{y}{x}\) to \(0,\) seeing \[3y^2 + 6y = 0 \implies y = -2 \cma y = 0 \pd\] When \(y = 0\) the implicit equation becomes \(x^4 - 0 = 0,\) meaning \(x = 0.\) Yet when \(x = 0\) and \(y = 0,\) both the numerator and denominator in \(\textderiv{y}{x}\) are \(0,\) so we ignore the point \((0, 0).\) On the other hand, substituting \(y = -2\) into the implicit equation \(x^4 - 3y^2 = y^3\) reveals \[ \ba x^4 - 3(-2)^2 &= (-2)^3 \nl x^4 &= 4 \nl \implies x &= \pm \sqrt[4]{4} \pd \ea \] The curve has vertical tangents at the points \(\boxed{\par{-\sqrt[4]{4}, -2}}\) and \(\boxed{\par{\sqrt[4]{4}, -2}}\) because only the denominator of \(\textderiv{y}{x}\) is \(0\) at those points.
EXERCISE 32
Let \(f(x) = 5x^6 + 7x^3 - 6x + 8.\) Calculate \(\par{\inv f}'(8).\)
SOLUTION It is impossible to determine the inverse function \(\inv f.\) Simply put, we cannot simply find the inverse function \(\inv f(x)\) algebraically and then differentiate it. To calculate the derivative of an inverse function at a point, we use the formula \[ \par{\inv f}'(8) = \frac{1}{f'(\inv f(8))} \pd \] Observe that \(f(0) = 8,\) so \(\inv f(8) = 0.\) Also, \[ f'(x) = 30x^5 + 21x^2 - 6 \cma \so f'(0) = -6 \pd \] Thus, \[ \par{\inv f}'(8) = \frac{1}{f'(0)} = \boxed{-\tfrac{1}{6}} \]
EXERCISE 33
Suppose \(f\) and \(g\) are inverse functions such that \(f(2) = -3\) and \(f'(2) = 1/4.\) Calculate \(g'(-3).\)
SOLUTION Since \(f\) and \(g\) are inverse functions, we have \[g'(x) = \frac{1}{f'(g(x))} \pd\] Thus, we see \[g'(-3) = \frac{1}{f'(g(-3))} \pd\] Because \(f(2) = -3,\) it follows that \(g(-3) = 2\) and so \[g'(3) = \frac{1}{f'(2)} = \frac{1}{1/4} = \boxed 4\]

EXERCISE 37
For
\[\ds \acsc x = -\frac{1}{\abs{x} \sqrt{x^2 - 1}}\]
prove the given formula.
SOLUTION Let \(y = \acsc x\) \(\iffArrow x = \csc y.\) We see \(\textderiv{x}{y} = -\csc y \cot y\) (recall from Section 2.3); thus, \[\deriv{y}{x} = \frac{1}{\textderiv{x}{y}} = -\frac{1}{\csc y \cot y} \pd\] To relate this expression to \(x = \csc y,\) we use the Pythagorean identity: Because \(\cot^2 y\) \(= \csc^2 y - 1,\) we have \[\cot y = \pm \sqrt{\csc^2 y - 1} = \pm \sqrt{x^2 - 1} \pd\] The inverse cosecant function is defined for \(y\) in \((-\pi/2, \pi/2).\) Over \(0 \leq y \lt \pi/2\) both cosecant and cotangent are positive (or \(0\)); thus, we choose the positive square root and so \[\deriv{}{x} \par{\acsc x} = -\frac{1}{x \sqrt{x^2 - 1}} \pd\] But over \(-\pi/2 \lt y \leq 0,\) both cosecant and cotangent are negative (or \(0\)). So we pick the negative square root solution, resulting in \[\deriv{}{x} \par{\acsc x} = -\frac{1}{x \par{-\sqrt{x^2 - 1}}} \pd\] Observe that the \(x\) in the denominator is negative (since \(x = \csc y\) and \(-\pi/2 \lt y \leq 0\)), so we write \(x = - \abs x.\) Doing so gives \[\deriv{}{x} \par{\acsc x} = -\frac{1}{- \abs x \par{-\sqrt{x^2 - 1}}} = -\frac{1}{\abs x \sqrt{x^2 - 1}}\pd\]

EXERCISE 56
Find the point on the graph of \(y = \ln x + 1\) to which the tangent line passes through the origin.
SOLUTION Let's call the point of tangency \((a, \ln a + 1).\) The slope of the tangent at \(a\) is \(1/a,\) so an equation of the tangent line at \(x = a\) is \[y - \par{\ln a + 1} = \tfrac{1}{a} (x - a) \pd\] This line must pass through the origin, so \((0, 0)\) must be a valid point; substituting \(x = 0\) and \(y = 0\) shows \[ \ba 0 - \par{\ln a + 1} &= \tfrac{1}{a} (0 - a) \nl \ln a &= 0 \nl \implies a &= 1 \pd \ea \] Then \(y = \ln 1 + 1\) \(= 1.\) So the point on the curve is \(\boxed{(1, 1)}.\)
EXERCISE 57
For what value of \(C\) does the equation \(x^2 = C + \ln x\) have only one solution?
SOLUTION For the curves of \(f(x) = x^2\) and \(g(x) = C + \ln x\) to intersect once, they must be tangent at some point \(x = a.\) Recall that the word tangent means \(f(a) = g(a)\) and \(f'(a) = g'(a).\) Observe that \(f'(x) = 2x\) and \(g'(x) = 1/x.\) Equating the derivatives at \(a\) shows \[2a = \tfrac{1}{a} \implies a = \tfrac{1}{\sqrt 2} \pd\] Now we substitute this value into the equation \(x^2 = C + \ln x,\) seeing \[ \ba \par{\tfrac{1}{\sqrt 2}}^2 &= C + \ln \tfrac{1}{\sqrt 2} \nl \implies C &= \boxed{\tfrac{1}{2} - \ln \tfrac{1}{\sqrt 2}} \ea \]
EXERCISE 58
Determine the area of the triangle bounded between the \(y\)-axis, the tangent line to \(y = \ln x\) at \(x = e,\) and the normal line to \(y = \ln x\) at \(x = e.\)
SOLUTION The bounded region is a triangle of width \(e.\) To find its height, we must calculate the \(y\)-intercept of the normal line. Recall from Section 2.2 that a normal line is perpendicular to a tangent line. The derivative of \(\ln x\) is \(1/x,\) so the tangent to the curve at \(x = e\) has slope \(1/e.\) Accordingly, the normal's slope is the negative reciprocal of this value, that is, \(-e.\) An equation of the normal line is therefore \[y - \ln e = -e(x - e) \or y = 1 - e(x - e) \pd\] When \(x = 0\) we see \[y = 1 - e(0 - e) = 1 + e^2 \pd\] The \(y\)-intercept \((1 + e^2)\) is the height of the bounded triangle. Thus, its area is \[\tfrac{1}{2} (e)(1 + e^2) = \boxed{\tfrac{1}{2} (e + e^3)}\]
EXERCISE 59
The function \(f(x) = 1/x + \atan x\) is invertible for \(x \gt 0.\) Show that \[\lim_{x \to (\pi/2)^+} \par{\inv f}'(x) = -\infty \pd\]
SOLUTION Observe that \(f(x) \to (\pi/2)^+\) as \(x \to \infty.\) Consequently, \(\inv f(x) \to \infty\) as \(x \to (\pi/2)^+.\) We have \[ \baf &&f'(x) &= -\frac{1}{x^2} + \frac{1}{x^2 + 1} = -\frac{1}{x^4 + x^2} &\nl \laWord{and} &&\lim_{x \to (\pi/2)^+} \par{\inv f}'(x) &= \lim_{x \to (\pi/2)^+} \frac{1}{f' \par{\inv f(x)}} \pd \eaf \] To evaluate this limit, we substitute \(u = \inv f(x).\) Then \(u \to \infty\) as \(x \to (\pi/2)^+,\) so our limit becomes \[ \ba \lim_{u \to \infty} \frac{1}{f'(u)} &= \lim_{u \to \infty} \frac{1}{-\dfrac{1}{u^4 + u^2}} \nl &= \lim_{u \to \infty} - \par{u^4 + u^2} \nl &= -\infty \pd \ea \]
