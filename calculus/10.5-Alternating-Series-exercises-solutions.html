10.5 Exercise Solutions â€” Alternating Series

EXERCISE 1
For
\[\ds \sum_{n = 1}^\infty \frac{(-1)^n}{\sqrt[3] n}\]
determine whether the series converges or diverges.
SOLUTION We see, for positive \(n,\) \[\frac{1}{\sqrt[3]{n + 1}} \leq \frac{1}{\sqrt[3] n} \pd\] Thus, \(1/\sqrt[3] n\) is decreasing for all \(n \geq 1.\) And since \(\lim_{n \to \infty} \par{1/\sqrt[3] n} = 0,\) the Alternating Series Test states that \[\sum_{n = 1}^\infty \frac{(-1)^n}{\sqrt[3] n} \convBoxed\]
EXERCISE 2
For
\[\ds \sum_{n = 0}^\infty \frac{(-1)^n}{n^2 + 1}\]
determine whether the series converges or diverges.
SOLUTION For \(n \geq 0\) we see \[\frac{1}{(n + 1)^2 + 1} \leq \frac{1}{n^2 + 1} \pd\] Therefore, \(1/\par{n^2 + 1}\) is decreasing for all \(n \geq 0.\) Also, note that \(\lim_{n \to \infty} \parbr{1/(n^2 + 1)} = 0.\) So the Alternating Series Test says that \[\sum_{n = 0}^\infty \frac{(-1)^n}{n^2 + 1} \convBoxed\]
EXERCISE 3
For
\[\ds \sum_{n = 1}^\infty \frac{(-1)^n \sqrt n}{2 \sqrt{n + 7}}\]
determine whether the series converges or diverges.
SOLUTION Observe that \[\lim_{n \to \infty} \frac{\sqrt n}{2 \sqrt{n + 7}} = \lim_{n \to \infty} \frac{\sqrt n}{2 \sqrt n} = \frac{1}{2} \ne 0 \pd\] So the series does not meet the conditions for the Alternating Series Test. We therefore cannot use the test to assert convergence or divergence. Yet we see \[\lim_{n \to \infty} \frac{(-1)^n \sqrt n}{2 \sqrt{n + 7}}\] does not exist. [As \(n \to \infty,\) the factor \((-1)^n\) forces the function to continually output values between \(-1/2\) and \(1/2.\)] So by the Divergence Test, \[\sum_{n = 1}^\infty \frac{(-1)^n \sqrt n}{2 \sqrt{n + 7}} \divBoxed\]
EXERCISE 4
For
\[\ds \sum_{n = 1}^\infty \frac{n^{-2/3}}{(-1)^n} \]
determine whether the series converges or diverges.
SOLUTION In its current form, the series seems arcane. But let's rewrite it as \[\sum_{n = 1}^\infty \frac{(-1)^n}{n^{2/3}} \pd\] [Note that \(1/(-1)^n = (-1)^n\) because the alternator only serves to change the sign.] For \(n \geq 1\) we see \[\frac{1}{(n + 1)^{2/3}} \leq \frac{1}{n^{2/3}} \and \lim_{n \to \infty} \frac{1}{n^{2/3}} = 0 \pd\] The Alternating Series Test therefore asserts that \[\sum_{n = 1}^\infty \frac{n^{-2/3}}{(-1)^n} \convBoxed\]
EXERCISE 5
For
\[\ds \sum_{n = 1}^\infty \frac{n^3 - 8}{\cos(\pi n) \, n^2} \]
determine whether the series converges or diverges.
SOLUTION Because \(\cos(\pi n)\) \(= (-1)^n,\) let's rewrite the series as \[\sum_{n = 1}^\infty \frac{(-1)^n \par{n^3 - 8}}{n^2} \pd\] We see that \[\lim_{n \to \infty} \frac{(-1)^n \par{n^3 - 8}}{n^2}\] does not exist. [As \(n\) grows large, the ratio \(\par{n^3 - 8}/n^2\) grows without bound.] Accordingly, the Divergence Test asserts that \[\sum_{n = 1}^\infty \frac{n^3 - 8}{\cos(\pi n) \, n^2} \divBoxed\]
EXERCISE 6
For
\[\ds \sum_{n = 1}^\infty \frac{(-1)^{2n}}{n}\]
determine whether the series converges or diverges.
SOLUTION Observe that \[(-1)^{2n} = \parbr{(-1)^2}^n = (1)^n = 1 \pd\] Hence, \[\sum_{n = 1}^\infty \frac{(-1)^{2n}}{n} = \sum_{n = 1}^\infty \frac{1}{n} \cma\] which diverges because it is the Harmonic Series.
EXERCISE 7
For
\[\ds \sum_{n = 0}^\infty \frac{(-1)^{n - 1}}{6n + 3}\]
determine whether the series converges or diverges.
SOLUTION Note that \[\lim_{n \to \infty} \frac{1}{6n + 3} = 0 \pd\] For \(n \geq 0\) we see \[\frac{1}{6(n + 1) + 3} = \frac{1}{6n + 9} \leq \frac{1}{6n + 3} \cma\] meaning \(1/(6n + 3)\) is decreasing on \([0, \infty).\) By the Alternating Series Test, \[\sum_{n = 0}^\infty \frac{(-1)^{n - 1}}{6n + 3} \convBoxed\]
EXERCISE 8
For
\[\ds \sum_{n = 0}^\infty \frac{(-1)^n \, n}{n^2 + 4}\]
determine whether the series converges or diverges.
SOLUTION We assert that \[\lim_{n \to \infty} \frac{n}{n^2 + 4} = 0 \pd\] But it isn't clear whether this function is decreasing. Let's compare it to the continuous function \(f(x) = x/\par{x^2 + 4},\) whose derivative is \[f'(x) = \frac{(1)\par{x^2 + 4} - x(2x)}{\par{x^2 + 4}^2} = \frac{4 - x^2}{\par{x^2 + 4}^2} \pd\] Observe that \(f' \lt 0\) on \((2, \infty),\) meaning \(f(x)\) is decreasing for \(2 \leq x \lt \infty.\) It follows that, for \(n \geq 2,\) \[\frac{n + 1}{(n + 1)^2 + 4} \leq \frac{n}{n^2 + 4} \pd\] The Alternating Series Test therefore asserts that \[\sum_{n = 2}^\infty \frac{(-1)^n \, n}{n^2 + 4} \conv \implies \sum_{n = 0}^\infty \frac{(-1)^n \, n}{n^2 + 4} \convBoxed\] (Adding a finite number of terms to a series does not change whether it converges or diverges.)
EXERCISE 9
For
\[\ds \sum_{n = 0}^\infty (-2)^{-n}\]
determine whether the series converges or diverges.
SOLUTION We see \[(-2)^{-n} = \frac{1}{(-2)^n} = \frac{1}{(-1)^n \, 2^n} = \frac{(-1)^n}{2^n} \pd\] Thus, we rewrite the series as \[\sum_{n = 0}^\infty \frac{(-1)^n}{2^n} \pd\] This is a geometric series of common ratio \(r = -1/2.\) Since \(\abs r \lt 1,\) \[\sum_{n = 0}^\infty (-2)^{-n} \convBoxed\] Alternatively, we could attain the same result by applying the Alternating Series Test: For all \(n \geq 0,\) we see \[\frac{1}{2^{n + 1}} \leq \frac{1}{2^n} \cma\] and \(\lim_{n \to \infty} \par{1/2^n} = 0.\) Hence, the test says that the series is convergent.
EXERCISE 10
For
\[\ds \sum_{n = 1}^\infty \frac{\cos(2 \pi n)}{\sqrt[4] n}\]
determine whether the series converges or diverges.
SOLUTION Observe that \(\cos(2 \pi n) = 1\) for all integers \(n.\) So the series is equivalent to \[\sum_{n = 1}^\infty \frac{1}{\sqrt[4] n} \cma\] which diverges because it is a \(p\)-series with \(p = 1/4 \lt 1.\)
EXERCISE 11
For
\[\ds \sum_{n = 1}^\infty (-1)^n \frac{5n^3 + 1}{n^3 + 4n^2 + 8}\]
determine whether the series converges or diverges.
SOLUTION Observe that \[\lim_{n \to \infty} \frac{5n^3 + 1}{n^3 + 4n^2 + 8} = \lim_{n \to \infty} \frac{5n^3}{n^3} = 5 \ne 0 \pd\] So the series does not meet the conditions for the Alternating Series Test. We therefore cannot use the test to assert convergence or divergence. Yet we see \[\lim_{n \to \infty} (-1)^n \frac{5n^3 + 1}{n^3 + 4n^2 + 8}\] does not exist. [As \(n \to \infty,\) the factor \((-1)^n\) forces the function to continually output values between \(-5\) and \(5\)]. So by the Divergence Test, \[\sum_{n = 1}^\infty (-1)^n \frac{5n^3 + 1}{n^3 + 4n^2 + 8} \divBoxed\]
EXERCISE 12
For
\[\ds \sum_{n = 1}^\infty \frac{(-1)^n \, n^3}{n^4 + 4n}\]
determine whether the series converges or diverges.
SOLUTION We observe that \[\lim_{n \to \infty} \frac{n^3}{n^4 + 4n} = 0 \pd\] To establish that the function is decreasing, let's compare it to the continuous function \(f(x) = x^3/\par{x^4 + 4x},\) whose derivative is \[f'(x) = \frac{3x^2 \par{x^4 + 4x} - x^3 \par{4x^3 + 4}}{\par{x^4 + 4x}^2} = \frac{-x^6 + 8x^3}{\par{x^4 + 4x}^2} = \frac{-x^3 \par{x^3 - 8}}{\par{x^4 + 4x}^2} \pd \] Since \(f'(x) \lt 0\) for \(x \gt 2,\) \(f\) is decreasing on \([2, \infty).\) It therefore follows that, for \(n \geq 2,\) \[\frac{(n + 1)^3}{(n + 1)^4 + 4(n + 1)} \leq \frac{n^3}{n^4 + 4n} \pd\] The Alternating Series Test therefore asserts that \[\sum_{n = 2}^\infty \frac{(-1)^n \, n^3}{n^4 + 4n} \conv \implies \sum_{n = 1}^\infty \frac{(-1)^n \, n^3}{n^4 + 4n} \convBoxed\] (Adding a finite number of terms to a series does not change whether it converges or diverges.)
EXERCISE 13
For
\[\ds S = \sum_{n = 1}^\infty \frac{(-1)^n}{\sqrt n} \cmaa S_3 = \sum_{i = 1}^3 \frac{(-1)^i}{\sqrt i}\]
use the Alternating Series Error Bound to calculate the maximum error in the partial sum approximation.
SOLUTION Using the Alternating Series Error Bound, we assert that \(S_3\) differs from the true value of \(S\) by no more than the magnitude of the first omitted term, \[b_4 = \frac{1}{\sqrt 4} = \boxed{\frac{1}{2}}\]

SOLUTION Using the Alternating Series Error Bound, we assert that \(S_4\) differs from the true value of \(S\) by no more than the magnitude of the first excluded term, \[b_5 = \frac{1}{5^3} = \boxed{\frac{1}{625}}\]
EXERCISE 15
For

use the Alternating Series Error Bound to calculate the maximum error in the partial sum approximation.
SOLUTION The error in the approximation \(S_8\) is no greater than the magnitude of the first omitted term: \[b_9 = \frac{\sqrt 9}{9 + 3} = \frac{3}{12} = \boxed{\frac{1}{4}}\]
EXERCISE 16
For
\[\ds S = \sum_{n = 1}^\infty \frac{n^2 \cos(\pi n)}{n^2 + 4n - 1} \cmaa S_8 = \sum_{i = 1}^8 \frac{i^2 \cos(\pi i)}{i^2 + 4i - 1}\]
use the Alternating Series Error Bound to calculate the maximum error in the partial sum approximation.
SOLUTION We use the Alternating Series Error Bound to bound the error by the magnitude of the first excluded term, \[b_9 = \frac{9^2}{9^2 + 4(9) - 1} = \frac{81}{81 + 36 - 1} = \boxed{\frac{81}{116}} \approx 0.698 \pd\]
EXERCISE 17
Let \(S = \sum_{n = 1}^\infty (-1)^n/n^2.\) Using the Alternating Series Error Bound for the partial sum \(S_3 = \sum_{i = 1}^3 (-1)^i/i^2,\) find an interval in which \(S\) must reside.
SOLUTION Observe that \[S_3 = -1 + \frac{1}{4} - \frac{1}{9} = -\frac{31}{36} \pd\] According to the Alternating Series Error Bound, the error in the approximation \(S_3\) is no greater than the magnitude of the first omitted term, which is \[b_4 = \frac{1}{4^2} = \frac{1}{16} \pd\] In other words, the estimate \(S_3\) differs from \(S\) by no more than \(1/16.\) So we have \(\abs{S - S_3} \leq 1/16,\) or \[S_3 - \frac{1}{16} \leq S \leq S_3 + \frac{1}{16} \iffArrow \boxed{-\frac{32}{36} \leq S \leq -\frac{30}{36}}\]
EXERCISE 18
Does the partial sum \(S_2 = \sum_{i = 0}^2 (-1)^{i} /(3i + 1)\) underestimate or overestimate the value of \(S = \sum_{n = 0}^\infty (-1)^n/(3n + 1) \ques\)
SOLUTION Performing the arithmetic, we see \(S_2 = 25/28 \gt 0.\) The first term we exclude is \[\frac{(-1)^{3}}{3(3) + 1} = -\frac{1}{28} \pd\] Since this number is negative, the remainder \(\sum_{n = 3}^\infty (-1)^{n + 1}/\par{n^2 + 1}\) is negative. In other words, the true value of \(S\) must be smaller than our estimate \(S_2.\) Thus, the partial sum \(S_2\) overestimates \(S.\)
EXERCISE 19
Does the partial sum \(S_4 = \sum_{i = 0}^4 (-1)^{i + 1} / \par{i^2 + 1}\) underestimate or overestimate the value of \(S = \sum_{n = 0}^\infty (-1)^{n + 1} / \par{n^2 + 1} \ques\)
SOLUTION If you perform the arithmetic, then you see \(S_4 = -56/85\) \(\lt 0.\) The first term we omit is \[\frac{(-1)^{5 + 1}}{5^2 + 1} = \frac{1}{26} \pd\] Since this number is positive, the remainder \(\sum_{n = 5}^\infty (-1)^{n + 1}/\par{n^2 + 1}\) is positive. In other words, the true value of \(S\) must be larger than our estimate \(S_4.\) Hence, the partial sum \(S_4\) underestimates \(S.\)
EXERCISE 20
For what values of \(N\) does the partial sum \(S_N = \sum_{i = 0}^N (-1)^{i + 1}/\sqrt{2i + 5}\) approximate \(S = \sum_{n = 0}^\infty (-1)^{n + 1}/\sqrt{2n + 5}\) with an error of less than \(0.2 \ques\)
SOLUTION Using the Alternating Series Error Bound, we find an error bound to the estimate \(S_N\) to be \[\abs{\frac{(-1)^{(N + 1) + 1}}{\sqrt{2(N + 1) + 5}}} = \frac{1}{\sqrt{2N + 7}} \pd\] Equating this expression to be less than \(0.2,\) we see \[\frac{1}{\sqrt{2N + 7}} \leq 0.2 \implies \sqrt{2N + 7} \geq 5 \implies 2N + 7 \geq 25 \implies \boxed{N \geq 9}\]
EXERCISE 21
Determine the values of \(N\) for which the partial sum \(S_N = \sum_{i = 3}^N (-1)^i/\par{i^2 - 3i + 1}\) approximates \(S = \sum_{n = 3}^\infty (-1)^n/\par{n^2 - 3n + 1}\) with an error of less than \(1/19.\)
SOLUTION The Alternating Series Error Bound says that the error is less than or equal to \[\abs{\frac{(-1)^{(N + 1) + 1}}{(N + 1)^2 - 3(N + 1) + 1}} = \frac{1}{N^2 - N - 1} \pd\] (This fraction is positive for \(N \geq 3,\) so we omit the absolute value bars.) Equating this result to be less than \(1/19,\) we see \[ \ba \frac{1}{N^2 - N - 1} & \leq \frac{1}{19} \nl N^2 - N - 1 & \geq 19 \nl N^2 - N - 20 & \geq 0 \nl (N - 5)(N + 4) & \geq 0 \implies \boxed{N \geq 5} \ea \]
EXERCISE 22
Consider the alternating series \(S = \sum_{n = 0}^\infty (-2/3)^n.\)
  1. The series is approximated by the partial sum \[S_3 = 1 - \frac{2}{3} + \frac{4}{9} - \frac{8}{27} \pd\] Use the Alternating Series Error Bound to find a number \(K\) for which \(\abs{S - S_3} \leq K.\)
  2. Show that the series converges to \(S = 3/5.\) (This case is unique; it is very rare to know the exact value of an alternating series.)
  3. Calculate the magnitude of the error \(\abs{R_3}\) of \(S_3.\)
  4. What percent of the error bound \(K\) does \(\abs{R_3}\) equal?
SOLUTION
  1. The Alternating Series Error Bound asserts that the error in \(S_3\) is no greater than the magnitude of the first term we exclude (the next term in the approximation)â€”that is, \(b_4 = 16/81.\) So \[\abs{S - S_3} \leq \underbrace{\boxed{\frac{16}{81}}}_{K}\] In words, it is guaranteed that the partial sum \(S_3\) differs from the infinite sum \(S\) by no more than \(K = 16/81.\)
  2. This alternating series turns out to be geometric with common ratio \(r = -2/3.\) Since \(\abs r \lt 1,\) the series converges to \[\frac{1}{1 + \frac{2}{3}} = \boxed{\frac{3}{5}}\]
  3. Note that \[S_3 = 1 - \frac{2}{3} + \frac{4}{9} - \frac{8}{27} = \frac{13}{27} \pd\] The magnitude of the error is therefore \[\abs{R_3} = \abs{S - S_3} = \abs{\frac{3}{5} - \frac{13}{27}} = \boxed{\frac{16}{135}} \approx 0.119 \pd\]
  4. The percentage is \[\frac{\abs{R_3}}{K} = \frac{16/135}{16/81} = \frac{81}{135} = \boxed{60\%}\] In words, the error bound \(K\) overestimates the error in the approximationâ€”the actual error is only \(60\%\) of this bound.
EXERCISE 23
For what values of \(p\) does the series \(\sum_{n = 1}^\infty (-1)^n/n^p\) converge?
SOLUTION For positive \(p,\) we have \(\lim_{n \to \infty} \par{1/n^p} = 0\) and \(1/(n + 1)^p \leq 1/n^p.\) These are the conditions needed to satisfy the Alternating Series Test. Hence, the family of series converges for \(\boxed{p \gt 0}.\)
EXERCISE 24
Determine the values of \(k\) for which \(\sum_{n = 1}^\infty (-1)^{n + 1} \, n^{4k + 3}\) converges.
SOLUTION According to the Alternating Series Test, the series converges if \[\lim_{n \to \infty} n^{4k + 3} = 0\] and \(n^{4k + 3}\) is decreasing. These conditions are satisfied when the exponent \(4k + 3\) is negativeâ€”namely, when \[4k + 3 \lt 0 \implies \boxed{k \lt -\tfrac{3}{4}}\]
