8.1 Exercise Solutions: Slope Fields and Euler's Method

EXERCISE 1
Classify the following differential equations by order.
  1. \(\ds 8 \derivOrder{y}{x}{2} - 5 \deriv{y}{x} = 2\)
  2. \(\ds \deriv{y}{x} = y + 4x\)
  3. \(\ds x^8 - y''' + \cos y = 4 e^{2x}\)
  4. \(\ds y^{(5)} + 4 y^{(6)} - 15x^4 = y''\)
SOLUTION
Leibniz notation \((\textderiv{y}{x})\) and the prime notation \((y')\) are interchangeable. It's important to be flexible in both sets of notation.
  1. Second-order. The highest-degree derivative that appears in the equation is second-order: \(8 \, \textderivOrder{y}{x}{2}.\)
  2. First-order. The highest-degree derivative that appears in the equation is first-order: \(\textDeriv{y}{x}.\)
  3. Third-order. The highest-degree derivative that appears in the equation is third-order: \(y'''.\)
  4. Sixth-order. The highest-degree derivative that appears in the equation is sixth-order: \(4 \, y^{(6)}.\)
EXERCISE 2
Consider the first-order differential equation \[\deriv{y}{x} = 6e^x - 3x^2 + 5 \pd\]
  1. Using integration, determine the general solution to the differential equation.
  2. Find the particular solution \(y = f(x)\) that satisfies the initial condition \(y(0) = 3.\)
SOLUTION See Section 4.1 to review the procedure for solving a differential equation by integration.
  1. The general solution to the differential equation is \[y = \int \par{6e^x - 3x^2 + 5} \di x = \boxed{6e^x - x^3 + 5x + C}\] for some constant \(C.\)
  2. We solve for \(C\) such that \(y(0) = 3\) is satisfied. Doing so, we see \[6e^0 - (0)^3 + 5(0) + C = 3 \implies C = -3 \pd\] Thus, the particular solution is \[f(x) = \boxed{6e^x - x^3 + 5x - 3}\]

EXERCISE 5
For
\[\ds 2y'' - 9y' + y - 8 = 0 \scoll y = 8\]
determine whether the given function is a solution to the differential equation.
SOLUTION Because \(y = 8\) is constant, we have \(y'\) \(= y'' = 0.\) We therefore see \[ \underbrace{2(0)}_{2y''} - \underbrace{9(0)}_{9y'} + \underbrace{8}_{y} - 8 \equalsCheck 0 \pd \] So \(y = 8\) is a solution.

EXERCISE 8

determine whether the given function is a solution to the differential equation.
SOLUTION We note that \(\textderiv{y}{t} = 3e^{-3t}.\) The differential equation is satisfied because \[\underbrace{3e^{-3t}}_{\textderiv{y}{t}} \equalsCheck \underbrace{5 - 3 \par{\frac{5}{3} - e^{-3t}}}_{5 - 3y} \pd\] So \(y = 5/3 - e^{-3t}\) is a solution.
EXERCISE 9
In many phenomena, a quantity changes at a rate proportional to itself. The differential equation that models this behavior is \[\deriv{y}{x} = ky\] for some constant \(k.\) Show that the family of exponential functions \(y = Ce^{kx}\) satisfies this differential equation. (We will examine exponential models in Section 8.3.)
SOLUTION Differentiating the exponential function \(y = Ce^{kx},\) we see \[\deriv{y}{x} = C \par{ke^{kx}} \equalsCheck \underbrace{k \par{Ce^{kx}}}_{ky} \pd\] Thus, the differential equation is satisfied by \(y = Ce^{kx}.\)
EXERCISE 10
Show that the particular solution to the initial value problem \[y' = x + 2y \cmaa y(1) = 3 \cma\] is \(y = \tfrac{1}{4} \par{-2x + 15e^{2x - 2} - 1}.\)
SOLUTION Differentiating the given \(y = \tfrac{1}{4} \par{-2x + 15e^{2x - 2} - 1},\) we get \[y' = -\tfrac{1}{2} + \tfrac{15}{2} e^{2x - 2} \pd\] We see \[\underbrace{-\tfrac{1}{2} + \tfrac{15}{2} e^{2x - 2}}_{y'} \equalsCheck x + \underbrace{2 \parbr{\tfrac{1}{4} \par{-2x + 15e^{2x - 2} - 1}}}_{2y} \pd\] In addition, the initial condition \(y(1) = 3\) is met: \[y(1) = \tfrac{1}{4} \parbr{-2(1) + 15e^{2(1) - 2} - 1} \equalsCheck 3 \pd\] Thus, the given function solves the initial value problem because both the differential equation and initial condition are satisfied.
EXERCISE 11
Determine whether each function is a solution to the fourth-order differential equation \[y^{(4)} - 16y = 0 \pd\]
  1. \(\ds y = Ce^{2x}\)
  2. \(\ds y = 2 \ln x\)
  3. \(\ds y = x^5\)
  4. \(\ds y = C_1 e^{2x} + C_2 e^{-2x} + C_3 \cos 2x + C_4 \sin 2x\)
SOLUTION
  1. Yes. Taking four derivatives of \(y = Ce^{2x},\) we see \[ \ba y' &= 2Ce^{2x} \cma \nl y'' &= 4Ce^{2x} \cma \nl y''' &= 8Ce^{2x} \cma \nl y^{(4)} &= 16Ce^{2x} \pd \ea \] Then \[\underbrace{16Ce^{2x}}_{y^{(4)}} - \underbrace{16 \par{Ce^{2x}}}_{16y} \equalsCheck 0 \pd\]
  2. No. If we take four derivatives of \(y = 2 \ln x,\) then we get \[ \ba y' &= \frac{2}{x} \cma \nl y'' &= -\frac{2}{x^2} \cma \nl y''' &= \frac{4}{x^3} \cma \nl y^{(4)} &= -\frac{12}{x^4} \pd \ea \] Observe that \[\underbrace{-\frac{12}{x^4}}_{y^{(4)}} - \underbrace{16 \par{5 \ln x}}_{16y} \ne 0 \pd\]
  3. No. Differentiating \(y = x^5\) four times, we attain the following results: \[ \ba y' &= 5x^4 \cma \nl y'' &= 20x^3 \cma \nl y''' &= 60x^2 \cma \nl y^{(4)} &= 120 x \pd \ea \] Examine that \[\underbrace{120 x}_{y^{(4)}} - \underbrace{16 \par{x^5}}_{16y} \ne 0 \pd\]
  4. Yes. We take four derivatives of \(y = C_1 e^{2x} + C_2 e^{-2x} + C_3 \cos 2x + C_4 \sin 2x \col\) \[ \ba y' &= 2 C_1 e^{2x} - 2 C_2 e^{-2x} - 2 C_3 \sin 2x + 2 C_4 \cos 2x \cma \nl y'' &= 4 C_1 e^{2x} + 4 C_2 e^{-2x} - 4 C_3 \cos 2x - 4 C_4 \sin 2x \cma \nl y''' &= 8 C_1 e^{2x} - 8 C_2 e^{-2x} + 8 C_3 \sin 2x - 8 C_4 \cos 2x \cma \nl y^{(4)} &= 16 C_1 e^{2x} + 16 C_2 e^{-2x} + 16 C_3 \cos 2x + 16 C_4 \sin 2x \pd \ea \] Observe that \[\small \underbrace{16 C_1 e^{2x} + 16 C_2 e^{-2x} + 16 C_3 \cos 2x + 16 C_4 \sin 2x}_{\large y^{(4)}} - \underbrace{16 \par{C_1 e^{2x} + C_2 e^{-2x} + C_3 \cos 2x + C_4 \sin 2x}}_{\large 16y} \equalsCheck 0 \pd\] In fact, this function is the most general solution to the differential equation \(y^{(4)} - 16y = 0.\)
EXERCISE 12
An equilibrium solution has a derivative of \(0\) at all \(x.\) Find the equilibrium solutions to the differential equation \[\deriv{y}{x} = y^3 - 5y^2 + 6y \pd\]
SOLUTION We equate \(\textderiv{y}{x} = 0,\) as follows: \[ \ba y^3 - 5y^2 + 6y &= 0 \nl y (y - 3)(y - 2) &= 0 \nl \implies y &= 0 \cma 2 \cma 3 \pd \ea \] The equilibrium solutions are therefore \(\boxed{y = 0},\) \(\boxed{y = 2},\) and \(\boxed{y = 3}.\)

EXERCISE 26
In an autonomous first-order differential equation, the rate does not depend on the independent variable; that is, it can be written in the form \[\deriv{y}{x} = f(y) \pd\] Determine whether the following differential equations are autonomous.
  1. \(\ds y' = xy \sin y\)
  2. \(\ds y' = A\) for any constant \(A\)
  3. \(\ds 3y' + 6y + 7 = 0\)
  4. \(\ds y' + x = 2 - y\)
SOLUTION
  1. No. The derivative \(y'\) includes \(x,\) so the differential equation is not autonomous.
  2. Yes. The rate \(y'\) does not depend on \(x.\)
  3. Yes. We solve for \(y',\) obtaining \[y' = -2y - \tfrac{7}{3} \cma\] which does not depend on \(x.\)
  4. No. Solving for \(y'\) gives \[y' = 2 - x - y \cma\] which includes \(x.\) The differential equation is therefore not autonomous.

EXERCISE 29
In the slope field for \(y' = 1/x\) and \(x \gt 0,\) the line segments' slopes are independent of \(y\) and decrease as \(x\) increases, as shown by Figure 10. Sketch the particular solution curve that passes through the point \((1, 2).\) Then use integration to solve the initial value problem and therefore determine the identity of the graph.
SOLUTION We begin the particular solution curve at the point \((1, 2)\) and follow the direction of the line segments. This curve decreases in slope as \(x\) increases. Next, we find the identity of this slope curve by integration (using the methods from Section 4.1): The general solution to the differential equation \(y' = 1/x\) is \[y = \int \frac{1}{x} \di x = \ln \abs x + C\] for some constant \(C.\) (But we ignore the absolute value bars because the slope field is restricted to \(x \gt 0.\)) Substituting the initial condition \(y(1) = 2\) shows \[y(1) = \ln 1 + C = 2 \implies C = 2 \pd\] The particular solution (the identity of the curve) is therefore \[\boxed{y = 2 + \ln x}\]

  2. Calculate the object's terminal velocity \(v_T,\) the maximum velocity it attains.
  3. What percentage of the terminal velocity is the velocity attained from part (a)?
SOLUTION
The particular solution to this differential equation turns out to be \[v(t) = \frac{40.1248 e^{1.60499 t} - 40.1248}{e^{1.60499 t} + 1} \pd\] It is very difficult to attain this solution by hand, so Euler's Method provides us a good approximation—and an even better estimate if we lower \(h.\)
  1. Since the object is initially stationary, we have \(v(0) = 0.\) With \(\beta = 0.02\) the differential equation becomes \[\deriv{v}{t} = 32.2 - 0.02 v^2 \pd\] The applications of Euler's Method are shown below: \[ \baat{2} v(0.1) &\approx v_1 &&= 0 + 0.1 \parbr{32.2 - 0.02(0)^2} = 3.22 \undiv{ft}{sec} \cma \nl v(0.2) &\approx v_2 &&= 3.22 + 0.1 \parbr{32.2 - 0.02(3.22)^2} = 6.419 \undiv{ft}{sec} \cma \nl v(0.3) &\approx v_3 &&= 6.419 + 0.1 \parbr{32.2 - 0.02(6.419)^2} = 9.557 \undiv{ft}{sec} \cma \nl v(0.4) &\approx v_4 &&= 9.557 + 0.1 \parbr{32.2 - 0.02(9.557)^2} = 12.594 \undiv{ft}{sec} \cma \nl v(0.5) &\approx v_5 &&= 12.594 + 0.1 \parbr{32.2 - 0.02(12.594)^2} = 15.497 \undiv{ft}{sec} \cma \nl v(0.6) &\approx v_6 &&= 15.497 + 0.1 \parbr{32.2 - 0.02(15.497)^2} = 18.237 \undiv{ft}{sec} \cma \nl v(0.7) &\approx v_7 &&= 18.237 + 0.1 \parbr{32.2 - 0.02(18.237)^2} = 20.791 \undiv{ft}{sec} \cma \nl v(0.8) &\approx v_8 &&= 20.792 + 0.1 \parbr{32.2 - 0.02(20.791)^2} = 23.147 \undiv{ft}{sec} \cma \nl v(0.9) &\approx v_9 &&= 23.147 + 0.1 \parbr{32.2 - 0.02(23.147)^2} = 25.295 \undiv{ft}{sec} \cma \nl v(1) &\approx v_{10} &&= 25.295 + 0.1 \parbr{32.2 - 0.02(25.295)^2} = \boxed{27.236 \undiv{ft}{sec}} \nl \eaat \]
  2. The object's velocity is maximized when it stops accelerating—that is, when \(\textderiv{v}{t} = 0.\) Solving for \(v_T\) shows \[0 = 32.2 - 0.02 v_T^2 \implies v_T = \sqrt{\frac{32.2}{0.02}} \approx \boxed{40.125 \undiv{ft}{sec}}\]
  3. The percentage of terminal velocity is \[\frac{27.236}{40.125} \times 100\% \approx \boxed{67.878 \%}\] After \(1\) second, the object is more than halfway to reaching terminal velocity!
EXERCISE 31
In a circuit, an inductor is a device that opposes a change in current. A battery of electromotive force \(20\) volts is connected to a circuit with resistance \(10\) ohms and inductance \(5\) henries (Figure 11). Let \(I\) be the circuit's current, which changes with time \(t,\) measured in seconds. The voltage drops are \(10 I\) across the resistor and \(5 \, \textderiv{I}{t}\) across the inductor. Applying one of Kirchhoff's Laws gives the differential equation \[20 - 10I - 5 \deriv{I}{t} = 0 \pd\] The initial current is \(0\) amps.
  1. Using Euler's Method with step size \(h = 0.25,\) approximate the current after \(1\) second has elapsed.
  2. What is the current after a very long time (that is, as \(t \to \infty\))?
  3. The particular solution to the differential equation turns out to be \[I(t) = 2 \par{1 - e^{-2t}} \pd\] Verify this result. Does this function make physical sense?
SOLUTION
  1. We have the initial condition \(I(0) = 0.\) Let's rewrite the given differential equation as \[\deriv{I}{t} = 4 - 2I \pd\] The first step of Euler's Method gives \[I_1 = 0 + 0.25 \parbr{4 - 2(0)} = 1 \un A \pd\] Likewise, subsequent approximations are as follows: \[ \ba I_2 &= 1 + 0.25 \parbr{4 - 2(1)} = 1.5 \un A \cma \nl I_3 &= 1.5 + 0.25 \parbr{4 - 2(1.5)} = 1.75 \un A \cma \nl I(1) \approx I_4 &= 1.75 + 0.25 \parbr{4 - 2(1.75)} = \boxed{1.875 \un A} \ea \]
  2. We see \(\textderiv{I}{t} \to 0\) as \(I \to 2,\) meaning \(I\) cannot increase past \(2 \un A.\) Thus, the current after a very long time is \(\boxed{2 \un A}.\)
  3. Differentiating \(I,\) we attain \[I'(t) = 4e^{-2t} \pd\] We then see \[20 - \underbrace{10 \parbr{2\par{1 - e^{-2t}}}}_{10 I} - \underbrace{5 \par{4e^{-2t}}}_{5 \, \textderiv{I}{t}} \equalsCheck 0 \pd\] In addition, the initial condition \(I(0) = 0\) is met: \[I(0) = 2 \par{1 - e^0} \equalsCheck 0 \pd\] Hence, \(I(t) = 2 \par{1 - e^{-2t}}\) is a particular solution because it satisfies the differential equation with \(I(0) = 0.\) The solution makes physical sense because the inductor gradually allows more current to flow in the circuit. Initially, the inductor opposes most of the current from flowing, but it eventually allows the current to grow to its maximum of \(2 \un A.\) With this behavior, inductors are used as shut-off devices—for example, in surge protectors—to prevent electronic damage by a current overload.
