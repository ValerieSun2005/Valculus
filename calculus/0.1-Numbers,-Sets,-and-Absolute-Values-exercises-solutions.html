0.1 Exercise Solutions: Numbers, Sets, and Absolute Values

EXERCISE 1
Define an integer, a rational number, an irrational number, and a real number.
SOLUTION An integer is a number that is not a fraction (and therefore has no decimal component). A rational number is a fraction of integers, whereas an irrational number cannot be expressed as a fraction of integers. Integers are rational numbers. A real number is either a rational number or irrational number. The set of all real numbers is denoted \(\RR.\) (In this text, we work exclusively with real numbers, so \(\RR\) the most diverse set we consider.)

EXERCISE 15
The sum of two irrational numbers is always an irrational number.
SOLUTION False. For example, the numbers \(\pi\) and \(-\pi\) are both irrational. But their sum is \[\pi + (-\pi) = 0 \cma\] which is rational.
EXERCISE 16
Multiplying two irrational numbers always yields an irrational number.
SOLUTION False. For example, the number \(\sqrt 3\) is irrational. But \[(\sqrt 3)(\sqrt 3) = (\sqrt 3)^2 = 3 \cma\] which is rational.
EXERCISE 17
Adding, subtracting, multiplying, and dividing rational numbers (with the exception of division by \(0\)) always yields a rational number.
SOLUTION True. Consider two rational numbers \(a/b\) and \(c/d,\) where \(a,\) \(b,\) \(c,\) and \(d\) are integers such that \(b \ne 0\) and \(d \ne 0.\) Adding these rational numbers shows \[\frac{a}{b} + \frac{c}{d} = \frac{ad + cb}{bd} \pd\] The quantities \(ad,\) \(cb,\) and \(bd\) are all integers because multiplying two integers yields an integer. Hence, the sum is a fraction of integers and is therefore rational. Similar reasoning enables us to verify the fact for subtracting, multiplying, and dividing rational numbers.
EXERCISE 18
For
\[\ds S = \{1, 2, 5, 6, 9\} \scoll T = \{2, 3, 4, 7, 9\}\]
find \(S \cup T\) and \(S \cap T\).
SOLUTION The union of \(S\) and \(T\) is the set of numbers that are in \(S\) or in \(T\) (or in both): \[S \cup T = \boxed{\{1, 2, 3, 4, 5, 6, 7, 9\}}\] The intersection of \(S\) and \(T\) is the set of numbers that are in both \(S\) and \(T \col\) \[S \cap T = \boxed{\{2, 9\}}\]
EXERCISE 19

\[\ds S = \{3, 5, 7, 8\} \scoll T = \{4, 6, 9, 11\}\]

SOLUTION The union of \(S\) and \(T\) is the set of numbers that are in \(S\) or in \(T\) (or in both): \[S \cup T = \boxed{\{3, 4, 5, 6, 7, 8, 9, 11\}}\] The intersection of \(S\) and \(T\) is the set of numbers that are in both \(S\) and \(T.\) Yet there are no shared numbers in \(S\) and \(T,\) so their intersection is the empty set: \[S \cap T = \boxed{\emptyset}\]

EXERCISE 43
If \(\abs{x + 2} \lt 5\) and \(\abs{y + 4} \lt 9,\) then use the Triangle Inequality to bound \(\abs{x + y + 6}.\)
SOLUTION We have \[ \ba \abs{x + y + 6} &= \abs{(x + 2) + (y + 4)} \nl &\leq \abs{x + 2} + \abs{y + 4} \nl &\lt 5 + 9 = 14 \pd \ea \] Hence, \[\boxed{\abs{x + y + 6} \lt 14}\]
EXERCISE 44
Given \(\abs{P + 4} \lt 0.3\) and \(\abs{Q - 4} \lt 0.8,\) use the Triangle Inequality to bound \(\abs{P + Q}.\)
SOLUTION We have \[ \ba \abs{P + Q} &= \abs{(P + 4) + (Q - 4)} \nl &\leq \abs{P + 4} + \abs{Q - 4} \nl &\lt 0.3 + 0.8 = 1.1 \pd \ea \] Hence, \[\boxed{\abs{P + Q} \lt 1.1}\]
