\(41.\)
There are several ways to evaluate \(\int \sin x \cos x \di x.\) Evaluate the integral by
  1. substituting \(u = \sin x\)
  2. substituting \(u = \cos x\)
  3. using the identity \(\sin 2x = 2 \sin x \cos x\)
\(42.\)
Let \(b\) be a positive constant. Evaluate the family of integrals \(\int b^x \di x.\)
\(43.\)
Let \(f\) be a function whose derivative \(f'\) is continuous. Suppose that \(f(3) = 2\) and \(f(5) = 6.\) Calculate \(\int_0^1 f'(2x + 3) \di x.\)
\(44.\)
Assume \(g'\) is continuous. If \(g(-2) = 3\) and \(g(6) = -5,\) then find \(\int_{-1}^1 g'(4x + 2) \di x.\)
\(45.\)
Let \(a\) be a positive constant. Prove the following formulas.
  1. \(\ds \int \frac{1}{x^2 + a^2} \di x = \frac{1}{a} \atan \par{\frac{x}{a}} + C\)
  2. \(\ds \int \frac{1}{\sqrt{a^2 - x^2}} \di x= \asin \par{\frac{x}{a}} + C\)
\(46.\)
Show that \(\int \csc x \di x\) \(= -\ln \abs{\csc x + \cot x} + C\) by reexpressing the integral as \[\int \frac{\csc x (\csc x + \cot x)}{\csc x + \cot x} \di x\] and substituting \(u = \csc x + \cot x.\)
\(47.\)
Calculate the area of the region bounded by the curve \(y = x^3 \sqrt{x^4 + 2},\) the \(x\)-axis, the \(y\)-axis, and the line \(x = 1.\)
\(48.\)
A power supply unit outputs a power load that varies with time according to the function \(P(t) = 60 t \sin \par{t^2}.\) How much energy does the unit supply over \(0 \leq t \leq 1 \ques\)
\(49.\)
A particle moves along the \(x\)-axis with time \(t \geq 0\) as given by the velocity function \(v(t) = t^2 \, e^{-t^3}.\) If the particle's initial position is \(x = 2,\) then calculate its position at \(t = 1.\)
\(50.\)
If \(f\) is continuous on \([a, b],\) then use the Substitution Rule to prove that \[\int_a^b f(x) \di x = \int_a^b f(a + b - x) \di x \pd\]
\(51.\)
This exercise examines the Weierstrass substitution, famously called the world's sneakiest substitution.
  1. Prove the trigonometric identities \[ \sin x = \frac{2 \tan \frac{x}{2}}{1 + \tan^2 \frac{x}{2}} \and \cos x = \frac{1 - \tan^2 \frac{x}{2}}{1 + \tan^2 \frac{x}{2}} \pd \] (Hint: Start with the double-angle identities for sine and cosine; then use the Pythagorean identity \(1 + \tan^2 \theta\) \(= 1/\cos^2 \theta.\))
  2. If \(t = \tan(x/2),\) then show that \[\dd x = \frac{2}{t^2 + 1} \di t \pd\]
  3. Use the substitution \(t = \tan(x/2)\) to evaluate \[\int_0^{\pi/2} \frac{1}{1 + \sin x + \cos x} \di x \pd\]
