6.2 Exercise Solutions: Trigonometric Integrals

EXERCISE 1
For
\[\ds \int \sin x \cos^2 x \di x\]
evaluate the integral.
SOLUTION Since the integrand contains an extra factor \(\sin x,\) we substitute \(u = \cos x\) to strip it, given that \(\dd u = - \sin x \di x.\) The integral then becomes \[ \ba \int -u^2 \di u &= -\tfrac{1}{3} u^3 + C \nl &= \boxed{-\tfrac{1}{3} \cos^3 x + C} \ea \]
EXERCISE 2
For
\[\ds \int \sin^3 x \cos^4 x \di x\]
evaluate the integral.
SOLUTION Sine's power is odd, so we save an extra term \(\sin x.\) We then use \(\sin^2 x = 1 - \cos^2 x\) to rewrite the integrand in terms of cosine: \[ \ba \int \sin^3 x \cos^4 x \di x &= \int \sin^2 x \cos^4 x \sin x \di x \nl &= \int (1 - \cos^2 x) \cos^4 x \sin x \di x \pd \ea \] Substituting \(u = \cos x,\) we get \(\dd u = - \sin x \di x\) and so the integral becomes \[ \ba \int [- (1 - u^2) \, u^4] \di u &= \int \par{u^6 - u^4} \di u \nl &= \tfrac{1}{7} u^7 - \tfrac{1}{5} u^5 + C \nl &= \boxed{\tfrac{1}{7} \cos^7 x - \tfrac{1}{5} \cos^5 x + C} \ea \]
EXERCISE 3
For
\[\ds \int_0^{\pi/4} \frac{1}{1 - \sin^2 x} \di x\]
evaluate the integral.
SOLUTION The Pythagorean identity gives \(\sin^2 x + \cos^2 x = 1\) \(\iffArrow \cos^2 x = 1 - \sin^2 x.\) So the integral becomes \[ \ba \int_0^{\pi/4} \frac{1}{\cos^2 x} \di x &= \int_0^{\pi/4} \sec^2 x \di x \nl &= \tan x \intEval_0^{\pi/4} \nl &= 1 - 0 = \boxed 1 \ea \]
EXERCISE 4
For
\[\ds \int \sin^2 x \di x\]
evaluate the integral.
SOLUTION We rewrite the integrand using the power-reduction formula for sine: \[ \ba \int \tfrac{1}{2} (1 - \cos 2x) \di x &= \tfrac{1}{2} \par{x - \tfrac{1}{2} \sin 2x} + C \nl &= \boxed{\tfrac{1}{2} x - \tfrac{1}{4} \sin 2x + C} \ea \]
EXERCISE 5
For
\[\ds \int \sin^4 x \di x\]
evaluate the integral.
SOLUTION We use the power-reduction formula for sine, \(\sin^2 x = \tfrac{1}{2} (1 - \cos 2x),\) to rewrite the integral as \[\int \parbr{\tfrac{1}{2}(1 - \cos 2x)}^2 \di x = \tfrac{1}{4} \int \par{1 - 2 \cos 2x + \cos^2 2x} \di x \pd \] From \(\cos^2 x = \tfrac{1}{2} (1 + \cos 2x),\) it follows that \(\cos^2 2x = \tfrac{1}{2} (1 + \cos 4x).\) The integral is therefore \[ \ba \tfrac{1}{4} \int \parbr{1 - 2 \cos 2x + \tfrac{1}{2} (1 + \cos 4x)} \di x &= \tfrac{1}{4} \par{\tfrac{3}{2} x - \sin 2x + \tfrac{1}{8} \sin 4x} + C \nl &= \boxed{\tfrac{3}{8} x - \tfrac{1}{4} \sin 2x + \tfrac{1}{32} \sin 4x + C} \ea \]
EXERCISE 6
For
\[\ds \int \cos^3 x \sin^4 x \di x\]
evaluate the integral.
SOLUTION Cosine's power is odd, so we preserve an extra factor \(\cos x.\) We then use \(\cos^2 x = 1 - \sin^2 x\) to rewrite the integrand in terms of sine: \[ \int \cos^2 x \sin^4 x \cos x \di x = \int (1 - \sin^2 x) \sin^4 x \cos x \di x \pd \] We let \(u = \sin x\) because then \(\dd u = \cos x \di x\) and so the integral becomes \[ \ba \int (1 - u^2) \, u^4 \di u &= \int \par{u^4 - u^6} \di u \nl &= \tfrac{1}{5} u^5 - \tfrac{1}{7} u^7 + C \nl &= \boxed{\tfrac{1}{5} \sin^5 x - \tfrac{1}{7} \sin^7 x + C} \ea \]
EXERCISE 7
For
\[\ds \int \sin^2 x \cos^2 x \di x\]
evaluate the integral.
SOLUTION Since sine and cosine both have even powers, let's write the integrand fully in terms of cosine by using \(\sin^2 x = 1 - \cos^2 x \col\) \[ \int \par{1 - \cos^2 x} \cos^2 x \di x = \int \par{\cos^2 x - \cos^4 x} \di x \pd \] In Example 3 and Example 4, we found \[ \ba \int \cos^2 x \di x &= \tfrac{1}{2} x + \tfrac{1}{4} \sin 2x + C \cma \nl \int \cos^4 x \di x &= \tfrac{3}{8} x + \tfrac{1}{4} \sin 2x + \tfrac{1}{32} \sin 4x + C \pd \ea \] Thus, the integral is \[ \par{\tfrac{1}{2} x + \tfrac{1}{4} \sin 2x} - \par{\tfrac{3}{8} x + \tfrac{1}{4} \sin 2x + \tfrac{1}{32} \sin 4x} + C \nl = \boxed{\tfrac{1}{8} x - \tfrac{1}{32} \sin 4x + C} \]
EXERCISE 8
For
\[\ds \int \sec^3 x \tan^3 x \di x\]
evaluate the integral.
SOLUTION Tangent's power is odd, so we preserve an extra factor \(\sec x \tan x\) and use \(\tan^2 x = \sec^2 x - 1\) to express the remaining factors in terms of secants: \[\int \sec^2 x \tan^2 x \sec x \tan x \di x = \int \sec^2 x (\sec^2 x - 1) \sec x \tan x \di x \pd\] To strip the extra factor \(\sec x \tan x,\) we substitute \(u = \sec x\) because then \(\dd u = \sec x \tan x \di x.\) The integral is now \[ \ba \int u^2 \, (u^2 - 1) \di u &= \int \par{u^4 - u^2} \di u \nl &= \tfrac{1}{5} u^5 - \tfrac{1}{3} u^3 + C \nl &= \boxed{\tfrac{1}{5} \sec^5 x- \tfrac{1}{3} \sec^3 x + C} \ea \]
EXERCISE 9
For
\[\ds \int \sec^2 x \tan x \di x\]
evaluate the integral.
SOLUTION The power of secant is even, and the power of tangent is odd. We therefore have two ways to evaluate the integral.

Solution 1 We substitute \(u = \tan x\) because then \(\dd u = \sec^2 x \di x\) and the integral becomes \[ \ba \int u \di u &= \tfrac{1}{2} u^2 + C \nl &= \boxed{\tfrac{1}{2} \tan^2 x + C} \ea \]

Solution 2 We rewrite the integral as \[\int \sec x \sec x \tan x \di x \pd\] Then we substitute \(u = \sec x\) so that \(\dd u = \sec x \tan x \di x.\) The integral then becomes \[ \ba \int u \di u &= \tfrac{1}{2} u^2 + C \nl &= \boxed{\tfrac{1}{2} \sec^2 x + C} \ea \]

Both answers are valid antiderivatives; they therefore only differ by a constant.
EXERCISE 10
For
\[\ds \int \sec^6 x \tan^3 x \di x\]
evaluate the integral.
SOLUTION The power of secant is even, and the power of tangent is odd. We therefore have two ways to evaluate the integral.

Solution 1 We conserve an extra factor \(\sec^2 x\) and use \(\sec^2 x = \tan^2 x + 1\) to reexpress the remaining factors in terms of tangents: \[ \ba \int \sec^6 x \tan^3 x \di x &= \int \sec^4 x \tan^3 x \sec^2 x \di x \nl &= \int (\tan^2 x + 1)^2 \tan^3 x \sec^2 x \di x \pd \ea \] Substituting \(u = \tan x\) enables us to clear the extra \(\sec^2 x,\) given that \(\dd u = \sec^2 x \di x.\) Then the integral becomes \[ \ba \int \par{u^2 + 1}^2 \, u^3 \di u &= \int \par{u^7 + 2u^5 + u^3} \di u \nl &= \tfrac{1}{8} u^8 + \tfrac{1}{3} u^6 + \tfrac{1}{4} u^4 + C \nl &= \boxed{\tfrac{1}{8} \tan^8 x + \tfrac{1}{3} \tan^6 x + \tfrac{1}{4} \tan^4 x + C} \ea \]

Solution 2 Conserving an extra factor \(\sec x \tan x\) and using \(\tan^2 x = \sec^2 x - 1\) to reexpress the remaining factors in terms of secants, we rewrite the integral as \[\int \sec^5 x \tan^2 x \sec x \tan x \di x = \int \sec^5 x \par{\sec^2 x - 1} \sec x \tan x \di x \pd \] Substituting \(u = \sec x,\) we have \(\dd u = \sec x \tan x \di x\) and so the integral becomes \[ \ba \int u^5 \par{u^2 - 1} \di u &= \int \par{u^7 - u^5} \di u \nl &= \tfrac{1}{8} u^8 - \tfrac{1}{6} u^6 + C \nl &= \boxed{\tfrac{1}{8} \sec^8 x - \tfrac{1}{6} \sec^6 x + C} \ea \]

Both answers are valid antiderivatives; they therefore only differ by a constant.
EXERCISE 11
For
\[\ds \int \frac{\cos^3 \theta}{\sin \theta} \di \theta\]
evaluate the integral.
SOLUTION This integral is unorthodox, but we take some inspiration from the strategy we devised to evaluate \(\int \sin^m x \cos^n x \di x.\) Cosine's power is odd, so we conserve an extra factor \(\cos \theta\) and use \(\cos^2 \theta\) \(= 1 - \sin^2 \theta\) to express the remaining factors in terms of sines: \[ \int \frac{\cos^2 \theta \cos \theta}{\sin \theta} \di \theta = \int \frac{(1 - \sin^2 \theta) \cos \theta}{\sin \theta} \di \theta \pd \] Substituting \(u = \sin \theta,\) we get \(\dd u = \cos \theta \di \theta\) and so the integral becomes \[ \ba \int \frac{1 - u^2}{u} \di u &= \int \par{\frac{1}{u} - u} \di u \nl &= \ln \abs{u} - \tfrac{1}{2} u^2 + C \nl &= \boxed{\ln \abs{\sin \theta} - \tfrac{1}{2} \sin^2 \theta + C} \ea \]
EXERCISE 12
For
\[\ds \int \frac{\sin^4 \theta}{\cos \theta} \di \theta\]
evaluate the integral.
SOLUTION
\(\int \sec \theta \di \theta\) \(= \ln \abs{\sec \theta + \tan \theta} + C\)
This integral is unorthodox, but we take some inspiration from the strategy we devised to evaluate \(\int \sin^m x \cos^n x \di x.\) The power of sine is even, so we rewrite \(\sin^4 \theta\) as \((1 - \cos^2 \theta)^2\) to get \[ \ba \int \frac{(1 - \cos^2 \theta)^2}{\cos \theta} \di \theta &= \int \frac{1 - 2 \cos^2 \theta + \cos^4 \theta}{\cos \theta} \di \theta \nl &= \int \par{\sec \theta - 2 \cos \theta + \cos^3 \theta} \di \theta \nl &= \ln \abs{\sec \theta + \tan \theta} - 2 \sin \theta + \sin \theta - \tfrac{1}{3} \sin^3 \theta + C \nl &= \boxed{\ln \abs{\sec \theta + \tan \theta} - \sin \theta - \tfrac{1}{3} \sin^3 \theta + C} \ea \] (We evaluated \(\int \cos^3 \theta \di \theta\) in Example 1.)
EXERCISE 13
For
\[\ds \int \sin^5 x \sec^6 x \di x\]
evaluate the integral.
SOLUTION Integrals with sines and secants are unorthodox, yet the integral is equivalent to the orthodox integral \[\int \tan^5 x \sec x \di x \pd\] Since tangent's power is odd, we save an extra factor \(\sec x \tan x\) and use \(\tan^2 x = \sec^2 x - 1\) to express the remaining factors in terms of secants: \[ \ba \int \tan^5 x \sec x \di x &= \int \tan^4 x \sec x \tan x \di x \nl &= \int (\sec^2 x - 1)^2 \sec x \tan x \di x \pd \ea \] Substituting \(u = \sec x,\) we have \(\dd u = \sec x \tan x \di x\) and so the integral becomes \[ \ba \int (u^2 - 1)^2 \di u &= \int \par{u^4 - 2u^2 + 1} \di u \nl &= \tfrac{1}{5} u^5 - \tfrac{2}{3} u^3 + u + C \nl &= \boxed{\tfrac{1}{5} \sec^5 x - \tfrac{2}{3} \sec^3 x + \sec x + C} \ea \]
EXERCISE 14
For
\[\ds \int_0^{\pi/4} \frac{\sin^4 x \tan^2 x}{\cos^8 x} \di x\]
evaluate the integral.
SOLUTION The integral is equivalent to the standard integral \[\int_0^{\pi/4} \tan^6 x \sec^4 x \di x \pd\] Because secant's power is even, we preserve an extra factor \(\sec^2 x\) and use \(\sec^2 x\) \(= \tan^2 x + 1\) to reexpress the remaining factors in terms of tangents, as follows: \[\int_0^{\pi/4} \tan^6 x \sec^2 x \sec^2 x \di x = \int_0^{\pi/4} \tan^6 x (\tan^2 x + 1) \sec^2 x \di x \pd\] Substituting \(u = \tan x,\) we have \(\dd u = \sec^2 x \di x.\) When \(x = 0,\) \(u = 0;\) when \(x = \pi/4,\) \(u = 1.\) So the integral becomes \[ \ba \int_0^1 u^6 (u^2 + 1) \di u &= \int_0^1 \par{u^8 + u^6} \di u \nl &= \par{\tfrac{1}{9} u^9 + \tfrac{1}{7} u^7} \intEval_0^1 \nl &= \boxed{\tfrac{16}{63}} \ea \]
EXERCISE 15
For
\[\ds \int \sec^4 x \di x\]
evaluate the integral.
SOLUTION As secant's power is even, we preserve an extra factor \(\sec^2 x\) and write the integral as \[\int (\tan^2 x + 1) \sec^2 x \di x \pd\] Substituting \(u = \tan x,\) we get \(\dd u = \sec^2 x \di x\) and so the integral becomes \[ \ba \int \par{u^2 + 1} \di u &= \tfrac{1}{3} u^3 + u + C \nl &= \boxed{\tfrac{1}{3} \tan^3 x + \tan x + C} \ea \]
EXERCISE 16
For
\[\ds \int \sec^8 t \di t\]
evaluate the integral.
SOLUTION Secant's power is even, so we conserve an extra factor \(\sec^2 t\) and use \(\sec^2 t = \tan^2 t + 1\) to convert the remaining secants in terms of tangents. The integral therefore becomes \[\int \sec^6 t \sec^2 t \di t = \int (\tan^2 t + 1)^3 \sec^2 t \di t \pd\] Substituting \(u = \tan t,\) we get \(\dd u= \sec^2 t \di t.\) The integral therefore becomes \[ \ba \int \par{u^2 + 1}^3 \di u &= \int \par{u^6 + 3u^4 + 3u^2 + 1} \di u \nl &= \tfrac{1}{7} u^7 + \tfrac{3}{5} u^5 + u^3 + u + C \nl &= \boxed{\tfrac{1}{7} \tan^7 t + \tfrac{3}{5} \tan^5 t + \tan^3 t + \tan t + C} \ea \]
EXERCISE 17
For
\[\ds \int \csc^8 y \di y\]
evaluate the integral.
SOLUTION Cosecant's power is even, so we conserve an extra factor \(\csc^2 y\) and use \(\csc^2 y\) \(= \cot^2 y + 1\) to reexpress the remaining cosecants in terms of cotangents. Doing so, we get \[\int \csc^6 y \csc^2 y \di y = \int (\cot^2 y + 1)^3 \csc^2 y \di y \pd\] Substituting \(u = \cot y,\) we have \(\dd u = -\csc^2 y \di y\) and so the integral becomes \[ \ba -\int \par{u^2 + 1}^3 \di u &= -\int \par{u^6 + 3u^4 + 3u^2 + 1} \di u \nl &= -\tfrac{1}{7} u^7 - \tfrac{3}{5} u^5 - u^3 - u + C \nl &= \boxed{-\tfrac{1}{7} \cot^7 y - \tfrac{3}{5} \cot^5 y - \cot^3 y - \cot y + C} \ea \]
EXERCISE 18
For
\[\ds \int_{\pi/6}^{\pi/3} \frac{1}{\cos^6 \theta} \di \theta\]
evaluate the integral.
SOLUTION The integral is equivalent to \[\int_{\pi/6}^{\pi/3} \sec^6 \theta \di \theta \pd\] Secant's power is even, so we conserve an extra factor \(\sec^2 \theta\) and use \(\sec^2 \theta\) \(= \tan^2 \theta + 1\) to express the remaining factors in terms of tangents. Doing so, we rewrite the integral as \[\int_{\pi/6}^{\pi/3} \sec^4 \theta \sec^2 \theta \di \theta = \int_{\pi/6}^{\pi/3} (\tan^2 \theta + 1)^2 \sec^2 \theta \di \theta \pd\] Substituting \(u = \tan \theta,\) we have \(\dd u = \sec^2 \theta \di \theta.\) When \(\theta = \pi/6,\) \(u = \sqrt{3}/3;\) when \(\theta = \pi/3,\) \(u = \sqrt 3.\) The integral therefore becomes \[ \ba \int_{\sqrt 3/3}^{\sqrt 3} (u^2 + 1)^2 \di u &= \int_{\sqrt 3/3}^{\sqrt 3} \par{u^4 + 2u^2 + 1} \di u \nl &= \par{\tfrac{1}{5} u^5 + \tfrac{2}{3} u^3 + u} \intEval_{\sqrt 3/3}^{\sqrt 3} \nl &= \boxed{\tfrac{592}{135} \sqrt 3} \ea \]
EXERCISE 19
For
\[\ds \int \csc^4 x \cot x \di x\]
evaluate the integral.
SOLUTION Cosecant's power is even and cotangent's power is odd, so there are two good ways to evaluate the integral.

Solution 1 We keep an extra factor \(\csc^2 x\) and use \(\csc^2 x = \cot^2 x + 1\) to express the remaining factors in terms of cotangents: \[ \ba \int \csc^4 x \cot x \di x &= \int \csc^2 x \cot x \csc^2 x \di x \nl &= \int (\cot^2 x + 1) \cot x \csc^2 x \di x \pd \ea \] Substituting \(u = \cot x,\) we have \(\dd u = -\csc^2 x \di x\) and so the integral becomes \[ \ba \int [-(u^2 + 1) \, u] \di u &= \int \par{-u^3 - u} \di u \nl &= -\tfrac{1}{4} u^4 - \tfrac{1}{2} u^2 + C \nl &= \boxed{-\tfrac{1}{4} \cot^4 x - \tfrac{1}{2} \cot^2 x + C} \ea \]

Solution 2 We rewrite the integral as \[\int \csc^3 x \csc x \cot x \di x \pd\] Substituting \(u = \csc x,\) we get \(\dd u = - \csc x \cot x \di x\) and the integral becomes \[-\int u^3 \di u = -\tfrac{1}{4} u^4 + C = \boxed{-\tfrac{1}{4} \csc^4 x + C}\]

Both answers are valid antiderivatives; they therefore only differ by a constant.
EXERCISE 20
For
\[\ds \int \csc^4 x \cot^3 x \di x\]
evaluate the integral.
SOLUTION The power of cosecant is even, so we save an extra factor \(\csc^2 x\) and use \(\csc^2 x = \cot^2 x + 1\) to express the remaining factors in terms of \(\cot x \col\) \[ \int \csc^2 x \cot^3 x \csc^2 x \di x = \int (\cot^2 x + 1) \cot^3 x \csc^2 x \di x \pd \] Substituting \(u = \cot x\) gives \(\dd u = - \csc^2 x \di x\) and so the integral becomes \[ \ba \int [-(u^2 + 1) \, u^3] \di u &= \int \par{-u^5 - u^3} \di u \nl &= - \tfrac{1}{6} u^6 - \tfrac{1}{4} u^4 + C \nl &= \boxed{-\tfrac{1}{6} \cot^6 x - \tfrac{1}{4} \cot^4 x + C} \ea \]
EXERCISE 21
For
\[\ds \int \tan^3 x \di x\]
evaluate the integral.
SOLUTION Using \(\tan^2 x\) \(= \sec^2 x - 1,\) we rewrite the integral as \[ \ba \int \tan x \tan^2 x \di x &= \int \tan x \left(\sec^2 x - 1\right) \di x \nl &= \int \tan x \sec^2 x \di x - \int \tan x \di x \pd \ea \] The right integral is \(\int \tan x \di x = \ln \abs{\sec x} + C,\) and we write the left integral as \[\int \sec x \sec x \tan x \di x \pd \] Substituting \(u = \sec x,\) we get \(\dd u = \sec x \tan x \di x\) and the integral becomes \[ \ba \int u \di u &= \tfrac{1}{2} u^2 + C \nl &= \tfrac{1}{2} \sec^2 x + C \pd \ea \] Therefore, \[ \int \tan^3 x \di x = \boxed{\tfrac{1}{2} \sec^2 x - \ln \abs{\sec x} + C} \]
EXERCISE 22
For
\[\ds \int \tan^4 x \di x\]
evaluate the integral.
SOLUTION Using \(\tan^2 x = \sec^2 x - 1,\) we rewrite the integral as \[ \ba \int \tan^2 x \tan^2 x \di x &= \int \tan^2 x (\sec^2 x - 1) \di x \nl &= \int \tan^2 x \sec^2 x \di x - \int \tan^2 x \di x \nl &= \int \tan^2 x \sec^2 x \di x - \int \sec^2 x \di x + \int \dd x \pd \ea \] To evaluate \(\int \tan^2 x \sec^2 x \di x,\) we let \(u = \tan x\) because then \(\dd u = \sec^2 x \di x;\) that is, we have \[\int u^2 \di u = \tfrac{1}{3} u^3 + C = \tfrac{1}{3} \tan^3 x + C \pd\] Since \(\textderiv{}{x} (\tan x)\) \(= \sec^2 x,\) we have \[ \int \tan^4 x \di x = \boxed{\tfrac{1}{3} \tan^3 x - \tan x + x + C} \]
EXERCISE 23
For
\[\ds \int \cos 14 \theta \cos 4 \theta \di \theta\]
evaluate the integral.
SOLUTION Rewriting the product of trigonometric functions as a sum, we get \[ \ba \tfrac{1}{2} \int [\cos(14 \theta - 4 \theta) + \cos(14 \theta + 4 \theta)] \di \theta &= \tfrac{1}{2} \int \par{\cos 10 \theta + \cos 18 \theta} \di \theta \nl &= \boxed{\tfrac{1}{20} \sin 10 \theta + \tfrac{1}{36} \sin 18 \theta + C} \ea \]

EXERCISE 25
For
\[\ds \int \cos 8 x \cos 10 x \di x\]

SOLUTION Rewriting the product of trigonometric functions as a sum, we get \[ \ba \tfrac{1}{2} \int [\cos(8 x - 10 x) + \cos(8 x + 10 x)] \di x &= \tfrac{1}{2} \int [\cos (-2 x) + \cos 18 x] \di x \nl &= -\tfrac{1}{4} \sin (-2 x) + \tfrac{1}{36} \sin 18 x + C \nl &= \boxed{\tfrac{1}{4} \sin 2 x + \tfrac{1}{36} \sin 18 x + C} \ea \] Note that \(\sin(-2x)\) \(= -\sin 2x\) because sine is an odd function.

EXERCISE 27

EXERCISE 30
For
\[\ds \int \frac{1}{1 - \cos \theta} \di \theta\]
evaluate the integral.
SOLUTION Multiplying the numerator and denominator each by the conjugate, \((1 + \cos \theta),\) we see \[ \ba \int \frac{1}{1 - \cos \theta} \di \theta &= \int \frac{1}{1 - \cos \theta} \par{\frac{1 + \cos \theta}{1 + \cos \theta}} \di \theta \nl &= \int \frac{1 + \cos \theta}{1 - \cos^2 \theta} \di \theta \nl &= \int \frac{1 + \cos \theta}{\sin^2 \theta} \di \theta \nl &= \int \par{\csc^2 \theta + \csc \theta \cot \theta} \di \theta \nl &= \boxed{-\cot \theta - \csc \theta + C} \ea \]
EXERCISE 31
For
\[\ds \int \frac{\cos 2 \theta}{\sin^4 \theta} \di \theta\]
