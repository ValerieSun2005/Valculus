10.1 Exercise Solutions — Infinite Sequences

EXERCISE 1
Define a sequence.
SOLUTION A sequence is a list of numbers—for example, \[\{1, 2, 3, \dots\} \pd\]
EXERCISE 2
For \[\{1, 3, 5, 7, \dots \}\] write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION Each term is an increment by \(2.\) If \(n\) starts at \(0,\) then the sequence is represented by \[a_n = \boxed{1 + 2n}\]
EXERCISE 3
For \[\{2, 6, 18, 54, \dots\}\] write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION Each term is \(3\) times its preceding term. Thus, \[a_n = \boxed{2(3)^n}\]
EXERCISE 4
For \[\ds \left\{\frac{2}{3}, -\frac{3}{4}, \frac{4}{5}, -\frac{5}{6}, \dots \right\}\] write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION If \(n\) increases by \(1,\) then each term's numerator and denominator both increase by \(1.\) We must also account for each term's alternating sign. The correct choice of \(a_n\) is \[a_n = \boxed{\frac{2 + n}{3 + n} (-1)^n}\]
EXERCISE 5
For \[\ds \left\{\frac{1}{2}, -\frac{2}{3}, \frac{3}{4}, -\frac{4}{5}, \dots \right\}\] write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION If \(n\) increases by \(1,\) then each term's numerator and denominator both increase by \(1.\) The sign of each term alternates. We therefore have \[a_n = \boxed{\frac{1 + n}{2 + n} (-1)^n}\]
EXERCISE 6
For \[\{-2, 2, -2, 2, \dots\}\] find the value to which the sequence converges or state that it diverges.
SOLUTION The terms of the sequence alternate and never settle to any final value. Thus, the sequence diverges.
EXERCISE 7
For \[\ds \left\{\frac{n - 2}{n + 5} \right\} \subsuper{1}{\infty}\] find the value to which the sequence converges or state that it diverges.
SOLUTION We take the limit as \(n \to \infty\) of the \(n\)th term: \[\lim_{n \to \infty} \frac{n - 2}{n + 5} = 1 \pd\] Therefore, the sequence converges to \(1.\)
EXERCISE 8
For \[\ds \{0, -3, -6, -9, \dots\}\] find the value to which the sequence converges or state that it diverges.
SOLUTION As \(n \to \infty,\) the terms of the sequence approach \(-\infty.\) Thus, the sequence diverges.
EXERCISE 9
For \[\ds \{b_n\} \subsuper{0}{\infty} \cma b_n = \frac{n^2 + 4n - 8}{3n^4 + 5}\] find the value to which the sequence converges or state that it diverges.
SOLUTION We find \[\lim_{n \to \infty} b_n = \lim_{n \to \infty} \frac{n^2 + 4n - 8}{3n^4 + 5} = 0 \pd\] Thus, sequence \(\{b_n\}\) converges to \(0.\)
EXERCISE 10
For \[\ds \{a_n\} \subsuper{1}{\infty} \cma a_n = \frac{n^3 - 8n^2 + 12n - 5}{9n^2 + 5n - 12}\] find the value to which the sequence converges or state that it diverges.
SOLUTION We find \[\lim_{n \to \infty} \frac{n^3 - 8n^2 + 12n - 5}{9n^2 + 5n - 12} = \infty \pd\] Since \(\{a_n\}\) does not approach any value, the sequence diverges.
EXERCISE 11
For \[\ds \left\{\frac{3}{4}, \frac{9}{16}, \frac{27}{64}, \frac{81}{256}, \dots\right\}\] find the value to which the sequence converges or state that it diverges.
SOLUTION The general term of this sequence is \[a_n = \par{\frac{3}{4}}^n \cma\] starting from \(n = 1.\) Then \(\lim_{n \to \infty} a_n = 0,\) so the sequence converges to \(0.\)
