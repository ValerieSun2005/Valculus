EXERCISE 2
For
\[2x^3 + 4x^2 + x - 3 \cmaa x_1 = 0\]
use two iterations of Newton's Method with the given initial approximation to estimate the function's zero.
SOLUTION Differentiating gives \(f'(x) = 6x^2 + 8x + 1.\) So the first iteration of Newton's Method yields \[ \ba x_2 &= x_1 - \frac{f(x_1)}{f'(x_1)} \nl &= 0 - \frac{f(0)}{f'(0)} \nl &= 0 - \frac{2(0)^3 + 4(0)^2 + (0) - 3}{6(0)^2 + 8(0) + 1} \nl &= 3 \pd \ea \] The next iteration then gives \[ \ba x_3 &= x_2 - \frac{f(x_2)}{f'(x_2)} \nl &= 3 - \frac{f(3)}{f'(3)} \nl &= 3 - \frac{2(3)^3 + 4(3)^2 + (3) - 3}{6(3)^2 + 8(3) + 1} \nl &= \frac{147}{79} \approx \boxed{1.861} \ea \]

EXERCISE 14
Consider the polynomial \(f(x) = x^7 + x - 1.\)
  1. Prove that \(f\) has exactly one zero in \([0, 1].\)
  2. Use Newton's Method with three iterations to estimate the zero, starting from \(x_1 = 0.\)
SOLUTION
  1. Observe that \(f(0) = -1\) and \(f(1) = 1.\) Because \(f(x)\) is a polynomial, it is continuous for all \(x,\) including on \([0, 1].\) Note that \(0\) is between \(-1\) and \(1.\) Thus, the Intermediate Value Theorem (see Section 1.4) guarantees a number \(r\) in \([0, 1]\) such that \(f(r) = 0.\) So \(f\) has a zero in \([0, 1].\) In addition, \(f'(x) = 7x^6 + 1 \gt 0,\) so \(f\) is increasing for all \(x.\) Therefore, \(f\) must only have one zero in \([0, 1].\)
  2. Newton's Method gives \[ \ba x_2 &= x_1 - \frac{f(x_1)}{f'(x_1)} \nl &= 0 - \frac{(0)^7 + (0) - 1}{7(0)^6 + 1} \nl &= 1 \pd \ea \] The next iteration yields \[ \ba x_3 &= x_2 - \frac{f(x_2)}{f'(x_2)} \nl &= 1 - \frac{(1)^7 + (1) - 1}{7(1)^6 + 1} \nl &= 0.875 \pd \ea \] Our final approximation for the zero is \[ \ba x_4 &= x_3 - \frac{f(x_3)}{f'(x_3)} \nl &= 0.875 - \frac{(0.875)^7 + (0.875) - 1}{7(0.875)^6 + 1} \nl &\approx \boxed{0.810} \ea \]
EXERCISE 15
For positive \(k,\) let \(f(x) = (1 + x)^k - 1.\) Prove that the \(n\)th iteration of Newton's Method gives \[x_{n + 1} = x_n + \frac{1 - (1 + x_n)^k}{k (1 + x_n)^{k - 1}} \pd\]
SOLUTION Note that \(f'(x) = k(1 + x)^{k - 1}.\) We therefore have \[ \ba x_{n + 1} &= x_n - \frac{f(x_n)}{f'(x_n)} \nl &= x_n - \frac{(1 + x_n)^k - 1}{k (1 + x_n)^{k - 1}} \nl &= x_n + \frac{1 - (1 + x_n)^k}{k (1 + x_n)^{k - 1}} \cma \ea \] as requested.
EXERCISE 16
Using Newton's Method, calculate the critical numbers of \[f(x) = x^5 - \ln \abs x\] correct to four decimal places.
SOLUTION Let's first find the critical numbers algebraically and then use Newton's Method to approximate their values.

Critical Numbers Recall, from Section 3.1, that \(f\) has a critical point at \(x\) if \(f'(x) = 0\) or if \(f(x)\) is defined but \(f'(x)\) is undefined. Differentiating shows \[f'(x) = 5x^4 - \frac{1}{x} \pd\] Note that \(f'(0)\) is undefined, yet \(x = 0\) is not a critical number since \(f(0)\) is also undefined. Instead, solving \(f'(x) = 0\) shows \[ \ba 5x^4 - \frac{1}{x} &= 0 \nl 5x^5 &= 1 \nl \implies x &= \sqrt[5]{\frac{1}{5}} \pd \ea \]

Newton's Method The number \(\sqrt[5]{1/5}\) is the zero of \(g(x) = x^5 - 1/5.\) Note that \(g'(x) = 5x^4.\) Let's use Newton's Method with any nonzero starting approximation—say, \(x_1 = 1.\) So \[ \ba x_2 &= x_1 - \frac{g(x_1)}{g'(x_1)} \nl &= 1 - \frac{(1)^5 - \tfrac{1}{5}}{5(1)^4} \nl &= \frac{21}{25} = 0.84 \pd \ea \] Subsequent approximations are \[ \ba x_3 &= 0.75234 \cma \nl x_4 &= 0.72673 \cma \nl x_5 &= 0.72479 \pd \ea \] Let's cease the approximation since \(x_4\) and \(x_5\) both agree to the desired four decimal places. The critical number is therefore \(\sqrt[5]{1/5} \approx \boxed{0.7247}.\)

EXERCISE 18
Let \(f(x) = \tfrac{1}{5} x^5 + 8x^2.\) Use Newton's Method to approximate the inflection points of \(f\) correct to five decimal places.
SOLUTION Let's first locate the inflection points algebraically and then use Newton's Method to approximate their values.

Inflection Points Recall, from Section 3.3, that \(f\) has an inflection point at \(x\) if \(f''\) changes sign at \(x.\) Differentiating shows \[ \ba f'(x) &= x^4 + 16x \cma \nl f''(x) &= 4x^3 + 16 \pd \ea \] The solution to \(f''(x) = 0\) is \(x = \sqrt[3]{-4}.\) Note that \(f''(x) \lt 0\) for \(x \lt \sqrt[3]{-4}\) and \(f''(x) \gt 0\) for \(x \gt \sqrt[3]{-4}.\) Thus, \(f\) has an inflection point at this value of \(x.\)

Newton's Method The number \(\sqrt[3]{-4}\) is the zero of \(g(x) = x^3 + 4.\) Note that \(g'(x) = 3x^2.\) Let's use Newton's Method with any nonzero starting approximation—say, \(x_1 = 1.\) Then \[ \ba x_2 &= x_1 - \frac{g(x_1)}{g'(x_1)} \nl &= 1 - \frac{(1)^3 + 4}{3(1)^2} \nl &= -\frac{2}{3} \approx -0.666667 \pd \ea \] Subsequent approximations give \[ \ba x_3 &= -3.444444 \cma \nl x_4 &= -2.408679 \cma \nl x_5 &= -1.835602 \cma \nl x_6 &= -1.619449 \cma \nl x_7 &= -1.588031 \cma \nl x_8 &= -1.587401 \cma \nl x_9 &= -1.587401 \pd \ea \] Let's cease the approximation since \(x_8\) and \(x_9\) both agree to the desired five decimal places. The inflection point is therefore at \(x = \sqrt[3]{-4} \approx \boxed{-1.58740}.\)

EXERCISE 19
Let \(f(x) = x^5 + x + 1.\) Use Newton's Method to approximate, correct to four decimal places, the values of \(c\) that satisfy the Mean Value Theorem for \(f\) on \([-1, 1].\)
SOLUTION Let's first solve for \(c\) algebraically and then use Newton's Method to attain numerical estimates.

Mean Value Theorem See Section 3.2 to review the hypothesis of the Mean Value Theorem. Since \(f\) is a polynomial, it is differentiable for all \(x.\) Thus, \(f\) is continuous on \([-1, 1]\) and differentiable on \((-1, 1)\) and so the Mean Value Theorem guarantees at least one value \(c\) in \((-1, 1)\) such that \[ \ba f'(c) &= \frac{f(1) - f(-1)}{1 - (-1)} \nl &= \frac{3 - (-1)}{1 - (-1)} = 2 \pd \ea \] Differentiating gives \(f'(x) = 5x^4 + 1.\) Therefore, we have \[ \ba f'(c) = 5c^4 + 1 &= 2 \nl 5c^4 &= 1 \nl \implies c &= \pm \sqrt[4]{\frac{1}{5}} \pd \ea \]

Newton's Method The number \(\sqrt[4]{1/5}\) is the positive zero of \(g(x) = x^4 - 1/5.\) Note that \(g'(x) = 4x^3.\) Let's use Newton's Method with any nonzero starting approximation—say, \(x_1 = 1.\) (This should be positive to avoid converging to the negative zero, \(x = -\sqrt[4]{1/5}.\)) Then \[ \ba x_2 &= x_1 - \frac{g(x_1)}{g'(x_1)} \nl &= 1 - \frac{(1)^4 - \tfrac{1}{5}}{4(1)^3} \nl &= \frac{4}{5} = 0.8 \pd \ea \] Subsequent approximations give \[ \ba x_3 &= 0.69766 \cma \nl x_4 &= 0.67049 \cma \nl x_5 &= 0.66875 \cma \nl x_6 &= 0.66874 \pd \ea \] Let's cease the approximation since \(x_5\) and \(x_6\) both agree to the desired four decimal places. Given \(c = \pm \sqrt[4]{1/5},\) the values of \(c\) are approximated to be \(\boxed{c \approx -0.6687}\) and \(\boxed{c \approx 0.6687}.\)

EXERCISE 20
Using Newton's Method for the equation \(x^2 - a = 0,\) show that \(\sqrt a\) is approximated by the algorithm \[x_{n + 1} = \frac{1}{2} \par{x_n + \frac{a}{x_n}} \pd\] Then use this formula to calculate \(\sqrt{500}\) correct to four decimal places.
SOLUTION Letting \(f(x) = x^2 - a,\) we have \(f'(x) = 2x.\) Then Newton's Method gives \[ \ba x_{n + 1} &= x_n - \frac{f(x_n)}{f'(x_n)} \nl &= x_n - \frac{\par{x_n}^2 - a}{2 x_n} \nl &= x_n - \frac{x_n}{2} + \frac{a}{2 x_n} \nl &= \frac{x_n}{2} + \frac{a}{2 x_n} \nl &= \frac{1}{2} \par{x_n + \frac{a}{x_n}} \cma \ea \] as requested. Let's begin our approximations with some reasonable guess, such as \(x_1 = 20.\) Then \[ \baat{2} x_2 &= \frac{1}{2} \par{20 + \frac{500}{20}} &&= 22.5 \cma \nl x_3 &= \frac{1}{2} \par{22.5 + \frac{500}{22.5}} &&\approx 22.36111 \cma \nl x_4 &= \frac{1}{2} \par{22.36111 + \frac{500}{22.36111}} &&\approx 22.36068 \cma \nl x_5 &= \frac{1}{2} \par{22.36068 + \frac{500}{22.36068}} &&\approx 22.36068 \pd \nl \eaat \] Note that \(x_4\) and \(x_5\) both agree to four decimal places. Thus, we estimate \(\sqrt{500} \approx \boxed{22.3606}.\)
EXERCISE 21
If \(x_1 \ne 0,\) then prove that Newton's Method completely fails to locate the zero of \(f(x) = \sqrt x.\)
SOLUTION Note that \(f'(x) = 1/(2 \sqrt x)\) and so \[ \ba x_2 &= x_1 - \frac{f(x_1)}{f'(x_1)} \nl &= x_1 - \frac{\sqrt{x_1}}{\dfrac{1}{2 \sqrt{x_1}}} \nl &= x_1 - 2 x_1 \nl &= -x_1 \pd \ea \] But the domain of \(f(x) = \sqrt x\) is \([0, \infty).\) Thus, any choice of \(x_1\) (apart from \(0\)) must be positive; then \(x_2 = -x_1\) is negative. So Newton's Method gives an approximation that is outside of the domain of \(f,\) thus failing completely.
EXERCISE 22
Determine the interval of starting approximations \(x_1\) such that Newton's Method yields invalid approximations for the zero of \(f(x) = 1 + \ln x.\)
SOLUTION Note that \(f'(x) = 1/x.\) So Newton's Method gives \[ \ba x_2 &= x_1 - \frac{f(x_1)}{f'(x_1)} \nl &= x_1 - \frac{1 + \ln x_1}{1/x_1} \nl &= x_1 - x_1 - x_1 \ln x_1 \nl &= - x_1 \ln x_1 \pd \ea \] The domain of \(f(x) = 1 + \ln x\) is \(x \gt 0.\) Newton's Method therefore yields invalid zeros if \(x_2 \leq 0\)—that is, \[ -x_1 \ln x_1 \leq 0 \iffArrow x_1 \ln x_1 \geq 0 \pd \] Note that \(\ln x_1 \geq 0\) for \(\boxed{x_1 \geq 1}.\)
EXERCISE 23
Let \(f(x) = x^3 - 5.\) Find all the starting approximations \(x_1\) for which Newton's Method yields a second approximation of \(x_2 = 1.\)
SOLUTION Differentiating gives \(f'(x) = 3x^2.\) So Newton's Method gives \[ \ba x_2 &= x_1 - \frac{f(x_1)}{f'(x_1)} \nl &= x_1 - \frac{\par{x_1}^3 - 5}{3 \par{x_1}^2} \nl &= x_1 - \frac{x_1}{3} + \frac{5}{3 \par{x_1}^2} \nl &= \frac{2 x_1}{3} + \frac{5}{3 \par{x_1}^2} \pd \ea \] Equating \(x_2 = 1\) gives \[ \ba 1 &= \frac{2 x_1}{3} + \frac{5}{3 \par{x_1}^2} \nl 3 \par{x_1}^2 &= 2 \par{x_1}^3 + 5 \pd \ea \] By inspection, a solution to the preceding equation is \(x_1 = -1.\) Using polynomial division, we find \[2 \par{x_1}^3 - 3 \par{x_1}^2 + 5 = \par{x_1 + 1} \parbr{2 \par{x_1}^2 - 5 x_1 + 5} \pd\] The discriminant of the expression in brackets is \((-5)^2 - 4(2)(5)\) \(= -15.\) Since this value is negative, the quadratic expression has no zeros. Hence, the only solution is \(\boxed{x_1 = -1}.\)
EXERCISE 24
Determine all the starting approximations \(x_1\) in \([0, 1]\) for which Newton's Method fails to converge to the zero of \(f(x) = x^3 + 2.\)
SOLUTION Note that \(f'(x) = 3x^2.\) The graph of \(f\) therefore has a horizontal tangent when \(f'(x) = 0,\) that is, at \(x = 0.\) So \(x_1 = 0\) causes Newton's Method to fail because the tangent line never strikes the \(x\)-axis. For other values of \(x_1,\) the iterative formula is \[ \ba x_2 &= x_1 - \frac{f(x_1)}{f'(x_1)} \nl &= x_1 - \frac{\par{x_1}^3 + 2}{3 \par{x_1}^2} \nl &= x_1 - \frac{x_1}{3} - \frac{2}{3 \par{x_1}^2} \nl &= \frac{2 x_1}{3} - \frac{2}{3 \par{x_1}^2} \pd \ea \] Newton's Method fails to converge if it yields \(x_2 = 0,\) the location of the horizontal tangent. Solving shows \[ \ba 0 &= \frac{2 x_1}{3} - \frac{2}{3 \par{x_1}^2} \nl 0 &= \par{x_1}^3 - 1 \nl \implies x_1 &= 1 \pd \ea \] Newton's Method also fails if it yields \(x_2 = 1,\) since the following approximation would be \(x_3 = 0.\) The function \(2x_1/3 - 2/[3(x_1)^2]\) is increasing for positive \(x_1,\) so the solution to \[ 1 = \frac{2 x_1}{3} - \frac{2}{3 \par{x_1}^2} \] satisfies \(x_1 \gt 1\) (the solution is \(x_1 \approx 1.806\)). Newton's Method also fails if it yields \(x_2 \approx 1.806\) (because then \(x_3 = 1\) and \(x_4 = 0\)), yet the solution to \[ 1.806 = \frac{2 x_1}{3} - \frac{2}{3 \par{x_1}^2} \] satisfies \(x_1 \gt 1.806\) (the solution is \(x_1 \approx 2.834\)). This pattern is repeating—subsequent defective values of \(x_1\) are all outside \([0, 1].\) But inside \([0, 1],\) there are no other cases in which Newton's Method fails. Note that \(f'\) is nonnegative, so \(f\) is always increasing. Hence, \(f\) has only one zero, and Newton's Method cannot converge to another zero. Additionally, in Example 3 Newton's Method fails because the approximations bounce back and forth. But \(f\) is increasing, so the tangent lines always have nonnegative slope, thus avoiding the bouncing issue. Hence, the defective starting approximations \(x_1 \in [0, 1]\) are \(\boxed{x_1 = 0}\) and \(\boxed{x_1 = 1}.\)
