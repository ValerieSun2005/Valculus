3.4 — Using the First and Second Derivatives

When we analyze graphs, it is imperative that we be able to describe their properties. From algebra class, we categorized a graph either as increasing or decreasing over an interval. But calculus provides us one more property—concavity, which describes the change in a function's rate of change. In this section we discuss the following:

  • Connecting \(f'\) to \(f\)
  • Concavity: Connecting \(f''\) to \(f\)
  • The Second-Derivative Test

Connecting \(f'\) to \(f\)

Let \(x_1\) and \(x_2\) be two numbers such that \(x_1 \lt x_1.\) If a function \(f\) is increasing over an interval, then \(f(x_1) \lt f(x_2)\) for \(x_1\) and \(x_2\) in that interval. Conversely, \(f\) is decreasing if \(f(x_1) \gt f(x_2).\) In Figure 1, the graph of \(y = f(x)\) is decreasing between points \(A\) and \(B,\) increasing between \(B\) and \(C,\) and decreasing between \(C\) and \(D.\) When \(f\) is increasing, we see that tangents to the curve have positive slope; thus, \(f' \gt 0.\) But when \(f\) is decreasing, the tangents have negative slope and so \(f' \lt 0.\) We therefore conclude the following: \(f\) is increasing if \(f' \gt 0\) and \(f\) is decreasing if \(f' \lt 0.\) We prove these facts using the Mean Value Theorem (from Section 3.3).

HOW \(f'\) AFFECTS \(f\)
  1. If \(f' \gt 0\) on an interval, then \(f\) is increasing on that interval.
  2. If \(f' \lt 0\) on an interval, then \(f\) is decreasing on that interval.

PROOF Consider the interval \([x_1, x_2]\) with \(x_1 \lt x_2.\) We must prove that if \(f'(x) \gt 0\) in this interval, then \(f(x_1) \lt f(x_2)\) and so \(f\) is increasing. Then we aim to show that if \(f'(x) \lt 0\) in this interval, then \(f(x_1) \gt f(x_2),\) indicating that \(f\) is decreasing. We assume that \(f\) is continuous over \(x_1 \leq x \leq x_2\) and differentiable over \(x_1 \lt x \lt x_2.\) Thus, the Mean Value Theorem guarantees a number \(c\) in \((x_1, x_2)\) such that \begin{equation} f(x_2) - f(x_1) = f'(c) (x_2 - x_1) \pd \label{eq:proof-f'-mvt} \end{equation} If \(f' \gt 0\) over \((x_1, x_2),\) then the right-hand side of \(\eqref{eq:proof-f'-mvt}\) becomes a product of positive numbers because \(f'(c) \gt 0\) and \(x_2 - x_1 \gt 0.\) Consequently, \[f(x_2) - f(x_1) \gt 0 \or f(x_1) \lt f(x_2) \cma\] so \(f\) is increasing on \([x_1, x_2].\) Hence, part (a) is proved. If \(f' \lt 0\) over \((x_1, x_2),\) then in \(\eqref{eq:proof-f'-mvt}\) \(f'(c) \lt 0,\) so the right-hand side is a negative quantity. Thus, \[f(x_2) - f(x_1) \lt 0 \or f(x_1) \gt f(x_2) \cma\] meaning \(f\) is decreasing over \([x_1, x_2].\) Part (b) is proved. \[\qedproof\]

EXAMPLE 1
Let \(f(x) = \tfrac{1}{3} x^3 - 2x^2 + 3x + 2.\) Find the intervals on which \(f\) is increasing and on which \(f\) is decreasing.
The function \(f\) is increasing if \(f'(x) \gt 0\) and decreasing if \(f'(x) \lt 0.\) Differentiating \(f\) gives \[f'(x) = x^2 - 4x + 3 = (x - 3)(x - 1) \pd\] We find that \(f'(x) \lt 0\) for \(x\) in \((1, 3),\) so \(f(x)\) is increasing over \([1, 3].\) Also, \(f'(x) \gt 0\) for \(x\) in \((-\infty, 1)\) and \((3, \infty),\) so \(f\) is decreasing for \((-\infty, 1]\) and \([3, \infty).\) (See Figure 2.) Note that we write intervals of increase or decrease as closed intervals when \(f\) continuous at the endpoints.
EXAMPLE 2
In Figure 3, the graph of \(y = g'(x)\) is shown. Where is \(g\) increasing, and where is \(g\) decreasing?
This graph is of \(g',\) not \(g.\) Function \(g\) increases when \(g'(x) \gt 0\) and decreases when \(g'(x) \lt 0.\) We see \(g'(x) \gt 0\) for \(-4 \lt x \lt 3,\) so \(g(x)\) is increasing over \([-4, 3].\) Conversely, \(g'(x) \lt 0\) for \(-6 \leq x \lt -4\) and \(3 \lt x \leq 6.\) Thus, \(g(x)\) is decreasing over \([-6, -4]\) and \([3, 6].\)

Concavity: Connecting \(f''\) to \(f\)

We can determine whether a function is increasing or decreasing over an interval. But we delve one step deeper to consider concavity. In Figure 4A and Figure 4B, both functions \(f\) and \(g\) are increasing, but they are bent differently: We call \(f\) concave upward (or simply concave up) and \(g\) concave downward (or concave down). The word concave means curving inward; it's easy to remember the phrase "cave in." The graph of \(f\) appears to bend inward when viewed from above, whereas the graph of \(g\) looks to arch inward if we view it from below.

If we draw tangents to the graph of \(f,\) whose graph is concave up, then we see that the graph lies above all the tangents (Figure 5A). Thus, the rate of change of \(f\) is increasing. In other words, the derivative \(f'\) is an increasing function; accordingly, the second derivative \(f''\) is positive. By contrast, drawing tangents to the graph of \(g\) (whose graph is concave down) shows that the graph lies below all the tangents (Figure 5B). Because the slopes of the tangents are decreasing, \(f'\) is a decreasing function and so \(f''\) is negative.

CONCAVITY
  1. If \(f''(x) \gt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave up on that interval.
  2. If \(f''(x) \lt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave down on that interval.
EXAMPLE 3
Let \(f(x) = \tfrac{1}{6}x^3 - x^2 + 6x + 1.\) Describe the characteristics of the graph of \(f(x)\) when \(x = 1.\)
We first need to determine whether \(f\) is increasing or decreasing when \(x = 1;\) we'll then identify the graph's concavity. Taking derivatives, we find \[f'(x) = \tfrac{1}{2} x^2 - 2x + 6 \and f''(x) = x - 2 \pd\] Then \[ \ba f'(1) &= \tfrac{1}{2}(1)^2 - 2(1) + 6 = \tfrac{5}{2} \nl f''(1) &= 1 - 2 = -1 \pd \ea \] Because \(f'(1) \gt 0,\) \(f\) is increasing when \(x = 1.\) Also, \(f''(1) \lt 0,\) meaning the graph of \(f\) is concave down at \(x = 1.\) Drawing a tangent line to \(f\) at \(x = 1,\) we see that the graph lies below this tangent—the geometric meaning of \(f\) being concave down. (See Figure 6.)
EXAMPLE 4
Let \(f(x) = \tfrac{1}{3}x^3 - 2x^2 + x + 4.\) Where is the graph of \(f\) concave up, and where is it concave down?
The graph of \(f\) is concave up when \(f''(x) \gt 0\) and concave down when \(f''(x) \lt 0.\) Differentiating \(f,\) we find \[ f'(x) = x^2 - 4x + 1 \and f''(x) = 2x - 4 \pd \] Then \(f''(x) \gt 0\) when \(x \gt 2\) and \(f''(x) \lt 0\) when \(x \lt 2.\) Thus, the graph of \(f(x)\) is concave up over \((2, \infty)\) and concave down over \((-\infty, 2).\) In Figure 7, the tangents to the graph of \(f\) are above the graph for \(x \lt 2\) and below the graph for \(x \gt 2.\)

A point at which the graph of \(f\) changes concavity—from concave up to concave down or from concave down to concave up—is called an inflection point. For example, in Example 4 the graph of \(f\) switches from concave down to concave up at \(x = 2.\) Thus, \(x = 2\) is the location of an inflection point of \(f.\) An easy way to estimate the location of an inflection point of \(f\) is to identify a point at which \(f\) crosses its tangent.

INFLECTION POINT
The function \(f\) has an inflection point at \(x = a\) if the graph of \(f\) switches concavity at \(x = a.\)
EXAMPLE 5
Let \(f(x) = x^4 - 2x^3 - 12x^2 - 17x + 4.\) Find all inflection points of \(f.\)
At an inflection point, the graph of \(f\) changes concavity, so we seek sign changes in \(f''.\) Because \(f''\) is continuous, we find the solutions to \(f''(x) = 0.\) At these values of interest, we analyze whether \(f''(x)\) changes sign. Differentiating gives \[f'(x) = 4x^3 - 6x^2 - 24x - 17 \and f''(x) = 12x^2 - 12x - 24 \pd\] We then solve \(f''(x) = 0 \col\) \[ \ba 12x^2 - 12x - 24 &= 0 \nl x^2 - x - 2 &= 0 \nl (x - 2)(x + 1) &= 0 \nl \implies x &= -1 \cma x = 2 \pd \ea \] Because \(f''(x) \gt 0\) for \(x \lt -1\) and \(f''(x) \lt 0\) for \(-1 \lt x \lt 2,\) we confirm that \(x = -1\) is an inflection point because at that point \(f\) changes from concave up to concave down. Likewise, \(f''(x) \gt 0\) for \(x \gt 2;\) accordingly, \(f\) has an inflection point at \(x = 2\) because \(f\) switches from concave down to concave up at that point. (See Figure 8.)

To find inflection points of \(f,\) it is not sufficient to solve \(f''(x) = 0.\) You must also determine whether \(f''(x)\) changes sign at that point.

The Second-Derivative Test

Another application of second derivatives is the Second-Derivative Test, shown below. This test is an alternative to the First-Derivative Test (from Section 3.2).

SECOND-DERIVATIVE TEST
Let \(f''(x)\) be continuous near \(x = c.\)
  1. If \(f'(c) = 0\) and \(f''(c) \gt 0,\) then \(f\) has a relative minimum at \(x = c.\)
  2. If \(f'(c) = 0\) and \(f''(c) \lt 0,\) then \(f\) has a relative maximum at \(x = c.\)
  3. If \(f'(c) = 0\) and \(f''(c) = 0\) or \(f''(c)\) does not exist, then the test is inconclusive.
The Second-Derivative Test has its limitations: The test fails if \(f''(c)\) equals \(0\) or does not exist. In either case, we must instead use the First-Derivative Test.

In part (a), because \(f''(c) \gt 0,\) \(f\) is concave up at \(c.\) Accordingly, the graph of \(f\) must lie above its tangent at \(c,\) so \(f\) has a relative minimum at \(c.\) (See Figure 9A.) Conversely, in part (b) \(f''(c) \lt 0,\) meaning \(f\) is concave down at \(c.\) So the graph of \(f\) lies below its tangent at \(c,\) meaning \(c\) must be a relative maximum of \(f,\) as in Figure 9B. Part (c) states that \(f\) can have either a relative minimum, relative maximum, or neither at the critical number \(c\) if \(f''(c)\) equals \(0\) or does not exist.

EXAMPLE 6
Find and classify the relative extrema of \(f(x) = 2x^3 - 6x^2 - 18x + 5\) using the Second-Derivative Test.
We locate the critical numbers of \(f\) by letting \(f'(x) = 0.\) Differentiating gives \[f'(x) = 6x^2 - 12x - 18 = 6(x - 3)(x + 1) \cma\] whose zeros are \(x = -1\) and \(x = 3.\) Thus, these are the critical numbers of \(f.\) We will use the Second-Derivative Test to determine whether each critical number corresponds to a relative minimum, relative maximum, or neither of \(f.\) The second derivative of \(f\) is \[f''(x) = 12x - 12 \pd \] We have \[f''(-1) = -24 \and f''(3) = 24 \pd\] Because \(f''(-1) \lt 0,\) \(f\) has a relative maximum at \(x = -1.\) Since \(f''(3) \gt 0,\) \(f\) has a relative minimum at \(x = 3.\) (See Figure 10.)

Connecting \(f'\) to \(f\) The function \(f',\) the first derivative of \(f,\) signifies whether \(f\) is increasing or decreasing over an interval.

  1. If \(f' \gt 0\) on an interval, then \(f\) is increasing on that interval.
  2. If \(f' \lt 0\) on an interval, then \(f\) is decreasing on that interval.

Concavity: Connecting \(f''\) to \(f\) We use the second derivative \(f''\) to determine the concavity of the graph of \(f.\) The graph of \(f\) is concave up over an interval if the graph lies above its tangents on that interval. Conversely, the graph is concave down if the graph lies below its tangents.

  1. If \(f''(x) \gt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave up on that interval.
  2. If \(f''(x) \lt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave down on that interval.

The function \(f\) has an inflection point at \(x = a\) if the graph of \(f\) switches concavity at \(x = a.\)

Second-Derivative Test The Second-Derivative Test enables us to classify critical points of \(f\) as relative minima or relative maxima.

  1. If \(f'(c) = 0\) and \(f''(c) \gt 0,\) then \(f\) has a relative minimum at \(x = c.\)
  2. If \(f'(c) = 0\) and \(f''(c) \lt 0,\) then \(f\) has a relative maximum at \(x = c.\)
  3. If \(f'(c) = 0\) and \(f''(c) = 0\) or \(f''(c)\) does not exist, then the test is inconclusive.
