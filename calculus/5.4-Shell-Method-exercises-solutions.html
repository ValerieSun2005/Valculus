5.4 Exercise Solutions: Shell Method

EXERCISE 1
Fill in the blanks: Using the Shell Method, when we rotate a region about a vertical line, we consider ___ approximating rectangles and integrate with respect to \(x.\) But when we rotate a region about a horizontal line and use the Shell Method, we consider ___ approximating rectangles and integrate with respect to \(y.\)
SOLUTION vertical; horizontal
EXERCISE 2
Let \(R\) be the region bounded by \(y = \sin \par{x^2}\) and the \(x\)-axis (Figure 8). Write an integral expression that gives the volume of the solid generated by rotating \(R\)
  1. about the \(y\)-axis
  2. about the \(x\)-axis
SOLUTION
  1. A gap exists between \(R\) and the \(y\)-axis, so we can only use the Washer Method or the Shell Method. But to use the Washer Method, we would need to integrate with respect to \(y;\) finding a suitable integral setup is therefore very difficult. But it's far easier to use the Shell Method because we integrate with respect to \(x,\) and our function is given in terms of \(x.\) The curve hits the \(x\)-axis in the first quadrant when \(x = 0\) and when \[\sin \par{x^2} = 0 \implies x^2 = \pi \implies x = \sqrt{\pi} \pd\] So our integral setup is \[V = \boxed{2 \pi \int_0^{\sqrt \pi} x \sin \par{x^2} \di x}\] Even more conveniently, the integrand is easy to antidifferentiate by substituting \(u = x^2.\)
  2. No gap exists between \(R\) and the \(x\)-axis, so we could use either the Disk Method or the Shell Method. But to use the Shell Method, we would have to reexpress the region \(R\) in terms of \(y.\) This process is very difficult. Instead, using the Disk Method is far easier; doing so gives the integral \[V = \boxed{\pi \int_0^{\sqrt \pi} \sin^2 \par{x^2} \di x}\]
EXERCISE 3
For
\(\ds y = x - x^3 \cmaa x = 0 \cmaa y = 0 \scoll \abtAxis{y}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION The function \(x - x^3\) has zeros in the first quadrant when \[x \par{1 - x^2} = 0 \implies x = 0 \cma x = 1 \pd\] The region is therefore bounded from \(x = 0\) to \(x = 1.\) Using the Shell Method with \(f(x) = x - x^3,\) we calculate the volume to be \[ \ba V &= 2 \pi \int_0^1 x \par{x - x^3} \di x \nl &= 2 \pi \int_0^1 \par{x^2 - x^4} \di x \nl &= 2 \pi \par{\tfrac{1}{3} x^3 - \tfrac{1}{5} x^5} \intEval_0^1 \nl &= \boxed{\frac{4 \pi}{15}} \approx 0.838 \pd \ea \] Note: This problem could also be solved by using the Washer Method. But doing so would be challenging. We'd need to express all quantities in terms of \(y\) (since we rotate about a vertical line); solving for \(x\) in \(y = x - x^3\) is impossible without using special methods (beyond the scope of this text).
EXERCISE 4

\(\ds y = 1 + \sqrt x \cmaa y = 0 \cmaa x = 1 \cmaa x = 4 \scoll \abtAxis{y}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION The region is bounded between \(x = 1\) and \(x = 4.\) Using the Shell Method with \(f(x) = 1 + \sqrt x,\) we find the volume to be \[ \ba V &= 2 \pi \int_1^4 x \par{1 + \sqrt x} \di x \nl &= 2 \pi \int_1^4 \par{x + x^{3/2}} \di x \nl &= 2 \pi \par{\tfrac{1}{2} x^2 + \tfrac{2}{5} x^{5/2}} \intEval_1^4 \nl &= \boxed{\frac{199 \pi}{5}} \approx 125.036 \pd \ea \]
EXERCISE 5
For
\(\ds y = \frac{1}{x^2} \cmaa y = 0 \cmaa x = 1 \cmaa x = 2 \scoll \abtAxis y\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION The region is bounded between \(x = 1\) and \(x = 2.\) We use the Shell Method with \(f(x) = 1/x^2\) to calculate the volume as follows: \[ \ba V &= 2 \pi \int_1^2 x \par{\frac{1}{x^2}} \di x \nl &= 2 \pi \int_1^2 \frac{1}{x} \di x \nl &= 2 \pi \ln \abs x \intEval_1^2 \nl &= \boxed{2 \pi \ln 2} \approx 4.355 \pd \ea \]
EXERCISE 6
For
\(\ds y = \sqrt x \cmaa y = 0 \cmaa x = 4 \scoll \abtAxis x\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Since we rotate about a horizontal line (\(y = 0\)), we need to consider horizontal approximating rectangles. Because we slice horizontally, we reexpress all our quantities in terms of \(y \col\) We have \(x = y^2,\) which intersects the line \(x = 4\) when \(y = 2.\) The region is therefore bounded from \(y = 0\) to \(y = 2.\) An approximating rectangle at \(y\) has width \(h(y) = y^2\) and is located a distance \(r(y) = y\) from the \(x\)-axis. So the Shell Method gives a volume of \[ \ba V &= 2 \pi \int_0^2 y \par{y^2} \di y \nl &= 2 \pi \par{\tfrac{1}{4} y^4} \intEval_0^2 \nl &= \boxed{8 \pi} \approx 25.133 \pd \ea \] Compare this solution to Example 5.3-1, in which we used the Disk Method to calculate the volume. (Using the Disk Method is easier for this problem.)
EXERCISE 7
For
\(\ds y = x \cmaa y = x^2 \scoll \abt{x = 4}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Equating \(x = x^2\) shows that the region is bounded between \(x = 0\) and \(x = 1.\) Since we rotate about a vertical line, we construct vertical approximating rectangles. (Remember that the rectangles' orientations are parallel to the axis of rotation.) A vertical approximating rectangle at \(x\) has height \(h(x) = x - x^2\) and is located a distance of \(r(x) = 4 - x\) away from the line \(x = 4.\) Thus, by the Shell Method the solid of revolution has volume \[ \ba V &= 2 \pi \int_0^1 (4 - x) \par{x - x^2} \di x \nl &= 2 \pi \int_0^1 \par{x^3 - 5x^2 + 4x} \di x \nl &= 2 \pi \par{\tfrac{1}{4} x^4 - \tfrac{5}{3} x^3 + 2x^2} \intEval_0^1 \nl &= \boxed{\frac{7 \pi}{6}} \approx 3.665 \pd \ea \] Compare this solution to Example 2, in which we rotated the same region about the \(y\)-axis.
EXERCISE 8
For
\(\ds x = 4 \sqrt y \cmaa x = 0 \cmaa y = 1 \scoll \abt{y = 4}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION We are rotating about a horizontal line (\(y = 4\)), so to use the Shell Method we construct horizontal approximating rectangles. A typical horizontal approximating rectangle at \(y\) has width \(h(y) = 4 \sqrt y\) and is a distance of \(r(y) = 4 - y\) away from the line \(y = 4.\) And the region is bounded from \(y = 0\) to \(y = 1,\) so the Shell Method gives a volume of \[ \ba V &= 2 \pi \int_0^1 \par{4 - y} \par{4 \sqrt y} \di y \nl &= 2 \pi \int_0^1 \par{16 y^{1/2} - 4 y^{3/2}} \di y \nl &= 2 \pi \par{\tfrac{32}{3} y^{3/2} - \tfrac{8}{5} y^{5/2}} \intEval_0^1 \nl &= \boxed{\frac{272 \pi}{15}} \approx 56.968 \pd \ea \] Compare this solution to Example 3, in which we rotated a similar region about the \(x\)-axis (the line \(y = 0\)).
EXERCISE 9
For
\(\ds y = 4x \cmaa y = 2 \sqrt x \scoll \abtAxis x\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Since we rotate about a horizontal line (the line \(y = 0\)), let's construct horizontal approximating rectangles. We should also express all our quantities in terms of \(y \col\) We have \(x = y/4\) and \(x = y^2/4.\) These functions intersect when \(y = 0\) and \(y = 1,\) so the region is bounded from \(y = 0\) to \(y = 1.\) At any \(y,\) an approximating rectangle has width \(h(y) = y/4 - y^2/4\) and is a distance of \(r(y) = y\) away from the \(x\)-axis (the axis of rotation). Hence, the Shell Method gives the volume to be \[ \ba V &= 2 \pi \int_0^1 y \par{\frac{y - y^2}{4}} \di y \nl &= \frac{\pi}{2} \int_0^1 \par{y^2 - y^3} \di y \nl &= \frac{\pi}{2} \par{\tfrac{1}{3} y^3 - \tfrac{1}{4} y^4} \intEval_0^1 \nl &= \boxed{\frac{\pi}{24}} \approx 0.131 \pd \ea \]
EXERCISE 10
For
\(\ds x = 3 \cmaa x = y^2 - 1 \scoll \abt{y = -5}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Since we rotate about a horizontal line (the line \(y = -5\)), let's construct horizontal approximating rectangles. We calculate the intersection points of the two functions as follows: \[3 = y^2 - 1 \iffArrow y^2 = 4 \iffArrow y = -2 \cma y = 2 \pd\] The region is therefore bounded between \(y = -2\) and \(y = 2.\) At any \(y,\) a horizontal approximating rectangle has width \[h(y) = 3 - \par{y^2 - 1} = 4 - y^2\] and is a distance of \(r(y) = y + 5\) away from the axis of rotation, \(y = -5.\) The Shell Method therefore gives the volume to be \[ \ba V &= 2 \pi \int_{-2}^2 (y + 5) \par{4 - y^2} \di y \nl &= 2 \pi \int_{-2}^2 \par{-y^3 - 5y^2 + 4y + 20} \di y \nl &= 2 \pi \par{-\tfrac{1}{4} y^4 - \tfrac{5}{3} y^3 + 2y^2 + 20} \intEval_{-2}^2 \nl &= \boxed{\frac{320 \pi}{3}} \approx 335.103 \pd \ea \]
EXERCISE 11
For
\(\ds y = x^2 \cmaa y = 8 - x^2 \scoll \abt{x = 6}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION We first calculate the \(x\)-values at which the curves intersect: \[x^2 = 8 - x^2 \iffArrow x^2 = 4 \iffArrow x = -2 \cma x = 2 \pd\] So the region is bounded above by \(y = 8 - x^2\) and below by \(y = x^2\) from \(x = -2\) to \(x = 2.\) Since we rotate about a vertical line (the line \(x = 4\)), let's construct vertical approximating rectangles. A typical approximating rectangle at \(x\) has height \[h(x) = \par{8 - x^2} - \par{x^2} = 8 - 2x^2\] and is located a distance of \(r(x) = 6 - x\) away from the axis of rotation, \(x = 6.\) Accordingly, the Shell Method gives a volume of \[ \ba V &= 2 \pi \int_{-2}^2 (6 - x) \par{8 - 2x^2} \di x \nl &= 2 \pi \int_{-2}^2 \par{2x^3 - 12x^2 - 8x + 48} \di x \nl &= 2 \pi \par{\tfrac{1}{2} x^4 - 4 x^3 - 4x^2 + 48 x} \intEval_{-2}^2 \nl &= \boxed{256 \pi} \approx 804.248 \pd \ea \]
EXERCISE 12
For
\(\ds y = \sqrt x \cmaa y = 2 - x \cmaa y = 0 \scoll \abt{y = -1}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Since we rotate about a horizontal line (the line \(y = -1\)), we must construct horizontal approximating rectangles. Hence, we reexpress our functions as \(x = y^2\) and \(x = 2 - y,\) which intersect when \(y = 1.\) The region is therefore bounded from \(y = 0\) to \(y = 1.\) At any \(y,\) a horizontal approximating rectangle has width \[h(y) = (2 - y) - \par{y^2} = 2 - y - y^2\] and is a distance of \(r(y) = y + 1\) away from the axis of rotation, \(y = -1.\) The Shell Method therefore gives the volume to be \[ \ba V &= 2 \pi \int_0^1 (y + 1) \par{2 - y - y^2} \di y \nl &= 2 \pi \int_0^1 \par{-y^3 - 2y^2 + y + 2} \di y \nl &= 2 \pi \par{-\tfrac{1}{4} y^4 - \tfrac{2}{3} y^3 + \tfrac{1}{2} y^2 + 2y} \intEval_0^1 \nl &= \boxed{\frac{19 \pi}{6}} \approx 9.948 \pd \ea \]
EXERCISE 13
For
\(\ds y = x + 3 \cmaa y = 6 - 2x \cmaa y = -1 \scoll \abt{y = -7}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Because we rotate about a horizontal line, let's consider horizontal approximating rectangles. Since we slice horizontally, we express all quantities in terms of \(y \col\) We first reexpress the lines as \(x = y - 3\) and \(x = 3 - \tfrac{1}{2} y,\) which intersect when \[y - 3 = 3 - \tfrac{1}{2} y \implies y = 4 \pd\] So the region is bounded from \(y = -1\) to \(y = 4.\) An approximating rectangle at \(y\) has width \[h(y) = \par{3 - \tfrac{1}{2} y} - (y - 3) = 6 - \tfrac{3}{2} y\] and is located a distance of \(r(y) = y + 7\) away from the line \(y = -7.\) So the Shell Method gives \[ \ba V &= 2 \pi \int_{-1}^4 (y + 7) \par{6 - \tfrac{3}{2} y} \di y \nl &= 2 \pi \int_{-1}^4 \par{-\tfrac{3}{2} y^2 - \tfrac{9}{2} y + 42} \di y \nl &= 2 \pi \par{-\tfrac{1}{2} y^3 - \tfrac{9}{4} y^2 + 42y} \intEval_{-1}^4 \nl &= \boxed{\frac{575 \pi}{2}} \approx 903.208 \pd \ea \]
EXERCISE 14
For
\(\ds y = e^{-x^2/2} \cmaa y = 0 \cmaa x = 0 \cmaa x = 4 \scoll \abtAxis y\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION The region is bounded from \(x = 0\) to \(x = 4.\) Using the Shell Method with \(f(x) = e^{-x^2/2}\) gives the integral setup \[ V = 2 \pi \int_0^4 x e^{-x^2/2} \di x \pd \] Substituting \(u = -x^2/2,\) we find \(\dd u = -x \di x.\) When \(x = 0,\) \(u = 0;\) when \(x = 4,\) \(u = -8.\) The integral therefore becomes \[ \ba V &= -2 \pi \int_0^{-8} e^u \di u \nl &= 2 \pi \int_{-8}^0 e^u \di u \nl &= 2 \pi \par{e^u} \intEval_{-8}^0 \nl &= \boxed{2 \pi - 2 \pi e^{-8}} \approx 6.281 \pd \ea \]
EXERCISE 15
For
\(\ds x = \sqrt{y + 6} \cmaa y = x \cmaa y = -5 \scoll \abt{y = -6}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Since we rotate about a horizontal line (the line \(y = -6\)), let's construct horizontal approximating rectangles. We calculate the intersection point of the two functions as follows: \[\sqrt{y + 6} = y \implies y + 6 = y^2 \implies y^2 - y - 6 = 0 \implies y = 3 \pd\] (We ignore the negative solution \(y = -2\) because the square root expression \(\sqrt{y + 6}\) can never equal a negative number.) The region is therefore bounded between \(y = -5\) and \(y = 3.\) At any \(y,\) a horizontal approximating rectangle has width \(h(y) = \sqrt{y + 6} - y\) and is a distance of \(r(y) = y + 6\) away from the axis of rotation, \(y = -6.\) The Shell Method therefore gives the volume to be \[ \ba V &= 2 \pi \int_{-5}^3 (y + 6) \par{\sqrt{y + 6} - y} \di y \nl &= 2 \pi \int_{-5}^3 \parbr{(y + 6)^{3/2} - y^2 - 6y} \di y \nl &= 2 \pi \parbr{\tfrac{2}{5} (y + 6)^{5/2} - \tfrac{1}{3} y^3 - 3y^2} \intEval_{-5}^3 \nl &= \boxed{\frac{2824 \pi}{15}} \approx 591.457 \pd \ea \]
EXERCISE 16
For
\(\ds y = \frac{2}{x^2 + 1} \cmaa y = 1 \cmaa x = 0 \scoll \abtAxis{y}\)
use the Shell Method to find the volume of the solid generated by rotating the bounded region about the specified line.
SOLUTION Equating \(y = 2/\par{x^2 + 1}\) to \(y = 1\) gives \(x = 1.\) The region is therefore bounded between \(x = 0\) and \(x = 1.\) Since we rotate about a vertical line (the line \(x = 0\)), we construct vertical approximating rectangles. (In using the Shell Method, remember that the rectangles' orientations are parallel to the axis of rotation.) A vertical approximating rectangle at \(x\) has height \[h(x) = \frac{2}{x^2 + 1} - 1\] and is located a distance of \(r(x) = x\) away from the \(y\)-axis. Hence, the Shell Method gives the volume to be \[ \ba V &= 2 \pi \int_0^1 x \par{\frac{2}{x^2 + 1} - 1} \di x \nl &= 2 \pi \int_0^1 \frac{2x}{x^2 + 1} \di x - 2 \pi \int_0^1 x \di x \pd \ea \] Let's evaluate the first integral by substituting \(u = x^2 + 1;\) then \(\dd u = 2x \di x.\) When \(x = 0,\) \(u = 1;\) when \(x = 1,\) \(u = 2.\) So we see \[ \ba \int_0^1 \frac{2x}{x^2 + 1} \di x &= \int_1^2 \frac{1}{u} \di u \nl &= \ln \abs u \intEval_1^2 \nl &= \ln 2 \pd \ea \] Therefore, we get \[ \ba V &= 2 \pi \par{\ln 2} - 2 \pi \par{\tfrac{1}{2} x^2} \intEval_0^1 \nl &= \boxed{2 \pi \par{\ln 2 - \tfrac{1}{2}}} \approx 1.214 \pd \ea \]
EXERCISE 17

\(\ds y = \cos x \cmaa y = - \cos x \cmaa x = 0 \scoll \abt{x = -2}\)

SOLUTION The two curves intersect when \[\cos x = - \cos x \implies 2 \cos x = 0 \implies x = \frac{\pi}{2} \pd\] The region is therefore bounded from \(x = 0\) to \(x = \pi/2.\) As we rotate about a vertical line, we consider a vertical approximating rectangle at \(x,\) whose height is \[h(x) = \cos x - \par{-\cos x} = 2 \cos x\] and whose distance from \(x = -2\) is \(r(x) = x + 2.\) So we have \[V = 2 \pi \int_0^{\pi/2} (x + 2) (2 \cos x) \di x \approx \boxed{32.306}\]

EXERCISE 20
For
\(\ds y = e^{2x} \cmaa y = 4 + \sqrt x \cmaa y = 1 \cmaa y = 5 \scoll \abt{y = -0.4}\)
use a graphing calculator to find the volume of the solid generated by rotating the region about the specified line. Give your answer to three decimal places.
SOLUTION As we rotate about a horizontal line, we consider horizontal approximating rectangles. So we reexpress the functions as \(x = \tfrac{1}{2} \ln y\) and \(x = (y - 4)^2,\) \(y \geq 4.\) For \(1 \leq y \leq 4\) an approximating rectangle at \(y\) has width \(h(y) = \tfrac{1}{2} \ln y.\) But for \(4 \leq y \leq 5\) an approximating rectangle at \(y\) has width \(h(y) = \tfrac{1}{2} \ln y - (y - 4)^2.\) In each interval, the distance to \(y = -0.4\) is \(r(y) = y + 0.4.\) Because the widths change, we split the region at \(y = 4\) and use the Shell Method twice, as follows: \[ \ba V &= 2 \pi \int_1^4 (y + 0.4) \par{\tfrac{1}{2} \ln y} \di y + 2 \pi \int_4^5 (y + 0.4) \parbr{\tfrac{1}{2} \ln y - (y - 4)^2} \di y \nl &\approx \boxed{38.653} \ea \]
EXERCISE 21
For
\[\ds 2 \pi \int_0^{\pi/2} x \cos x \di x\]
the integral equals the volume of a solid of revolution. Identify the axis of rotation and describe the revolved region.
SOLUTION We identify \(r(x) = x\) and \(h(x) = \cos x.\) Hence, the axis of rotation is the \(y\)-axis. The revolved region is bounded above by \(y = \cos x\) and below by the \(x\)-axis from \(x = 0\) to \(x = \pi/2.\)
