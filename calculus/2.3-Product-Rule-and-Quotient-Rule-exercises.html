\(32.\)
By using the double-angle identity for sine, \(\sin 2x = 2 \sin x \cos x,\) determine \[\deriv{}{x} (\sin 2x) \pd\]
\(33.\)
Prove that the derivative of \(\csc x\) is \(-\csc x \cot x.\)
\(34.\)
Prove that the derivative of \(\cot x\) is \(-\csc^2 x.\)
\(35.\)
If \(g\) is differentiable, then the Reciprocal Rule for Derivatives states that \[\deriv{}{x} \parbr{\frac{1}{g(x)}} = -\frac{g'(x)}{[g(x)]^2} \pd\] Using the Quotient Rule, prove this Rule.
\(36.\)
Find all the \(x\)-coordinates at which the curve \(y = x^3 e^x\) has horizontal tangents.
\(37.\)
Determine the \(x\)-coordinates at which the curve \(y = e^x \sqrt x\) has horizontal tangents.
\(38.\)
Suppose \(f\) is a differentiable function that satisfies \(f(3) = -1\) and \(f'(3) = 2.\) If \(a(x) = x^2 \cdot f(x),\) then calculate \(a'(3).\)
\(39.\)
Let \(f\) be a differentiable function that satisfies \(f(3) = 3\) and \(f'(3) = 2.\) Let \(k(x) = x^2 \cdot f(x).\) Find \(k'(3).\)
\(40.\)
Let \(f\) be a differentiable function that satisfies \(f(3) = 2.\) The line \(y = 2 + \frac{1}{2}(x - 3)\) is tangent to the graph of \(y = f(x)\) at \(x = 3.\) Let \(k(x) = 2x^2 \cdot f(x).\) Find \(k'(3).\)
\(41.\)
Let \(f\) and \(g\) be differentiable functions such that \(f(2) = 2,\) \(f'(2) = -1,\) \(g(2) = 3,\) and \(g'(2) = -4.\) Find \(k'(2)\) for each function given.
  1. \(\ds k(x) = f(x) g(x)\)
  2. \(\ds k(x) = \frac{f(x)}{g(x)}\)
  3. \(\ds k(x) = \frac{g(x)}{f(x)}\)
  4. \(\ds k(x) = \frac{f(x)}{g(x) + f(x)}\)
\(42.\)
Let \(f(x) = 2xe^x.\) For \(n \geq 2,\) show that \[f^{(n)}(x) = 2e^x + f^{(n - 1)}(x) \pd\]
\(43.\)
Consider the family of functions \(f(x) = 1/(x^2 + 4x + k),\) where \(k\) is a constant.
  1. Show that \[f'(x) = \frac{-2x - 4}{\par{x^2 + 4x + k}^2} \pd\]
  2. If \(k = 4,\) then find the only vertical asymptote to the graph of \(y = f(x).\)
  3. For \(k \ne 4,\) determine the \(x\)-coordinate at which the graph of \(y = f(x)\) has a horizontal tangent.
  4. Determine and interpret \[\lim_{x \to -\infty} f'(x) \and \lim_{x \to \infty} f'(x) \pd\]
  5. Show that \(f\) has no vertical asymptotes if \(k \gt 4.\)
