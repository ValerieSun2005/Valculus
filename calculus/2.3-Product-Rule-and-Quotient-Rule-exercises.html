For each of exercises 1–12, find the derivative.

\(1.\) \(\ds x \sin x\) \(2.\) \(\ds x^2 \cos x\) \(3.\) \(\ds (\sin x) \sqrt{x} \)
\(4.\) \(\ds x \ln x\) \(5.\) \(\ds e^x \sin x\) \(6.\) \(\ds \frac{1}{x}\tan x\)
\(7.\) \(\ds \frac{\sin x}{x + 4}\) \(8.\) \(\ds \frac{\cos x}{x^2 + 4}\) \(9.\) \(\ds \frac{x^2}{\tan x}\)

\(13.\)
Let \(f\) be a differentiable function that satisfies \(f(3) = 2.\) The line \(y = 2 + \frac{1}{2}(x - 3)\) is tangent to the graph of \(y = f(x)\) at \(x = 3.\) Let \(k(x) = 2x^2 f(x).\) Find \(k'(3).\)
\(14.\)
Let \(f\) and \(g\) be differentiable functions such that \(f(2) = 2,\) \(f'(2) = -1,\) \(g(2) = 3,\) and \(g'(2) = -4.\) Find \(k'(2).\)
  1. \(\ds k(x) = f(x) g(x)\)
  2. \(\ds k(x) = \frac{f(x)}{g(x)}\)
  3. \(\ds k(x) = \frac{g(x)}{f(x)}\)
  4. \(\ds k(x) = \frac{f(x)}{g(x) + f(x)}\)
