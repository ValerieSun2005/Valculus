1.4 Exercise Solutions: Continuity

EXERCISE 1
In Figure 10, identify and classify the discontinuities of \(f\) on \((-4, 4).\) Describe the continuity at the endpoints, \(x = -4\) and \(x = 4.\)
SOLUTION The function \(f(x)\) has an infinite discontinuity at \(x = -1,\) a jump discontinuity at \(x = 1,\) and a removable discontinuity at \(x = 2.\) Also, \(f\) is continuous from the left at \(4\) because \[\lim_{x \to 4^-} f(x) = f(4) \pd\] Yet \(f\) is discontinuous at \(-4\) because \(f(-4)\) is undefined.
EXERCISE 2
Let's investigate the function \[f(x) = \bc x^2 + 1 &x \lt 1 \nl 3 - x &x \geq 1 \pd \ec \]
  1. Determine \(f(1).\)
  2. Find \(\lim_{x \to 1^-} f(x)\) and \(\lim_{x \to 1^+} f(x).\)
  3. Is \(f(x)\) continuous at \(x = 1 \ques\)
SOLUTION
  1. We see \[f(1) = 3 - 1 = \boxed 2\]
  2. Since \(f(x) = x^2 + 1\) for \(x \lt 1,\) we see \[\lim_{x \to 1^-} f(x) = \lim_{x \to 1^-} \par{x^2 + 1} = 1^2 + 1 = \boxed 2\] Likewise, because \(f(x) = 3 - x\) for \(x \geq 1,\) we have \[\lim_{x \to 1^+} f(x) = \lim_{x \to 1^+} \par{3 - x} = 3 - 1 = \boxed 2\]
  3. Because the one-sided limits of \(f\) at \(1\) both equal \(2,\) we have \(\lim_{x \to 1} f(x) = 2.\) We therefore say \[\lim_{x \to 1} f(x) = f(1) = 2 \pd\] By the definition of continuity, \(f\) is continuous at \(2.\)
EXERCISE 3
Consider the function \[f(x) = \frac{x^3 + x^2}{x} \pd\]
  1. Justify why \(f\) has a removable discontinuity at \(x = 0.\)
  2. Define a new function \(g\) that matches \(f\) for all \(x \ne 0\) but is continuous at \(0.\)
SOLUTION
  1. Observe that \(f(0)\) is undefined. But we see \[ \ba \lim_{x \to 0} f(x) &= \lim_{x \to 0} \frac{\cancel{x} \par{x^2 + x}}{\cancel x} \nl &= \lim_{x \to 0} \par{x^2 + x} \nl &= 0^2 + 0 = 0 \pd \ea \] Since \(\lim_{x \to 0} f(x)\) exists but \(f(0)\) is undefined, \(f\) has a removable discontinuity at \(0.\)
  2. Since \(\lim_{x \to 0} f(x) = 0,\) the graph of \(f\) has a hole at \((0, 0).\) To restore continuity, we fill in this hole by defining the new function \[ \ba g(x) &= \bc f(x) &x \ne 0 \nl 0 &x = 0 \ec \nl &= \boxed{\bc \dfrac{x^3 + x^2}{x} &x \ne 0 \nl 0 &x = 0 \ec} \ea \]
EXERCISE 4
If \(f\) is continuous at \(a,\) then \(f(a)\) is defined.
SOLUTION True. The function \(f\) is continuous at \(a\) if and only if \[\lim_{x \to a} f(x) = f(a) \pd\] This definition therefore implies that \(f(a)\) must be defined if \(f\) is continuous at \(a.\)
EXERCISE 5
If \(f\) is discontinuous at \(a,\) then \(\lim_{x \to a} f(x)\) does not exist.
SOLUTION False. The function \(f\) could have a removable discontinuity at \(a;\) in this scenario, \(\lim_{x \to a} f(x)\) exists but does not equal \(f(a)\) [or \(f(a)\) is undefined].

EXERCISE 9
For
\[\ds f(x) = \sqrt{2 - 4x} \cmaa x = -\frac{1}{2}\]
determine whether \(f\) is continuous at the given value.
SOLUTION Radical functions are continuous over all values in their domain. Since the root has \(n = 2\) (an even number), it is defined for all positive inputs. We therefore find its domain to be \(2 - 4x \geq 0,\) or \(x \leq 1/2.\) Because \(x = -1/2\) is in the domain of \(f(x),\) the function is continuous at \(x = -1/2.\)
EXERCISE 10
For
\[ \ds f(x)= \bc x^2 - 3x &x \lt -1 \nl \sin(x + 1) &x \geq -1 \cmaa \ec \; x = -1 \]
determine whether \(f\) is continuous at the given value.
SOLUTION The function \(f\) is continuous at \(-1\) if and only if \(\lim_{x \to -1} f(x) = f(-1).\) Analyzing the one-sided limits, we see \[ \baat{2} \lim_{x \to -1^-} f(x) &=(-1)^2 - 3(-1) &&= 4 \cma \nl \lim_{x \to -1^+} f(x) &= \sin(-1 + 1) &&= 0 \pd \eaat \] Because the one-sided limits are not equivalent, \(\lim_{x \to -1} f(x)\) does not exist and so \(f(x)\) is discontinuous at \(x = -1.\)
EXERCISE 11
For
\[ \ds f(x)= \bc \sqrt{1 - x} &x \leq 1 \nl \log_4(x) &x \gt 1 \cmaa \ec \; x = 1 \]
determine whether \(f\) is continuous at the given value.
SOLUTION If \(f\) is continuous at \(1,\) then \[\lim_{x \to 1} f(x) = f(1) = 0 \pd \] We consider the one-sided limits: \[ \baat{2} \lim_{x \to 1^-} f(x) &= \sqrt{1 - (1)} &&= 0 \cma \nl \lim_{x \to 1^+} f(x) &= \log_4(1) &&= 0 \pd \nl \eaat \] Thus, because \(\lim_{x \to 1} f(x) = f(1),\) we conclude that \(f(x)\) is continuous at \(x = 1.\)

Note that cosecant is defined, and therefore continuous, at \(2^2 = 4.\) Hence, the composite function \(\csc \par{x^2}\) is continuous at \(x = 2.\)
EXERCISE 13

EXERCISE 14
For
\[\ds f(x) = \frac{1}{x - 2} \cmaa x = 2\]
classify the discontinuity of \(f(x)\) at the given \(x\).
SOLUTION We have \[\lim_{x \to 2^-} f(x) = -\infty \and \lim_{x \to 2^+} f(x) = \infty \pd\] Since these one-sided limits are infinite, the graph of \(y = 1/(x - 2)\) has a vertical asymptote at \(x = 2.\) Thus, \(f\) has an infinite discontinuity at \(x = 2.\)
EXERCISE 15
For
\[\ds f(x) = \frac{x^2 - 5x + 6}{x - 3} \cmaa x = 3\]
classify the discontinuity of \(f(x)\) at the given \(x\).
SOLUTION Factoring the numerator of \(f(x),\) we see \[ \ba \lim_{x \to 3} f(x) &= \lim_{x \to 3} \frac{\cancel{(x - 3)}(x - 2)}{\cancel{x - 3}} \nl &= \lim_{x \to 3} (x - 2) \nl &= (3 - 2) = 1 \pd \ea \] Thus, \(x = 3\) is the location of a removable discontinuity of \(f(x)\) because \(\lim_{x \to 3} f(x)\) exists but \(f(3)\) is undefined.
EXERCISE 16
For
\[\ds f(x) = \frac{\abs{x}}{x} \cmaa x = 0\]
classify the discontinuity of \(f(x)\) at the given \(x\).
SOLUTION For \(x \lt 0,\) \(f(x) = -1.\) For \(x \gt 0,\) \(f(x) = 1.\) Hence, \[\lim_{x \to 0^-} \frac{\abs{x}}{x} = -1 \and \lim_{x \to 0^+} \frac{\abs{x}}{x} = 1 \pd\] Because these one-sided limits equal different numbers, \(f\) has a jump discontinuity at \(0.\)
EXERCISE 17
For
\[\ds f(x) = \frac{4x^2 - 9}{2x + 3} \cmaa x = -\frac{3}{2}\]
classify the discontinuity of \(f(x)\) at the given \(x\).
SOLUTION We factor the numerator as follows: \[4x^2 - 9 = (2x + 3)(2x - 3) \pd\] So we see \[ \ba \lim_{x \to -3/2} f(x) &= \lim_{x \to -3/2} \frac{\cancel{(2x + 3)} (2x - 3)}{\cancel{2x + 3}} \nl &= \lim_{x \to -3/2} (2x - 3) \nl &= 2 \par{-\tfrac{3}{2}} - 3 = -6 \pd \ea \] This limit exists but \(f(-3/2)\) is undefined, so \(f(x)\) has a removable discontinuity at \(x = -3/2.\)
EXERCISE 18
For
\[\ds f(x) = \frac{2 + 2x^2}{1 + x} \cmaa x = -1\]
classify the discontinuity of \(f(x)\) at the given \(x\).
SOLUTION As \(x \to -1,\) the denominator \((1 + x)\) approaches \(0\) but the numerator \(\par{2 + 2x^2}\) doesn't approach \(0.\) By making \(x\) closer to \(-1,\) we make the denominator of \(f(x)\) closer to \(0.\) Dividing by a smaller number causes \(f(x)\) to approach \(\infty\) or \(-\infty,\) depending on the sign of \(f(x).\) For \(x \lt -1,\) \(f(x) \lt 0.\) for \(x \gt -1,\) \(f(x) \gt 0.\) So we have \[\lim_{x \to -1^-} f(x) = -\infty \and \lim_{x \to -1^+} f(x) = \infty \pd\] The graph of \(f\) therefore has a vertical asymptote at \(x = -1,\) meaning \(f\) has an infinite discontinuity at \(-1.\)

EXERCISE 28
For
\[\ds \frac{9 \log x}{8 - 4x}\]
determine the interval on which the function is continuous.
SOLUTION The quotient of two continuous functions is another continuous function. We need to find where \(9 \log x\) and \(8 - 4x\) are both continuous, and avoid any point at which \(8 - 4x = 0.\) The logarithm is continuous for all \(x\) in its domain, \(x \gt 0.\) The polynomial \(8 - 4x\) is continuous for all \(x.\) But \(8 - 4x = 0\) when \(x = 2,\) so this point is excluded in the domain of the quotient of functions. Hence, \((9 \log x)/(8 - 4x)\) is continuous on \[\boxed{(0, 2) \cup (2, \infty)}\]
EXERCISE 29

EXERCISE 50
Let \[ g(x)= \bc k^2 - 4x &x \leq 2 \nl kx &x \gt 2 \pd \ec \] Find \(k\) for which \(g(x)\) is continuous at \(x = 2.\)
SOLUTION The function \(g\) is continuous at \(2\) if and only if \[\lim_{x \to 2} g(x) = g(2) \pd\] Observe that \(\lim_{x \to 2^-} g(x) = g(2),\) so we need to find \(k\) such that \[\lim_{x \to 2^-} g(x) = \lim_{x \to 2^+} g(x) \pd\] The one-sided limits at \(2\) are as follows: \[ \ba \lim_{x \to 2^-} g(x) &= k^2 - 4(2) = k^2 - 8 \cma \nl \lim_{x \to 2^+} g(x) &= 2k \pd \ea \] We equate these expressions to find \(k,\) as follows: \[ \ba k^2 - 8 = 2k &\iffArrow k^2 - 2k - 8 = 0 \nl &\iffArrow (k - 4)(k + 2) = 0 \cma \ea \] whose solutions are \[\boxed{k = -2} \and \boxed{k = 4}\]
EXERCISE 51
Determine \(k\) such that \[ f(x)= \bc 8k - x^2 &x \lt 4 \nl k^2 \sqrt{x} - k^3 &x \geq 4 \ec \] is continuous at \(x = 4.\)
SOLUTION The function \(f\) is continuous at \(4\) if and only if \[\lim_{x \to 4} f(x) = f(4) \pd\] Also observe that \(\lim_{x \to 4^+} f(x) = f(4).\) So we need to find \(k\) to satisfy \[\lim_{x \to 4^-} f(x) = \lim_{x \to 4^+} f(x) \pd\] The left-hand limit is \[ \lim_{x \to 4^-} f(x) = 8k - (4)^2 = 8k - 16 \pd \] The right-hand limit is \[ \lim_{x \to 4^+} f(x) = k^2 \sqrt{4} - k^3 = 2k^2 - k^3 \pd \] We equate these expressions and solve for \(k,\) as follows: \[ \ba 8k - 16 &= 2k^2 - k^3 \nl k^3 - 2k^2 + 8k - 16 &= 0 \nl k^2(k - 2) + 8(k - 2) &= 0 \nl (k^2 + 8)(k - 2) &= 0 \nl \implies k &= 2 \pd \ea \] So for \(\boxed{k = 2},\) \(f(x)\) is continuous at \(x = 4.\)
EXERCISE 52
Find \(p\) for which \[ k(x)= \bc 6 - x &x \leq p \nl x^2 + 4 &x \gt p \ec \] is continuous at \(x = p.\)
SOLUTION THe function \(k(x)\) is continuous at \(x = p\) if and only if \[\lim_{x \to p} k(x) = k(p) \pd\] But since \(\lim_{x \to p^-} k(x) = k(p),\) our goal is to find \(p\) for which \[\lim_{x \to p^-} k(x) = \lim_{k \to p^+} k(x) \pd\] The left-hand limit of \(k\) is \[\lim_{x \to p^-} k(x) = 6 - p \pd \] Moreover, the right-hand limit is \[\lim_{x \to p^+} k(x) = p^2 + 4 \pd \] Equating these expressions and solving for \(p\) produce \[ 6 - p = p^2 + 4 \iffArrow p^2 + p - 2 = 0 \cmaa \] whose solutions are \[\boxed{p = -2} \and \boxed{p = 1}\]
EXERCISE 53
Let \[ f(x)= \bc e^{2x} + 7 &x \leq q \nl 5e^x + 1 &x \gt q \pd \ec \] Find \(q\) for which \(f(x)\) is continuous at \(x = q.\)
SOLUTION The function \(f\) is continuous at \(q\) if and only if \[\lim_{x \to q} f(x) = f(q) \pd\] But because \(\lim_{x \to q^-} f(x) = f(q),\) we focus on finding \(q\) to satisfy \[\lim_{x \to q^-} f(x) = \lim_{x \to q^+} f(x) \pd\] The one-sided limits of \(f\) at \(q\) are \[ \ba \lim_{x \to q^-} f(x) &= e^{2q} + 7 \cma \nl \lim_{x \to q^+} f(x) &= 5e^q + 1 \pd \ea \] We equate these expressions to obtain \[ \begin{aligned} e^{2q} + 7 &= 5e^q + 1 \\ e^{2q} - 5e^q + 6 &= 0 \cmaa \\ \ea \] which we factor as \[(e^q - 2)(e^q - 3) = 0\pd\] Now we equate each factor to \(0\) to solve for \(q \col\) \[ \baat{2} e^q - 2 &= 0 \lspace &e^q - 3&=0 \nl e^q &= 2 \lspace &e^q &= 3 \nl q &= \ln 2 \lspace &q &= \ln 3 \pd \eaat \] Therefore, \(\boxed{q = \ln 2}\) and \(\boxed{q = \ln 3}\) permit \(f(x)\) to be continuous at \(x = q.\)
EXERCISE 54
Find \(a\) and \(b\) so that the following function is continuous: \[ f(x) = \bc 4x + 2 &x \lt 2 \nl ax + b &2 \leq x \lt 5 \nl \sqrt{6 - x} &x \geq 5 \pd \ec \]
SOLUTION The piecewise function \(f(x)\) has splits at \(x = 2\) and \(x = 5.\) If \(f\) is continuous, then it's continuous at these splits. Thus, we need \[\lim_{x \to 2} f(x) = f(2) \and \lim_{x \to 5} f(x) = f(5) \pd\] The one-sided limits of \(f\) at \(2\) are \[ \ba \lim_{x \to 2^-} f(x) &= 4(2) + 2 = 10 \cmaa \nl \lim_{x \to 2^+} f(x) &= 2a + b \pd \ea \] These one-sided limits must be equal, so we have \begin{equation} 2a + b = 10 \pd \label{eq:f(x)-3-piecewise-eq-1} \end{equation} Likewise, the one-sided limits of \(f\) at \(5\) are \[ \ba \lim_{x \to 5^-} f(x) &= 5a + b \cma \nl \lim_{x \to 5^+} f(x) &= \sqrt{6 - 5} = 1 \pd \ea \] These one-sided limits must be equal, so we find \begin{equation} 5a + b = 1 \pd \label{eq:f(x)-3-piecewise-eq-2} \end{equation} We now have the following system of equations: \begin{align} 2a + b &= 10 \cma \eqlabel{eq:f(x)-3-piecewise-eq-1} \nl 5a + b &= 1 \cma \eqlabel{eq:f(x)-3-piecewise-eq-2} \end{align} whose solution is \[\boxed{a = -3} \and \boxed{b = 16}\]
EXERCISE 55
Functions \(f,\) \(g,\) and \(h\) satisfy \(g(x) \leq f(x) \leq h(x)\) for \(-3 \leq x \leq -1.\) It is known that \(f(-2) = 8\) and \[\lim_{x \to -2} g(x) = \lim_{x \to -2} h(x) = 8 \pd\] Is \(f(x)\) continuous at \(x = -2 \ques\)
SOLUTION Yes. By the Squeeze Theorem (from Section 1.3), we have \(\lim_{x \to -2} f(x) = 8.\) Since this limit equals \(f(-2) = 8,\) \(f\) is continuous at \(-2.\)
EXERCISE 56
On \(2 \leq x \leq 5,\) three functions are defined such that \(p(x) \leq r(x) \leq q(x).\) Functions \(p\) and \(q\) are both continuous at \(4,\) and suppose that \(p(4) = q(4)\) \(= 0.\) At \(x = 4,\) prove that \(r(x)\) is continuous and has a zero.
SOLUTION Functions \(p\) and \(q\) are each given to be continuous at \(x = 4.\) And we are given \(p(4) = q(4) = 0,\) so we have \[ \baat{2} \lim_{x \to 4} p(x) &= p(4) &&= 0 \cma \nl \lim_{x \to 4} q(x) &= q(4) &&= 0 \pd \eaat \] By the Squeeze Theorem, \(\lim_{x \to 4} r(x) = 0.\) In addition, we have \(p(4) \leq r(4) \leq q(4)\) \(\iffArrow 0 \leq r(4) \leq 0,\) so \(r(4) = 0.\) Because \(\lim_{x \to 4} r(x) = r(4),\) \(r\) is continuous at \(x = 4.\)
