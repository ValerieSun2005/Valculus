EXERCISE 1
If \(f\) is continuous over \([a, b],\) then what is the expression for the average value of \(f\) over this interval?
SOLUTION The expression is \begin{equation} \avg f = \frac{1}{b - a} \int_a^b f(x) \di x \pd \label{eq:f-avg} \end{equation}
EXERCISE 2
What does the Mean Value Theorem for Integrals state?
SOLUTION Let \(\avg f\) be the average value of the continuous function \(f\) over \([a, b].\) The Mean Value Theorem for Integrals asserts that at sometime in this interval, \(f\) equals its average value. In other words, there is a value \(c\) in \([a, b]\) such that \begin{equation} f(c) = \frac{1}{b - a} \int_a^b f(x) \di x \pd \label{eq:mvt-integrals} \end{equation}
EXERCISE 3
For
\[f(x) = x \cma 0 \leq x \leq 2\]
calculate the average value of the function over the given interval.
SOLUTION Using \(\eqref{eq:f-avg},\) the average value is \[ \ba \avg f &= \frac{1}{2 - 0} \int_0^{2} x \di x \nl &= \tfrac{1}{2} \par{\tfrac{1}{2} x^2} \intEval_0^2 \nl &= \tfrac{1}{2} \parbr{\tfrac{1}{2}(2)^2 - 0} \nl &= \boxed 1 \ea \]
EXERCISE 4
For
\[g(t) = t^2 + 1 \cma 0 \leq t \leq 1\]
calculate the average value of the function over the given interval.
SOLUTION By \(\eqref{eq:f-avg}\) the average value is \[ \ba \avg g &= \frac{1}{1 - 0} \int_0^1 t^2 + 1 \di t \nl &= \par{\tfrac{1}{3} t^3 + t} \intEval_0^1 \nl &= \tfrac{1}{3} + 1 \nl &= \boxed{\tfrac{4}{3}} \ea \]
EXERCISE 5
For
\[f(x) = e^x \cma 0 \leq x \leq \ln 3\]
calculate the average value of the function over the given interval.
SOLUTION By \(\eqref{eq:f-avg}\) the average value is \[ \ba \avg f &= \frac{1}{(\ln 3) - 0} \int_0^{\ln 3} e^x \di x \nl &= \frac{1}{\ln 3} \par{e^x} \intEval_0^{\ln 3} \nl &= \frac{1}{\ln 3} \par{e^{\ln 3} - e^0} \nl &= \frac{1}{\ln 3} \par{3 - 1} \nl &= \boxed{\frac{2}{\ln 3}} \ea \]
EXERCISE 6
For
\[\ds h(x) = \sin x \cma 0 \leq x \leq \frac{\pi}{2}\]
calculate the average value of the function over the given interval.
SOLUTION We use \(\eqref{eq:f-avg}\) to get \[ \ba \avg h &= \frac{1}{(\pi/2) - 0} \int_0^{\pi/2} \sin x \di x \nl &= \tfrac{2}{\pi} \par{-\cos x} \intEval_0^{\pi/2} \nl &= \tfrac{2}{\pi} \parbr{\par{-\cos \tfrac{\pi}{2}} - (-\cos 0)} \nl &= \boxed{\tfrac{2}{\pi}} \ea \]
EXERCISE 7
For
\[\ds f(y) = \frac{1}{y} \cma 1 \leq y \leq 4\]
calculate the average value of the function over the given interval.
SOLUTION We use \(\eqref{eq:f-avg}\) to get \[ \ba \avg f &= \frac{1}{4 - 1} \int_1^4 \frac{1}{y} \di y \nl &= \tfrac{1}{3} (\ln y) \intEval_1^4 \nl &= \tfrac{1}{3} \parbr{(\ln 4) - (\ln 1)} \nl &= \boxed{\tfrac{1}{3} \ln 4} \ea \]
EXERCISE 8
Let \(f\) be a continuous function. For \(0 \leq x \leq 3,\) the average value of \(f\) is \(7.\) Find the value of \(\int_0^3 f(x) \di x.\)
SOLUTION We have \[\avg f = \frac{1}{3 - 0} \int_0^3 f(x) \di x = 7 \pd\] Multiplying both sides by \(3\) thus shows \[\int_0^3 f(x) \di x = 7(3) = \boxed{21}\]
EXERCISE 9
For \(1 \leq x \leq 6,\) the average value of the positive, continuous function \(g(x)\) is \(5.\) What is the area under the graph of \(y = g(x)\) from \(x = 1\) to \(x = 6 \ques\)
SOLUTION The area under the graph of the positive function \(g\) from \(1\) to \(6\) is the definite integral \(\int_1^6 g(x) \di x.\) The average value of \(g\) over this interval is \[\avg g = \frac{1}{6 - 1} \int_1^6 g(x) \di x = 5 \pd\] Thus, we multiply both sides by \(6 - 1,\) finding the area to be \[\int_1^6 g(x) \di x = 5(6 - 1) = \boxed{25}\]
EXERCISE 10
Let \(f(x) = 1/x^2.\)
  1. Show that the average value of \(f\) over \([1, 3]\) is \(1/3.\)
  2. What value of \(c\) satisfies the Mean Value Theorem for Integrals for \(f\) over \([1,3] \ques\)
SOLUTION
  1. We integrate \(f\) from \(1\) to \(3\) and divide by \((3 - 1) \col\) \[ \ba \avg f &= \frac{1}{3 - 1} \int_1^3 \frac{1}{x^2} \di x \nl &= \tfrac{1}{2} \par{-\tfrac{1}{x}} \intEval_1^3 \nl &= \tfrac{1}{2} \parbr{\par{-\tfrac{1}{3}} - \par{-1}} \nl &= \tfrac{1}{3} \cma \ea \] as requested.
  2. By the Mean Value Theorem for Integrals, there is a value \(c\) in \([1, 3]\) such that \(f(c) = \avg f\) \(= 1/3.\) We find \[f(c) = \frac{1}{c^2} = \frac{1}{3} \implies c = \boxed{\sqrt 3}\]
EXERCISE 11
A city's temperature \(t\) hours after midnight is modeled by \[T(t) = 60 + 14 \sin \frac{\pi t}{12} \cma\] measured in \(\farenheit.\) Find the city's average temperature between midnight and noon.
SOLUTION It is midnight when \(t = 0,\) and it is noon when \(t = 12.\) We therefore seek the average value of \(T\) over \(0 \leq t \leq 12.\) Using \(\eqref{eq:f-avg},\) we find \[ \ba \avg T &= \tfrac{1}{12} \int_0^{12} 60 + 14 \sin \frac{\pi t}{12} \di t \nl &= \tfrac{1}{12} \par{60t - \frac{168}{\pi} \cos \frac{\pi t}{12}} \intEval_0^{12} \nl &= \tfrac{1}{12} \parbr{60(12) - \frac{168}{\pi} (-1 + 1)} \nl &= \boxed{60 \farenheit} \ea \]
EXERCISE 12
Prove that the average value of \(f(x) = x\) over \(a \leq x \leq b\) is \(\tfrac{1}{2}(a + b).\)
SOLUTION The expression for the average value is \[\avg f = \frac{1}{b - a} \int_a^b x \di x \pd\] Integrating gives \[ \ba \avg f &= \frac{1}{b - a} \par{\tfrac{1}{2}x^2} \intEval_a^b \nl &= \frac{1}{2(b - a)} \par{b^2 - a^2} \nl &= \frac{1}{2(b - a)} (b + a)(b - a) \nl &= \tfrac{1}{2} (b + a) \cma \ea \] as requested.
EXERCISE 13
Assume that \(f(x) = 1/x^2\) is continuous for \(a \leq x \leq b.\) Prove that the average value of \(f\) over \([a, b]\) is \(\avg f = 1/ab.\) Then prove that \(c = \sqrt{ab}\) satisfies the Mean Value Theorem for Integrals for \(f\) over \([a, b].\)
SOLUTION The expression for the average value is \[\avg f = \frac{1}{b - a} \int_a^b \frac{1}{x^2} \di x \pd\] We integrate, finding \[ \ba \avg f &= \frac{1}{b - a} \par{-\frac{1}{x}} \intEval_a^b \nl &= \frac{1}{b - a} \par{\frac{1}{a} - \frac{1}{b}} \nl &= \frac{1}{b - a} \par{\frac{b - a}{ab}} \nl &= \frac{1}{ab} \cma \ea \] as requested. Moreover, we solve \(f(c) = \avg f\) for \(c \col\) \[f(c) = \frac{1}{c^2} = \frac{1}{ab} \implies c = \sqrt{ab} \pd\] Thus, \(c = \sqrt{ab}\) satisfies the Mean Value Theorem for \(f\) over \([a, b],\) as requested.
