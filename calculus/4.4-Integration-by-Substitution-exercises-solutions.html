4.4 Exercise Solutions — Integration by Substitution

EXERCISE 1
For \[\ds \int 3x^2 \sin(x^3) \di x\] evaluate the indefinite integral.
SOLUTION Our goal is to perform a substitution to simplify the integral. For the integral of a composite function, we want the derivative of the inner function to be in the integrand. In this case, the inner function of the composition \(\sin(x^3)\) is \(x^3,\) whose derivative is \(3x^2.\) This exact factor is in the integrand, so we let \(u = x^3\) because \(\dd u = 3x^2 \di x.\) The integral then simplifies to \[\int \sin(\underbrace{x^3}_{u}) \underbrace{(3x^2) \di x}_{\dd u} = \int \sin u \di u = -\cos u + C\] for some constant \(C.\) Note that as we execute the integration in the "\(u\) world," no \(x\) terms remain—this is perfect. In addition, the new integral is the trivial \(\int \sin u \di u,\) for which we readily identify the solution \(-\cos u + C.\) Finally, we express our answer in terms of \(x.\) Since \(u = x^3,\) our final answer is \[\boxed{-\cos(x^3) + C}\] We can check this answer by differentiating it; doing so returns us the integrand \(3x^2 \sin(x^3).\)
EXERCISE 2
For \[\ds \int 4x \cos(2x^2) \di x\] evaluate the indefinite integral.
SOLUTION We want to simplify the integral to a trivial form. The inner function of the composition \(\cos(2x^2)\) is \(2x^2,\) whose derivative is \(4x.\) This term perfectly lies in the integrand, so we substitute \(u = 2x^2.\) Then \(\dd u = 4x \di x\) and so the integral becomes \[\int \cos(\underbrace{2x^2}_{u}) \underbrace{(4x) \di x}_{\dd u} = \int \cos u \di u = \sin u + C\] for any constant \(C.\) Lastly, we substitute back; we let \(u = 2x^2,\) so our final answer is \[\boxed{\sin(2x^2) + C}\] If you differentiate this answer, then you obtain the integrand \(4x \cos(2x^2).\) Hence, this expression is correct.
EXERCISE 3
For \[\ds \int \sin(-x) \di x\] evaluate the indefinite integral.
SOLUTION The inner function is \(-x,\) whose derivative is \(-1.\) Let us rewrite the integral as \[\int -(-1) \sin(-x) \di x \pd\] Then we substitute \(u = -x,\) from which \(\dd u = (-1) \di x.\) We therefore have \[\int -\underbrace{\sin(-x)}_{\sin u} \underbrace{(-1) \di x}_{\dd u} = - \int \sin u \di u = -(-\cos u) + C = \cos u + C\] for some constant \(C.\) Finally, we reexpress our answer in terms of \(x,\) giving \[\boxed{\cos(-x) + C}\]
EXERCISE 4
For \[\ds \int e^{-4x} \di x\] evaluate the indefinite integral.
SOLUTION The inner function of \(e^{-4x}\) is \(-4x,\) whose derivative is \(-4.\) It is okay if the derivative is a constant; our sole goal is to transform all expressions in the integral in terms of \(u.\) The integral becomes \[ \ba \int \par{-\tfrac{1}{4}} \underbrace{e^{-4x}}_{e^u} \underbrace{(-4) \di x}_{\dd u} &= -\tfrac{1}{4} \int e^u \di u \nl &= -\tfrac{1}{4} e^u + C \pd \ea \] The constant \(C\) is arbitrary. Converting back to \(x,\) we attain \[\boxed{-\tfrac{1}{4} e^{-4x} + C}\]
EXERCISE 5
For \[\ds \int 2x e^{-x^2} \di x\] evaluate the indefinite integral.
SOLUTION The derivative of \(x^2\) is \(2x,\) which is in the integrand. It is most useful to substitute \(u = -x^2,\) since doing so transforms \(e^{-x^2}\) to \(e^u,\) whose antiderivative we immediately recognize. Then \(\dd u = -2x \di x\) and so the integral becomes \[\int -e^u \di u = -\int e^u \di u = -e^u + C\] for some constant \(C.\) Expressing our answer back in terms of \(x,\) we attain \[\boxed{-e^{-x^2} + C}\]
EXERCISE 6
For \[\ds \int \frac{1}{x - 2} \di x\] evaluate the indefinite integral.
SOLUTION Substituting \(u = x - 2\) gives \(\dd u = \dd x.\) The integral therefore becomes \[\int \frac{1}{u} \di u = \ln \abs u + C\] for any constant \(C.\) We always convert back to our original variable \((x),\) so we write the final answer as \[\boxed{\ln \abs{x - 2} + C}\]
EXERCISE 7
For \[\ds \int \sin^2 \theta \cos \theta \di \theta\] evaluate the indefinite integral.
SOLUTION Think of the integrand as \((\sin \theta)^2 \cos \theta.\) The inner function of the composition \((\sin \theta)^2\) is \(\sin \theta,\) whose derivative is \(\cos \theta.\) This term resides in the integral, so we substitute \(u = \sin \theta\) to "clear out" the factor \(\dd u = \cos \theta \di \theta.\) Doing so gives \[\int u^2 \di u = \tfrac{1}{3} u^3 + C\] for some constant \(C.\) Now we substitute back to \(\theta\) to attain our final answer; we substituted \(u = \sin \theta,\) so we have \[\boxed{\tfrac{1}{3} \sin^3 \theta + C}\]
EXERCISE 8
For \[\ds \int \frac{1}{t \ln t} \di t\] evaluate the indefinite integral.
SOLUTION We rewrite the integral as \[\int \frac{1}{t} \cdot \frac{1}{\ln t} \di t \pd\] It then becomes apparent that \(1/t,\) the derivative of \(\ln t,\) is in the integrand. So we let \(u = \ln t\) because then \(\dd u = (1/t) \di t,\) and the integral becomes \[\int \frac{1}{u} \di u = \ln \abs u + C\] for some arbitrary constant \(C.\) Now we convert our answer to the original variable \(t.\) We chose \(u = \ln t,\) so our final answer is \[\boxed{\ln(\ln t) + C}\]
EXERCISE 9
For \[\ds \int \frac{\cos(\ln y)}{y} \di y\] evaluate the indefinite integral.
SOLUTION The derivative of \(\ln y\) is \(1/y,\) which resides in the integral. Therefore, to "clear out" this factor, we substitute \(u = \ln y.\) Then \(\dd u = (1/y) \di y\) and so our integral becomes \[\int \cos u \di u = \sin u + C\] for an arbitrary number \(C.\) Substituting back (we chose \(u = \ln y\)) gives \[\boxed{\sin(\ln y) + C}\]
EXERCISE 10
For \[\ds \int \frac{e^h}{1 + e^h} \di h\] evaluate the indefinite integral.
SOLUTION Let us write the integrand as \[\int e^h \cdot \frac{1}{1 + e^h} \di h \pd\] We substitute \(u = 1 + e^h\) because then \(\dd u = e^h \di h,\) so our integral becomes \[\int \frac{1}{u} \di u = \ln \abs u + C\] for any number \(C.\) Substituting back gives \[\boxed{\ln(1 + e^h) + C}\] (We don't need absolute value bars, because \(1 + e^h\) is always positive.)
EXERCISE 11
For \[\ds \int \frac{x + 3}{x + 2} \di x\] evaluate the indefinite integral.
SOLUTION In general, to find an antiderivative of a rational expression, we want to use polynomial division to simplify the expression. Alternatively, we could rewrite the fraction as \[\frac{(x + 2) + 1}{x + 2} = 1 + \frac{1}{x + 2} \pd\] Now the integral becomes \[\int \par{1 + \frac{1}{x + 2}} \di x \pd\] By the Sum Property of Antiderivatives (see Section 4.1), the integral is \[\int 1 \di x + \int \frac{1}{x + 2} \di x \cma\] or \begin{equation} x + \int \frac{1}{x + 2} \di x \pd \label{eq:x+3/x+2} \end{equation} To evaluate \(\int \di x/(x + 2),\) we perform a substitution with \(u = x + 2.\) Then \(\dd u = \di x\) and so the integral becomes \[\int \frac{1}{u} \di u = \ln \abs u + C = \ln \abs{x + 2} + C \pd\] So \(\eqref{eq:x+3/x+2}\) becomes our final answer: \[\boxed{x + \ln \abs{x + 2} + C}\]
EXERCISE 12
For \[\ds \int \frac{x - 6}{x - 4} \di x\] evaluate the indefinite integral.
SOLUTION To antidifferentiate a rational expression, we generally want to simplify the expression. We could do so using polynomial division, but it's easiest to rewrite the fraction as \[\frac{(x - 4) - 2}{x - 4} = 1 - \frac{2}{x - 4} \pd\] Now the integral becomes \[\int \par{1 - \frac{2}{x - 4}} \di x \pd\] By the Difference Property of Antiderivatives (see Section 4.1), the integral becomes \[\int 1 \di x - \int \frac{2}{x - 4} \di x \cma\] or \begin{equation} (x) - \par{2 \int \frac{1}{x - 4} \di x} \pd \label{eq:x-5/x-4} \end{equation} To evaluate \(2\int \di x/(x - 4),\) we perform a substitution with \(u = x - 4.\) Then \(\dd u = \di x;\) the integral becomes \[2\int \frac{1}{u} \di u = 2\ln \abs u + C = 2\ln \abs{x - 4} + C \pd\] So \(\eqref{eq:x-5/x-4}\) becomes our final answer: \[\boxed{x - 2\ln \abs{x - 4} + C}\]
EXERCISE 13
For \[\ds \int \frac{e^{\sqrt x}}{\sqrt x} \di x\] evaluate the indefinite integral.
SOLUTION We think of the integral as \[\int \frac{1}{\sqrt x} \cdot e^{\sqrt x} \di x \pd\] The derivative of \(\sqrt x\) is \(1/(2 \sqrt x);\) thus, we "clear out" the factor \(1/\sqrt x\) by substituting \(u = \sqrt x.\) Then \(\dd u = 1/(2 \sqrt x) \di x,\) and we attain \[2 \int e^u \di u = 2e^u + C \cma\] where \(C\) is an arbitrary constant. Substituting back gives \[\boxed{2e^{\sqrt x} + C}\]
EXERCISE 14
For \[\ds \int \frac{x}{3 + 2x^2} \di x\] evaluate the indefinite integral.
SOLUTION We write the integral as \[\int \frac{1}{3 + 2x^2} \cdot x \di x \pd\] The derivative of \(3 + 2x^2\) is \(4x.\) You should recognize that although \(4\) doesn't appear in the integrand, \(x\) does appear. We therefore "strip" the \(x\) by substituting \(u = 3 + 2x^2,\) since \(\dd u = 4x \di x\) or \(\dd x = (1/4x) \di u.\) We attain \[\int \frac{1}{u} \cdot \tfrac{1}{4} \di u = \tfrac{1}{4} \ln \abs u + C \cma\] where \(C\) is an arbitrary constant. Substituting back gives \[\boxed{\tfrac{1}{4} \ln(3 + 2x^2) + C}\] (We don't need absolute value bars around the natural logarithm, because its argument is always positive.)
EXERCISE 15
For \[\ds \int \frac{1}{(x \ln x) \ln(\ln x)} \di x\] evaluate the indefinite integral.
SOLUTION Letting \(u = \ln(\ln x),\) we find \(\dd u = \dd x/(x \ln x)\) and so the integral becomes \[\int \frac{1}{u} \di u = \ln \abs u + C = \boxed{\ln \abs{\ln(\ln x)} + C}\] where \(C\) is an arbitrary constant.
EXERCISE 16
For \[\ds \int y \sqrt{y - 1} \di y\] evaluate the indefinite integral.
SOLUTION The term \(\sqrt{y - 1}\) is problematic to us; we want to use a substitution to convert it to \(\sqrt u.\) Hence, we let \(u = y - 1.\) Then \(\dd u = \dd y\) and \(y = u + 1.\) In the integrand, we replace the term \(y\) with \(u + 1.\) Doing so gives \[\int (u + 1) \sqrt u \di u = \int \par{u^{3/2} + u^{1/2}} \di u \pd\] Antidifferentiating, we attain \[\tfrac{2}{5} u^{5/2} + \tfrac{2}{3} u^{3/2} + C = \boxed{\tfrac{2}{5} (y - 1)^{5/2} + \tfrac{2}{3} (y - 1)^{3/2} + C}\] if \(C\) is an arbitrary constant.
EXERCISE 17
For \[\ds \int \frac{x}{1 + x^4} \di x\] evaluate the indefinite integral.
SOLUTION Recall that \[\deriv{}{u} \parbr{\tan^{-1} u} = \frac{1}{1 + u^2} \pd\] This indefinite integral mimics this form; the only difference is the term \(x\) in the numerator. To rectify this factor, we substitute \(u = x^2\) because then \(\dd u = 2x \di x,\) or \(\dd x = (1/2x) \di u.\) The integral then becomes \[ \ba \int \frac{x}{1 + (u)^2} \cdot \frac{1}{2x} \di u &= \tfrac{1}{2} \int \frac{1}{1 + u^2} \di u \nl &= \tfrac{1}{2} \tan^{-1} u + C \ea \] for any value \(C.\) Our final step is to reexpress this answer in terms of \(x.\) We chose \(u = x^2,\) so our final answer is \[\boxed{\tfrac{1}{2} \tan^{-1} (x^2) + C}\]
EXERCISE 18
There are several ways to evaluate \(\int \sin x \cos x \di x.\) Evaluate the integral by
  1. substituting \(u = \sin x\)
  2. substituting \(u = \cos x\)
  3. using the identity \(\sin 2x = 2 \sin x \cos x\)
SOLUTION
  1. Substituting \(u = \sin x\) gives \(\dd u = \cos x \di x,\) so the integral becomes \[\int \underbrace{\sin x}_{u} \, \underbrace{\cos x \di x}_{\dd u } = \int u \di u = \tfrac{1}{2} u^2 + C\] for an arbitrary constant \(C.\) We now substitute back to obtain \[\boxed{\tfrac{1}{2} \sin^2 x + C}\]
  2. When we substitute \(u = \cos x\) we get \(\dd u = -\sin x \di x,\) or \(\dd x = (-1/\sin x) \di u.\) Hence, the integral becomes \[\int -u \di u = -\tfrac{1}{2} u^2 + C \cma\] where \(C\) is an arbitrary constant. Substituting back then gives \[\boxed{-\tfrac{1}{2} \cos^2 x + C}\]
  3. We rewrite the integral as \[\tfrac{1}{2} \int \sin 2x \di x \pd\] We substitute \(u = 2x;\) then \(\dd u = 2 \di x,\) or \(\dd x = (1/2) \di u.\) The integral then becomes \[\tfrac{1}{2} \int \tfrac{1}{2} \sin u \di u = -\tfrac{1}{4} \cos u + C\] for any number \(C.\) Substituting back yields \[\boxed{-\tfrac{1}{4} \cos 2x + C}\]
All answers are valid antiderivatives; their graphs therefore differ by only a constant.
EXERCISE 19
Let \(f\) be a function whose derivative \(f'\) is continuous. Suppose that \(f(3) = 2\) and \(f(5) = 6.\) Calculate \(\int_0^1 f'(2x + 3) \di x.\)
SOLUTION We convert the integral to a form whose antiderivative we readily identify. Let \(u = 2x + 3;\) then \(\dd u = 2 \di x.\) It is easiest to convert the bounds now: When \(x = 0,\) \(u = 3.\) When \(x = 1,\) \(u = 5.\) Hence, our integral becomes \[ \ba \int_3^5 f'(u) \di u &= f(u) \intEval_3^5 \nl &= f(5) - f(3) \nl &= 6 - 2 = \boxed 4 \ea \] Note that after we converted the bounds, we performed all subsequent operations in terms of \(u.\)
EXERCISE 20
Assume \(g'\) is continuous. If \(g(-2) = 3\) and \(g(6) = -5,\) then find \(\int_{-1}^1 g'(4x + 2) \di x.\)
SOLUTION We convert the integral to a form whose antiderivative we readily identify. Let \(u = 4x + 2;\) then \(\dd u = 4 \di x.\) It is easiest to convert the bounds now: When \(x = -1,\) \(u = -2.\) When \(x = 1,\) \(u = 6.\) Hence, our integral becomes \[ \ba \int_{-2}^6 g'(u) \di u &= g(u) \intEval_{-2}^6 \nl &= g(6) - g(-2) \nl &= -5 - 3 = \boxed{-8} \ea \] After we converted the bounds, note that we performed all subsequent operations in terms of \(u.\)
EXERCISE 21
Show that \(\int \csc x \di x\) \(= -\ln \abs{\csc x + \cot x} + C\) by reexpressing the integral as \[\int \frac{\csc x (\csc x + \cot x)}{\csc x + \cot x} \di x\] and substituting \(u = \csc x + \cot x.\)
SOLUTION In this new form, we have \[\int \frac{\csc^2 x + \csc x \cot x}{\csc x + \cot x} \di x \pd\] Using the provided substitution, we find \(\dd u = (-\csc x \cot x + \csc^2 x) \di x.\) The numerator of the integral is therefore \(-\dd u,\) so our integral becomes \[-\int \frac{\dd u}{u} = -\ln \abs u + C = -\ln \abs{\csc x + \cot x} + C \cma\] as given.
