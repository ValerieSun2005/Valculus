10.4 Exercises — Absolute Convergence and the Ratio and Root Tests

Click on a problem number to see its solution.

\(1.\)
Define absolute convergence and conditional convergence.
\(2.\)
If \(\sum \abs{a_n}\) is convergent, then what is true about \(\sum a_n \ques\)
\(3.\)
How do you use the Ratio Test to test \(\sum a_n\) for convergence or divergence?
\(4.\)
Using the Root Test, how do you test \(\sum a_n\) for convergence or divergence?
\(5.\)
What does the Ratio Test conclude about \(\sum a_n\) in each of the following cases?
  1. \(\ds \lim_{n \to \infty} \abs{\frac{a_{n + 1}}{a_n}} = 0.2\)
  2. \(\ds \lim_{n \to \infty} \abs{\frac{a_{n + 1}}{a_n}} = 1.2\)
  3. \(\ds \lim_{n \to \infty} \abs{\frac{a_{n + 1}}{a_n}} = 1\)
\(6.\)
By the Root Test, what do we know about \(\sum a_n\) in each of the following cases?
  1. \(\ds \lim_{n \to \infty} \sqrt[n]{\abs{a_n}} = 0.5\)
  2. \(\ds \lim_{n \to \infty} \sqrt[n]{\abs{a_n}} = 3.5\)
  3. \(\ds \lim_{n \to \infty} \sqrt[n]{\abs{a_n}} = 1\)

For each of exercises 7–13, determine whether the series converges absolutely, converges conditionally, or diverges.

\(7.\) \(\ds \sum_{n = 1}^{\infty} \frac{(-1)^n}{n^3}\) \(8.\) \(\ds \sum_{n = 1}^{\infty} \frac{(-1)^n}{n^6}\) \(9.\) \(\ds \sum_{n = 3}^{\infty} \frac{(-1)^n}{\sqrt{n}}\)
\(10.\) \(\ds \sum_{n = 0}^{\infty} (-2)^n\) \(11.\) \(\ds \sum_{n = 1}^\infty \frac{1}{n^3}\) \(12.\) \(\ds \sum_{n = 1}^{\infty} \frac{(-1)^n}{n^{3/2}}\)
\(13.\) \(\ds \sum_{n = 1}^{\infty} \frac{\cos(\pi n)}{\sqrt[3]{n}}\)
\(14.\)
Show that \(\sum_{n = 1}^\infty (\cos n)/n^2\) converges.
\(15.\)
Show that \(\sum_{n = 1}^\infty (\sin n)/n^3\) converges.
\(16.\)

\(17.\)
Prove that \(\sum_{n = 1}^\infty (-1)^n/n^p\) converges conditionally if \(0 \lt p \leq 1\) and converges absolutely if \(p \gt 1.\)

\(28.\) \(\ds \sum_{n = 0}^\infty 7^n\) \(29.\) \(\ds \sum_{n = 1}^\infty \par{\frac{2n - 2}{3n + 1}}^n\) \(30.\) \(\ds \sum_{n = 2}^\infty \par{\frac{n + 4}{n + 8}}^n\)
\(31.\) \(\ds \sum_{n = 1}^\infty \frac{(-3)^n}{n}\) \(32.\) \(\ds \sum_{n = 1}^\infty \frac{(-1)^n}{n^n}\) \(33.\) \(\ds \sum_{n = 1}^\infty \frac{n^5}{4^n}\)
\(34.\) \(\ds \sum_{n = 1}^\infty (-1)^n \, \frac{3^n}{n^6}\)
\(35.\)
A series' terms are defined recursively by the equation \[a_{n + 1} = (-1)^n \, \frac{3n + 4}{e^n} \, a_n \pd\] Determine whether \(\sum a_n\) converges or diverges.
\(36.\)
An infinite geometric series \(\sum a_n\) satisfies the following recursive formula: \[a_{n + 1} = r \, a_n \cma\] where \(r\) is the common ratio. Find \(\lim_{n \to \infty} \abs{a_{n + 1}/a_n}.\) Using the Ratio Test, find the values of \(r\) for which \(\sum a_n\) converges absolutely. Is this result expected?
\(37.\)
Let \(\sum a_n\) be a series such that \(\lim_{n \to \infty} \abs{a_{n + 1}/a_n} \lt 1.\) Using the Ratio Test, prove that \(\sum (a_n/n)\) is convergent.
\(38.\)
Suppose that the Ratio Test asserts that \(\sum a_n\) is divergent. Prove that \(\sum na_n\) is also divergent.
\(39.\)
Prove that \(\sum \par{n^k/n!}\) converges for all \(k.\)
\(40.\)
For what values of \(k\) does \(\sum \par{k^n/n}\) converge?
\(41.\)
Justify why the sum of a polynomial is divergent. Then show that the Ratio Test fails to prove that the sum of a polynomial is divergent.
\(42.\)
Let \(P(n)\) be a polynomial. Prove that \(\sum P(n) \, e^{-n}\) is absolutely convergent.
