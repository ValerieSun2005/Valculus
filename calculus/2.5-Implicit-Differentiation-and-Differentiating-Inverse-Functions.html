2.5 — Implicit Differentiation and Differentiating Inverse Functions

We have differentiated functions in the form \(y = f(x)\). In this section, however, we discuss how to differentiate implicit equations.

An equation of the form \(y = f(x)\) is said to be explicit; examples of explicit equations are \(y = x - 4\) and \(y = x^2 - 4x.\) Conversely, an implicit equation relates \(x\) and \(y\) through an equation in which \(y\) does not stand alone. For example, \[x^2 + xy = 6 \and 9x - 4y = 0\] are examples of implicit equations.

Many curves are best expressed as explicit equations. For example, the equation of an entire unit circle is \begin{equation} x^2 + y^2 = 1 \pd \label{eq:unit-circle} \end{equation} To determine the slope of this circle at any point \((x, y),\) we need \(\textDeriv{y}{x}.\) We apply implicit differentiation: We treat \(y\) as \(y(x),\) a function of \(x,\) so \(\eqRefer{eq:unit-circle}\) should be thought of as \[x^2 + [y(x)]^2 = 1 \period\] Now we differentiate both sides with respect to \(x.\) Applying the Chain Rule to differentiate \(y(x),\) we get \[2x + 2y \, \deriv{y}{x} = 0 \period\] Now we solve for \(\textDeriv{y}{x} \col\) \[\deriv{y}{x} = -\frac{x}{y} \period\]

Figure 1 shows a diagram of a unit circle with a tangent line. At \((0, -1)\) and \((0, 1),\) the circle has horizontal tangents because \(\textderiv{y}{x} = 0.\) And at \((-1, 0)\) and \((1, 0),\) the circle has vertical tangents and so \(\textderiv{y}{x} = 0\) is undefined. Hence, \(\textDeriv{y}{x} = -x/y\) appears mathematically accurate.

In some cases, we can solve for \(y\) directly in an implicit equation. For example, in \(xy = 1\) it is easy to see \(y = 1/x,\) so \(\textderiv{y}{x}\) can be found without using implicit differentiation. But in most cases, solving for \(y\) explicitly isn't possible and so implicit differentiation is needed to acquire \(\textderiv{y}{x}.\) Let's establish a set of steps to perform implicit differentiation.

STEPS OF IMPLICIT DIFFERENTIATION
  1. Differentiate both sides of the implicit equation with respect to \(x.\)
  2. On the left side, collect all terms containing the factor \(\textderiv{y}{x}.\) Move all other terms to the right side of the equation.
  3. Factor \(\textderiv{y}{x}\) out of the left side of the equation.
  4. Solve for \(\textderiv{y}{x}.\)
EXAMPLE 1
Consider the implicit equation \(2x^3y -xy^2 = 16.\) Find \(\textDeriv{y}{x}.\)
This equation is implicit, and solving for \(y\) isn't feasible. Thus, we use the method of implicit differentiation.
  1. We first differentiate both sides of the equation with respect to \(x.\) We use the Chain Rule to differentiate \(y,\) which we treat as a function of \(x.\) \[ \ba \deriv{}{x} (2x^3y - xy^2) &= \deriv{}{x}(16) \nl \deriv{}{x} \par{2x^3 y} - \deriv{}{x} \par{xy^2} &= 0 \nl 2 \left(3x^2 y + x^3 \deriv{y}{x} \right) - \left(y^2 + 2xy \deriv{y}{x} \right)&= 0 \pd \ea \]
  2. The second step is to bring all terms containing \(\textderiv{y}{x}\) to the left side. All other terms should be moved to the right side. Doing so, we get \[2x^3 \deriv{y}{x} - 2xy \deriv{y}{x} = y^2 - 6x^2 y \pd\]
  3. Factoring out \(\textderiv{y}{x},\) we attain \[\deriv{y}{x} \par{2x^3 - 2xy} = y^2 - 6x^2y \pd\]
  4. To solve for \(\textderiv{y}{x},\) we divide both sides by \(2x^3 - 2xy.\) Doing so gives \[\deriv{y}{x} = \boxed{\frac{y^2 - 6x^2 y}{2x^3 - 2xy}}\]
EXAMPLE 2
For the curve given by \(x^2 y = 2 - xy^2,\) find the points at which the curve has horizontal tangents and the points at which the curve has vertical tangents.

Let's first find \(\textderiv{y}{x}.\)

  1. Differentiating both sides of the implicit equation with respect to \(x,\) we get \[ 2x y + x^2 \deriv{y}{x} = (0) - y^2 - 2xy \deriv{y}{x} \pd \]
  2. We move every term involving \(\textderiv{y}{x}\) to the left side of the equation; we place all the other terms on the right side. Doing so gives \[x^2 \deriv{y}{x} + 2xy \deriv{y}{x} = -y^2 - 2xy \pd\]
  3. On the left side, we factor out the factor \(\textderiv{y}{x} \col\) \[\deriv{y}{x} \par{x^2 + 2xy} = -y^2 - 2xy \pd\]
  4. Solving for \(\textderiv{y}{x},\) we see \begin{equation} \deriv{y}{x} = \frac{-y^2 - 2xy}{x^2 + 2xy} \pd \label{eq:dy/dx-tangents} \end{equation}

The equation \(-y^2 - 2xy\) \(= 0\) also has the solution \(y = 0.\) But \(y = 0\) doesn't satisfy \(x^2 y = 2 - xy^2,\) so we disregard this value and only use \(x = -\tfrac{1}{2} y.\)

Horizontal Tangents The curve has a horizontal tangent when \(\textderiv{y}{x} = 0.\) For an implicit curve, we must describe locations as points—that is, we need both \(x\) and \(y.\) In \(\eqref{eq:dy/dx-tangents}\) we observe that \(\textderiv{y}{x} = 0\) when the numerator is \(0\) and the denominator is nonzero. So solving shows \[-y^2 - 2xy = 0 \implies x = -\tfrac{1}{2} y \pd\] Plugging this expression into \(x^2 y = 2 - xy^2,\) we see \[ \ba \par{-\tfrac{1}{2} y}^2 y &= 2 - \par{-\tfrac{1}{2} y} y^2 \nl \tfrac{1}{4} y^3 &= 2 + \tfrac{1}{2} y^3 \nl y^3 &= -8 \nl \implies y &= -2 \pd \ea \] The \(x\)-coordinate corresponding to \(y = -2\) is \(x = -\tfrac{1}{2}(-2)\) \(= 1.\) Thus, the curve has a horizontal tangent at the point \(\boxed{(1, -2)}.\)

Vertical Tangents The curve has a vertical tangent when the denominator of \(\textderiv{y}{x}\) is \(0\) and the numerator is nonzero. In \(\eqref{eq:dy/dx-tangents}\) equating \(x^2 + 2xy\) to \(0\) shows \[x(x + 2y) = 0 \implies x = -2y \pd\] (We ignore the solution \(x = 0\) because it does not satisfy the implicit equation \(x^2 y = 2 - xy^2.\)) Substituting \(x = -2y\) into the implicit equation shows \[ \ba \par{-2y}^2 y &= 2 - (-2y) y^2 \nl 2y^3 &= 2 \nl \implies y &= 1 \pd \ea \] Thus, \(x = -2(1) = -2.\) So the curve has a vertical tangent at the point \(\boxed{(-2, 1)}.\) (See Figure 2.)

EXAMPLE 3
Find \(\textderivOrder{y}{x}{2}\) for \[2x^3 + 9y^2 - 12y = 5 \pd\]

Finding \(\textderiv{y}{x}\) We begin by finding \(\textderiv{y}{x},\) as follows: \begin{align} \deriv{}{x} \par{2x^3 + 9y^2 - 12y} &= \deriv{}{x}(5) \nonum \nl 6x^2 + 18 y \deriv{y}{x} - 12 \deriv{y}{x} &= 0 \nonum \nl \deriv{y}{x} \par{18y - 12} &= 6x^2 \nonum \nl \deriv{y}{x} \par{6y - 2} &= x^2 \nonum \nl \implies \deriv{y}{x} &= \frac{x^2}{3y - 2} \pd \label{eq:second-deriv-ex} \end{align}

Finding \(\textderivOrder{y}{x}{2}\) To attain the second derivative, we differentiate \(\textderiv{y}{x}\) with respect to \(x.\) In doing so, we use the Quotient Rule while applying the Chain Rule to differentiate \(y\) with respect to \(x.\)
