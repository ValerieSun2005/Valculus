How many real zeros does the function \(g(x) = 4x^2 + 3x + 12\) have?
\(5.\)
How many real zeros does the function \(h(x) = -2x^2 + 16x - 5\) have?
\(6.\)

\(7.\)
Let \(g(x) = x^2 + bx + 2\). For what values of \(b\) does \(g\) have no real zeros?
