1.3 — Evaluating Limits Algebraically

This section defines indeterminate forms and covers how to evaluate limits of the indeterminate form \(\indZero.\) We define The Indeterminate Form \(\indZero\) and evaluate limits of the indeterminate form \(\indZero\) by Factorization, Rationalization, and Using Trigonometric Identities.

The Indeterminate Form \(\indZero\)

We begin by defining an indeterminate form. An limit is of an indeterminate form if direct substitution doesn't yield an answer. This section focuses on the indeterminate form \(\indZero.\) Consider the family of limits \begin{equation} L = \lim_{x \to a} \frac{f(x)}{g(x)} \comma g(x) \ne 0 \period \label{eq:family-of-limits} \end{equation} If both \(f(x)\) and \(g(x)\) approach \(0\) as \(x\) tends to \(a,\) For example, the limit \(\lim_{x \to 0} x^2/x\) is of the indeterminate form \(\indZero\) because as \(x \to 0,\) \[x^2 \to 0 \and x \to 0 \period \] Performing a direct substitution—that is, plugging \(x = 0\) into \(x^2/x\)—yields \(\indZero,\) a meaningless expression, meaning we need to use further analysis. It's important to distinguish between a function that approaches \(0\) and a function that equals \(0.\) In \(\eqRefer{eq:family-of-limits},\) if \(L\) is of an indeterminate form \(\indZero\) or \(\indInfty,\) then both \(f(x)\) and \(g(x)\) are changing; neither function is static. For example, the limit \(\lim_{x \to 0} 0/x\) is not of the indeterminate form \(\indZero,\) since the numerator is fixed at \(0.\) And this limit equals \(0.\)

INDETERMINATE FORM \(\indZero\)
Consider the limit \[L = \lim_{x \to a} \frac{f(x)}{g(x)} \comma g(x) \ne 0 \pd\] If both \(f(x)\) and \(g(x)\) approach \(0\) as \(x\) approaches \(a,\) then \(L\) is in the indeterminate form \(\indZero.\)

Factorization

When we have the indeterminate form \(\indZero,\) our primary resource is to cancel common factors. As an example, consider the function \[f(x) = \frac{x^2 - 25}{x + 5} \period \] The limit \(\lim_{x \to -5} f(x)\) is of the indeterminate form \(\indZero.\) But the numerator of \(f(x)\) is a difference of squares: \[x^2 - 25 = (x + 5)(x - 5) \period\] Thus, \[ \ba \lim_{x \to -5} \frac{x^2 - 25}{x + 5} &= \lim_{x \to -5} \frac{(x + 5)(x - 5)}{x + 5} \nl &= \lim_{x \to -5} (x - 5) \nl &= \boxed{-10} \ea \]

The graph of \(y = f(x)\) is continuous for all \(x\) except at \(x = -5,\) the location of a removable discontinuity (Figure 1). When we cancel the common factor \((x + 5),\) we transform \(f(x)\) into an equivalent function whose limit as \(x \to 5\) may be found. This value is \(\lim_{x \to -5} f(x)\) \(= -10,\) the \(y\)-value of the removable discontinuity.

When factoring polynomials, we often have differences of squares, differences of cubes, and sums of cubes. Therefore, remember the following formulas: \begin{align} a^2 - b^2 &= (a + b)(a - b) \label{eq:diff-of-squares} \nl a^3 - b^3 &= (a - b)(a^2 + ab + b^2) \label{eq:diff-of-cubes} \nl a^3 + b^3 &= (a + b)(a^2 - ab + b^2) \period \label{eq:sum-of-cubes} \end{align}

INDETERMINATE FORM \(\indZero\)
Suppose that the limit of a rational expression is in the indeterminate form \(\indZero.\)
  1. Identify any common factors in the numerator and denominator. If any are visible by inspection, then cancel them.
  2. If no common factors are present, then factor the numerator and denominator.
EXAMPLE 1
For each of (i)–(iii), evaluate the limit.
  1. \(\ds \lim_{x \to -2} \frac{x^2 - 4}{x + 2} \)
  2. \(\ds \lim_{x \to 4} \frac{x^3 - 64}{x - 4} \)
  3. \(\ds \lim_{x \to -3} \frac{x^3 + 27}{x + 3} \)
  1. Trying direct substitution immediately gives the indeterminate form \(\indZero,\) so we try to find any common factors. The numerator is a difference of squares, \(x^2 - 2^2,\) so we use \(\eqRefer{eq:diff-of-squares}\) to get \[x^2 - 4 = (x + 2)(x - 2) \period \] Re-expressing the numerator and canceling the common factor \((x + 2)\) provide \[ \ba \lim_{x \to -2} \frac{(x + 2)(x - 2)}{x + 2} &= \lim_{x \to -2} (x - 2) \nl &= \boxed{-4} \ea \]
  2. An immediate direct substitution yields the form \(\indZero,\) so we try to expose common factors. Since the numerator is a difference of cubes, \(x^3 - 4^3,\) we use \(\eqRefer{eq:diff-of-cubes}\) to re-express the numerator as \[ (x - 4)(x^2 + 4x + 16) \period \] Then we get \[ \ba \lim_{x \to 4} \frac{x^3 - 64}{x - 4} &= \lim_{x \to 4} \frac{(x - 4)(x^2 + 4x + 16)}{x - 4} \nl &= \lim_{x \to 4} (x^2 + 4x + 16) \nl &= \boxed{48} \ea \]
  3. Since the limit is in the indeterminate form \(\indZero,\) our goal is to expose any common factors. Using \(\eqRefer{eq:sum-of-cubes},\) we rewrite the numerator as \[(x + 3)(x^2 - 3x + 9) \period \] The limit then becomes \[ \ba \lim_{x \to -3} \frac{(x + 3)(x^2 - 3x + 9 )}{x + 3} &= \lim_{x \to -3} (x^2 - 3x + 9) \nl &= \boxed{27} \ea \]

Rationalization

A limit in an indeterminate form may have no exposed common factors. But if the expression has a radical, then rationalization may expose a common factor. As an example, consider \[g(x) = \frac{\sqrt{2x + 5} - 1}{x + 2} \period \] The limit \(\lim_{x \to -2} g(x)\) is in the indeterminate form \(\indZero.\) The function \(g(x)\) has a radical in the numerator, so we try to rationalize it: We multiply the numerator and denominator each by the conjugate of \((\sqrt{2x + 5} - 1),\) which is \((\sqrt{2x + 5} + 1).\) Doing so yields \[ \ba g(x) &= \frac{\sqrt{2x + 5} - 1}{x + 2} \cdot \frac{\sqrt{2x + 5} + 1}{\sqrt{2x + 5} + 1} \nl &= \frac{(2x + 5) - 1}{(x + 2) \left(\sqrt{2x + 5} + 1\right)} \nl &= \frac{2(x + 2)}{(x + 2) \left(\sqrt{2x + 5} + 1\right)} \nl &= \frac{2}{\sqrt{2x + 5} + 1} \comma x \ne -2 \period \ea \] We cleared the common factor \((x + 2)\) in the numerator and denominator, so \[ \ba \lim_{x \to -2} g(x) &= \lim_{x \to -2} \frac{2}{\sqrt{2x + 5} + 1} \nl &= \boxed{1} \ea \]

EXAMPLE 2
For each of (i) and (ii), evaluate the limit.
  1. \(\ds \lim_{x \to 9} \frac{3x - 27}{\sqrt{x - 8} + 1} \)
  2. \(\ds \lim_{x \to 9/4} \frac{\sqrt{4x - 5} - 2}{16x - 36} \)
  1. The limit is in the indeterminate form \(\indZero.\) The denominator has a radical, so we try to rationalize the denominator. The conjugate of the denominator is \((\sqrt{x - 8} - 1),\) and we multiply the numerator and denominator each by this factor to get \[ \ba \lim_{x \to 9} \frac{3x - 27}{\sqrt{x - 8} + 1} &= \lim_{x \to 9} \frac{3x - 27}{\sqrt{x - 8} + 1} \cdot \frac{\sqrt{x - 8} - 1}{\sqrt{x - 8} - 1} \nl &= \lim_{x \to 9} \frac{(3x - 27)(\sqrt{x - 8} - 1)}{(x - 8) - 1} \nl &= \lim_{x \to 9} \frac{3(x - 9)(\sqrt{x - 8} - 1)}{x - 9} \nl &= \lim_{x \to 9} 3 \left(\sqrt{x - 8} - 1\right) \nl &= \boxed{6} \ea \]
  2. This limit is in the indeterminate form \(\indZero.\) The numerator has a radical, so we attempt to rationalize the numerator. We do so multiplying the numerator and denominator by \((\sqrt{4x - 5} + 2),\) obtaining \[ \ba \lim_{x \to 9/4} \frac{\sqrt{4x - 5} - 2}{16x - 36} \cdot \frac{\sqrt{4x - 5} + 2}{\sqrt{4x - 5} + 2} &= \lim_{x \to 9/4} \frac{(4x - 5) - 4}{(16x - 36)(\sqrt{4x - 5} + 2) } \nl &= \lim_{x \to 9/4} \frac{4x - 9}{4(4x - 9)(\sqrt{4x - 5} + 2)} \nl &= \lim_{x \to 9/4} \frac{1}{4(\sqrt{4x - 5} + 2)} \nl &= \boxed{\frac{1}{16}} \ea \]

Using Trigonometric Identities

We now discuss limits of indeterminate forms that involve trigonometric expressions. We simplify limits using the following trigonometric identities: \begin{align} 1 &= \sin^2 x + \cos^2 x \label{eq:pythag-identity} \nl \sin 2x &= 2 \sin x \cos x \label{eq:sin-double-angle} \nl \cos 2x &= \cos^2 x - \sin^2 x \label{eq:cos-double-angle} \period \end{align}

EXAMPLE 3
For each of (i)–(iii), evaluate the limit.
  1. \(\ds \lim_{x \to \pi/2} \frac{\sin 2x }{\cos x} \)
  2. \(\ds \lim_{x \to 0} \frac{1 - \cos x}{3 \sin x} \)
  3. \(\ds \lim_{x \to \pi/4} \frac{\cos 2x}{\cos x - \sin x} \)
  1. Using \(\eqRefer{eq:sin-double-angle},\) we rewrite \(\sin 2x\) as \(2 \sin x \cos x.\) Then \[ \ba \lim_{x \to \pi/2} \frac{\sin 2x}{\cos x} &= \lim_{x \to \pi/2} \frac{2 \sin x \cos x}{\cos x} \nl &= \lim_{x \to \pi/2} 2 \sin x \nl &= 2 \sin \par{\frac{\pi}{2}} \nl &= \boxed{2} \ea \]
  2. We multiply the numerator and denominator each by \((1 + \cos x),\) from which we find \[ \ba \lim_{x \to 0} \frac{1 - \cos x}{3 \sin x} &= \lim_{x \to 0} \frac{1 - \cos x}{3 \sin x} \cdot \frac{1 + \cos x}{1 + \cos x} \nl &= \lim_{x \to 0} \frac{1 - \cos^2 x}{3 \sin x(1 + \cos x)} \period \ea \] Then using \(\eqRefer{eq:pythag-identity},\) we have \[\sin^2 x + \cos^2 x = 1 \implies \sin^2 x = 1 - \cos^2 x \period\] Replacing \(1 - \cos^2 x\) with \(\sin^2 x\) in the numerator yields \[ \ba \lim_{x \to 0} \frac{\sin^2 x}{3 \sin x(1 + \cos x)} &= \lim_{x \to 0} \frac{\sin x}{3 (1 + \cos x)} \nl &= \frac{\sin 0}{3\left(1 + \cos 0 \right)} \nl &= \boxed{0} \ea \]
  3. From \(\eqRefer{eq:cos-double-angle},\) we have \(\cos 2x = \cos^2 x - \sin^2 x.\) Since this expression is a difference of squares, we further rewrite it as \[(\cos x + \sin x)(\cos x - \sin x) \period\] Substituting in this expression, we find \[ \ba \lim_{x \to \pi/4} \frac{\cos 2x}{\cos x - \sin x} &= \lim_{x \to \pi/4} \frac{\left(\cos x + \sin x\right) \left(\cos x - \sin x\right)}{\cos x - \sin x} \nl &= \lim_{x \to \pi/4} \left(\cos x + \sin x \right) \nl &= \cos \par{\frac{\pi}{4}} + \sin \par{\frac{\pi}{4}} \nl &= \boxed{\sqrt{2}} \ea \]

Factoring, rationalization, and using trigonometric identities are all strategies we use to manipulate many limits of the indeterminate form \(\indZero.\) It's important to remain vigilant for any common factors.

The Indeterminate Form \(\indZero\) Factorization, Rationalization, and using Trigonometric Identities

The Indeterminate Form \(\indZero\) Consider the limit \[L = \lim_{x \to a} \frac{f(x)}{g(x)} \comma g(x) \ne 0 \pd\] If both \(f(x)\) and \(g(x)\) approach \(0\) as \(x\) approaches \(a,\) then \(L\) is in the indeterminate form \(\indZero.\)

Factorization The fundamental method we use to resolve limits in the indeterminate form \(\indZero\) is to cancel a common factor in the numerator and denominator. For rational functions, it is convenient to memorize the following formulas: \begin{align*} a^2 - b^2 &= (a + b)(a - b) \tag*{\eqref{eq:diff-of-squares}} \nl a^3 - b^3 &= (a - b)(a^2 + ab + b^2) \tag*{\eqref{eq:diff-of-cubes}} \nl a^3 + b^3 &= (a + b)(a^2 - ab + b^2) \pd \tag*{\eqref{eq:sum-of-cubes}} \end{align*}

Rationalization If a limit of the indeterminate form \(\indZero\) contains a radical, then try multiplying the numerator and denominator by the conjugate. Doing so may expose a common factor to cancel.
