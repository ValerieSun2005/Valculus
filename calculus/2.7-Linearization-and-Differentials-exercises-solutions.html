2.7 Exercise Solutions: Linearization and Differentials

EXERCISE 1
If \(y = f(x),\) then define and interpret the differential \(\dd y.\)
SOLUTION The differential \(\dd y\) is given by \[\dd y = f'(x) \di x \pd\] In this form, \(\dd x\) is an independent variable whose value we choose. Then \(\dd y\) is a dependent variable measuring the amount by which the tangent line rises or falls as \(x\) is changed by some amount \(\dd x.\) This rise or fall in the tangent line matches the rise or fall of the curve \(f\)—that is, the value \(\Delta y\)—for small values of \(\dd x.\) Hence, for infinitely small values of \(\dd x,\) the tangent line behaves identically to the curve \(f.\)
EXERCISE 2
What is linearization?
SOLUTION Linearization is the process by which we approximate the values of a function \(f\) by using a tangent line to \(f\) at some center \(x = a.\) An equation of the line tangent to \(f\) at \(x = a\) is given by \[y - f(a) = f'(a) (x - a) \pd\] The output of this line at \(x\) is an estimate of \(f(x)\)—that is, \[f(x) \approx f(a) + f'(a) (x - a) \pd\] In words, we find an equation of the tangent line to \(f\) at \(a\) and then plug in some \(x\) to obtain an estimate for \(f(x).\) We could also use differentials as a linear approximation if we take \(\dd x = x - a\) and \(\dd y \approx f(x) - f(a).\)

EXERCISE 8
Walkthrough of Linearization Let's use linearization to estimate the value of \(\sqrt{5}.\) The number \(5\) is closest to the perfect square \(4,\) so we center our linearization at \(x = 4.\)
  1. We start by constructing a tangent line to the curve. Write an equation for the line tangent to \(y = \sqrt x\) at the center of the linearization, \(x = 4.\)
  2. Determine the \(y\)-value that your tangent line reaches when \(x = 5.\) What does this value represent?
  3. Linearization is effective for \(x\) near the center \(a = 4\) but loses accuracy for \(x\) far away from \(a.\) We know that \(\sqrt{100} = 10,\) but what value does your linearization provide for \(\sqrt{100} \ques\)
SOLUTION
  1. To construct a tangent line to \(y = \sqrt x,\) we use point-slope form, for which we need a point on the line and its slope. (See Section 2.2 to review how to write an equation for a tangent line.) The line intersects the curve at the point \((4, 2)\) with slope \[\deriv{}{x} \par{\sqrt x \,} \intEval_{x = 4} = \frac{1}{2 \sqrt 4} = \frac{1}{4} \pd\] Hence, an equation for this tangent line is \[y - 2 = \tfrac{1}{4} (x - 4) \or y = 2 + \tfrac{1}{4} (x - 4) \pd\]
  2. We substitute \(x = 5\) into the equation of the tangent line, finding \[y = 2 + \tfrac{1}{4}(5 - 4) = \boxed{\tfrac{9}{4}} = 2.25 \pd\] This value is our approximation of \(\sqrt 5.\) We chose to center our linearization at the perfect square \(x = 4\) because we approximated \(\sqrt 5\) by adding simple, rational numbers.
  3. Plugging \(x = 100\) into our equation of the tangent line gives \[y = 2 + \tfrac{1}{4} (100 - 4) = \boxed{26}\] So this linearization estimates the value of \(\sqrt{100}\) to be \(26,\) which is very far away from the actual value of \(10.\) This disparity shows that linearization often leads to inaccuracy for inputs far away from its center.

EXERCISE 19
For
\[\ds \sqrt{23}\]
use a tangent line approximation (or differentials) to estimate the value.
SOLUTION Using either linearization or differentials yields the same result. Select the method with which you feel most comfortable.
Both methods—using a tangent line and using differentials—yield the same estimate \(\sqrt{23} \approx 4.8.\) The actual value of \(\sqrt{23}\) is roughly \(4.796\)—very close!

Using Linearization We have the freedom to decide where to center our linearization. (Different centers yield different estimates.) Notice that \(23\) is close to \(25,\) a perfect square. Thus, we use a linearization centered at \(a = 25;\) we construct a tangent line to \(y = \sqrt x\) at \(x = 25\) and evaluate it at \(x = 23.\) The slope of this tangent line is \[\deriv{y}{x} \intEval_{x = 25} = \frac{1}{2 \sqrt{25}} = \frac{1}{10} \pd\] Observe that \(\sqrt{25} = 5,\) so an equation of this tangent line is \[y - 5 = \tfrac{1}{10} (x - 25) \or y = 5 + \tfrac{1}{10} (x - 25) \pd\] Substituting \(x = 23,\) we see \[\sqrt{23} \approx 5 + \tfrac{1}{10} (23 - 25) = \boxed{4.8}\] We chose the perfect square \(a = 25\) because the value of \(\sqrt{25}\) is easily obtained. Accordingly, our estimate of \(\sqrt{23}\) only required adding rational numbers.

Using Differentials We use the differential \(\dd y\) to estimate \(\Delta y,\) the amount by which \(\sqrt x\) changes as \(x\) decreases from \(25\) to \(23.\) Since \(y = \sqrt x,\) the differential \(\dd y\) is given by \[\dd y = \frac{1}{2 \sqrt x} \di x \pd\] The value \(23\) is \(2\) less than the center of the approximation, \(a = 25,\) so we use \(\dd x = -2.\) Then \[\Delta y \approx \dd y = \frac{1}{2 \sqrt{25}} \cdot (-2) = -\frac{1}{5} \pd\] Because \(\Delta y = \sqrt{23} - \sqrt{25},\) we have \[\sqrt{23} = \sqrt{25} + \Delta y \approx 5 - \tfrac{1}{5} = \boxed{4.8}\]

EXERCISE 27
Let \(f(x) = \sqrt{2x + 3}.\)
  1. Show that a linearization for \(f\) centered at \(x = 3\) is \(f(x) \approx 2 + \tfrac{x}{3}.\)
  2. Determine the interval of \(x\) for which the linearization differs from \(f\) by no more than \(0.3.\)
SOLUTION
  1. The slope to \(f\) at \(x = 3\) is given by \[f'(3) = \frac{1}{\sqrt{2x + 3}} \intEval_{x = 3} = \frac{1}{3} \pd\] Because \(f(3) = 3,\) an equation of the line tangent to \(f\) at \(x = 3\) is \[y - 3 = \tfrac{1}{3}(x - 3) \or y = 2 + \tfrac{x}{3} \pd\] The outputs of this tangent line approximate \(f,\) so the linearization centered at \(x = 3\) is \[f(x) \approx 2 + \tfrac{x}{3} \cma\] as requested.
  2. Because the linearization must be a distance of no more than \(0.3\) from \(f,\) we establish the inequality \[\abs{f(x) - \par{2 + \tfrac{x}{3}}} \leq 0.3 \or \abs{\sqrt{2x + 3} - \par{2 + \tfrac{x}{3}}} \leq 0.3 \pd\] Using a graphing calculator, we see that this interval is satisfied for \[\boxed{-0.125 \leq x \leq 7.925}\]

EXERCISE 29
The sides of a cube are each \(10\) centimeters long. Using differentials, calculate the amount by which the cube's volume increases when each side is increased by \(0.01\) centimeter.
SOLUTION Let \(s\) be one of the cube's sides and \(V\) be its volume. The maximum measurement error of \(s\) is \(\Delta s = 0.01.\) We have \(V = s^3\) and \[\dd V = 3s^2 \di s \pd\] We use the approximation \(\Delta V \approx \dd V,\) for which we take \(\dd s = \Delta s\) \(= 0.01.\) So we see \[\Delta V \approx 3(10)^2 (0.01) = \boxed{3 \un{cm}^3}\]
EXERCISE 30
A square's sides are measured to be \(4\) inches each with a maximum error of \(0.05\) inch. Use differentials to approximate the maximum percent error in the square's area.
SOLUTION We will call \(s\) one of the square's sides and \(A\) its area. Then \(A = s^2\) and so \[\dd A = 2s \di s \pd\] We use the approximation \(\Delta A \approx \dd A,\) for which we let \(\dd s = 0.05\) be the maximum error in the length measurement. The relative error is the error in \(A\) divided by the measured value of \(A \col\) \[ \ba \frac{\Delta A}{A} &\approx \frac{\dd A}{A} \nl &= \frac{2s \di s}{s^2} \nl &= \frac{2 \di s}{s} \nl &= \frac{2(0.05)}{4} = 0.025 \pd \ea \] So the maximum percent error in \(A\) is \(0.025 \times 100\%\) \(= \boxed{2.5\%}.\)
EXERCISE 31
A circle's radius is measured to be \(5.6 \pm 0.3\) inches. Using differentials, approximate the maximum percent error in the circle's area.
SOLUTION The notation \(5.6 \pm 0.3\) means that the radius is measured to be \(5.6 \un{in}\) with an error of no more than \(0.3 \un{in}.\) Let \(r\) be the circle's radius and \(A\) be its area, so \(A = \pi r^2.\) We consider the maximum measurement error of \(r\) to be \(\Delta r = \dd r.\) The maximum error in \(A\) is \(\Delta A,\) for which we construct the approximation \[\dd A = 2 \pi r \di r \pd\] The percent error in \(A\) is given by the error in \(A\) divided by the measured value: \[ \frac{\Delta A}{A} \approx \frac{\dd A}{A} = \frac{2 \pi r \di r}{\pi r^2} = \frac{2 \di r}{r} \pd \] With \(r = 5.6\) and \(\dd r = 0.3,\) this expression becomes \[\frac{2(0.3)}{5.6} \times 100\% \approx \boxed{10.714 \%}\]
EXERCISE 32
The side \(s\) of a square is increased in length by \(1\%.\) Using a linear approximation, estimate the square's new area in terms of \(s.\)
SOLUTION Let \(A\) denote the square's area. Then \(A = s^2,\) from which we attain \[\dd A = 2s \di s \pd\] Since \(s\) is increased by \(1\%,\) \(\dd s = 0.01s\) and so we see \[\dd A = 2s(0.01s) = 0.02 s^2 \pd\] The new area of the square is therefore \[ \ba A &= s^2 + \Delta A \nl & \approx s^2 + \dd A \nl &= s^2 + 0.02s^2 = \boxed{1.02s^2} \ea \]
EXERCISE 33
Earth can be approximated by a sphere whose radius is roughly \(6360\) miles. If Earth's radius were shrunk by \(1\) mile, then approximately by what percent would its surface area decrease?
SOLUTION A sphere of radius \(r\) has surface area given by the formula \(A = 4 \pi r^2.\) The differential \(\dd A\) is given by \[\dd A = 8 \pi r \di r \pd\] If \(r\) decreases by \(1\) mile, then \(\dd r = -1.\) The percent change in surface area is given by \(\Delta A/A,\) which we estimate to be \[ \ba \frac{\dd A}{A} &= \frac{8 \pi r \di r}{4 \pi r^2} \nl &= \frac{2 \di r}{r} \nl &= \frac{2(-1)}{6360} \times 100\% \nl &\approx \boxed{0.003 \%} \ea \]
EXERCISE 34
Let \(g\) be a differentiable function that satisfies \(g(2) = 7\) and \(g'(x) = x^2 - 1.\) Approximate \(g(2.1).\)
SOLUTION We don't know the function \(g;\) we only have its derivative \(g'.\) Hence, we use a linear approximation to estimate \(g(2.1) \col\) The slope to the graph of \(g\) at \(x = 2\) is \[g'(2) = (2)^2 - 1 = 3 \pd\] Thus, an equation of the line tangent to the graph of \(g\) at \(x = 2\) is \[y - g(2) = g'(2) (x - 2) \or y = 7 + 3(x - 2) \pd\] Accordingly, we use the linearization \[g(x) \approx 7 + 3 (x - 2) \pd\] Plugging in \(x = 2.1\) gives \[ \ba g(2.1) &\approx 7 + 3(2.1 - 2) \nl &= \boxed{7.3} \ea \]
EXERCISE 35
For nonzero \(x,\) let \(Q(x) = x^k\) where \(k \gt 0.\) Show that the maximum relative error of \(Q\) is \(k\) times the maximum relative error of \(x.\)
SOLUTION The maximum relative error of \(x\) is \(\Delta x/x,\) and the maximum relative error of \(Q\) is \(\Delta Q/Q.\) We approximate the numerators in each expression as \(\dd x\) and \(\dd Q,\) respectively. Since \(Q'(x) = k x^{k - 1},\) we see \[\dd Q = k x^{k - 1} \di x \pd\] So the relative error of \(Q\) is \[\frac{\dd Q}{Q} = \frac{k x^{k - 1} \di x}{x^k} = \frac{k \di x}{x} \pd\] This expression represents \(k\) times the relative error of \(x.\)
EXERCISE 36
Let \(f(x) = x^2 + C\) and \(T(x) = 6x - 5.\) Find the value of \(C\) such that \(T\) is the linearization of \(f\) at \(x = 3.\)
SOLUTION The line \(T(x)\) is the linearization of \(f(x)\) at \(x = 3\) if and only if \(T\) is tangent to \(f\) at \(3\)—that is, \(T(3) = f(3)\) and \(T'(3) = f'(3).\) Observe that \[f(3) = 9 + C \and T(3) = 13 \pd\] So we see \[9 + C = 13 \implies C = 4 \pd\] Also, \(f'(3) = T'(3) = 6.\) Thus, the graphs of \(f\) and \(T\) are tangent at \(x = 3\) for \(\boxed{C = 4}.\) You can view an interactive plot of \(f\) and \(T\) at https://www.desmos.com/calculator/4lpyxm3m1v?lang=en.
EXERCISE 37
The line \(T(x) = -4x - 3\) is a linearization of the parabola \(f(x) = x^2 + 1.\) Find the \(x\)-value at which the linearization is centered.
SOLUTION A linearization \(T(x)\) is centered at \(x = a\) if and only if \(T\) is tangent to \(f\) at \(a.\) The word tangent means that \(T(a) = f(a)\) and \(T'(a) = f'(a).\) Thus, we equate \(T\) to \(f\) to find their intersection point: \[ \ba x^2 + 1 &= -4x - 3 \nl x^2 + 4x + 4 &= 0 \nl \implies x &= -2 \pd \ea \] We also see that \(f'(-2) = T'(-2)\) \(= -4,\) meaning the two graphs are tangent when \(x = -2.\) Hence, the linearization \(T(x) = -4x - 3\) for \(f\) is centered at \(\boxed{x = -2}.\)
