8.2 Exercise Solutions: Separation of Variables

EXERCISE 1
For
\[\ds \deriv{y}{x} = 3x^2 + 4\]
solve the differential equation.
SOLUTION Performing Separation of Variables, we place all the \(y\)-terms on the left and all the \(x\)-terms on the right. We multiply both sides by \(\dd x\) to get \[\dd y = (3x^2 + 4) \dd x \pd\] We now integrate both sides, as follows: \[ \ba \int \dd y &= \int (3x^2 + 4) \di x \nl y + C_1 &= x^3 + 4x + C_2 \pd \ea \] No reason exists to write two constants of integration, so we simply write one constant on the right: \[ \boxed{y = x^3 + 4x + C} \] where \(C = C_2 - C_1.\)
EXERCISE 2
For
\[\ds \deriv{y}{x} = 4x^3 + 9x^2 - 8x + 4\]
solve the differential equation.
SOLUTION Separating variables, we multiply both sides by \(\dd x\) and integrate to get \[ \ba \int \dd y &= \int (4x^3 + 9x^2 - 8x + 4) \di x \nl y + C_1 &= x^4 + 3x^2 - 4x^2 + 4x + C_2 \ea \] No reason exists to write two constants of integration, so for simplicity we let \(C = C_2 - C_1.\) The solution is therefore \[\boxed{y = x^4 + 3x^2 - 4x^2 + 4x + C}\]
EXERCISE 3
For
\[\ds \deriv{y}{x} = x - \sin x \]
solve the differential equation.
SOLUTION Multiplying both sides by \(\dd x,\) we obtain \[ \dd y = (x - \sin x) \di x \pd \] Now we integrate both sides to get \[ \boxed{y = \tfrac{1}{2}x^2 + \cos x + C} \]
EXERCISE 4
For
\[\ds t \, \deriv{y}{t} = 2\]
solve the differential equation.
SOLUTION On both sides we divide by \(t\) and multiply by \(\dd t\) to separate the variables: \[\int \dd y = \int \frac{2}{t} \di t \pd\] Our solution to the differential equation is therefore \[\boxed{y = 2 \ln \abs t + C}\]
EXERCISE 5
For
\[\ds y' = 2x \cmaa y(0) = 2\]
solve the differential equation.
SOLUTION Performing Separation of Variables, we write \(y'\) as \(\textDeriv{y}{x}\) and multiply both sides by \(\dd x \col\) \[\dd y = 2x \di x \pd\] Integrating both sides gives \[y = x^2 + C \pd\] (We combine the constant of integration on the left with the constant on the right.) A particular solution has a value for \(C\) that fits the initial condition \(y(0) = 2.\) We see \[2 = 0^2 + C \implies C = 2 \pd\] Thus, the particular solution is \[\boxed{y = x^2 + 2}\]
EXERCISE 6
For
\[\ds \deriv{y}{x} = 2y\]
solve the differential equation.
SOLUTION We place all the \(y\)-terms on one side and the \(x\)-terms on the other side. Thus, on both sides we multiply by \(\dd x\) and divide by \(y\) to get \[\frac{1}{y} \di y = 2 \di x \pd\] Integrating both sides gives \[\ln \abs y = 2x + C_1 \pd\] We exponentiate both sides using the base \(e\) to get \[\abs y = e^{2x + C_1} = e^{C_1} e^{2x} \pd\] Letting \(C = \pm e^{C_1},\) we write the solution as \[\boxed{y = Ce^{2x}}\]
EXERCISE 7
For
\[\ds \deriv{y}{t} \, \cos y = 4t + 3 \cmaa y(1) = \pi\]
solve the differential equation.
SOLUTION We multiply both sides by \(\dd t\) to separate the variables: \[ \ba \int \cos y \di y &= \int (4t + 3) \di t \nl \sin y &= 2t^2 + 3t + C \pd \ea \] We are interested in the particular solution that satisfies the initial condition \((1, \pi).\) Substituting this point and solving for \(C\) yield \[\sin \pi = 2(1)^2 + 3(1) + C \implies C = -5 \pd\] Thus, we have \[\boxed{\sin y = 2t^2 + 3t - 5}\] Note: In this implicit equation, \(y\) can be any real number. But if we express \(y\) using the inverse sine function, then we impose the restriction \(-\pi/2 \leq y \leq \pi/2,\) which causes a loss of generality. We therefore choose to present our solution as an implicit equation.
EXERCISE 8
For
\[t \, y' = 1 + y^2\]
solve the differential equation.
SOLUTION We first rewrite \(y'\) as \(\textDeriv{y}{t}.\) Then on both sides we divide by \(t \par{1 + y^2}\) to get \[\frac{1}{1 + y^2} \di y = \frac{1}{t} \di t \pd\] Integrating both sides gives \[\tan^{-1} y = \ln \abs{t} + C \or \boxed{y = \tan \par{\ln \abs t + C}}\] provided that \[- \frac{\pi}{2} \lt \ln \abs t + C \lt \frac{\pi}{2} \pd\]
EXERCISE 9
For
\[\ds y \, y' \sec x = 2\]
solve the differential equation.
SOLUTION We rewrite \(y'\) as \(\textDeriv{y}{x}.\) To separate the \(x\)- and \(y\)-terms, on each side we divide by \(y\) and multiply by \(\sec x \di x.\) Doing so gives \[y \di y = 2 \cos x \pd\] Integrating both sides gives \[\tfrac{1}{2} y^2 = 2 \sin x + C_1 \or \boxed{y = \pm \sqrt{4 \sin x + C}}\] where \(C = 2 C_1.\)
EXERCISE 10
For
\[\ds x^2 \, y' = y + 3\]
solve the differential equation.
SOLUTION Treating \(y'\) as \(\textDeriv{y}{x},\) we isolate the \(x\)- and \(y\)-terms to find \[ \ba \int \frac{1}{y + 3} \di y &= \int \frac{1}{x^2} \di x \nl \ln \abs{y + 3} &= -\frac{1}{x} + C_1 \pd \ea \] Let \(C = \pm e^{C_1}.\) To solve for \(y,\) we exponentiate both sides using \(e \col\) \[ \abs{y + 3} = e^{-1/x + C_1} \implies \boxed{y = Ce^{-1/x} - 3} \]
EXERCISE 11
For
\[\ds \deriv{y}{x} = \frac{x}{\sqrt{y + 2}}\]
solve the differential equation.
SOLUTION We multiply both sides by \(\sqrt{y + 2} \di x\) to get \[\sqrt{y + 2} \di y = x \di x \pd\] Integrating both sides, we attain \[\tfrac{2}{3} (y + 2)^{3/2} = \tfrac{1}{2} x^2 + C_1 \pd\] Letting \(C = \tfrac{3}{2} C_1\) and solving for \(y,\) we get \[\boxed{y = \par{\tfrac{3}{4} x^2 + C}^{2/3} - 2}\]
EXERCISE 12
For
\[\ds y' = e^{x + y}\]
solve the differential equation.
SOLUTION We use exponent properties to write \(y' = e^x e^y.\) Then we perform Separation of Variables as follows: \[ \ba \int e^{-y} \di y &= \int e^x \di x \nl -e^{-y} &= e^x + C_1 \pd \ea \] Letting \(C = -C_1,\) we solve for \(y\) as follows: \[e^{-y} = C - e^x \implies \boxed{y = - \ln \par{C - e^x}}\]
EXERCISE 13
For
\[\ds \sqrt{4xy - 9x} = \deriv{y}{x}\]
solve the differential equation.
SOLUTION Let's write the equation as \[ \ba \sqrt{4y - 9} \, \sqrt x &= \deriv{y}{x} \nl \frac{\dd y}{\sqrt{4y - 9}} &= \sqrt x \di x \pd \ea \] Integrating both sides therefore gives \[\tfrac{1}{2} \sqrt{4y - 9} = \tfrac{2}{3} x^{3/2} + C_1 \pd\] Letting \(C = 2 C_1,\) we have \[\sqrt{4y - 9} = \tfrac{4}{3} x^{3/2} + C \implies \boxed{y = \frac{9 + \par{\tfrac{4}{3} x^{3/2} + C}^2}{4}}\]
EXERCISE 14
For
\[\ds y \, \deriv{y}{t} = t \, \sqrt{1 - y^2}\]
solve the differential equation.
SOLUTION We divide both sides by \(\sqrt{1 - y^2}\) and multiply both sides by \(\dd t \col\) \[ \frac{y}{\sqrt{1 - y^2}} \di y = t \di t \pd \] Now we integrate both sides: To integrate the left side, we substitute \(u = 1 - y^2.\) Then \(\dd u = -2y \di y\) and so \[ \ba \int \frac{y}{\sqrt{1 - y^2}} \di y &= -\tfrac{1}{2} \int \frac{1}{\sqrt{u}} \di u \nl &= - \sqrt{u} = -\sqrt{1 - y^2} \pd \ea \] (We neglected the constant of integration.) So the solution to the differential equation is \[-\sqrt{1 - y^2} = \tfrac{1}{2} t^2 + C \pd\] Now we solve for \(y\) as follows: \[ 1 - y^2 = \par{\tfrac{1}{2} t^2 + C}^2 \implies \boxed{y = \pm\sqrt{1 - \par{\tfrac{1}{2} t^2 + C}^2}} \]
EXERCISE 15
For
\[\ds \par{x^4 + 3} \par{y^2 + 1} = x \deriv{y}{x}\]
solve the differential equation.
SOLUTION Moving all terms with \(y\) to the left and all terms with \(x\) to the right, we attain \[ \ba \frac{\dd y}{y^2 + 1} &= \frac{x^4 + 3}{x} \di x \nl \frac{\dd y}{y^2 + 1} &= \par{x^3 + \frac{3}{x}} \di x \pd \ea \] Now we have \[ \ba \int \frac{\dd y}{y^2 + 1} &= \int \par{x^3 + \frac{3}{x}} \di x \nl \atan y &= \tfrac{1}{4} x^4 + 3 \ln \abs x + C \pd \ea \] We could write our solution explicitly as \[ \boxed{y = \tan \par{\tfrac{1}{4} x^4 + 3 \ln \abs x + C}} \] provided that \[-\frac{\pi}{2} \lt \tfrac{1}{4} x^4 + 3 \ln \abs x + C \lt \frac{\pi}{2} \pd\]
EXERCISE 16
For
\[\ds 8 \ln x \di x + 3 x \di y = 0\]
solve the differential equation.
SOLUTION Let's separate the variables as follows: \[ \ba 3 x \di y &= - 8 \ln x \di x \nl \int \dd y &= \int -\frac{8 \ln x}{3x} \di x \pd \ea \] On the right, we let \(u = \ln x\) and \(\dd u = (1/x) \di x.\) Therefore, \[ \ba \int -\frac{8 \ln x}{3x} \di x &= -\tfrac{8}{3} \int u \di u \nl &= -\tfrac{8}{3} \par{\tfrac{1}{2} u^2} + C \nl &= -\tfrac{4}{3} (\ln x)^2 + C \pd \ea \] Our solution to the differential equation is therefore \[\boxed{y = C -\tfrac{4}{3} (\ln x)^2}\]
EXERCISE 17
For
\[\ds \cos 4y \deriv{y}{x} - x \ln x = 0\]
solve the differential equation.
SOLUTION Let's move all terms with \(y\) to the left side and all terms with \(x\) to the right side. Doing so gives \[ \int \cos 4y \di y = \int x \ln x \di x \pd \] We evaluate \(\int x \ln x \di x\) by parts with \[ \baat{2} u &= \ln x \lspace &\dd v &= x \di x \nl \dd u &= \frac{1}{x} \di x \lspace &v &= \frac{x^2}{2} \pd \eaat \] So we see \[ \ba \int x \ln x \di x &= (\ln x) \par{\frac{x^2}{2}} - \int \par{\frac{x^2}{2}} \par{\frac{1}{x}} \di x \nl &= \tfrac{1}{2} x^2 \ln x - \tfrac{1}{2} \int x \di x \nl &= \tfrac{1}{2} x^2 \ln x - \tfrac{1}{4} x^2 + C \pd \ea \] The solution to the differential equation is therefore \[\boxed{\tfrac{1}{4} \sin 4y = \tfrac{1}{2} x^2 \ln x - \tfrac{1}{4} x^2 + C}\] (Solving for \(y\) causes a loss of generality; keeping the solution in implicit form conserves the generality of the solutions.)
EXERCISE 18
For
\[\ds \deriv{y}{t} - \frac{y}{t^2 - 4} = 0\]
solve the differential equation.
SOLUTION We separate variables to get \[ \int \frac{\dd y}{y} = \int \frac{1}{t^2 - 4} \di t \pd \] The integral on the right can be evaluated by either the trigonometric substitution \(t = 2 \sec \theta\) or by using partial fraction decomposition. Let's use partial fraction decomposition: We factor \(t^2 - 4\) as \((t + 2)(t - 2)\) and so find \[\frac{1}{t^2 - 4} = \frac{A}{t + 2} - \frac{B}{t - 2} \pd\] We therefore have \[1 = A(t - 2) + B(t + 2) \pd\] Substituting \(t = 2\) shows \[1 = A(0) + 4B \implies B = \tfrac{1}{4} \pd\] Likewise, letting \(t = -2\) yields \[1 = -4A + B(0) \implies A = -\tfrac{1}{4} \pd\] So we get \[ \ba \int \frac{1}{t^2 - 4} \di t &= \int \par{\frac{-1/4}{t + 2} + \frac{1/4}{t - 2}} \di t \nl &= -\tfrac{1}{4} \ln \abs{t + 2} + \tfrac{1}{4} \ln \abs{t - 2} + C_1 \nl &= \tfrac{1}{4} \ln \abs{\frac{t - 2}{t + 2}} + C_1 \pd \ea \] The solution to the differential equation is therefore \[ \ln \abs y = \tfrac{1}{4} \ln \abs{\frac{t - 2}{t + 2}} + C_1 \pd \] Letting \(C = \pm e^{C_1},\) we have \[ y = C \abs{\frac{t - 2}{t + 2}}^{1/4} \or \boxed{y = C \sqrt[4]{\abs{\frac{t - 2}{t + 2}}}} \]
EXERCISE 19
For
\[\ds \sqrt{r - 3} = \par{16 x^2 + 1} \deriv{r}{x}\]
solve the differential equation.
SOLUTION We separate variables to get \[ \int \frac{\dd r}{\sqrt{r - 3}} = \int \frac{\dd x}{16x^2 + 1} \pd \] On the right, substituting \(u = 4x\) gives \(\dd u = 4 \di x\) and so \[ \ba \int \frac{\dd x}{16x^2 + 1} &= \tfrac{1}{4} \int \frac{1}{u^2 + 1} \di u \nl &= \tfrac{1}{4} \atan u + C_1 \nl &= \tfrac{1}{4} \atan 4x + C_1 \pd \ea \] So the solution to the differential equation is \[ 2 \sqrt{r - 3} = \tfrac{1}{4} \atan 4x + C_1 \or \boxed{r = 3 + \par{\tfrac{1}{8} \atan 4x + C}^2} \] where \(C = C_1/2.\)
EXERCISE 20
For
\[\ds x \sqrt[3]{5y - 6} \deriv{y}{x} = \ln x\]
solve the differential equation.
SOLUTION Separation of Variables gives \[ \int \sqrt[3]{5y - 6} \di y = \int \frac{\ln x}{x} \di x \pd \] The left integral can be evaluated by substituting \(u = 5y - 6;\) then \(\dd u = 5 \di y.\) So we see \[ \ba \int \sqrt[3]{5y - 6} \di y &= \tfrac{1}{5} \int \sqrt[3]{u} \di u \nl &= \tfrac{3}{20} u^{4/3} + C_1 \nl &= \tfrac{3}{20} (5y - 6)^{4/3} + C_1 \pd \ea \] To evaluate \(\int (\ln x)/x \di x,\) we substitute \(u = \ln x\) to get \(\dd u = (1/x) \di x.\) The integral therefore becomes \[\int u \di u = \tfrac{1}{2} u^2 + C_2 = \tfrac{1}{2} (\ln x)^2 + C_2 \pd\] Hence, the solution to the differential equation is \[\tfrac{3}{20} (5y - 6)^{4/3} + C_1 = \tfrac{1}{2} (\ln x)^2 + C_2 \pd\] Solving for \(y\) and rewriting the constants \(C_1\) and \(C_2\) as one constant \(C,\) we have \[ (5y - 6)^{4/3} = \tfrac{10}{3} (\ln x)^2 + C \implies \boxed{y = \tfrac{6}{5} + \tfrac{1}{5} \parbr{\tfrac{10}{3} (\ln x)^2 + C}^{3/4}} \]
EXERCISE 21
For
\[\ds \deriv{y}{t} = \frac{5 \sqrt{y + 8}}{t^2 - 4t + 3}\]
solve the differential equation.
SOLUTION Separation of Variables gives \[ \int \frac{\dd y}{\sqrt{y + 8}} = \int \frac{5}{t^2 - 4t + 3} \di t \pd \] Since \(t^2 - 4t + 3\) \(= (t - 3)(t - 1),\) we decompose the rational expression into partial fractions: \[\frac{5}{t^2 - 4t + 3} = \frac{A}{t - 3} + \frac{B}{t - 1} \pd\] To solve for \(A\) and \(B,\) we write \[5 = A(t - 1) + B(t - 3) \pd\] Substituting \(t = 3\) gives \[5 = A(3 - 1) + B(0) \implies A = \tfrac{5}{2} \pd\] Likewise, letting \(t = 1\) shows \[5 = A(1 - 1) + B(1 - 3) \implies B = -\tfrac{5}{2} \pd\] We therefore have \[ \ba \int \frac{5}{t^2 - 4t + 3} \di t &= \int \par{\frac{5/2}{t - 3} - \frac{5/2}{t - 1}} \di t \nl &= \tfrac{5}{2} \ln \abs{t - 3} - \tfrac{5}{2} \ln \abs{t - 1} + C_1 \nl &= \tfrac{5}{2} \ln \abs{\frac{t - 3}{t - 1}} + C_1 \pd \ea \] Also note that \[\int \frac{\dd y}{\sqrt{y + 8}} = 2 \sqrt{y + 8} + C_2 \pd\] The solution to the differential equation is therefore \[ 2 \sqrt{y + 8} + C_2 = \tfrac{5}{2} \ln \abs{\frac{t - 3}{t - 1}} + C_1 \pd \] Letting \(C = (C_1 - C_2)/2\) and solving for \(y,\) we get \[ \boxed{y = \par{\tfrac{5}{4} \ln \abs{\frac{t - 3}{t - 1}} + C}^2 - 8} \]
EXERCISE 22
For
\[\ds \deriv{\theta}{t} \par{\tan^2 \theta + 1} = t \sqrt{2t + 6} \cos \theta\]
solve the differential equation.
SOLUTION Observe that \(\tan^2 \theta + 1\) \(= \sec^2 \theta\) and \(\cos \theta = 1/\sec \theta.\) Performing Separation of Variables, we move all terms with \(\theta\) to the left and all terms with \(t\) to the right: \[\int \sec^3 \theta \di \theta = \int t \sqrt{2t + 6} \di t \pd\] To evaluate the right integral, we substitute \(u = 2t + 6.\) Then \(t = \tfrac{1}{2} u - 3\) and \(\dd u = 2 \di t,\) and we see \[ \ba \int t \sqrt{2t + 6} \di t &= \tfrac{1}{2} \int \par{\tfrac{1}{2} u - 3} \sqrt u \di u \nl &= \int \par{\tfrac{1}{4} u^{3/2} - \tfrac{3}{2} u^{1/2}} \di u \nl &= \tfrac{1}{10} u^{5/2} - u^{3/2} + C \nl &= \tfrac{1}{10} (2t + 6)^{5/2} - (2t + 6)^{3/2} + C \pd \ea \] In Example 6.2-12 we found \[\int \sec^3 \theta \di \theta = \tfrac{1}{2} \sec \theta \tan \theta + \tfrac{1}{2} \ln \abs{\sec \theta + \tan \theta} + C \pd\] Hence, our solution to the differential equation is \[ \boxed{\tfrac{1}{2} \sec \theta \tan \theta + \tfrac{1}{2} \ln \abs{\sec \theta + \tan \theta} = \tfrac{1}{10} (2t + 6)^{5/2} - (2t + 6)^{3/2} + C} \] (Solving for \(\theta\) is clearly impossible, so we settle for an implicit equation.)
EXERCISE 23
For
\[\ds y = kx^2\]
find the orthogonal trajectory to the family of functions, where \(k\) is a nonzero constant.
SOLUTION By differentiation, the slope to \(y = kx^2\) at any \((x, y)\) is \[\deriv{y}{x} = 2k x \pd\] Since the orthogonal trajectory intersects the family of parabolas at right angles, its slope is the negative reciprocal of \(2kx\)—that is, \[\deriv{y}{x} = -\frac{1}{2kx} \pd\] We use Separation of Variables to solve this differential equation, as follows: \[ \int \dd y = \int -\frac{1}{2kx} \di x \implies \boxed{y = -\frac{1}{2k} \ln \abs x + C} \]
EXERCISE 24
For
\[\ds y = k\ln x\]
find the orthogonal trajectory to the family of functions, where \(k\) is a nonzero constant.
SOLUTION At any \((x, y),\) the slope to \(y = k \ln x\) is \[\deriv{y}{x} = \frac{k}{x} \pd\] The orthogonal trajectory's slope is therefore the negative reciprocal of this expression; namely, it satisfies the differential equation \[\deriv{y}{x} = -\frac{x}{k} \pd\] We solve this differential equation by performing Separation of Variables, as follows: \[ \int \dd y = \int -\frac{x}{k} \di x \implies \boxed{y = -\frac{x^2}{2k} + C} \]
EXERCISE 25
For
\[\ds y = \cos kx\]
find the orthogonal trajectory to the family of functions, where \(k\) is a nonzero constant.
SOLUTION The slope to \(y = \cos kx\) at any \((x, y)\) is \[\deriv{y}{x} = - k \sin kx \pd\] Since the orthogonal trajectory's slope is the negative reciprocal of the cosine curve's slope, the orthogonal trajectory satisfies the differential equation \[\deriv{y}{x} = -\frac{1}{-k \sin kx} = \frac{1}{k \sin kx} \pd\] Performing Separation of Variables then gives \[\int \dd y = \int \frac{\dd x}{k \sin kx} = \int \frac{1}{k} \csc kx \di x \pd\] The orthogonal trajectory is therefore \[\boxed{y = C -\frac{1}{k^2} \ln \abs{\csc kx + \cot kx}}\]
EXERCISE 26
For
\[\ds x^2 = e^{ky}\]
find the orthogonal trajectory to the family of functions, where \(k\) is a nonzero constant.
SOLUTION We use Implicit Differentiation to find \[2x = k e^{ky} \deriv{y}{x} \implies \deriv{y}{x} = \frac{2x}{k e^{ky}} \pd\] The orthogonal trajectory is perpendicular to the curve \(x^2 = e^{ky},\) so its slope is the negative reciprocal; that is, it satisfies the differential equation \[\deriv{y}{x} = -\frac{k e^{ky}}{2x} \pd\] Now we use Separation of Variables to get \[ \ba \int e^{-ky} \di y &= \int -\frac{k}{2x} \di x \nl -\frac{1}{k} e^{-ky} &= -\frac{k}{2} \ln \abs x + C_1 \pd \ea \] Letting \(C = -kC_1,\) we find the orthogonal trajectory to be \[ e^{-ky} = \frac{k^2 \ln \abs x}{2} - k C_1 \implies \boxed{y = -\frac{1}{k} \ln \par{\frac{k^2 \ln \abs x}{2} + C}} \]
EXERCISE 27
For
\[\ds x = k^y\]
find the orthogonal trajectory to the family of functions, where \(k\) is a nonzero constant.
SOLUTION Let's rewrite the equation as \(x = e^{y \ln k}.\) To find \(\textderiv{y}{x}\) we use Implicit Differentiation as follows: \[ 1 = e^{y \ln k} (\ln k) \deriv{y}{x} \implies \deriv{y}{x} = \frac{1}{e^{y \ln k} \ln k} \pd \] The orthogonal trajectory is perpendicular to the curve \(x = k^y,\) so its slope is the negative reciprocal; that is, it satisfies the differential equation \[\deriv{y}{x} = -e^{y \ln k} \ln k \pd\] Performing Separation of Variables, we attain \[ \ba \int e^{-y \ln k} \di y &= \int (-\ln k) \di x \nl -\frac{e^{-y \ln k}}{\ln k} &= -x \ln k + C_1 \nl \frac{1}{k^y \ln k} &= x \ln k - C_1 \pd \ea \] We solve for \(y\) as follows: \[k^y = \frac{1}{x (\ln k)^2 - C_1 \ln k} \implies \boxed{y = \log_k \par{\frac{1}{x (\ln k)^2 + C}}}\] where \(C = -C_1 \ln k.\)
EXERCISE 28
A cup holds \(100\) milliliters of water, in which \(5\) grams of salt is initially dissolved. Saline water with a concentration of \(0.02\) gram of salt per milliliter of water is poured into the cup at a constant rate of \(10\) milliliters per minute. Water is drained from the cup such that the cup maintains its volume of \(100\) milliliters. How much salt is in the cup after \(8\) minutes?
SOLUTION Let \(y\) be the mass, in grams, of salt in the cup. Also let \(t\) be time in minutes. Note the initial condition \(y(0) = 5.\) The rate at which salt enters the cup is \[\text{rate in} = \frac{10 \un{mL}}{1 \un{min}} \times \frac{0.02 \un{g}}{1 \un{mL}} = \frac{0.2 \un{g}}{1 \un{min}} \pd\] At time \(t\) the concentration of salt in the cup, because its volume is constantly \(100 \un{mL},\) is \(y(t)/100.\) Liquid must exit the cup at the same entrance rate, \(10 \undiv{mL}{min},\) to maintain the same volume. So the rate at which salt exits the cup is \[\text{rate out} = \frac{10 \un{mL}}{1 \un{min}} \times \frac{y(t) \un{g}}{100 \un{mL}} = \frac{y(t)}{10} \, \frac{\un{g}}{\un{min}} \pd \] Hence, we attain the differential equation \[ \ba \deriv{y}{t} &= (\text{rate in}) - (\text{rate out}) \nl &= 0.2 - \frac{y}{10} = \frac{2 - y}{10} \pd \ea \] Performing Separation of Variables gives \[ \ba \int \frac{\dd y}{2 - y} &= \int \frac{\dd t}{10} \nl - \ln \abs{2 - y} &= \frac{t}{10} + C \pd \ea \] Substituting the initial condition \(y(0) = 5\) gives \[- \ln \abs{-3} = \frac{0}{10} + C \implies C = - \ln 3 \pd\] We therefore find \[ \ba - \ln \abs{2 - y} &= \frac{t}{10} - \ln 3 \nl \ln \abs{2 - y} &= \ln 3 - \frac{t}{10} \nl \abs{2 - y} &= 3 e^{-t/10} \pd \ea \] But since \(y(0) = 5,\) we have \((2 - y) \lt 0\) for \(t \geq 0.\) Accordingly, \(\abs{2 - y}\) \(= -(2 - y)\) and so we get \[ \ba -(2 - y) &= 3 e^{-t/10} \nl y &= 2 + 3 e^{-t/10} \pd \ea \] Hence, the amount of salt after \(8\) minutes is \[y(2) = \boxed{2 + 3 e^{-8/10}} \approx 3.348 \un g \pd\]
EXERCISE 29
Brine with a concentration of \(0.2\) kilogram of salt per liter of water flows into a tank at a rate of \(15\) liters per hour. Liquid in the tank is pumped out the same rate. The tank initially contains \(20\) kilograms of salt dissolved in \(600\) liters of water. How much salt remains in the tank after \(10\) hours?
SOLUTION Let \(y\) be the mass, in kilograms, of salt in the tank. Also let \(t\) be time in hours. Note the initial condition \(y(0) = 20.\) The rate at which salt enters the tank is \[\text{rate in} = \frac{15 \un{L}}{1 \un{hr}} \times \frac{0.2 \un{kg}}{1 \un{L}} = \frac{3 \un{kg}}{1 \un{hr}} \pd\] At time \(t\) the concentration of salt in the tank, since its volume is always \(600 \un{L},\) is \(y(t)/600.\) Liquid must exit at the same entrance rate, \(15 \undiv{L}{hr},\) to maintain the same volume in the tank. The rate at which salt exits is therefore \[\text{rate out} = \frac{15 \un{L}}{1 \un{hr}} \times \frac{y(t) \un{kg}}{600 \un{L}} = \frac{y(t)}{40} \, \frac{\un{kg}}{\un{hr}} \pd \] We therefore write the differential equation \[ \ba \deriv{y}{t} &= (\text{rate in}) - (\text{rate out}) \nl &= 3 - \frac{y}{40} = \frac{120 - y}{40} \pd \ea \] Performing Separation of Variables gives \[ \ba \int \frac{\dd y}{120 - y} &= \int \frac{\dd t}{40} \nl - \ln \abs{120 - y} &= \frac{t}{40} + C \pd \ea \] Substituting the initial condition \(y(0) = 20\) gives \[- \ln 100 = \frac{0}{40} + C \implies C = - \ln 100 \pd\] We therefore find \[ \ba - \ln \abs{120 - y} &= \frac{t}{40} - \ln 100 \nl \ln \abs{120 - y} &= \ln 100 - \frac{t}{40} \nl \abs{120 - y} &= 100 e^{-t/40} \pd \ea \] But since \(y(0) = 20\) and \(100 e^{-t/40} \gt 0\) for all \(t,\) we have \(\abs{120 - y}\) \(= 120 - y.\) So we obtain \[ \ba 120 - y &= 100 e^{-t/40} \nl y &= 120 - 100 e^{-t/40} \pd \ea \] Hence, the amount of salt after \(10\) hours is \[y(10) = \boxed{120 - 100 e^{-10/40}} \approx 42.120 \un{kg} \pd\]

A water tank contains \(4\) grams of a toxic chemical dissolved in \(500\) liters of water. To dilute the liquid, clean water is pumped into the tank at a rate of \(10\) liters per hour; water is drained from the tank at the same rate. After \(5\) days, what mass of the toxic chemical remains in the tank?
SOLUTION Let \(y\) be the mass, in grams, of the toxic chemical in the tank. Also let \(t\) be time in hours. Note the initial condition \(y(0) = 4.\) The rate at which the chemical enters the tank is \(0 \undiv{g}{hr}.\) At time \(t\) the concentration of chemical in the tank, because its volume is constantly \(500 \un{L},\) is \(y(t)/500.\) Liquid must exit the tank at the same entrance rate, \(10 \undiv{L}{hr},\) to maintain the same volume. So the rate at which the chemical is pumped out is \[\text{rate out} = \frac{10 \un{L}}{1 \un{hr}} \times \frac{y(t) \un{g}}{500 \un{L}} = \frac{y(t)}{50} \, \frac{\un{g}}{\un{hr}} \pd \] Thus, we attain the differential equation \[ \ba \deriv{y}{t} &= (\text{rate in}) - (\text{rate out}) \nl &= 0 - \frac{y}{50} \pd \ea \] Performing Separation of Variables gives \[ \ba \int \frac{\dd y}{y} &= \int -\frac{\dd t}{50} \nl \ln \abs y &= -\frac{t}{50} + C \pd \ea \] Substituting the initial condition \(y(0) = 4\) gives \[\ln 4 = -\frac{0}{50} + C \implies C = \ln 4 \pd\] So we have \[ \ba \ln \abs y &= -\frac{t}{50} + \ln 4 \nl \abs y &= 4 e^{-t/50} \pd \ea \] But since \(4 e^{-t/50} \gt 0\) for all \(t,\) we get \[ \ba y &= 4 e^{-t/50} \pd \ea \] Hence, the amount of chemical after \(5\) days—or \(24 \times 5\) \(= 120\) hours—is \[y(120) = \boxed{4 e^{-120/50}} \approx 0.363 \un g \pd\]

Perform the substitution \(u = x + y\) to solve the differential equation \[\deriv{y}{x} = x + y \pd\]
SOLUTION By differentiating both sides of \(u = x + y,\) we observe that \[\deriv{u}{x} = 1 + \deriv{y}{x} \iffArrow \deriv{y}{x} = \deriv{u}{x} - 1 \pd\] So the differential equation becomes \[\deriv{u}{x} - 1 = u \or \deriv{u}{x} = u + 1 \pd\] We perform Separation of Variables to get \[ \ba \int \frac{\dd u}{u + 1} &= \int \dd x \nl \ln \abs{u + 1} &= x + C_1 \nl u + 1 &= C e^x \nl u &= C e^x - 1 \cma \ea \] where \(C = \pm e^{C_1}.\) Reexpressing our answer in terms of \(x\) and \(y,\) we have \[x + y = C e^x - 1 \or \boxed{y = Ce^x - x - 1}\]
EXERCISE 32
By substituting \(y' = u,\) show that a general solution to \(y'' + y' = 0\) is, for arbitrary constants \(C_1\) and \(C_2,\) \[y = C_1 e^{-x} + C_2 \pd\]
SOLUTION The substitution \(y' = u\) gives \[u' + u = 0 \cma\] which we rewrite as the separable first-order differential equation \[\deriv{u}{x} = -u \pd\] On both sides we multiply by \(\dd x,\) divide by \(u,\) and then integrate. Doing so gives \[ \ba \int \frac{1}{u} \di u &= - \int \dd x \nl \ln \abs u &= -x + C_0 \nl u &= C_1 e^{-x} \cma \ea \] where \(C_1 = \pm e^{C_0}.\) Remember that \(u = y',\) so we substitute back to find \[ \ba \deriv{y}{x} &= C_1 e^{-x} \nl \int \dd y &= \int C_1 e^{-x} \di x \nl y &= -C_1 e^{-x} + C_2 \pd \ea \] Note that \(-C_1\) is semantically equivalent to \(C_1\) because they are both arbitrary constants. We therefore write \[\boxed{y = C_1 e^{-x} + C_2}\] Just as the general solution to a first-order differential equation has one constant of integration, the general solution of a second-order differential equation has two arbitrary constants.

Let's first find the general solution \(H(t).\) We isolate the variables, getting \begin{align} \int \frac{\dd H}{\sqrt H} &= \int -k \di t \nonumber \nl \tfrac{1}{2} \sqrt{H} &= -kt + C \pd \label{eq:tank-1} \end{align} We now solve for \(k\) and \(C \col\) When \(t = 0,\) \(H = 16.\) Thus, we get \[\tfrac{1}{2} \sqrt{16} = 0 + C \implies C = 2 \pd\] When \(t = 10,\) \(H = 9.\) We then find \[\tfrac{1}{2} \sqrt{9} = -10k + 2 \implies k = \tfrac{1}{20} \pd \] Therefore, \(\eqref{eq:tank-1}\) becomes \[ \tfrac{1}{2} \sqrt{H} = -\tfrac{1}{20} t + 2 \so \boxed{H(t) = \par{4 - \tfrac{1}{10}t}^2} \]
EXERCISE 35
Find all functions that satisfy the following property: the function's rate of change is inversely proportional to the square of itself.
SOLUTION We interpret rate of change to be \(\textderiv{y}{x},\) which is inversely proportional to \(y^2\)—that is, \[\deriv{y}{x} = \frac{k}{y^2}\] for any constant \(k.\) Performing Separation of Variables, we get \[ \ba \int y^2 \di y &= \int k \di x \nl \tfrac{1}{3} y^3 &= kx + c \nl \implies y &= \sqrt[3]{3kx + 3c} \ea \] Since \(c\) is an arbitrary constant, \(3c\) is also an arbitrary constant; likewise, \(3k\) is an arbitrary constant because \(k\) is any constant. We are given the initial condition \(Q(2) = 4,\) meaning \(Q\) must take on positive values. So let's choose the positive square root: \[Q = \sqrt{\frac{16t}{16 - 7t}} = \boxed{4 \sqrt{\frac{t}{16 - 7t}}}\]
EXERCISE 37
A particle moves along the \(x\)-axis with an acceleration given by \(a = -\tfrac{1}{2} v^2,\) where \(v\) is velocity. The particle begins its motion at the origin with a velocity of \(2.\) Find the particle's position, velocity, and acceleration functions in terms of time \(t \geq 0.\)
SOLUTION Note the initial conditions \(x(0) = 0\) and \(v(0) = 2.\)

Velocity Function Recall that acceleration is the time derivative of velocity—that is, \(a = \textderiv{v}{t}.\) Given \(a = -\tfrac{1}{2} v^2,\) we attain the differential equation \[ \deriv{v}{t} = -\tfrac{1}{2} v^2 \pd \] Performing Separation of Variables, we get \[ \ba \int \frac{1}{v^2} \di v &= \int -\tfrac{1}{2} \di t \nl -\frac{1}{v} &= -\tfrac{1}{2} t + C \pd \ea \] To solve for the constant \(C,\) we substitute the given initial condition \(v(0) = 2\) to find \[ -\tfrac{1}{2} = -\tfrac{1}{2} (0) + C \implies C = -\tfrac{1}{2} \pd \] Our velocity function is therefore \[v = \frac{1}{\tfrac{1}{2} t + \tfrac{1}{2}} = \boxed{\frac{2}{t + 1}}\]

Acceleration Function Given the relationship \(a = -\tfrac{1}{2} v^2,\) we substitute our velocity function to get \[a = -\tfrac{1}{2} \par{\frac{2}{t + 1}}^2 = \boxed{-\frac{2}{(t + 1)^2}}\]

Position Function The particle's position \(x\) at any time \(t\) is described by the differential equation \(v = \textderiv{x}{t}.\) Since we found \(v = 2/(t + 1),\) we get the differential equation \[ \deriv{x}{t} = \frac{2}{t + 1} \pd \] Integrating both sides gives \[x = 2 \ln \abs{t + 1} + D\] for some constant \(D.\) Substituting the initial condition \(x(0) = 0,\) we get \[0 = 2 \ln (0 + 1) + D \implies D = 0 \pd\] Hence, the position as a function of time is \[x = \boxed{2 \ln(t + 1)}\] (Since \(t \geq 0,\) we drop the absolute value bars around the logarithm.)

EXERCISE 38
A duck swims back and forth along a river with acceleration given by \(a^2 = 1 - 4v^2,\) where \(v\) is the duck's velocity. Let \(s\) be the duck's position from its nest in the river, and take east to be the positive direction. The duck begins at the nest and travels east with an initial velocity of \(1/2.\) Determine the duck's position, velocity, and acceleration as functions of time \(t \geq 0.\)
SOLUTION Note the initial conditions \(s(0) = 0\) and \(v(0) = 1/2.\)

Velocity Function Recall that acceleration is the time derivative of velocity—that is, \(a = \textderiv{v}{t}.\) Given \(a^2 = 1 - 4v^2,\) we have the following differential equation: \[ \deriv{v}{t} = \sqrt{1 - 4v^2} \pd \] (Once we attain the solution, we will see that it is irrelevant whether we choose the positive or negative square root.) Performing Separation of Variables gives \[\int \frac{1}{\sqrt{1 - 4v^2}} \di v = \int \dd t \pd\] To evaluate the integral on the left, we substitute \(u = 2v.\) Then \(\dd u = 2 \di v\) and so \[ \ba \int \frac{1}{\sqrt{1 - 4v^2}} \di v &= \int \frac{1/2}{\sqrt{1 - u^2}} \di u \nl &= \tfrac{1}{2} \asin u \nl &= \tfrac{1}{2} \asin 2v \cma \ea \] where we omitted the constant of integration. We therefore have \[\tfrac{1}{2} \asin 2v = t + C \pd\] Substituting the initial condition \(v(0) = 1/2,\) we find \[\tfrac{1}{2} \asin 1 = 0 + C \implies C = \frac{\pi}{4} \pd\] Hence, we have \[ \ba \tfrac{1}{2} \asin 2v &= t + \frac{\pi}{4} \nl \asin 2v &= 2t + \frac{\pi}{2} \nl 2v &= \sin \par{2t + \frac{\pi}{2}} \nl v(t) &= \boxed{\tfrac{1}{2} \sin \par{2t + \frac{\pi}{2}}} \ea \] (This function is even, so it was acceptable to pick the positive square root when decomposing \(a^2 = 1 - 4v^2.\))

Acceleration Function The acceleration function is given by \(a(t) = \textderiv{v}{t},\) namely, \[ \ba a(t) &= \deriv{}{t} \parbr{\tfrac{1}{2} \sin \par{2t + \frac{\pi}{2}}} \nl &= \boxed{\cos \par{2t + \frac{\pi}{2}}} \ea \]

Position Function The position function is given by \(s(t) = \int v(t) \di t,\) that is, \[s(t) = \int \tfrac{1}{2} \sin \par{2t + \frac{\pi}{2}} \di t \pd\] Substituting \(u = 2t + \pi/2,\) we have \(\dd u = 2 \di t\) and therefore \[ \ba \int \tfrac{1}{2} \sin \par{2t + \frac{\pi}{2}} \di t &= \int \tfrac{1}{4} \sin u \di u \nl &= - \tfrac{1}{4} \cos u + D \nl &= -\tfrac{1}{4} \cos \par{2t + \frac{\pi}{2}} + D \pd \ea \] Accordingly, the position function is \[s(t) = -\tfrac{1}{4} \cos \par{2t + \frac{\pi}{2}} + D \pd\] Since \(s(0) = 0,\) we find \[0 = -\tfrac{1}{4} \cos \par{0 + \frac{\pi}{2}} + D \implies D = 0 \pd\] Thus, the position function is \[s(t) = \boxed{-\tfrac{1}{4} \cos \par{2t + \frac{\pi}{2}}}\]
