10.1 Exercise Solutions: Sequences

EXERCISE 1
Define a sequence.
SOLUTION A sequence is a list of numbers written in a definite order—for example, \[\{1, 2, 3, \dots\} \pd\] We can write the general term of a sequence using the notation \(a_n,\) a function whose domain is nonnegative integers \(n.\)
EXERCISE 2
How do we determine whether a sequence \(\{a_n\}\) converges or diverges?
SOLUTION The sequence converges if its terms eventually settle to a final value \(L;\) otherwise, the sequence diverges. In particular, \(\{a_n\}\) is convergent if and only if \(\lim_{n \to \infty} a_n = L.\) So if this limit does not exist, then the sequence is divergent. The number \(L\) is called the limit of the sequence.
EXERCISE 3
Let \(m\) and \(M\) be any numbers such that \(m \leq M.\) For all \(n \geq 0,\) the sequence \(\{a_n\} \subsuper{0}{\infty}\) satisfies \(m \leq a_n \leq M\) and \(a_{n + 1} \lt a_n.\) What can you conclude about the sequence, and what is its limit (if it exists)?
SOLUTION Because \(m \leq a_n \leq M\) for all \(n \geq 0,\) the sequence \(\{a_n\}\) is bounded. And since \(a_{n + 1} \lt a_n\) for all \(n \geq 0,\) the sequence is decreasing (and is therefore monotonic). By the Monotonic Sequence Theorem, any sequence that is monotonic and bounded is convergent. Hence, \(\{a_n\}\) converges. Since its terms are decreasing, they must be approaching \(m\) as \(n \to \infty.\) So \[\lim_{n \to \infty} a_n = \boxed m\]
EXERCISE 4
For
\[\ds \parbrace{n + 4} \subsuper{0}{\infty}\]
write out the first three terms of the infinite sequence.
SOLUTION Let \(a_n = n + 4.\) We then see \[ \baat{2} a_0 &= 0 + 4 &&= 4 \cma \nl a_1 &= 1 + 4 &&= 5 \cma \nl a_2 &= 2 + 4 &&= 6 \pd \eaat \] So the expanded sequence is \[\boxed{\parbrace{4, 5, 6, \dots}}\]
EXERCISE 5
For
\[\ds \parbrace{\frac{(-1)^n}{n^2 + 2}} \subsuper{1}{\infty}\]
write out the first three terms of the infinite sequence.
SOLUTION Let \(a_n = (-1)^n/\par{n^2 + 2}.\) We then see \[ \baat{2} a_1 &= \frac{(-1)^1}{(1)^2 + 2} &&= -\frac{1}{3} \cma \nl a_2 &= \frac{(-1)^2}{(2)^2 + 2} &&= \frac{1}{6} \cma \nl a_3 &= \frac{(-1)^3}{(3)^2 + 2} &&= -\frac{1}{11} \pd \eaat \] So the expanded sequence is \[\boxed{\parbrace{-\frac{1}{3}, \frac{1}{6}, -\frac{1}{11}, \dots}}\]
EXERCISE 6
For
\[\ds \parbrace{\frac{n + 1}{3n + 2}} \subsuper{0}{\infty}\]
write out the first three terms of the infinite sequence.
SOLUTION Let \(a_n = (n + 1)/(3n + 2).\) We then see \[ \baat{2} a_0 &= \frac{(0) + 1}{3(0) + 2} &&= \frac{1}{2} \cma \nl a_1 &= \frac{(1) + 1}{3(1) + 2} &&= \frac{2}{5} \cma \nl a_2 &= \frac{(2) + 1}{3(2) + 2} &&= \frac{3}{8} \cma \nl \eaat \] So the expanded sequence is \[\boxed{\parbrace{\frac{1}{2}, \frac{2}{5}, \frac{3}{8}, \dots}}\]
EXERCISE 7
For
\[\ds a_{n + 1} = \tfrac{1}{6} a_n \cmaa a_1 = 5\]
write out the first three terms of the infinite sequence.
SOLUTION In a recursive sequence, each term is defined by a relationship to preceding terms. Writing out two more terms, we find \[ \baat{2} a_2 &= \tfrac{1}{6} a_1 &&= \tfrac{5}{6} \cma \nl a_3 &= \tfrac{1}{6} a_2 &&= \tfrac{5}{36} \pd \eaat \] The expanded sequence is therefore \[\boxed{\parbrace{5, \frac{5}{6}, \frac{5}{36}, \dots}}\] This sequence is called geometric because each term differs by a common ratio, \(1/6.\)
EXERCISE 8
For
\[\ds a_{n + 1} = -\tfrac{1}{2} \par{a_n + 4} \cmaa a_0 = 8\]
write out the first three terms of the infinite sequence.
SOLUTION In a recursive sequence, we relate the general term to the sequence's preceding terms. Writing out two more terms, we find \[ \baat{2} a_1 &= -\tfrac{1}{2} \par{a_0 + 4} &&= -6 \cma \nl a_2 &= -\tfrac{1}{2} \par{a_1 + 4} &&= 1 \pd \eaat \] The expanded sequence is therefore \[\boxed{\parbrace{8, -6, 1, \dots}}\]
EXERCISE 9
For
\[\{1, 3, 5, 7, \dots \}\]
write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION Each term is incremented by \(2.\) If \(n\) starts at \(0,\) then the sequence is represented by \[a_n = \boxed{1 + 2n}\]
EXERCISE 10
For
\[\{2, 6, 18, 54, \dots\}\]
write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION Each term is \(3\) times its preceding term. Thus, for a starting index of \(n = 0,\) \[a_n = \boxed{2(3)^n}\]
EXERCISE 11

write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION If \(n\) increases by \(1,\) then each term's numerator and denominator both increase by \(1.\) Each term alternates in sign, so we have \[a_n = \boxed{(-1)^n \, \frac{2 + n}{3 + n}}\]
EXERCISE 12
For
\[\ds \parbrace{\frac{1}{2}, -\frac{2}{3}, \frac{3}{4}, -\frac{4}{5}, \dots}\]
write the formula \(a_n\) starting at \(n = 0\) that models each sequence.
SOLUTION If \(n\) is increased by \(1,\) then each term's numerator and denominator both increase by \(1.\) The sign of each term alternates. We therefore have \[a_n = \boxed{(-1)^n \, \frac{1 + n}{2 + n}}\]
EXERCISE 13
For
\[\{-2, 2, -2, 2, \dots\}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION The terms of the sequence alternate between \(-2\) and \(2\) and do not approach any final value. So the sequence diverges.
EXERCISE 14
For
\[\ds \parbrace{\frac{n - 2}{n + 5}} \subsuper{1}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION The sequence converges if and only if \((n - 2)/(n + 5)\) approaches some real number as \(n \to \infty.\) We see \[\lim_{n \to \infty} \frac{n - 2}{n + 5} = \lim_{n \to \infty} \frac{n}{n} = 1 \pd\] Therefore, the sequence converges with a limit of \(\boxed 1.\)
EXERCISE 15
For
\[\ds \{0, -3, -6, -9, \dots\}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION The sequence's terms decrease toward \(-\infty,\) failing to settle to a final value. So the sequence diverges.
EXERCISE 16
For
\[\ds \parbrace{\frac{n^2 + 4n - 8}{3n^4 + 5}} \subsuper{0}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION We find \[\lim_{n \to \infty} \frac{n^2 + 4n - 8}{3n^4 + 5} = \lim_{n \to \infty} \frac{n^2}{3n^4} = \lim_{n \to \infty} \frac{1}{3n^2} = 0 \pd\] Thus, the sequence converges and has a limit of \(\boxed 0.\)
EXERCISE 17
For
\[\ds \parbrace{\frac{n^3 - 8n^2 + 12n - 5}{9n^2 + 5n - 12}} \subsuper{1}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION We find \[\lim_{n \to \infty} \frac{n^3 - 8n^2 + 12n - 5}{9n^2 + 5n - 12} = \lim_{n \to \infty} \frac{n^3}{9n^2} = \lim_{n \to \infty} \frac{n}{9} = \infty \pd\] Since this limit does not exist, the sequence's terms do not tend to a final value as \(n \to \infty.\) (In fact, they increase without bound.) The sequence therefore diverges.
EXERCISE 18
For
\[\ds \parbrace{\frac{3}{4}, \frac{9}{16}, \frac{27}{64}, \frac{81}{256}, \dots}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION
We could define \(a_n\) to begin at any starting index (such as \(n = 0\)); \(\lim_{n \to \infty} a_n = 0\) is true regardless of the starting index.
The general term of this sequence is \[a_n = \par{\frac{3}{4}}^n \cma\] starting from \(n = 1.\) Then \(\lim_{n \to \infty} a_n = 0,\) so the sequence converges and has a limit of \(\boxed 0.\)
EXERCISE 19
For
\[\ds \parbrace{\frac{\atan n}{n}} \subsuper{1}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION Observe that \[ \ba \lim_{n \to \infty} \frac{\atan n}{n} &= \par{\lim_{n \to \infty} \frac{1}{n}} \par{\lim_{n \to \infty} \atan n} \nl &= (0) \par{\frac{\pi}{2}} = 0 \pd \ea \] The sequence therefore converges, and its limit is \(\boxed{0}.\)
EXERCISE 20
For
\[\ds \parbrace{\sin kn} \subsuper{0}{\infty} \cmaa k \ne 0\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION Because \(\lim_{n \to \infty} \sin kn\) does not exist (by a similar argument to Example 1.1-7), the terms of \(\{\sin kn\}\) do not settle to any final value as \(n \to \infty.\) Hence, the sequence diverges.
EXERCISE 21
For
\[\ds \parbrace{\frac{(-1)^n}{n^k}} \subsuper{1}{\infty} \cmaa k \gt 0\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION Since \(-1 \leq (-1)^n \leq 1,\) we have \[-\frac{1}{n^k} \leq \frac{(-1)^n}{n^k} \leq \frac{1}{n^k} \pd\] If \(k \gt 0,\) then \[\lim_{n \to \infty} \par{-\frac{1}{n^k}} = \lim_{n \to \infty} \frac{1}{n^k} = 0 \pd\] [In Section 1.1 we justified the limit \(\lim_{x \to \infty} \par{1/x^k} = 0,\) where \(k\) is positive.] Thus, by the Squeeze Theorem for Sequences, \((-1)^n/n^k \to 0\) as \(n \to \infty.\) Hence, the sequence \(\{(-1)^n/n^k\}\) converges and has a limit of \(\boxed{0}.\)
EXERCISE 22
For
\[\ds \parbrace{\frac{\sqrt[4] n}{e^{n/2}}} \subsuper{0}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION The limit of the sequence is equivalent to the limit \[\lim_{x \to \infty} \frac{\sqrt[4] x}{e^{x/2}} \pd\] Since both the numerator and denominator approach \(\infty\) as \(x \to \infty,\) this limit is in the indeterminate form \(\indInfty.\) So using L'Hôpital's Rule, we see \[ \lim_{x \to \infty} \frac{\sqrt[4] x}{e^{x/2}} = \lim_{x \to \infty} \frac{\tfrac{1}{4} x^{-3/4}}{\tfrac{1}{2} e^{x/2}} = \lim_{x \to \infty} \frac{1}{2 x^{3/4} e^{x/2}} = 0 \pd \] (In the denominator, both terms \(x^{3/4}\) and \(e^{x/2}\) approach \(\infty\) as \(x \to \infty.\) The ratio therefore decreases to \(0.\)) Accordingly, the sequence converges; its limit is \(\boxed 0.\)
EXERCISE 23
For
\[\ds \parbrace{\frac{n^2 + 4}{n \ln n}} \subsuper{3}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION The limit of the sequence is equivalent to the limit \[\lim_{x \to \infty} \frac{x^2 + 4}{x \ln x} \pd\] Since both the numerator and denominator approach \(\infty\) as \(x \to \infty,\) this limit is in the indeterminate form \(\indInfty.\) We evaluate the limit by using L'Hôpital's Rule twice, as follows: \[\lim_{x \to \infty} \frac{x^2 + 4}{x \ln x} = \lim_{x \to \infty} \frac{2x}{\ln x + 1} = \lim_{x \to \infty} \frac{2}{1/x} = \lim_{x \to \infty} 2x = \infty \pd\] Since this limit does not exist, the limit of the sequence does not exist. So the sequence diverges.
EXERCISE 24
For
\[\ds \parbrace{\frac{\sin^3 \par{n^4 + 8}}{\sqrt[3] n}} \subsuper{1}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION The range of sine is \([-1, 1],\) so the range of the cube of sine is also \([-1, 1].\) We therefore see \[ \baat{2} -1 &\leq \sin^3 \par{n^4 + 8} &&\leq 1 \nl -\frac{1}{\sqrt[3] n} &\leq \frac{\sin^3 \par{n^4 + 8}}{\sqrt[3] n} &&\leq \frac{1}{\sqrt[3] n} \pd \eaat \] Notice that \[\lim_{n \to \infty} \par{-\frac{1}{\sqrt[3] n}} = \lim_{n \to \infty} \par{\frac{1}{\sqrt[3] n}} = 0 \pd\] By the Squeeze Theorem for Sequences, \[\lim_{n \to \infty} \frac{\sin^3 \par{n^4 + 8}}{\sqrt[3] n} = 0 \pd\] So the sequence converges and has a limit of \(\boxed{0}.\)
EXERCISE 25
For
\[\ds \parbrace{1 + \frac{\cos^{48} n}{n^3 + n}} \subsuper{1}{\infty}\]
determine whether the sequence converges or diverges; find its limit if it converges.
SOLUTION Because the range of cosine is \([-1, 1],\) we have \[ \baat{2} -1 &\leq \cos^{48} n &&\leq 1 \nl -\frac{1}{n^3 + n} &\leq \frac{\cos^{48} n}{n^3 + n} &&\leq \frac{1}{n^3 + n} \nl 1 - \frac{1}{n^3 + n} &\leq 1 + \frac{\cos^{48} n}{n^3 + n} &&\leq 1 + \frac{1}{n^3 + n} \pd \eaat \] Note that \[\lim_{n \to \infty} \par{1 - \frac{1}{n^3 + n}} = \lim_{n \to \infty} \par{1 + \frac{1}{n^3 + n}} = 1 \pd\] Thus, the Squeeze Theorem for Sequences gives \[\lim_{n \to \infty} \par{1 + \frac{\cos^{48} n}{n^3 + n}} = 1 \pd\] The sequence therefore converges and has a limit of \(\boxed 1.\)

describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.

EXERCISE 29
For
\[\ds \parbrace{\frac{2}{5}, -\frac{6}{10}, \frac{3}{5}, -\frac{7}{10}, \dots}\]
describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.
SOLUTION This sequence is neither increasing nor decreasing. It is also not monotonic. Observe that the terms with odd indexes—for example, the first and third terms, \(2/5\) and \(3/5\)—are growing to \(\infty.\) Conversely, the terms with even indexes—such as the second and fourth terms, \(-6/10\) and \(-7/10\)—are decreasing to \(-\infty.\) Hence, the sequence is not bounded.
EXERCISE 30
For
\[\ds \parbrace{\frac{n}{n + 2}} \subsuper{0}{\infty}\]
describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.
SOLUTION Let's write out a few terms to see a pattern: \[\parbrace{0, \frac{1}{3}, \frac{2}{4}, \frac{3}{5}, \frac{4}{6}, \dots} \pd\] The terms are increasing. To prove this pattern for all \(0 \leq n \lt \infty,\) we see \[ \ba \frac{(n + 1)}{(n + 1) + 2} &\gt \frac{n}{n + 2} \nl \frac{n + 1}{n + 3} &\gt \frac{n}{n + 2} \nl (n + 1)(n + 2) &\gt n (n + 3) \nl n^2 + 3n + 2 &\gt n^2 + 3n \nl 2 &\gt 0 \pd \ea \] Thus, the \((n + 1)\)st term is always greater than the \(n\)th term. Because the sequence is increasing, it is bounded below by \(m = 0.\) We also have \[\lim_{n \to \infty} \frac{n}{n + 2} = 1 \cma\] so the terms are increasing to \(1.\) Hence, the sequence is bounded between \(0\) and \(1.\) All increasing sequences are monotonic. The sequence is therefore increasing, monotonic, and bounded.
EXERCISE 31
For
\[\ds \parbrace{e^{-n}} \subsuper{1}{\infty}\]
describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.
SOLUTION Let's compare \(e^{-n}\) to the continuous function \(f(x) = e^{-x}.\) Differentiating shows \[f'(x) = -e^{-x} \cma\] which is negative for all \(x.\) The function \(f(x) = e^{-x}\) is therefore decreasing for all \(x,\) including for \(x \geq 1,\) so the sequence \(\parbrace{e^{-n}}\) is also decreasing for \(n \geq 1.\) For \(1 \leq n \lt \infty\) we see \(0 \lt e^{-n} \leq e^{-1}.\) Thus, the sequence is bounded, decreasing, and monotonic. (All decreasing sequences are monotonic.)
EXERCISE 32
For
\[\ds \parbrace{\frac{(-1)^{n + 1}}{n^2}} \subsuper{1}{\infty}\]
describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.
SOLUTION Let \(a_n = (-1)^{n + 1}/n^2.\) Writing out a few terms, we see \[ \baat{2} a_1 &= \frac{(-1)^{1 + 1}}{(1)^2} &&= 1 \cma \nl a_2 &= \frac{(-1)^{2 + 1}}{(2)^2} &&= -\frac{1}{4} \cma \nl a_3 &= \frac{(-1)^{3 + 1}}{(3)^2} &&= \frac{1}{9} \pd \nl \eaat \] Because each term alternates in sign, the sequence is not monotonic; it is therefore neither increasing nor decreasing. But since \(1/n^2 \to 0\) as \(n \to \infty,\) the sequence's terms are decreasing in magnitude to \(0.\) So the sequence must be bounded.
EXERCISE 33
For
\[\ds \parbrace{\frac{n^2}{2n^2 + 9}} \subsuper{2}{\infty}\]
describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.
SOLUTION Let \(f(x)\) be a function such that \(f(n) = n^2/\par{2n^2 + 9}\) for all integers \(n \geq 2.\) We then differentiate \(f\) to see \[ f'(x) = \frac{(2x) \par{2x^2 + 9} - x^2 \par{4x}}{\par{2x^2 + 9}^2} = \frac{18x}{\par{2x^2 + 9}^2} \pd \] Since \(f'(x) \gt 0\) for all \(x,\) the function \(f\) is increasing for all \(x.\) Accordingly, the sequence \(\{a_n\}\) is increasing for \(n \geq 2.\) Also observe that \[\lim_{n \to \infty} \frac{n^2}{2n^2 + 9} = \lim_{n \to \infty} \frac{n^2}{2n^2} = \frac{1}{2} \pd\] Because the sequence is increasing, its lower bound is \(m = a_2 = 4/17,\) and its upper bound is \(1/2.\) Hence, \(4/17 \leq a_n \lt 1/2\) for all \(n \geq 2.\) The sequence \(\{a_n\} \subsuper{2}{\infty}\) is therefore bounded, increasing, and monotonic. (All increasing sequences are monotonic.)
EXERCISE 34
For
\[\ds \parbrace{n e^{-4n}} \subsuper{1}{\infty}\]
describe the sequence in terms of monotonicity, increasing or decreasing, and boundedness.
SOLUTION Let \(f(x)\) be a function such that \(f(n) = n e^{-4n}\) for all integers \(n \geq 1.\) Differentiating \(f,\) we see \[f'(x) = e^{-4x} - 4x e^{-4x} = e^{-4x} \par{1 - 4x} \pd\] Observe that \(f'(x) \lt 0\) for all \(x \gt 1/4,\) meaning \(f(x)\) is decreasing on \([1/4, \infty).\) Accordingly, the sequence \(\parbrace{n e^{-4n}}\) is decreasing for all \(n \geq 1.\) To find the limit of the sequence, we take the limit of \(f(x)\) as \(x \to \infty\)—namely, \[\lim_{x \to \infty} xe^{-4x} = \lim_{x \to \infty} \frac{x}{e^{4x}} \pd\] Since the numerator and denominator both approach \(\infty\) as \(x \to \infty,\) the limit is in the indeterminate form \(\indInfty.\) Applying L'Hôpital's Rule gives the limit to be \[\lim_{x \to \infty} \frac{1}{4 e^{4x}} = 0 \pd\] The limit of the sequence is the same; that is, \(n e^{-4n} \to 0\) as \(n \to \infty.\) But because the sequence is decreasing and has positive terms, it is bounded below by \(0.\) And it is bounded above by \(m = (1) e^{-4(1)}\) \(= e^{-4}.\) Hence, the sequence is bounded. It is also decreasing; thus, it is monotonic.
EXERCISE 35
For
\(\{a_n\}\) converges but \(\{1/a_n\}\) diverges
provide an example of \(a_n\) that satisfies the property.
SOLUTION One example is \(a_n = 1/n;\) since \(\lim_{n \to \infty} (1/n) = 0,\) the sequence \(\{a_n\}\) converges. Yet \[\frac{1}{a_n} = \frac{1}{1/n} = n \cma\] and \(\lim_{n \to \infty} n\) does not exist. Thus, \(\{1/a_n\}\) diverges.
EXERCISE 36
For
\(\{a_n\}\) converges but \(\{n a_n\}\) diverges
provide an example of \(a_n\) that satisfies the property.
SOLUTION One example is \(a_n = 1/\sqrt n;\) since \(\lim_{n \to \infty} (1/\sqrt n) = 0,\) the sequence \(\{a_n\}\) converges. But \[n a_n = n \par{\frac{1}{\sqrt n}} = \sqrt n \cma\] and \(\lim_{n \to \infty} \sqrt n\) does not exist. Therefore, \(\{n a_n\}\) diverges.
EXERCISE 37
For
\(\{a_n\}\) diverges but \(\{a_{2n}\}\) converges
provide an example of \(a_n\) that satisfies the property.
SOLUTION An easy example is \(a_n = (-1)^n.\) The sequence \(\{a_n\}\) diverges (see Example 4). But observe that \[a_{2n} = (-1)^{2n} = \parbr{(-1)^2}^n = (1)^n = 1 \pd\] Hence, \(\lim_{n \to \infty} a_{2n} = 1,\) so \(\{a_{2n}\}\) converges.
EXERCISE 38
Provide an example of two divergent sequences \(\{a_n\}\) and \(\{b_n\}\) such that \(\{a_n b_n\}\) converges.
SOLUTION One example is \(a_n = (-1)^n\) and \(b_n = (-1)^n;\) in Example 4 we justified that \(\{(-1)^n\}\) is divergent. We see \[ \ba a_n b_n &= \parbr{(-1)^n} \parbr{(-1)^n} = (-1)^{2n} \nl &= \parbr{(-1)^2}^n = (1)^n = 1 \pd \ea \] Then \(\lim_{n \to \infty} (a_n b_n) = 1,\) so \(\{a_n b_n\}\) converges.
EXERCISE 39
Give an example of two convergent sequences \(\{a_n\}\) and \(\{b_n\}\) such that \(\{a_n/b_n\}\) diverges.
SOLUTION One example is \[a_n = \frac{1}{n} \and b_n = \frac{1}{n^2} \pd\] Since \(a_n \to 0\) and \(b_n \to 0\) as \(n \to \infty,\) the sequences \(\{a_n\}\) and \(\{b_n\}\) both converge. But we see \[\frac{a_n}{b_n} = \frac{1/n}{1/n^2} = n \cma\] which tends to \(\infty\) as \(n \to \infty.\) Since \(\lim_{n \to \infty} (a_n/b_n)\) does not exist, the sequence \(\{a_n/b_n\}\) diverges.
EXERCISE 40
If \(\{a_n\}\) diverges, then \(\lim_{n \to \infty} a_n\) does not exist.
SOLUTION True. The sequence \(\{a_n\}\) converges if and only if \(\lim_{n \to \infty} a_n\) exists. If the limit doesn't exist, then the sequence diverges.
EXERCISE 41
If two sequences \(\{a_n\}\) and \(\{b_n\}\) are both convergent, then the sequences \(\{a_n + b_n\},\) \(\{a_n - b_n\},\) \(\{a_n b_n\},\) and (if \(\lim_{n \to \infty} b_n \ne 0\)) \(\{a_n/b_n\}\) are all convergent.
SOLUTION True. The sequences \(\{a_n\}\) and \(\{b_n\}\) are convergent if and only if, for real numbers \(L_1\) and \(L_2,\) \[\lim_{n \to \infty} a_n = L_1 \and \lim_{n \to \infty} b_n = L_2 \pd\] So by the Limit Laws, \[ \ba \lim_{n \to \infty} \par{a_n + b_n} &= L_1 + L_2 \cma \nl \lim_{n \to \infty} \par{a_n - b_n} &= L_1 - L_2 \cma \nl \lim_{n \to \infty} \par{a_n b_n} &= L_1 L_2 \cma \nl \lim_{n \to \infty} \par{\frac{a_n}{b_n}} &= \frac{L_1}{L_2} \cmaa L_2 \ne 0 \pd \ea \] Thus, the sequences with these four combinations of \(\{a_n\}\) and \(\{b_n\}\) are all convergent.
EXERCISE 42
If \(\{a_n\}\) and \(\{b_n\}\) both diverge, then \(\{a_n + b_n\}\) also diverges.
SOLUTION False. For example, the sequences \(\{a_n\}\) and \(\{b_n\}\) defined by \[a_n = n \and b_n = -n\] both diverge because neither \(\lim_{n \to \infty} a_n\) nor \(\lim_{n \to \infty} b_n\) exists. But \(a_n + b_n = n + (-n)\) \(= 0,\) so we have \[\lim_{n \to \infty} \par{a_n + b_n} = \lim_{n \to \infty} 0 = 0 \pd\] Since this limit exists, the sequence \(\{a_n + b_n\}\) converges.
EXERCISE 43
All convergent sequences are bounded and monotonic.
SOLUTION False. The sequence \[a_n = \frac{(-1)^n}{n}\] converges because \(a_n \to 0\) as \(n \to \infty\) (as you may verify using the Squeeze Theorem for Sequences). But it is not monotonic because its terms oscillate in sign. Note: The Monotonic Sequence Theorem says that bounded, monotonic sequences are convergent, but not all convergent sequences must satisfy these criteria.
EXERCISE 44
If a sequence is divergent, then it cannot be bounded.
SOLUTION False. The sequence \[a_n = (-1)^n\] diverges but is bounded.
EXERCISE 45
If \(\{a_n\}\) converges, then prove that \(\{a_n/n\}\) converges to \(0.\)
SOLUTION The sequence \(\{a_n\}\) converges if and only if \(\lim_{n \to \infty} a_n = L\) for some real number \(L.\) By the Product Law for Limits, \[ \ba \lim_{n \to \infty} \frac{a_n}{n} &= \par{\lim_{n \to \infty} \frac{1}{n}} \par{\lim_{n \to \infty} a_n} \nl &= (0) (L) = 0 \pd \ea \] So the sequence \(\{a_n/n\}\) converges to \(0.\)
EXERCISE 46
Prove that if \(\{\abs{a_n}\}\) converges to \(0,\) then \(\{a_n\}\) also converges to \(0.\)
SOLUTION Our goal is to prove that if \(\lim_{n \to \infty} \abs{a_n} = 0,\) then \(\lim_{n \to \infty} a_n = 0\) too. We see \(-\abs{a_n} \leq a_n \leq \abs{a_n}.\) Since \[\lim_{n \to \infty} \abs{a_n} = \lim_{n \to \infty} \par{-\abs{a_n}} = 0 \cma\] the Squeeze Theorem for Sequences asserts that \(\lim_{n \to \infty} a_n = 0\) too. Hence, \(\{a_n\}\) converges to \(0.\)
EXERCISE 47
Let \(\{b_n\}\) be a bounded sequence. If \(\lim_{n \to \infty} a_n = 0,\) then prove that \(\lim_{n \to \infty} \par{a_n b_n} = 0.\)
SOLUTION Since \(\{b_n\}\) is bounded, we have \(m \leq b_n \leq M\) for any real \(m\) and \(M.\) It then follows that \[m a_n \leq a_n b_n \leq M a_n \pd\] As \(n \to \infty,\) \(a_n \to 0\) (as given); we therefore find \[ \baat{2} \lim_{n \to \infty} (m a_n) &= m \lim_{n \to \infty} a_n &&= 0 \cma \nl \lim_{n \to \infty} (M a_n) &= M \lim_{n \to \infty} a_n &&= 0 \pd \eaat \] So, as requested, \(\lim_{n \to \infty} \par{a_n b_n} = 0\) by the Squeeze Theorem for Sequences.
EXERCISE 48
Prove that the sequence defined recursively by \[a_1 = 1 \cmaa a_{n + 1} = \tfrac{2}{5} a_n - 3 \cma\] is decreasing.
SOLUTION The sequence is decreasing if and only if \(a_{n + 1} \lt a_n\) for all \(n \geq 1.\) Starting with the case \(n = 1,\) we have \[a_2 = \tfrac{2}{5} a_1 - 3 = \tfrac{2}{5} (1) - 3 = -\tfrac{13}{5} \pd\] Clearly, \(-13/5 \lt 1\) and so we conclude \(a_2 \lt a_1.\) Now we assume \(a_{n + 1} \lt a_n\) is true for \(n = k\) (where \(k \gt 1)\). Doing so, we find \[ \ba a_{k + 1} &\lt a_k \nl \tfrac{2}{5} a_{k + 1} &\lt \tfrac{2}{5} a_k \nl \tfrac{2}{5} a_{k + 1} - 3 &\lt \tfrac{2}{5} a_k - 3 \nl a_{k + 2} &\lt a_{k + 1} \pd \ea \] So \(a_{n + 1} \lt a_n\) is also true for \(n = k + 1.\) By mathematical induction, \(a_{n + 1} \lt a_n\) is true for all \(n \geq 1.\) Hence, \(\{a_n\}\) is decreasing.
EXERCISE 49
Using a Proof by Contradiction, prove that the sequence defined by \[a_1 = 3 \cmaa a_{n + 1} = -\frac{1}{a_n} \cma\] is divergent.
SOLUTION Let's begin by supposing that \(\{a_n\}\) converges—that is, by assuming \(\lim_{n \to \infty} a_n\) \(= L\) for any real \(L.\) We therefore have \(a_n \to L\) and \(a_{n + 1} \to L\) as \(n \to \infty.\) Taking the limits of both sides of the recursive relationship, we have \[ \ba \lim_{n \to \infty} a_{n + 1} &= \lim_{n \to \infty} \par{-\frac{1}{a_n}} \nl L &= -\frac{1}{L} \nl L^2 &= -1 \pd \ea \] But the equation \(L^2 = -1\) has no real solutions. Thus, we have a contradiction—no real value of \(L\) exists. So the sequence \(\{a_n\}\) cannot be convergent; it must instead diverge.
EXERCISE 50
Let \(a_n\) be defined by the recursive relationship \[a_1 = 9 \cmaa a_{n + 1} = \tfrac{1}{4} \par{a_n + 5} \pd\] Prove that \(\{a_n\}\) converges, and find its limit.
SOLUTION To gauge the sequence's behavior, let's first write out a few terms: \[ \baat{2} a_2 &= \tfrac{1}{4} \par{a_1 + 5} = 3.5 &\lspace a_3 &= \tfrac{1}{4} \par{a_2 + 5} = 2.125 \nl a_4 &= 1.78125 &\lspace a_5 &= 1.6953125 \nl a_6 &= 1.673828125 &\lspace a_7 &= 1.668457031 \nl a_8 &= 1.667114258 &\lspace a_9 &= 1.666778564 \pd \eaat \] It appears that the sequence is decreasing to \(5/3.\) Now let's prove that this guess is true.

Proving Convergence Let's show that \(\{a_n\}\) is monotonic and bounded; then the Monotonic Sequence Theorem asserts that the sequence is convergent. We use a Proof by Induction to prove that \(a_{n + 1} \lt a_n\) for all \(n \geq 1 \col\) Starting with the case \(n = 1,\) we see \(a_2 = 3.5\) \(\lt a_1 = 9.\) So \(a_{n + 1} \lt a_n\) is true for \(n = 1.\) Assuming that the inequality is true for \(n = k,\) where \(k \gt 1,\) we see \[ \ba a_{k + 1} &\lt a_k \nl \tfrac{1}{4} a_{k + 1} &\lt \tfrac{1}{4} a_k \nl \tfrac{1}{4} \par{a_{k + 1} + 5} &\lt \tfrac{1}{4} \par{a_k + 5} \nl a_{k + 2} &\lt a_{k + 1} \pd \ea \] Thus, \(a_{n + 1} \lt a_n\) is also true for \(n = k + 1.\) Accordingly, \(\{a_n\}\) is decreasing for all \(n \geq 1\) by mathematical induction. Next we show that the sequence is bounded: Since the sequence is decreasing, it has an upper bound of \(M = a_1 = 9.\) We suspect that the sequence has a lower bound of \(5/3,\) so let's verify this guess by using mathematical induction to prove that \(a_n \gt 5/3\) when \(n \geq 1.\) We see \(a_1 = 9 \gt 5/3,\) so this inequality is true for \(n = 1.\) Assuming that it is true for \(n = k\) (where \(k \gt 1\)), we see \[ \ba a_k &\gt \tfrac{5}{3} \nl \tfrac{1}{4} a_k &\gt \tfrac{5}{12} \nl \tfrac{1}{4} \par{a_k + 5} &\gt \tfrac{5}{12} + \tfrac{5}{4} = \tfrac{5}{3} \nl a_{k + 1} &\gt \tfrac{5}{3} \pd \ea \] Hence, \(a_n \gt 5/3\) is true for \(n = k + 1.\) So by mathematical induction, the inequality is true for all \(n \geq 1.\) We therefore have \(5/3 \lt a_n \leq 9,\) so the sequence is bounded. And because the sequence is monotonic (as it is decreasing), it converges by the Monotonic Sequence Theorem.

Limit of the Sequence Because we know \(\lim_{n \to \infty} a_n\) exists, let's suppose that the limit equals \(L.\) Then both \(a_{n + 1}\) and \(a_n\) approach \(L\) as \(n \to \infty.\) Using the recursive relationship between \(a_n\) and \(a_{n + 1},\) we observe \[ \ba \lim_{n \to \infty} a_{n + 1} &= \lim_{n \to \infty} \parbr{\tfrac{1}{4} \par{a_n + 5}} \nl L &= \tfrac{1}{4} L + \tfrac{5}{4} \nl L &= \tfrac{5}{3} \pd \ea \] So, as expected, the sequence converges to \(\boxed{5/3}.\)
