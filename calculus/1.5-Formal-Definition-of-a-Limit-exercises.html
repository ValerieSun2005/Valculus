\(\hspace{0.56em}1.\)
Because \(\lim_{x \to 1} \par{x^3 - 2x + 7}\) \(= 6,\) a number \(\delta \gt 0\) exists such that for any \(\varepsilon,\) \[\abs{\par{x^3 - 2x + 7} - 6} \lt \varepsilon \if \abs{x - 1} \lt \delta \pd\] Using a graphing calculator, find \(\delta\) if \(\varepsilon = 0.05.\)
\(\hspace{0.56em}2.\)
The limit \(\lim_{x \to 0} \par{1/x^4} = \infty\) means, for some \(M \gt 0,\) a number \(\delta \gt 0\) exists such that \[\frac{1}{x^4} \gt M \if \abs x \lt \delta \pd\] Using a graphing calculator, find the largest \(\delta\) for which
  1. \(M = 100\)
  2. \(M = 500\)
  3. \(M = 1000\)
\(\hspace{0.56em}3.\)

  1. \(M = 50\)
  2. \(M = 100\)
  3. \(M = 250\)
