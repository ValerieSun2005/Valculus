0.6: Modifying Functions; Inverse Functions

A basic understanding of functions raises questions about the next stepâ€”how do we modify functions and their graphs? Doing so enables us to graph new realms of functions grounded on known, foundational functions. What functions can we produce by combining two or more functions? Finally, how do we map a function's output back onto its input? These answers are explained through the following topics:

  • Transforming Functions
  • Combinations of Functions
  • Inverse Functions

Transforming Functions

A function transformation modifies the graph of \(y = f(x)\) by stretching, compressing, or relocating it. The simplest types are translations, which shift the graph vertically or horizontally. Let \(c\) be a positive constant. Then adding \(c\) to \(f(x)\) raises all the \(y\)-values by \(c,\) so the graph of \(y = f(x) + c\) is the graph of \(y = f(x)\) shifted up \(c\) units (Figure 1A). By the same logic, the graph of \(y = f(x) - c\) is the graph of \(y = f(x)\) moved \(c\) units down.

If \(c \gt 0,\) then each output value of \(f(x - c)\) is reached when \(x\) is \(c\) units greater than in \(f(x).\) Hence, the graph of \(y = f(x - c)\) is the graph of \(y = f(x)\) shifted \(c\) units to the right (Figure 1B). Equally counterintuitive, the graph of \(y = f(x + c)\) is the graph of \(y = f(x)\) shifted \(c\) units to the left.

TRANSLATING GRAPHS
Let \(c \gt 0.\)
New Function Translation
\(y = f(x) + c\) Shift \(y = f(x)\) up \(c\) units
\(y = f(x) - c\) Shift \(y = f(x)\) down \(c\) units
\(y = f(x - c)\) Shift \(y = f(x)\) to the right \(c\) units
\(y = f(x + c)\) Shift \(y = f(x)\) to the left \(c\) units

Next we consider stretching and compressing graphs. First suppose that \(a \gt 1.\) Accordingly, the output values of \(a f(x)\) are \(a\) times bigger than the output values of \(f(x).\) So the graph of \(y = a f(x)\) is the graph of \(y = f(x)\) stretched vertically by a factor of \(a\) (Figure 2A). Conversely, the output values of \(\tfrac{1}{a} f(x)\) are \(a\) times smaller than the output values of \(f(x).\) Hence, the graph of \(y = \tfrac{1}{a} f(x)\) is compressed vertically by a factor of \(a\) (Figure 2B). Through the same logic, because \(ax\) is \(a\) times larger than \(x,\) the output values of \(f(ax)\) are reached \(a\) times sooner than in \(f(x).\) Thus, the graph of \(y = f(ax)\) is \(y = f(x)\) compressed horizontally by a factor of \(a\) (Figure 3A). Similarly, \(x/a\) is \(a\) times smaller than \(x,\) meaning the output values of \(f(x/a)\) are attained \(a\) times later than in \(f(x).\) The graph of \(y = f(x/a)\) is therefore the result of stretching \(y = f(x)\) horizontally by a factor of \(a\) (Figure 3B).

DILATING GRAPHS
Let \(a \gt 1.\)
New Function Translation
\(y = a f(x)\) Stretch \(y = f(x)\) vertically by a factor of \(a\)
\(y = \tfrac{1}{a} f(x)\) Compress \(y = f(x)\) vertically by a factor of \(a\)
\(y = f(ax)\) Compress \(y = f(x)\) horizontally by a factor of \(a\)
\(y = f(x/a)\) Stretch \(y = f(x)\) horizontally by a factor of \(a\)

The output value of \(y = -f(x)\) has the opposite sign of \(y = f(x).\) Because \((x, y)\) is replaced by \((x, -y),\) the graph of \(y = -f(x)\) is the graph of \(y = f(x)\) reflected across the \(x\)-axis (Figure 4A). Likewise, because a point on the graph of \(y = f(x)\) is \((x, y),\) a point on \(y = f(-x)\) is \((-x, y).\) Hence, the graph of \(y = f(-x)\) is a reflection of \(y = f(x)\) across the \(y\)-axis (Figure 4B).

REFLECTING GRAPHS
New Function Translation
\(y = -f(x)\) Reflect \(y = f(x)\) across the \(x\)-axis
\(y = f(-x)\) Reflect \(y = f(x)\) across the \(y\)-axis

These rules of transformations enable us to hand-draw many custom functions starting from their parent functions. A parent function is the simplest form of function within a particular category of functions. For example, \(y = x\) is the parent function of linear functions; it's the foundation for all linear functions. When multiple transformations are present, we apply translations last. The following examples demonstrate these procedures.

EXAMPLE 1
Sketch the graph of \(y = 2 \sqrt{x + 4} - 3.\)
We begin with the parent function, \(y = \sqrt x,\) which we graphed in Example 0.4-4. (See Figure 5.) Then we stretch the graph vertically by a factor of \(2,\) attaining the graph of \(y = 2 \sqrt x\) in Figure 6A. Lastly, we shift this graph \(4\) units left and \(3\) units down to attain the final graph in Figure 6B.
EXAMPLE 2
Graph the function \(y = \abs{2x - 4} - 5.\)
The parent function is \(y = \abs x,\) which we graphed in Section 0.4. (See Figure 7.) Rewriting the function as \[y = \abs{2(x - 2)} - 5\] reveals that the parent graph must be compressed horizontally by a factor of \(2,\) and then translated \(2\) units right and \(5\) units down. Compressing \(y = \abs x\) by a factor of \(2\) yields the graph of \(y = \abs{2x}\) in Figure 8A. Finally, we shift this graph \(2\) units right and \(5\) units down to attain the graph in Figure 8B.
EXAMPLE 3
Plot the graph of \(y = \abs{x^2 - 1}.\)
The graph of \(y = x^2 - 1\) is the parabola \(y = x^2\) shifted down \(1\) unit (Figure 9). This graph lies under the \(x\)-axis for \(-1 \lt x \lt 1,\) so we reflect this portion across the \(x\)-axis to attain the graph of \(y = \abs{x^2 - 1}\) in Figure 10.

Combinations of Functions

We often combine two functions \(f\) and \(g\) to attain new functions \(f + g,\) \(f - g,\) \(fg,\) and \(f/g.\) Doing so is similar to the process for manipulating numbers. The sum and difference functions are defined by, respectively, \begin{align} (f + g)(x) &= f(x) + g(x) \cma \label{eq:func-sum} \nl (f - g)(x) &= f(x) - g(x) \pd \label{eq:func-diff} \end{align} Likewise, the product and quotient functions are defined by, respectively, \begin{align} (fg)(x) &= f(x) g(x) \cma \label{eq:func-prod} \nl \par{\frac{f}{g}} (x) &= \frac{f(x)}{g(x)} \cmaa g(x) \ne 0 \pd \label{eq:func-quot} \end{align} If the domain of \(f\) is \(D_1\) and the domain of \(g\) is \(D_2,\) then the domain of each function combination is the intersection \(D_1 \cap D_2\) because both functions must be defined. The domain of \(f/g\) has one additional restrictionâ€”namely, \(g(x) \ne 0\)â€”because division by \(0\) is impossible. Thus, the domain of \(f/g\) is \(\{x \in D_1 \cap D_2 \mid g(x) \ne 0\}.\)

COMBINING FUNCTIONS
If \(f\) and \(g\) are functions, then \begin{align} (f + g)(x) &= f(x) + g(x) \cma \eqlabel{eq:func-sum} \nl (f - g)(x) &= f(x) - g(x) \cma \eqlabel{eq:func-diff} \nl (fg)(x) &= f(x) g(x) \cma \eqlabel{eq:func-prod} \nl \par{\frac{f}{g}} (x) &= \frac{f(x)}{g(x)} \cmaa g(x) \ne 0 \pd \eqlabel{eq:func-quot} \end{align} Let the domain of \(f\) be \(D_1\) and the domain of \(g\) be \(D_2.\) The domains of \(f + g,\) \(f - g,\) and \(fg\) are \(\{x \in D_1 \cap D_2\},\) and the domain of \(f/g\) is \(\{x \in D_1 \cap D_2 \mid g(x) \ne 0\}.\)
EXAMPLE 4
If \(f(x) = x^2\) and \(g(x) = \sqrt{x + 2},\) then find
  1. \((f + g)(2)\)
  2. \(\ds \par{\frac{f}{g}}(7)\)
  3. \(\ds \par{\frac{g}{f}}(x)\) and its domain
  1. Note that \(f(2) = 2^2 = 4\) and \(g(2) = \sqrt{2 + 2} = 2.\) So \[ \ba (f + g)(2) &= f(2) + g(2) \nl &= 4 + 2 \nl &= \boxed 6 \ea \]
  2. We see \(f(7) = 49\) and \(g(7) = 3.\) Thus, \[ \ba \par{\frac{f}{g}}(7) &= \frac{f(7)}{g(7)} \nl &= \boxed{\tfrac{49}{3}} \ea \]
  3. We have \[ \ba \par{\frac{g}{f}}(x) &= \frac{g(x)}{f(x)} \nl &= \boxed{\frac{\sqrt{x + 2}}{x^2}} \ea \] The domain of \(g(x) = \sqrt{x + 2}\) is \([-2, \infty),\) and the domain of \(f(x) = x^2\) is \((-\infty, \infty).\) The intersection of these two sets is \[[-2, \infty) \cap (-\infty, \infty) = [-2, \infty) \pd\] Accordingly, the domain of \(g/f\) is \[ \ba \{x \in [-2, \infty) \mid x^2 \ne 0\} &= \{x \in [-2, \infty) \mid x \ne 0\} \nl &= \boxed{[-2, 0) \cup (0, \infty)} \ea \]

Next we examine composite functions, in which one function is inputted into another. Given two functions \(f\) and \(g,\) the composition of \(f\) and \(g\) is \begin{equation} (f \circ g)(x) = f(g(x)) \cma \label{eq:func-composite} \end{equation} read as \(f\) composed with \(g.\) Simply put, the function \(g(x)\) is substituted into the function \(f.\) We call \(g\) the inner function and \(f\) the outer function. The domain of \(f \circ g\) is all \(x\) such that \(g(x)\) and \(f(g(x))\) are both defined. The following examples show cases of composite functions.

COMPOSITE FUNCTIONS
For two functions \(f\) and \(g,\) the composite function \(f \circ g\) is defined by \begin{equation} (f \circ g)(x) = f(g(x)) \pd \eqlabel{eq:func-composite} \end{equation} The domain of \(f \circ g\) is all \(x\) for which \(g(x)\) and \(f(g(x))\) are both defined.
EXAMPLE 5
If \(f(x) = 2x - 1\) and \(g(x) = x^2,\) then find \((f \circ g)(x)\) and \((f \circ g)(3).\)
Let's rewrite \((f \circ g)(x)\) as \(f(g(x)).\) We substitute \(g(x) = x^2\) into \(f(x) = 2x - 1,\) as follows: \[ \ba f(g(x)) &= f \par{x^2} \nl &= 2 \par{x^2} - 1 \nl &= \boxed{2x^2 - 1} \ea \] Then \((f \circ g)(3)\) is \[f(g(3)) = 2(3)^2 - 1 = \boxed{17}\]
EXAMPLE 6
Given \(f(x) = \sqrt x\) and \(g(x) = x + 3,\) find each composite function and its domain.
  1. \(f \circ g\)
  2. \(g \circ f\)
  3. \(f \circ f\)
  4. \(g \circ g\)
  1. We have \[ \ba (f \circ g)(x) &= f(g(x)) \nl &= f(x + 3) \nl &= \boxed{\sqrt{x + 3}} \ea \] The domain is \(\boxed{[-3, \infty)}.\)
  2. We have \[ \ba (g \circ f)(x) &= g(f(x)) \nl &= g \par{\sqrt x} \nl &= \boxed{\sqrt x + 3} \ea \] The domain is \(\boxed{[0, \infty)}.\)
  3. Functions can be composed with each other: \[ \ba (f \circ f)(x) &= f(f(x)) \nl &= f \par{\sqrt x} \nl &= \sqrt{\sqrt{x}} \nl &= \par{x^{1/2}}^{1/2} \nl &= x^{1/4} = \boxed{\sqrt[4] x} \ea \] The domain is \(\boxed{[0, \infty)}.\)
  4. We see \[ \ba (g \circ g)(x) &= g(g(x)) \nl &= g(x + 3) \nl &= (x + 3) + 3 \nl &= \boxed{x + 6} \ea \] The domain is \(\boxed{(-\infty, \infty)}.\)

CAUTION Example 6 shows that the order of composition matters; generally, \(f \circ g \ne g \circ f.\)

EXAMPLE 7
Given \(f(x) = \sqrt x,\) \(g(x) = x^3 + x,\) and \(h(x) = 6 - x,\) find \((f \circ g \circ h)(x).\)
We have \[ \ba (f \circ g \circ h)(x) &= f(g(h(x))) \nl &= f(g(6 - x)) \nl &= f \par{(6 - x)^3 + (6 - x)} \nl &= \boxed{\sqrt{(6 - x)^3 + (6 - x)}} \ea \]

Inverse Functions

We appreciate how functions map one number \(x\) in the domain \(D\) onto another number \(f(x)\) in the range \(E.\) But how do we reverse this processâ€”going from a number in \(E\) to another in \(D \ques\)

To answer this question, we first construct a definition. If each input in the domain \(D\) is mapped to a single unique output in the range \(E,\) then the function \(f\) is called one-to-one. Formally, \(f\) satisfies \[f(x_1) \ne f(x_2) \for x_1 \ne x_2 \pd\] Simply put, \(y = f(x)\) is one-to-one if no \(y\)-value is outputted more than once. Accordingly, a horizontal line must not strike the graph more than once. This criterion is called the Horizontal Line Test, a geometric method for testing whether a function is one-to-one.

ONE-TO-ONE FUNCTIONS
A function \(f\) is one-to-one if and only if it never outputs the same value twice: \[f(x_1) \ne f(x_2) \for x_1 \ne x_2 \pd\] By the Horizontal Line Test, a function is one-to-one if and only if no horizontal line intersects its graph multiple times.

As an example, the function \(f(x) = x^2\) is not one-to-one because many horizontal lines strike its curve multiple times (Figure 11A). Conversely, \(g(x) = x^3\) is one-to-one since no horizontal line strikes its graph more than once (Figure 11B). A function that is increasing or decreasing over an interval is one-to-one over that interval. One-to-one functions enable us to discuss inverse functions, defined below.

The superscript in an inverse function does not denote a power, so \[\inv f(x) \ne \frac{1}{f(x)} \pd\] Instead, the reciprocal is denoted by \[[f(x)]^{-1} = \frac{1}{f(x)} \pd\]
INVERSE FUNCTIONS
Let \(f\) be a one-to-one function with domain \(D\) and range \(E.\) The inverse function \(\inv f\) has domain \(E\) and range \(D\) and satisfies \[\inv f(y) = x \iffArrow f(x) = y\] for any \(y\) in \(E.\)

In words, \(f\) maps \(x\) onto \(y\) and \(\inv f\) maps \(y\) back onto \(x.\) Since \(f\) is one-to-one, each output \(y\) in the range \(E\) can be routed back to uniquely one input \(x\) in the domain \(D.\) (See Figure 12.) Thus, the domain of \(\inv f\) is the range of \(f,\) and the range of \(\inv f\) is the domain of \(f.\)

As an example, suppose that a town's population as a function of time is \(P = f(t),\) where \(t\) is measured in years. If the population is \(600\) after \(2\) years, then \(f(2) = 600.\) But what if we require the time at which the population is some number? The inverse function therefore says \(\inv f(600) = 2,\) connecting the population of \(600\) back to the time, \(2\) years. In general, \(t = \inv f(P)\) gives the time at which the population is \(P,\) effectively reversing the input and output.

EXAMPLE 8
A one-to-one function \(f\) satisfies \(f(2) = 5,\) \(f(5) = 7,\) and \(f(6) = -4.\) Find \(\inv f(5),\) \(\inv f(7),\) and \(\inv f(-4).\)
The inverse function \(\inv f\) reverses the effect of \(f.\) We therefore have \[ \baat{4} &f(2) &&= 5 \, &\iffArrow \, && \inv f(5) &= \boxed 2 \nl &f(5) &&= 7 \, &\iffArrow \, && \inv f(7) &= \boxed 5 \nl &f(6) &&= -4 \, &\iffArrow \, && \inv f(-4) &= \boxed 6 \nl \eaat \] (See Figure 13.)

Because \(\inv f\) reverses the effect of \(f,\) the following cancelation equations are true: \begin{align} f \par{\inv f(x)} &= x \cma \label{eq:cancel-f} \nl \inv f \par{f(x)} &= x \pd \label{eq:cancel-f-1} \end{align} In both equations, \(x\) must be within the domains of \(f\) and \(\inv f.\) The following example reveals how this information can prove whether two functions are inverses.

EXAMPLE 9
Are \(g(x) = \sqrt[3]{x + 2}\) and \(h(x) = x^3 - 2\) inverse functions?
The two functions are inverses if and only if \(g(h(x)) = x\) and \(h(g(x)) = x.\) We see \[ \ba g(h(x)) &= g \par{x^3 - 2} \nl &= \sqrt[3]{(x^3 - 2) + 2} \nl &= \sqrt[3]{x^3} \nl &\equalsCheck x \pd \ea \] Also, \[ \ba h(g(x)) &= h \par{\sqrt[3]{x + 2}} \nl &= \par{\sqrt[3]{x + 2}}^3 - 2 \nl &= (x + 2) - 2 \nl &\equalsCheck x \pd \ea \] Thus, \(g\) and \(h\) are inverses of each other.

If \(f(a) = b,\) then \(\inv f(b) = a.\) Geometrically, the point \((a, b)\) lies on the graph of \(y = f(x),\) and the point \((b, a)\) lies on the graph of \(y = \inv f(x).\) Recall that swapping a point's \(x\)- and \(y\)-coordinates is equivalent to reflecting that point across the line \(y = x.\) Consequently, the graphs of \(f\) and \(\inv f\) are reflections of each other across the line \(y = x.\) (See Figure 14.)

With an understanding of inverse functions, we must discuss how to determine one. To find an inverse function algebraically, we execute the following procedure.

DETERMINING AN INVERSE FUNCTION ALGEBRAICALLY
To find the inverse function of \(f(x),\) use the following steps.
  1. Write \(y = f(x).\)
  2. Interchange \(x\) and \(y.\)
  3. Solve for \(y.\) The resulting equation is \(y = \inv f(x).\)
EXAMPLE 10
Find the inverse function of \(f(x) = 4x^5 - 8.\)
  1. We write \[y = 4x^5 - 8 \pd\]
  2. Swapping \(x\) and \(y\) yields \[x = 4y^5 - 8 \pd\]
  3. Solving for \(y\) in the equation in Step 2, we have \[ \ba x + 8 &= 4y^5 \nl y^5 &= \frac{x + 8}{4} \nl y &= \sqrt[5]{\frac{x + 8}{4}} \pd \ea \] Hence, the inverse function is \[\inv f(x) = \boxed{\sqrt[5]{\frac{x + 8}{4}}}\] (See Figure 15.)

Transforming Functions A function transformation modifies the graph of \(y = f(x)\) in some way, by stretching, compressing, or relocating it. A translation moves the graph vertically or horizontally. Let \(c\) be a positive constant; then the following rules of translations are applicable.

New Function Translation
\(y = f(x) + c\) Shift \(y = f(x)\) up \(c\) units
\(y = f(x) - c\) Shift \(y = f(x)\) down \(c\) units
\(y = f(x - c)\) Shift \(y = f(x)\) to the right \(c\) units
\(y = f(x + c)\) Shift \(y = f(x)\) to the left \(c\) units

If \(a \gt 1,\) then the following rules describe dilations to functions' graphs.

New Function Translation
\(y = a f(x)\) Stretch \(y = f(x)\) vertically by a factor of \(a\)
\(y = \tfrac{1}{a} f(x)\) Compress \(y = f(x)\) vertically by a factor of \(a\)
\(y = f(ax)\) Compress \(y = f(x)\) horizontally by a factor of \(a\)
\(y = f(x/a)\) Stretch \(y = f(x)\) horizontally by a factor of \(a\)

The following rules describe reflections of graphs across axes.

New Function Translation
\(y = -f(x)\) Reflect \(y = f(x)\) across the \(x\)-axis
\(y = f(-x)\) Reflect \(y = f(x)\) across the \(y\)-axis

Combinations of Functions If \(f\) and \(g\) are functions, then combinations of these functions are \begin{align} (f + g)(x) &= f(x) + g(x) \cma \eqlabel{eq:func-sum} \nl (f - g)(x) &= f(x) - g(x) \cma \eqlabel{eq:func-diff} \nl (fg)(x) &= f(x) g(x) \cma \eqlabel{eq:func-prod} \nl \par{\frac{f}{g}} (x) &= \frac{f(x)}{g(x)} \cmaa g(x) \ne 0 \pd \eqlabel{eq:func-quot} \end{align} Suppose that the domain of \(f\) is \(D_1\) and the domain of \(g\) is \(D_2.\) The domains of \(f + g,\) \(f - g,\) and \(fg\) are \(\{x \in D_1 \cap D_2\},\) and the domain of \(f/g\) is \(\{x \in D_1 \cap D_2 \mid g(x) \ne 0\}.\) A composite function involves one function inputted into another. For two functions \(f\) and \(g,\) the composite function \(f \circ g\) is defined by \begin{equation} (f \circ g)(x) = f(g(x)) \pd \eqlabel{eq:func-composite} \end{equation} The domain of \(f \circ g\) is all \(x\) for which \(g(x)\) and \(f(g(x))\) are both defined. The order of composition cannot be swapped; namely, \(f \circ g \ne g \circ f.\)

Inverse Functions A function \(f\) is one-to-one if and only if it never outputs the same value twice: \[f(x_1) \ne f(x_2) \for x_1 \ne x_2 \pd\] By the Horizontal Line Test, a function is one-to-one if and only if no horizontal line intersects its graph multiple times. An increasing or decreasing function is one-to-one. If \(f\) is a one-to-one function with domain \(D\) and range \(E,\) then the inverse function \(\inv f\) has domain \(E\) and range \(D\) and satisfies \[\inv f(y) = x \iffArrow f(x) = y\] for any \(y\) in \(E.\) The inverse function \(\inv f\) maps \(y\) back onto \(x,\) thus canceling the effect of \(f.\) As a result, the following cancelation equations are true: \begin{align} f \par{\inv f(x)} &= x \cma \eqlabel{eq:cancel-f} \nl \inv f \par{f(x)} &= x \cma \eqlabel{eq:cancel-f-1} \end{align} where \(x\) is within the domain of \(f.\) Note that \(\inv f(x) \ne 1/f(x).\) The graph of \(y = f(x)\) and \(y = \inv f(x)\) are reflections of each other across the line \(y = x.\) To find the inverse function of \(f(x)\) algebraically, we perform the following steps.

  1. Write \(y = f(x).\)
  2. Interchange \(x\) and \(y.\)
  3. Solve for \(y.\) The resulting equation is \(y = \inv f(x).\)
