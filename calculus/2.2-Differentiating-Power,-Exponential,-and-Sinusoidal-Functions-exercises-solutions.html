2.2 Exercise Solutions: Differentiating Power, Exponential, and Sinusoidal Functions

EXERCISE 1
What is the Power Rule?
SOLUTION The Power Rule enables us to differentiate power functions, in which the base is variable and the exponent is constant. If \(n\) is any integer, then the Power Rule gives \[\deriv{}{x} \par{x^n} = nx^{n - 1} \pd\]
EXERCISE 2
How do we write equations of tangent lines and normal lines?
SOLUTION

Tangent Lines We write an equation for a tangent line to \(f\) at \(a\) by using point-slope form: Since the tangent line passes the point \((a, f(a))\) with slope \(f'(a),\) its equation is given by \[y - f(a) = f'(a)(x - a) \pd\]

Normal Lines A normal line intersects the graph of \(f\) at an angle of \(90 \degree.\) Thus, a normal line is perpendicular to a tangent line. So its slope is the negative reciprocal of the tangent line's slopeâ€”namely, \(-1/f'(a).\) The equation for a normal line is similar to the equation for a tangent line, but we replace \(f'(a)\) with \(-1/f'(a) \col\) \[y - f(a) = -\frac{1}{f'(a)}(x - a) \pd\]

EXERCISE 12
For
\[\ds \deriv{}{x} \par{x^2 - 4x + 12}\]
evaluate the derivative.
SOLUTION To differentiate this polynomial, we use a combination of the Sum, Difference, and Constant Multiple Rules for Derivatives. The Sum and Difference Rules for Derivatives permit us to split the derivative into \[\deriv{}{x} \par{x^2} - \deriv{}{x} \par{4x} + \deriv{}{x} \par{12} \pd\] The Constant Rule for Derivatives asserts that the derivative of a constant is \(0.\) So we have \[\deriv{}{x} \par{x^2} - \deriv{}{x} \par{4x} \pd\] The Constant Multiple Rule for Derivatives enables us to write \[\deriv{}{x} \par{x^2} - 4 \deriv{}{x} \par{x} \pd\] Now applying the Power Rule gives the derivative of \(x^2 - 4x + 12\) to be \[\par{2x^{2 - 1}} - \par{4 \cdot 1 x^{1 - 1}} = \boxed{2x - 4}\]

EXERCISE 14
For
\[\ds \deriv{}{t} \par{t^3 + 8t^2 - t}\]
evaluate the derivative.

EXERCISE 15

\[\ds \deriv{}{x} \par{\tfrac{1}{2} x^4 + 4x^{5/2} - 7}\]

We find \[ \ba \deriv{}{x} \par{\tfrac{1}{2} x^4 + 4x^{5/2} - 7} &= \deriv{}{x} \par{\tfrac{1}{2} x^4} + \deriv{}{x} \par{4x^{5/2}} - \deriv{}{t} (7) && (\textrm{Sum and Difference Rules}) \nl &= \deriv{}{x} \par{\tfrac{1}{2} x^4} + \deriv{}{x} \par{4x^{5/2}} - 0 && (\textrm{Constant Rule}) \nl &= \tfrac{1}{2} \deriv{}{x} \par{x^4} + 4 \deriv{}{x} \par{x^{5/2}} && (\textrm{Constant Multiple Rule}) \nl &= \tfrac{1}{2} \par{4x^3} + 4 \par{\tfrac{5}{2} x^{3/2}} && (\textrm{Power Rule}) \nl &= \boxed{2x^3 + 10 \sqrt{x^3}} \ea \]
EXERCISE 16
For
\[\ds \deriv{}{x} \parbr{(2x + 1)^2}\]

EXERCISE 25
Consider the family of cubic functions whose coefficients are \(a,\) \(b,\) \(c,\) and \(d \col\) \[f(x) = ax^3 + bx^2 + cx + d \pd\] Determine \(f'(x)\) and \(f''(x).\)
SOLUTION The coefficients \(a,\) \(b,\) \(c,\) and \(d\) are simply constants. Thus, with the Power Rule we see \[ \ba f'(x) &= a \deriv{}{x} \par{x^3} + b \deriv{}{x} \par{x^2} + c \deriv{}{x} \par{x} + \deriv{}{x} \par{d} \nl &= a \par{3x^2} + b \par{2x} + c(1) + 0 \nl &= \boxed{3ax^2 + 2bx + c} \ea \] Differentiating again, we find \[ \ba f''(x) &= 3a \deriv{}{x} \par{x^2} + 2b \deriv{}{x} \par{x} + \deriv{}{x} \par{c} \nl &= 3a(2x) + 2b(1) + 0 \nl &= \boxed{6ax + 2b} \ea \]
EXERCISE 26
For nonzero constants \(p\) and \(q,\) let \(f(x) = q x^{p/q}.\) Show that \[f'(x) = \frac{p}{qx} \, f(x) \pd\]
SOLUTION Applying the Constant Rule and Power Rule, we see \[ \ba f'(x) &= q \deriv{}{x} \par{x^{p/q}} \nl &= q \par{\frac{p}{q} x^{p/q - 1}} \nl &= p x^{p/q - 1} \pd \ea \] Through algebraic manipulation, we see \[ \ba p x^{p/q - 1} &= p x^{p/q} \cdot x^{-1} \nl &= p x^{p/q} \cdot \frac{q}{q} \cdot x^{-1} \nl &= q x^{p/q} \cdot \frac{p}{qx} \nl &= f(x) \cdot \frac{p}{qx} \cma \ea \] as requested.
EXERCISE 27
Find the \(x\)-values at which \(y = x - 2 \sqrt x\) has horizontal tangents.
SOLUTION The graph has horizontal tangents when \(\textderiv{y}{x} = 0.\) Rewriting \(2 \sqrt x\) as \(2x^{1/2},\) we find \[ \ba \deriv{y}{x} &= \deriv{}{x} (x) - 2 \deriv{}{x} \par{x^{1/2}} \nl &= 1 - 2 \par{\frac{1}{2 \sqrt x}} \nl &= 1 - \frac{1}{\sqrt x} \pd \ea \] We see \[1 - \frac{1}{\sqrt x} = 0 \implies \sqrt x = 1 \implies \boxed{x = 1}\]
EXERCISE 28
Determine the \(x\)-values in \([0, 2 \pi]\) at which the graph of \(f(x) = 2x + 4 \sin x\) has horizontal tangents.
SOLUTION The graph of \(f\) has a horizontal tangent at some \(x\) if \(f'(x) = 0.\) We find \[ \ba f'(x) &= \deriv{}{x} (2x) + \deriv{}{x} \par{4 \sin x} \nl &= 2 + 4 \cos x \pd \ea \] So \[ \ba 2 + 4 \cos x &= 0 \nl \cos x &= -\tfrac{1}{2} \nl \implies x &= \boxed{\tfrac{2 \pi}{3}} \cma \boxed{\tfrac{4 \pi}{3}} \ea \]
EXERCISE 29
Let \(g(x) = x - e^x.\) Show that the graph of \(y = g(x)\) has a horizontal tangent when it intersects the \(y\)-axis.
SOLUTION The graph of \(g\) has a horizontal tangent when \(g'(x) = 0.\) We find \[ \ba g'(x) &= \deriv{}{x} (x) - \deriv{}{x} \par{e^x} \nl &= 1 - e^x \pd \ea \] So we see \[1 - e^x = 0 \implies x = 0 \pd\] Thus, \(g\) has a horizontal tangent when \(x = 0,\) the location of the \(y\)-axis.
EXERCISE 30
Determine the \(78\)th derivative of \(f(x) = \sin x.\)
SOLUTION
The notation \(f^{(n)}(x)\) means the \(n\)th derivative of \(f(x).\)
If we write out a few derivatives, then we see \[ \ba f'(x) &= \cos x \cma \nl f''(x) &= -\sin x \cma \nl f'''(x) &= - \cos x \cma \nl f^{(4)}(x) &= \sin x \pd \ea \] This loop repeats every four derivatives. The \(76\)th derivative (a number divisible by \(4\)) of \(\sin x\) is therefore \(\sin x,\) so we find \[ \ba f^{(76)}(x) &= \sin x \cma \nl f^{(77)}(x) &= \cos x \cma \nl f^{(78)}(x) &= \boxed{-\sin x} \nl \ea \]
EXERCISE 31
Find the \(23\)rd derivative of \(f(x) = \cos x.\)
SOLUTION
The notation \(f^{(n)}(x)\) means the \(n\)th derivative of \(f(x).\)
Writing out a few derivatives, we see \[ \ba f'(x) &= -\sin x \cma \nl f''(x) &= - \cos x \cma \nl f'''(x) &= \sin x \cma \nl f^{(4)}(x) &= \cos x \pd \ea \] Every fourth derivative of \(\cos x\) is \(\cos x.\) The \(20\)th derivative (a number divisible by \(4\)) of \(\cos x\) is therefore \(\cos x,\) so we find \[ \ba f^{(20)}(x) &= \cos x \cma \nl f^{(21)}(x) &= - \sin x \cma \nl f^{(22)}(x) &= - \cos x \cma \nl f^{(23)}(x) &= \boxed{\sin x} \nl \ea \]
EXERCISE 32
Write an equation of the line tangent to the graph of \(y = x^2 + 5x - 3\) when \(x = 1.\)
SOLUTION To construct an equation for the tangent line, we use point-slope form with the point of tangency and the slope. When \(x = 1,\) we see \[y = (1)^2 + 5(1) - 3 = 3 \pd\] So the point of tangency is \((1, 3).\) The derivative of the function is given by \[ \ba \deriv{}{x} \par{x^2 + 5x - 3} &= \deriv{}{x} \par{x^2} + 5 \deriv{}{x} (x) - \deriv{}{x} (3) \nl &= 2x + 5 \pd \ea \] The slope to the curve when \(x = 1\) is therefore \[2(1) + 5 = 7 \pd\] Thus, an equation of the tangent line is \[y - 3 = 7(x - 1) \or \boxed{y = 7x - 4}\]
EXERCISE 33
Let \(f(x) = 3 + 2 \sin x.\)
  1. Write an equation of the line tangent to \(y = f(x)\) at \(x = \pi/4.\)
  2. Determine an equation of the line normal to \(y = f(x)\) at \(x = \pi/4.\)
SOLUTION
  1. We see \[f \par{\tfrac{\pi}{4}} = 3 + 2 \sin \tfrac{\pi}{4} = 3 + \sqrt 2 \pd\] So the point of tangency is \(\par{\pi/4 \cma 3 + \sqrt 2}.\) At this point, the slope to \(f\) is \[ \ba f' \par{\tfrac{\pi}{4}} &= \deriv{}{x} \par{3 + 2 \sin x} \intEval_{x = \pi/4} \nl &= 2 \cos \tfrac{\pi}{4} \nl &= \sqrt 2 \pd \ea \] Thus, an equation of the tangent line is \[\boxed{y - \par{3 + \sqrt 2} = \sqrt 2 \, \par{x - \tfrac{\pi}{4}}}\]
  2. The normal line is perpendicular to the tangent line. The slope of the normal line is the negative reciprocal of the tangent line's slopeâ€”that is, \(-1/\sqrt 2.\) So an equation for the normal line is \[\boxed{y - \par{3 + \sqrt 2} = -\tfrac{1}{\sqrt 2} \, \par{x - \tfrac{\pi}{4}}}\]
EXERCISE 34
A line is normal to the curve \(y = e^x\) at \(x = a.\) In terms of \(a,\) find the point at which the normal line touches the \(x\)-axis.
SOLUTION The derivative of \(e^x\) is \(e^x,\) so the slope to the curve at \(x = a\) is \(e^a.\) Since the normal line is perpendicular to the tangent line, its slope is the negative reciprocal of the tangent slopeâ€”that is, \(-1/e^a.\) The point of tangency is \((a, e^a),\) so an equation of the normal line is \[y - e^a = -\frac{1}{e^a} \par{x - a} \pd\] The line strikes the \(x\)-axis when \(y = 0,\) from which we find \[0 - e^a = -\frac{1}{e^a} \par{x - a} \implies x = \boxed{e^{2a} + a}\]
EXERCISE 35
Find the point on the curve \(y = e^x\) to which the tangent line is parallel to the line \(y = 2x.\)
SOLUTION Since the tangent line is parallel to the line \(y = 2x,\) its slope must be \(2.\) Hence, the derivative of \(e^x,\) which is \(e^x,\) at some point must be \(2.\) Solving gives \[e^x = 2 \implies x = \ln 2 \pd\] At this point, \(y = e^{\ln 2} = 2.\) So the tangent line at \(\boxed{(\ln 2, 2)}\) is parallel to the line \(y = 2x.\)
EXERCISE 36
Suppose that an object moves back and forth along the \(x\)-axis with time \(t \geq 0\) as modeled by the position function \(x(t) = 9 \sin t.\)
  1. Determine the object's velocity function \(v(t)\) and acceleration function \(a(t).\)
  2. For \(0 \leq t \leq \pi,\) when is the object's acceleration equal to \(0 \ques\)
  3. When \(t = \pi/3,\) is the object speeding up or slowing down?
SOLUTION
\(v(t) = x'(t)\) and \(a(t) = v'(t)\) \(= x''(t).\)
  1. Velocity is the time derivative of position, so we find \[ \ba v(t) &= x'(t) \nl &= 9 \deriv{}{t} (\sin t) \nl &= \boxed{9 \cos t} \ea \] Acceleration is the time derivative of velocity; thus, we have \[ \ba a(t) &= v'(t) \nl &= \deriv{}{t} (9 \cos t) \nl &= \boxed{-9 \sin t} \ea \]
  2. Equating \(a(t) = 0\) shows \[-9 \sin t = 0 \implies t = \boxed{0} \cma \boxed{\pi}\]
  3. The object is speeding up if and only if its velocity and acceleration have the same sign; it is slowing down if and only if velocity and acceleration have different signs. When \(t = \pi/3,\) we see \[ \ba v \par{\tfrac{\pi}{3}} &= 9 \cos \tfrac{\pi}{3} \gt 0 \cma \nl a \par{\tfrac{\pi}{3}} &= -9 \sin \tfrac{\pi}{3} \lt 0 \pd \ea \] Since \(v\) and \(a\) have opposite signs, the object is slowing down when \(t = \pi/3.\)
EXERCISE 37
A particle moves along a straight line with time \(t \geq 0\) as modeled by the position function \(s(t)\) \(= -t + 2 \sin t.\)
  1. Find the velocity function \(v(t)\) and acceleration function \(a(t).\)
  2. What is the first time \(t\) at which the object is stationary?
  3. When \(t = \pi/4,\) is the object speeding up or slowing down?
SOLUTION
\(v(t) = s'(t)\) and \(a(t) = v'(t)\) \(= s''(t).\)
  1. Velocity is the time derivative of position, so we find \[ \ba v(t) &= s'(t) \nl &= -\deriv{}{t} (t) + 2 \deriv{}{t} (\sin t) \nl &= \boxed{-1 + 2 \cos t} \ea \] Acceleration is the time derivative of velocity; thus, we have \[ \ba a(t) &= v'(t) \nl &= \deriv{}{t} (-1 + 2 \cos t) \nl &= \deriv{}{t} (-1) + 2 \deriv{}{t} (\cos t) \nl &= \boxed{-2 \sin t} \ea \]
  2. The object is stationary when \(v = 0,\) so we find \[ \ba -1 + 2 \cos t &= 0 \nl \cos t &= \tfrac{1}{2} \nl \implies t &= \boxed{\tfrac{\pi}{3}} \ea \]
  3. A particle is speeding up if and only if its velocity and acceleration have the same sign; it is slowing down if and only if velocity and acceleration have different signs. When \(t = \pi/4,\) we see \[ \ba v \par{\tfrac{\pi}{4}} &= -1 + \cos \tfrac{\pi}{4} \gt 0 \cma \nl a \par{\tfrac{\pi}{4}} &= -2 \sin \tfrac{\pi}{4} \lt 0 \pd \ea \] Since \(v\) and \(a\) have opposite signs, the object is slowing down when \(t = \pi/4.\)
EXERCISE 38
A heavy object attached to a spring is extended a distance \(A\) from its equilibrium position (Figure 8). The object's vibration with time \(t\) is modeled by the position function \[s(t) = A \cos t \pd\]
  1. Determine the block's velocity function \(v(t)\) and acceleration function \(a(t).\)
  2. What is the earliest time when the block's speed equal to \(A/2 \ques\)
  3. When is the magnitude of the block's acceleration the greatest?
SOLUTION
From Section 2.1, \(v(t) = s'(t)\) and \(a(t) = v'(t)\) \(= s''(t).\)
  1. The velocity function \(v(t)\) is \(s'(t),\) or \[\deriv{}{t} \par{A \cos t} = A \deriv{}{t} \par{\cos t} = \boxed{-A \sin t}\] The acceleration function \(a(t)\) is given by \(v'(t) \col\) \[\deriv{}{t} \par{-A \sin t} = -A \deriv{}{t} \par{\sin t} = \boxed{-A \cos t}\]
  2. The block's speed function is \(\abs{v}.\) So we seek the solutions to \[\abs{v(t)} = A \abs{\sin t} = \frac{A}{2} \pd\] We see \[\abs{\sin t} = \tfrac{1}{2} \implies \boxed{t = \tfrac{\pi}{6}}\]
  3. Since \(s(t) = -a(t),\) the block's acceleration has greatest magnitude when its displacement from equilibrium is greatestâ€”that is, at the far-left and far-right positions.
EXERCISE 39
By recognizing the limit as a disguised derivative of a power function, evaluate \[\lim_{h \to 0} \frac{\dfrac{1}{(2 + h)^8} - \dfrac{1}{2^8}}{h} \pd\]
SOLUTION The limit definition of a derivative asserts that \[f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \pd\] (See Section 2.1.) We therefore see \(f(x) = 1/x^8 = x^{-8}.\) So this limit represents the derivative of \(x^{-8}\) evaluated at \(x = 2.\) The Power Rule gives \[\deriv{}{x} \par{x^{-8}} = -8 x^{-9} = -\frac{8}{x^9} \pd\] So the limit is \[-\frac{8}{2^9} = \boxed{-\frac{1}{64}}\]
EXERCISE 40

EXERCISE 41
The piecewise function \(f\) is defined by \[f(x) = \bc 2x^2 + 3x &x \lt 1 \nl x^7 + 4 &x \geq 1 \pd \ec \] Is \(f(x)\) differentiable at \(x = 1 \ques\)
SOLUTION The function \(f\) is differentiable at \(1\) if and only if \(f\) is continuous at \(1\) and the one-sided derivatives of \(f\) at \(1\) are equal. We see \(f\) is continuous at \(1\) because \(\lim_{x \to 1} f(x)\) \(= f(1) = 5 \col\) \[ \baf &&\lim_{x \to 1^-} f(x) &= \lim_{x \to 1^+} f(x) &\nl \laWord{or} && 2(1)^2 + 3(1) &= (1)^7 + 4 &\nl && &= 5 \pd \eaf \] (See Section 1.4 to review the definition of continuity.) The left-hand derivative of \(f\) at \(1\) is \[\deriv{}{x} \par{2x^2 + 3x} \intEval_{x = 1} = 4(1) + 3 = 7 \pd\] The right-hand derivative is \[\deriv{}{x} \par{x^7 + 4} \intEval_{x = 1} = 7(1)^6 = 7 \pd\] Since both one-sided derivatives are equal, \(f\) is differentiable at \(1.\)
EXERCISE 42
Let \[g(t) = \bc t^3 + t + 1 &t \lt 0 \nl e^t - 2 &t \geq 0 \pd \ec \] Is \(g(t)\) differentiable at \(t = 0 \ques\)
SOLUTION
Continuity is a requirement for differentiability but not a sufficient condition.
The function \(g\) is differentiable at \(0\) if and only if \(g\) is continuous at \(0\) and the one-sided derivatives of \(g\) at \(0\) are equal. We see \[ \baf &&\lim_{t \to 0^-} g(t) &= (0)^3 + (0) + 1 = 1 &\nl \laWord{and} && \lim_{t \to 0^+} g(t) &= e^0 - 2 = -1 \pd \eaf \] Since these one-sided limits aren't equal, \(\lim_{t \to 0} g(t)\) does not exist. Thus, \(g\) is discontinuous at \(0\) and so \(g\) cannot be differentiable at \(0.\)
EXERCISE 43
The function \(f(x) = ax^3 + 2x^2 - 6x + 1\) has a horizontal tangent when \(x = 2.\) Find the value of \(a.\)
SOLUTION The graph of \(f\) has a horizontal tangent at \(2\) if and only if \(f'(2) = 0.\) Differentiating \(f,\) we find \[f'(x) = 3ax^2 + 4x - 6 \pd\] Setting \(f'(2) = 0\) shows \[3a(2)^2 + 4(2) - 6 = 0 \implies 12a = -2 \cma\] whose solution is \(\boxed{a = -1/6}.\)

Calculate the values of \(a\) and \(b\) such that the line \(y = bx - 3\) is tangent to the parabola \(f(x) = ax^2 + 1\) when \(x = 2.\)
SOLUTION Because the line \(y = bx - 3\) is tangent to the parabola \(f(x) = ax^2 + 1\) at \(x = 2,\) the two graphs must intersect when \(x = 2\) with the same slope, \(f'(2).\) Substituting \(x = 2\) into each function and equating them yield \begin{flalign} &&2b - 3 &= 4a + 1 \nonumber &\nl \laWord{or} &&4a - 2b &= -4 \pd \label{eq:a-b-eq} \end{flalign} Observe that \[\deriv{}{x} (bx - 3) = b \and \deriv{}{x} \par{ax^2 + 1} = 2ax \pd\] Equating the derivatives at \(x = 2,\) we find \[b = 2a(2) \implies b = 4a \pd\] Hence, \(\eqref{eq:a-b-eq}\) shows \[(b) - 2b = -4 \implies b = 4 \pd\] So \(\boxed{a = 1}\) and \(\boxed{b = 4}.\) A link to an interactive graph in which you can toggle \(a\) and \(b\) can be found at https://www.desmos.com/calculator/meyjqazuig?lang=en.

EXERCISE 46

SOLUTION The slope to \(y = x^2\) when \(x = a\) is given by \[\deriv{}{x} \par{x^2} \intEval_{x = a} = 2a \pd\] Therefore, an equation of the tangent line to the parabola at \(x = a\) is given by \[y - a^2 = 2a(x - a) \or y = 2ax - a^2 \pd\] Since this line intersects the other line \(y = x - 6,\) we equate these functions to discover \[ \ba 2ax - a^2 &= x - 6 \nl x(2a - 1) &= a^2 - 6 \nl \implies x &= \frac{a^2 - 6}{2a - 1} \pd \ea \] Substituting this expression into the function \(y = x - 6\) shows \[y = \frac{a^2 - 6}{2a - 1} - 6 = \frac{a^2 - 6 - 6(2a - 1)}{2a - 1} = \frac{a^2 - 12a}{2a - 1} \pd\] Thus, these two lines intersect at the given point for \(a \ne 1/2.\) (In fact, for \(a = 1/2\) the lines are parallel to each other.)
EXERCISE 47
Let \(f(x) = x^2\) and \(g(x) = -x^2 - C.\) Find the value of \(C\) such that the line tangent to \(f\) at \(x = 2\) is also tangent to \(g.\)
SOLUTION The word tangent means that the line intersects \(g(x)\) at \(a\) with slope \(g'(a).\) We anticipate that the tangent line to \(f\) at \(x = 2\) is also tangent to \(g\) in the third quadrant. So we are considering only one tangent line. We see \(f(2) = 4\) and \(f'(2) = 4,\) so an equation of this tangent line is \[y - 4 = 4(x - 2) \or y = 4x - 4 \pd\] This line has slope \(4\) and must be tangent to \(g(x) = -x^2 - C,\) whose derivative is \(g'(x) = -2x.\) Equating slopes shows \[-2x = 4 \implies x = -2 \pd\] Thus, the line \(y = 4x - 4\) is tangent to \(f\) at \(x = 2\) and tangent to \(g\) at \(x = -2.\) The word tangent means that this line also intersects \(g(x)\) at \(x = -2.\) When \(x = -2,\) the line's \(y\)-value is \[4(-2) - 4 = -12 \cma\] so \(g(-2) = -12.\) Thus, we see \[g(-2) = -(-2)^2 - C = -12 \implies \boxed{C = 8}\]
EXERCISE 48
Find the value of \(b\) such that the parabola \(y = x^2 + b\) is tangent to \(y = \abs x.\)
SOLUTION The word tangent means that the graphs of \(y = x^2 + b\) and \(y = \abs x\) intersect each other with the same slope. The graph of \(\abs x\) has slope \(-1\) for \(x \lt 0\) and slope \(1\) for \(x \gt 0.\) The derivative function for the parabola is \[\deriv{}{x} \par{x^2 + b} = 2x \pd\] We equate this expression to \(1\) and \(-1\) to find \[ \baat{2} 2x &= 1 \lspace 2x &&= -1 \nl x &= \tfrac{1}{2} \scol \lspace x &&= -\tfrac{1}{2} \pd \eaat \] At each value of \(x,\) \(\abs x = 1/2.\) Thus, the parabola must also reach the \(y\)-value \(1/2\) when \(x = 1/2\) and \(x = -1/2 \col\) \[y = \par{\pm \tfrac{1}{2}}^2 + b = \tfrac{1}{2} \implies \boxed{b = \tfrac{1}{4}}\]
EXERCISE 49
For any real number \(c,\) use the limit definition of a derivative to prove that \[\deriv{}{x} \sin(x + c) = \cos(x + c) \pd\]
SOLUTION This proof is nearly identical to the proof discussed in the section. By the limit definition of a derivative, \[\deriv{}{x} \sin(x + c) = \lim_{h \to 0} \frac{\sin(x + c + h) - \sin(x + c)}{h} \pd\] We now use the double-angle identity for sine, treating \((x + c)\) as one term. Doing so gives \[ \ba \deriv{}{x} \sin(x + c) &= \lim_{h \to 0} \frac{\sin(x + c) \cos h + \cos(x + c) \sin h - \sin(x + c)}{h} \nl &= \lim_{h \to 0} \frac{\sin(x + c) (\cos h - 1)}{h} + \lim_{h \to 0} \frac{\sin h \cos(x + c)}{h} \nl &= \sin(x + c) \lim_{h \to 0} \frac{\cos h - 1}{h} + \cos(x + c) \lim_{h \to 0} \frac{\sin h}{h} \pd \ea \] Recall that as \(h \to 0,\) \((\cos h - 1)/h \to 0\) and \(\sin h/h \to 1.\) Thus, we see \[\deriv{}{x} \sin(x + c) = \sin(x + c) \cdot 0 + \cos(x + c) \cdot 1 = \cos(x + c) \cma\] as requested.
EXERCISE 50
Let \(R\) be the region in the first quadrant bounded between the coordinate axes and the line tangent to the curve \(y = 1/x\) at any point \((a, 1/a).\) Calculate the area of \(R\) and justify that it is independent of \(a.\)
SOLUTION Region \(R\) is a triangle whose hypotenuse is the tangent line. The derivative of \(1/x\) is, by the Power Rule, \[-x^{-2} = -\frac{1}{x^2} \pd\] Thus, the slope to the curve \(1/x\) at \(x = a\) is \(-1/a^2.\) The point of tangency is \((a, 1/a),\) so an equation of the tangent line is \[y - \frac{1}{a} = -\frac{1}{a^2}(x - a) \or y = -\frac{x}{a^2} + \frac{2}{a} \pd\] To find the triangle's base \(b,\) we set \(y = 0 \col\) \[0 = -\frac{b}{a^2} + \frac{2}{a} \implies b = 2a \pd\] The triangle's height \(h\) is found by letting \(x = 0 \col\) \[h = -\frac{0}{a^2} + \frac{2}{a} \implies h = \frac{2}{a} \pd\] Hence, the triangle's area is \[\tfrac{1}{2} bh = \tfrac{1}{2}(2a) \par{\tfrac{2}{a}} = \boxed{2}\] It is surprising that the triangle's area is always \(2,\) regardless of the value of \(a.\)
