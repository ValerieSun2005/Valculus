7.6 Exercise Solutions: Probability

EXERCISE 1
What is a probability density function for a continuous random variable?
SOLUTION A probability density function for a continuous random variable \(X\) is a function under which the area gives the probability that \(X\) lies between two numbers. Mathematically, \[P(a \leq X \leq b) = \int_a^b f(x) \di x \pd\] The function \(f\) is a probability density function if and only if and \(f(x) \geq 0\) and \(\int_{-\infty}^\infty f(x) \di x = 1.\)
EXERCISE 2
If \(f\) is a probability density function, then how do we find the mean of \(f \ques\)
SOLUTION The mean of a probability density function is given by \[\mu = \int_{-\infty}^\infty x f(x) \di x \pd\]

SOLUTION Recall that we interest ourselves in the area under a continuous probability density function; we have little use for the actual value the function outputs.

The requested probability \(P(0 \leq X \leq 8)\) is calculated by determining the area under the graph of \(y = f(x)\) from \(x = 0\) to \(x = 8;\) that is, \[P(0 \leq X \leq 8) = \int_0^8 f(x) \di x = \tfrac{1}{2}(0.2)(8) = \boxed{0.8}\] So under the distribution \(y = f(x),\) if we assign any random number to \(X,\) then there is an \(80 \%\) chance that the value of \(X\) is between \(0\) and \(8.\)
  3. Observe that \[ \ba P(X \leq 0) &= \int_{-\infty}^0 f(x) \di x = \int_{-2}^0 f(x) \di x \nl &= \tfrac{1}{2}(2)(0.2) = \boxed{0.2} \ea \] Hence, under the distribution \(y = f(x)\) the random variable \(X\) has a \(20\%\) chance of having a value less than \(0.\)

For
\[\ds f(x) = \bc \cos x & 0 \leq x \leq \frac{\pi}{2} \nl 0 & \text{otherwise} \ec \]
determine whether the function is a probability density function.
SOLUTION Yes, \(f(x)\) is a probability density function. The function \(f\) is a probability density function if and only if \(f(x) \geq 0\) and \(\int_{-\infty}^\infty f(x) \di x = 1.\) The function \(f\) is never negative. For
\[\ds f(x) = \bc e^{2x} & 0 \leq x \leq c \nl 0 & \text{otherwise} \ec \]
determine the value of \(c\) such that the function is a probability density function.
SOLUTION Observe that the function \(f(x)\) is never negative. We need the value \(c\) such that \(\int_{-\infty}^\infty f(x) \di x = 1.\) Because \(f(x) = 0\) for all \(x\) outside of \([0, c],\) the improper integral becomes \[ \ba \int_0^c e^{2x} \di x &= \tfrac{1}{2} e^{2x} \intEval_0^c \nl &= \tfrac{1}{2} \par{e^{2c} - 1} \pd \ea \] Equating this expression to \(1\) shows \[ \ba \tfrac{1}{2} \par{e^{2c} - 1} &= 1 \nl e^{2c} &= 3 \nl \implies c &= \boxed{\tfrac{1}{2} \ln 3} \ea \]

The value of \(c\) must satisfy \(\int_{-\infty}^\infty f(x) \di x = 1.\) Because \(f(x) = 0\) for all \(x\) outside of \([2, 4],\) the improper integral is equivalent to \[ \ba \int_2^4 cx \di x &= \tfrac{1}{2} cx^2 \intEval_2^4 \nl &= \tfrac{1}{2} \parbr{16c - 4c} \nl &= 6c \pd \ea \] Equating this expression to \(1\) gives \[ \ba 6c &= 1 \nl \implies c &= \boxed{\tfrac{1}{6}} \ea \]

EXERCISE 17
An store employee helps \(75\%\) of customers standing in line within the first minute. Using an exponential probability density function, calculate the probability that a customer is helped within \(3\) minutes.
SOLUTION We model the wait times using an exponential probability density function: \[ f(t) = \bc 0 & t \lt 0 \nl ke^{-kt} & t \geq 0 \cma \ec \] where \(t\) is time in minutes. We want to solve for \(k.\) Since \(75\%\) of customers are helped at the checkout counter within the first minute, we have \(\int_0^1 f(t) \di t = 0.75.\) We see \[ \ba \int_0^1 ke^{-kt} \di t &= 0.75 \nl \frac{k}{-k} e^{-kt} \intEval_0^1 &= 0.75 \nl e^0 - e^{-k} &= 0.75 \nl \implies k &= -\ln 0.25 \approx 1.386 \pd \ea \] Hence, the exponential probability density function is \[ f(t) = \bc 0 & t \lt 0 \nl 1.386e^{-1.386t} & t \geq 0 \pd \ec \] The probability that a customer is helped within \(3\) minutes is therefore \[ \ba P(0 \leq T \leq 3) &= \int_0^3 1.386e^{-1.386t} \di t \nl &= \frac{1.386}{-1.386} e^{-1.386t} \intEval_0^3 \nl &= e^{0} - e^{-1.386(3)} \nl &\approx \boxed{0.984} = 98.4 \% \pd \ea \]
EXERCISE 18
At a call center, the average waiting time for customers is \(3\) minutes. Using an exponential probability density function, calculate the probability that a customer is helped within \(5\) minutes.
SOLUTION We model the wait times using an exponential probability density function: \[ f(t) = \bc 0 & t \lt 0 \nl ke^{-kt} & t \geq 0 \cma \ec \] where \(t\) is time in minutes. We are given the mean of this function to be \(\mu = 3.\) Recall that the mean is \(\mu = 1/k,\) so we have \[ f(t) = \bc 0 & t \lt 0 \nl \dfrac{e^{-t/3}}{3} & t \geq 0 \pd \ec \] Accordingly, the probability that a customer is helped within \(5\) minutes is \[ \ba P(0 \leq T \leq 5) &= \int_0^5 \frac{e^{-t/3}}{3} \di t \nl &= - e^{-t/3} \intEval_0^5 \nl &= - \par{e^{-5/3} - e^0} \nl &\approx \boxed{0.811} = 81.1 \% \pd \ea \]
EXERCISE 19
In a call center, the manager plans to hire additional support agents if calls are answered in an average time of \(5\) minutes. If \(40\%\) of calls are answered \(3\) minutes, and an exponential probability density function models the time a customer waits, then should the manager hire more staff?
SOLUTION We model the wait times using an exponential probability density function: \[ f(t) = \bc 0 & t \lt 0 \nl ke^{-kt} & t \geq 0 \cma \ec \] where \(t\) is the waiting time in minutes. Since \(40\%\) of calls are answered within \(3\) minutes, we have \(\int_0^3 f(t) \di t = 0.4.\) We see \[ \ba \int_0^3 ke^{-kt} \di t &= 0.4 \nl \frac{k}{-k} e^{-kt} \intEval_0^3 &= 0.4 \nl e^0 - e^{-3k} &= 0.4 \nl \implies k &= -\tfrac{1}{3} \ln 0.6 \approx 0.17 \pd \ea \] The mean of the exponential probability density function is \[\mu = \frac{1}{k} = \frac{1}{0.17} = 5.882 \pd\] Accordingly, the average wait time is \(5.882\) minutes. Yes, the manager should hire more staff.
EXERCISE 20
The average lifespan of a light bulb is \(800\) hours. Using an exponential probability density function, calculate the probability that a light bulb lasts more than \(900\) hours.
SOLUTION We model the light bulb's longevity using an exponential probability density function: \[ f(t) = \bc 0 & t \lt 0 \nl ke^{-kt} & t \geq 0 \cma \ec \] where \(t\) is time in hours. The mean of this function is given to be \(\mu = 800.\) Recall that the mean is \(\mu = 1/k,\) so we have \[ f(t) = \bc 0 & t \lt 0 \nl \dfrac{e^{-t/800}}{800} & t \geq 0 \pd \ec \] Accordingly, the probability that a light bulb lasts more than \(900\) hours is given by the improper integral \[\int_{900}^\infty \frac{e^{-t/800}}{800} \di t \pd\] Using the methods of Section 6.5, we obtain \[ \ba P(T \geq 900) &= \lim_{b \to \infty} \int_{900}^b \frac{e^{-t/800}}{800} \di t \nl &= \lim_{b \to \infty} \par{- e^{-t/800}} \intEval_{900}^b \nl &= \par{\lim_{b \to \infty} - e^{-b/800}} - \par{-e^{-900/800}} \nl &= e^{-9/8} \approx \boxed{0.325} = 32.5 \% \pd \ea \]
EXERCISE 21
The heights of women in the United States are Normally distributed with an average height of \(64.5\) inches and a standard deviation of \(2.5\) inches. In a sample of \(250\) randomly chosen women, use the 68–95–99.7 Rule to determine
  1. the number of women whose heights are between \(62\) and \(67\) inches
  2. the number of women whose heights are between \(59.5\) and \(69.5\) inches
  3. the number of women whose heights are between \(57\) and \(72\) inches
SOLUTION
  1. The values \(62\) and \(67\) are each one standard deviation away from the mean in either direction. The 68–95–99.7 Rule asserts that \(68\%\) of data lie within \(1\) standard deviation of the mean. So we predict that the number of women between \(62\) and \(67\) inches tall is \[250 \times 0.68 = \boxed{170}\]
  2. The values \(59.5\) and \(69.5\) are each \(2\) standard deviations away from the mean in either direction. According to the 68–95–99.7 Rule, \(95\%\) of data lie within \(2\) standard deviations of the mean. Thus, we predict that the number of women between \(59.5\) and \(69.5\) inches tall is \[250 \times 0.95 \approx \boxed{238}\]
  3. The values \(57\) and \(72\) are each \(3\) standard deviations away from the mean in either direction. The 68–95–99.7 Rule estimates that \(99.7 \%\) of data (nearly all of the data) reside within \(3\) standard deviations of the mean. So we predict the number of women between \(57\) and \(72\) inches to be \[250 \times 0.997 \approx \boxed{249}\] This is nearly all the women in the sample.

The ACT is a standardized exam used in college admissions in the United States. It is scored on a scale from \(1\) to \(36.\) In \(2021,\) the average ACT score was \(20.3\) points with a standard deviation of \(6.3\) points. What percentage of students scored \(30\) points or higher?
SOLUTION With a mean of \(\mu = 20.3\) and a standard deviation of \(\sigma = 6.3,\) the probability density function for the score distribution is given by \[f(x) = \normalPdf{20.3}{6.3}{x} \pd\] We integrate this function from \(x = 30\) to \(x = 36\) (the highest score); using numerical methods or a calculator, we see \[ \ba P(30 \leq X \leq 36) &= \int_{30}^{36} \normalPdf{20.3}{6.3}{x} \di x \nl &\approx 0.0554 = \boxed{5.54\%} \ea \] Observe that \(100 - 5.54 = 94.46.\) A score of \(30\) is therefore the \(94.46\)th percentile.

EXERCISE 24
The heights of men in the United States are Normally distributed with a mean of \(69\) inches and a standard deviation of \(3\) inches. In a random sample of \(100\) men, predict how many are \(6\) foot \(2\) inches or taller.
SOLUTION The mean is \(\mu = 69,\) and the standard deviation is \(\sigma = 3.\) Hence, the probability density function for the heights of American men is \[f(x) = \normalPdf{69}{3}{x} \pd\] Since \(1 \un{ft} = 12 \un{in},\) we convert \(6 \un{ft}\) \(2 \un{in}\) to \(74 \un{in}.\) We integrate this function from \(x = 74\) to \(x = \infty\) to determine the probability that a man is \(74 \un{in}\) or taller. To avoid the improper integral, we replace \(x = \infty\) with some sufficiently large number as an upper bound—say, \(x = 90,\) which is located \(7\) standard deviations away from the mean and is therefore extremely rare. Using numerical methods or a calculator, we see \[ \ba P(X \geq 74) &\approx \int_{74}^{90} \normalPdf{69}{3}{x} \di x \nl &\approx 0.048 \pd \ea \] Accordingly, we expect \(100 \times 0.048 \approx \boxed{5}\) men to be \(6 \un{ft}\) \(2 \un{in}\) or taller. If you desired, then you could pick an even larger number for the upper bound. Yet doing so would not greatly alter the probability; for example, the difference between \(\int_{74}^{90} f(x) \di x\) and \(\int_{74}^{9999} f(x) \di x\) is \(1.3 \times 10^{-12}.\) This difference is completely insignificant. The moral of the story is to choose a bound far enough from the mean (several standard deviations); doing so enables easier computations while preserving accuracy.
EXERCISE 25
In July, a town has an average temperature of \(82 \fahrenheit,\) with a standard deviation of \(7 \fahrenheit.\) On how many days in July do you expect the temperature to be \(76 \fahrenheit\) or lower?
SOLUTION The mean is \(\mu = 82,\) and the standard deviation is \(\sigma = 7.\) Accordingly, the probability density function for the town's temperature is given by \[f(x) = \normalPdf{82}{7}{x} \pd\] To calculate the probability that the temperature is \(76 \fahrenheit\) or lower, we integrate \(f(x)\) from \(x = -\infty\) to \(x = 76.\) To avoid the improper integral, we replace \(x = -\infty\) with some sufficiently small number as a lower bound—say, \(x = 40,\) which is located \(6\) standard deviations away from the mean and is therefore exceptionally rare. Using numerical methods or a calculator, we see \[ \ba P(X \leq 76) &\approx \int_{40}^{76} \normalPdf{82}{7}{x} \di x \nl &\approx 0.196 \pd \ea \] The month of July has \(31\) days, so we expect \(31 \times 0.196 \approx \boxed{6}\) days to be \(76 \fahrenheit\) or cooler. If you desired, then you could pick an even smaller number for the lower bound. But observe the following: The difference between \(\int_{40}^{76} f(x) \di x\) and \(\int_{0}^{76} f(x) \di x\) is \(9.6 \times 10^{-10},\) a magnitude we safely ignore. Thus, it is acceptable to choose a bound just far enough from the mean (several standard deviations) to enable easier computations.
EXERCISE 26

A worker throws out any box that weighs \(475\) grams or less. In a sample of \(1000\) boxes, determine how many boxes the worker throws out.
SOLUTION The mean is \(\mu = 500,\) and the standard deviation is \(\sigma = 10.\) Hence, the probability density function for the boxes' weights is given by \[f(x) = \normalPdf{500}{10}{x} \pd\] We integrate this function from \(x = 0\) to \(x = 475\) to determine the probability that a box weighs less than \(475\) grams. But it is easier (and safe) to replace \(x = 0\) with some sufficiently small number as a lower bound—say, \(x = 440,\) which is located \(6\) standard deviations away from the mean and is therefore extremely unlikely. Using numerical methods or a calculator, we have \[ \ba P(X \leq 475) &\approx \int_{440}^{475} \normalPdf{500}{10}{x} \di x \nl &\approx 0.006 \pd \ea \] Accordingly, we expect \(1000 \times 0.006 \approx \boxed{6}\) boxes to be thrown out. You could be more conservative and pick an even smaller number for the lower bound, but doing so would not change the probability significantly; for example, the difference between \(\int_{0}^{475} f(x) \di x\) and \(\int_{440}^{475} f(x) \di x\) is \(1 \times 10^{-9}.\) This difference is completely negligible. The key takeaway is use a bound far enough from the mean (several standard deviations) to enable easier computations while preserving accuracy.
