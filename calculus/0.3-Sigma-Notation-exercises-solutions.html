EXERCISE 1
For
\[\ds \sum_{i = 1}^3 i^4\]
expand the summation and calculate the result.
SOLUTION Expanding out the terms shows \[ \ba \sum_{i = 1}^3 i^4 &= 1^4 + 2^4 + 3^4 \nl &= \boxed{98} \ea \]

EXERCISE 7
For
\[\ds \sum_{n = 1}^4 \frac{2}{n^2 + 2n}\]
expand the summation and calculate the result.
SOLUTION Expanding out the terms shows \[ \ba \sum_{n = 1}^4 \frac{2}{n^2 + 2n} &= \frac{2}{(1)^2 + 2(1)} + \frac{2}{(2)^2 + 2(2)} + \frac{2}{(3)^2 + 2(3)} + \frac{2}{(4)^2 + 2(4)} \nl &= \boxed{\frac{17}{15}} \ea \]
EXERCISE 8
For
\[\ds \sum_{i = 1}^5 (-1)^n\]
expand the summation and calculate the result.
SOLUTION Expanding out the terms shows \[ \ba \sum_{i = 1}^5 (-1)^n &= (-1)^1 + (-1)^2 + (-1)^3 + (-1)^4 + (-1)^5 \nl &= (-1) + 1 + (-1) + 1 + (-1) \nl &= \boxed{-1} \ea \]
EXERCISE 9
For
\[\ds \sum_{k = 1}^3 \par{\frac{1}{k} - \frac{1}{k + 1}}\]
expand the summation and calculate the result.
SOLUTION Expanding out the terms shows \[ \ba \sum_{k = 1}^3 \par{\frac{1}{k} - \frac{1}{k + 1}} &= \par{\frac{1}{1} - \frac{1}{2}} + \par{\frac{1}{2} - \frac{1}{3}} + \par{\frac{1}{3} - \frac{1}{4}} \nl &= 1 - \frac{1}{4} = \boxed{\frac{3}{4}} \ea \] This series is called telescoping due to the pattern in which terms are canceled.
EXERCISE 10
For
\[\ds \sum_{n = 0}^4 \cos \pi n\]
expand the summation and calculate the result.
SOLUTION Expanding out the terms shows \[ \ba \sum_{n = 0}^4 \cos \pi n &= \cos 0 + \cos \pi + \cos 2 \pi + \cos 3 \pi + \cos 4 \pi \nl &= 1 + (-1) + 1 + (-1) + 1 \nl &= \boxed 1 \ea \]
EXERCISE 11
For
\[\ds \sum_{i = 1}^7 9\]
evaluate the series by using properties of summations.
SOLUTION If the starting index is \(1\) and the summation function is a constant, then the sum is the upper bound multiplied by the constant; that is, \[\sum_{i = 1}^7 9 = 9(7) = \boxed{63}\]

EXERCISE 13
For
\[\ds \sum_{i = 1}^9 \par{i^3 + 2}\]
evaluate the series by using properties of summations.
SOLUTION Splitting the summation, we see \[ \ba \sum_{i = 1}^9 \par{i^3 + 2} &= \sum_{i = 1}^9 i^3 + \sum_{i = 1}^9 2 \nl &= \parbr{\frac{9(9 + 1)}{2}}^2 + 2(9) \nl &= \boxed{2043} \ea \]
EXERCISE 14
For
\[\ds \sum_{i = 1}^8 \par{6i^2 - 4i + 5}\]
evaluate the series by using properties of summations.
SOLUTION The series is decomposed as follows: \[ \ba \sum_{i = 1}^8 \par{6i^2 - 4i + 5} &= \sum_{i = 1}^8 6i^2 - \sum_{i = 1}^8 4i + \sum_{i = 1}^8 5 \nl &= 6 \sum_{i = 1}^8 i^2 - 4 \sum_{i = 1}^8 i + \sum_{i = 1}^8 5 \nl &= 6 \parbr{\frac{8(8 + 1)(16 + 1)}{6}} - 4 \parbr{\frac{8 (8 + 1)}{2}} + 5(8) \nl &= \boxed{1120} \ea \]
EXERCISE 15
For
\[\ds \sum_{i = 1}^6 \par{4i^2 + 6i - 10}\]
evaluate the series by using properties of summations.
SOLUTION The series is decomposed as follows: \[ \ba \sum_{i = 1}^6 \par{4i^2 + 6i - 10} &= \sum_{i = 1}^6 4i^2 + \sum_{i = 1}^6 6i - \sum_{i = 1}^6 10 \nl &= 4 \sum_{i = 1}^6 i^2 + 6 \sum_{i = 1}^6 i - \sum_{i = 1}^6 10 \nl &= 4 \parbr{\frac{6(6 + 1)(12 + 1)}{6}} + 6 \parbr{\frac{6 (6 + 1)}{2}} - 10(6) \nl &= \boxed{430} \ea \]
EXERCISE 16
Using a Proof by Induction, prove the formula \[\sum_{i = 1}^n i^3 = \parbr{\frac{n(n + 1)}{2}}^2 \pd\]
SOLUTION This proof is similar to Example 4. Let \(S_n\) \(= \sum_{i = 1}^n i^3\) \(= [n(n + 1)/2]^2.\) Our first goal is to show that \(S_1\) is true. Then we assume that \(S_k\) (where \(k\) is any positive integer) is true and hold that \(S_{k + 1}\) is true. First observe that \(S_1\) is true because \[\sum_{i = 1}^1 i^3 = 1^3 \equalsCheck \parbr{\frac{1(1 + 1)}{2}}^2 \pd\] Next we assume that \(S_k\) is true—namely, \[\sum_{i = 1}^k i^3 = \parbr{\frac{k(k + 1)}{2}}^2 \pd\] We must show that \(S_{k + 1}\) is consequently true—that is, \[\sum_{i = 1}^{k + 1} i^3 = \parbr{\frac{(k + 1)(k + 2)}{2}}^2 \pd\] We see \[ \ba \sum_{i = 1}^{k + 1} i^3 &= \sum_{i = 1}^k i^3 + (k + 1)^3 \nl &= \parbr{\frac{k(k + 1)}{2}}^2 + (k + 1)^3 \nl &= (k + 1)^2 \par{\frac{k^2}{4} + (k + 1)} \nl &= (k + 1)^2 \par{\frac{k^2 + 4k + 4}{4}} \nl &= (k + 1)^2 \frac{(k + 2)^2}{4} \nl &\equalsCheck \parbr{\frac{(k + 1)(k + 2)}{2}}^2 \pd \ea \] Therefore, by mathematical induction, \(S_n\) is true for all positive integers \(n.\)
