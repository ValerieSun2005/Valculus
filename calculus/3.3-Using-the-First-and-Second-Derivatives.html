3.3: Using the First and Second Derivatives

When we analyze graphs, it is imperative that we be able to describe their properties. From algebra class, we categorized a graph either as increasing or decreasing over an interval. But calculus provides us one more property—concavity, which describes the change in a function's rate of change. Moreover, how do we test whether a function has a relative minimum or maximum? A function's first and second derivatives play vital roles in answering these questions. In this section we discuss the following topics:

  • Increasing and Decreasing
  • First-Derivative Test
  • Concavity
  • Second-Derivative Test

Increasing and Decreasing

Let \(x_1\) and \(x_2\) be two numbers such that \(x_1 \lt x_1.\) If a function \(f\) is increasing over an interval, then \(f(x_1) \lt f(x_2)\) for \(x_1\) and \(x_2\) in that interval. Conversely, \(f\) is decreasing if \(f(x_1) \gt f(x_2).\) In Figure 1, the graph of \(y = f(x)\) is decreasing between points \(A\) and \(B,\) increasing between \(B\) and \(C,\) and decreasing between \(C\) and \(D.\) When \(f\) is increasing, we see that tangents to the curve have positive slope; thus, \(f' \gt 0.\) But when \(f\) is decreasing, the tangents have negative slope and so \(f' \lt 0.\) We therefore conclude the following: \(f\) is increasing if \(f' \gt 0\) and \(f\) is decreasing if \(f' \lt 0.\) We prove these facts using the Mean Value Theorem (from Section 3.2).

INCREASING AND DECREASING
  1. If \(f' \gt 0\) on an interval, then \(f\) is increasing on that interval.
  2. If \(f' \lt 0\) on an interval, then \(f\) is decreasing on that interval.

PROOF Consider the interval \([x_1, x_2]\) with \(x_1 \lt x_2.\) We must prove that if \(f'(x) \gt 0\) in this interval, then \(f(x_1) \lt f(x_2)\) and so \(f\) is increasing. Then we aim to show that if \(f'(x) \lt 0\) in this interval, then \(f(x_1) \gt f(x_2),\) indicating that \(f\) is decreasing. We assume that \(f\) is continuous over \([x_1, x_2]\) and differentiable over \((x_1, x_2).\) Thus, the Mean Value Theorem guarantees a number \(c\) in \((x_1, x_2)\) such that \begin{equation} f(x_2) - f(x_1) = f'(c) (x_2 - x_1) \pd \label{eq:proof-f'-mvt} \end{equation} If \(f' \gt 0\) over \((x_1, x_2),\) then the right-hand side of \(\eqref{eq:proof-f'-mvt}\) becomes a product of positive numbers because \(f'(c) \gt 0\) and \(x_2 - x_1 \gt 0.\) Consequently, \[f(x_2) - f(x_1) \gt 0 \or f(x_1) \lt f(x_2) \cma\] so \(f\) is increasing on \([x_1, x_2].\) Hence, part (a) is proved. If \(f' \lt 0\) over \((x_1, x_2),\) then in \(\eqref{eq:proof-f'-mvt}\) \(f'(c) \lt 0,\) so the right-hand side is a negative quantity. Thus, \[f(x_2) - f(x_1) \lt 0 \or f(x_1) \gt f(x_2) \cma\] meaning \(f\) is decreasing over \([x_1, x_2].\) Part (b) is therefore proved. \[\qedproof\]

EXAMPLE 1
Let \(f(x) = \tfrac{1}{3} x^3 - 2x^2 + 3x + 2.\) Find the intervals on which \(f\) is increasing and on which \(f\) is decreasing.
The function \(f\) is increasing if \(f'(x) \gt 0\) and decreasing if \(f'(x) \lt 0.\) Differentiating \(f\) gives \[f'(x) = x^2 - 4x + 3 = (x - 3)(x - 1) \pd\] We find that \(f'(x) \lt 0\) for \(x\) in \((1, 3),\) so \(f(x)\) is increasing over \([1, 3].\) Also, \(f'(x) \gt 0\) for \(x\) in \((-\infty, 1)\) and \((3, \infty),\) so \(f\) is decreasing for \((-\infty, 1]\) and \([3, \infty).\) (See Figure 2.) Note that we write intervals of increase or decrease as closed intervals when \(f\) continuous at the endpoints.
EXAMPLE 2
In Figure 3, the graph of \(y = g'(x)\) is shown. Where is \(g\) increasing, and where is \(g\) decreasing?
This graph is of \(g',\) not \(g.\) Function \(g\) increases when \(g'(x) \gt 0\) and decreases when \(g'(x) \lt 0.\) We see \(g'(x) \gt 0\) for \(-4 \lt x \lt 3,\) so \(g(x)\) is increasing over \([-4, 3].\) Conversely, \(g'(x) \lt 0\) for \(-6 \leq x \lt -4\) and \(3 \lt x \leq 6.\) Thus, \(g(x)\) is decreasing over \([-6, -4]\) and \([3, 6].\)

First-Derivative Test

In Section 3.1 we learned to find a function's relative extrema by analyzing its critical numbers. Yet how do we test whether the function has a relative minimum or relative maximum at the critical number? In Figure 4 the function \(f\) has a relative minimum at \((a, f(a))\) and a relative maximum at \((b, f(b)).\) Observe that \(f\) is decreasing to the left of \(a\) and increasing to the right of \(a.\) Through our discussion of Increasing and Decreasing, \(f'\) therefore flips sign from negative to positive at \(a.\) Conversely, \(f\) is increasing to the left of \(b\) and decreasing to the right of \(b.\) Consequently, \(f'\) changes sign from positive to negative at \(b.\) These observations describe the First-Derivative Test, defined below.

FIRST-DERIVATIVE TEST
If \(c\) is a critical number of \(f,\) then
  1. \(f\) has a relative minimum at \(c\) if \(f'\) changes sign from negative to positive at \(c.\)
  2. \(f\) has a relative maximum at \(c\) if \(f'\) changes sign from positive to negative at \(c.\)
  3. \(f\) has no relative extremum \(c\) if \(f'\) does not change sign at \(c.\)

The First-Derivative Test can also prove the nonexistence of a relative extremum. For example, the derivative of \(g(x) = x^3\) is \(g'(x) = 3x^2.\) Note that \(g'(0) = 0,\) but \(g'\) does not change sign at \(0\) because \(g'(x) = 3x^2\) is strictly nonnegative. Accordingly, the First-Derivative Test asserts that \(g(x) = x^3\) does not have a relative extremum at \(x = 0.\) (See Figure 5.)

EXAMPLE 3
Locate and classify all the relative extrema of \(g(x) = 2x^3 - 3x^2 - 1.\)
First we find the critical numbers of \(g\) by solving \(g'(x) = 0,\) as follows: \[ \ba g'(x) = 6x^2 - 6x &= 0 \nl 6x(x - 1) &= 0 \nl \implies x &= 0 \cma x = 1 \pd \ea \] Note that \(g'\) is never undefined, so \(0\) and \(1\) are the only critical numbers. The following table shows the sign of \(g'\) on intervals bounded by the critical numbers.
Interval of \(x\) \(g'(x) = 6x(x - 1)\)
\((-\infty, 0)\) \(+\)
\((0, 1)\) \(-\)
\((1, \infty)\) \(+\)
At \(x = 0,\) \(g'\) changes sign from positive to negative. Thus, the First-Derivative Test asserts that \(g\) has a relative maximum at \(x = 0;\) this value is \[g(0) = 2(0)^3 - 3(0)^2 - 1 = -1 \pd\] Conversely, \(g'\) switches sign from negative to positive at \(x = 1,\) so \(g\) has a relative minimum at \(x = 1.\) This value is \[g(1) = 2(1)^3 - 3(1)^2 - 1 = -2 \pd\] (See Figure 6.)
EXAMPLE 4
The function \(g\) is continuous on \([-6, 6].\) The graph of \(y = g'(x)\) is shown in Figure 7. Find the critical numbers of \(g.\) At each critical number, determine whether \(g\) has a relative minimum, a relative maximum, or neither.

A number \(c\) is a critical number of \(g\) if it's in the domain of \(g\) and if \(g'(c) = 0\) or \(g'(c)\) is undefined. From Figure 7, \(g'(x) = 0\) when \(x = -4\) and \(x = 5.\) Also, \(g'(x)\) is undefined at \(x = 1\) and \(x = 3,\) but both numbers are in the domain of \(g\) because \(g\) is given to be continuous on \([-6, 6].\) Therefore, the critical numbers of \(g\) are \(-4,\) \(1,\) \(3,\) and \(5.\) The following table shows the sign of \(g'\) for intervals bounded by these values.

Thus, \(g(x)\) has

  • a relative maximum at \(x = 1\) because the sign of \(g'\) flips from positive to negative at \(x = 1.\)
  • no extrema at \(x = 3\) or \(x = 5\) because \(g'\) does not change sign at those points.

Concavity

We can determine whether a function is increasing or decreasing over an interval. But we delve one step deeper to consider concavity. In Figure 8A and Figure 8B, both functions \(f\) and \(g\) are increasing, but they are bent differently: We call \(f\) concave upward (or simply concave up) and \(g\) concave downward (or concave down). The word concave means curving inward; it's easy to remember the phrase cave in. The graph of \(f\) appears to bend inward when viewed from above, whereas the graph of \(g\) looks to arch inward if we view it from below.

If we draw tangents to the graph of \(f,\) whose graph is concave up, then we see that the graph lies above all the tangents (Figure 9A). Thus, the rate of change of \(f\) is increasing. In other words, the derivative \(f'\) is an increasing function; accordingly, the second derivative \(f''\) is positive. By contrast, drawing tangents to the graph of \(g\) (whose graph is concave down) shows that the graph lies below all the tangents (Figure 9B). Because the slopes of the tangents are decreasing, \(f'\) is a decreasing function and so \(f''\) is negative.

CONCAVITY
  1. If \(f''(x) \gt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave up on that interval.
  2. If \(f''(x) \lt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave down on that interval.
EXAMPLE 5
Let \(f(x) = \tfrac{1}{6}x^3 - x^2 + 6x + 1.\) Describe the graph of \(f(x)\) at \(x = 1.\)
Taking derivatives, we find \[f'(x) = \tfrac{1}{2} x^2 - 2x + 6 \and f''(x) = x - 2 \pd\] Then \[ \ba f'(1) &= \tfrac{1}{2}(1)^2 - 2(1) + 6 = \tfrac{5}{2} \cma \nl f''(1) &= 1 - 2 = -1 \pd \ea \] Because \(f'(1) \gt 0,\) \(f\) is increasing at \(x = 1.\) Also, since \(f''(1) \lt 0,\) the graph of \(f\) is concave down at \(x = 1.\) Drawing a tangent line to \(f\) at \(x = 1,\) we see that the graph lies below this tangent—the geometric meaning of \(f\) being concave down. (See Figure 10.)
EXAMPLE 6
Let \(f(x) = \tfrac{1}{3}x^3 - 2x^2 + x + 4.\) Where is the graph of \(f\) concave up, and where is it concave down?
The graph of \(f\) is concave up when \(f''(x) \gt 0\) and concave down when \(f''(x) \lt 0.\) Differentiating \(f,\) we find \[ f'(x) = x^2 - 4x + 1 \and f''(x) = 2x - 4 \pd \] Then \(f''(x) \gt 0\) when \(x \gt 2\) and \(f''(x) \lt 0\) when \(x \lt 2.\) Thus, the graph of \(f(x)\) is concave up over \((2, \infty)\) and concave down over \((-\infty, 2).\) In Figure 11, the tangents to the graph of \(f\) are above the graph for \(x \lt 2\) and below the graph for \(x \gt 2.\)

A point at which the graph of \(f\) changes concavity—from concave up to concave down or from concave down to concave up—is called an inflection point. For example, in Example 6 the graph of \(f\) switches from concave down to concave up at \(x = 2.\) Thus, \(x = 2\) is the location of an inflection point of \(f.\) An easy way to estimate the location of an inflection point of \(f\) is to identify a point at which \(f\) crosses its tangent.

To find inflection points of \(f,\) it is not sufficient to solve \(f''(x) = 0.\) You must also determine whether \(f''(x)\) changes sign at that point.
INFLECTION POINT
The function \(f\) has an inflection point at \(x = a\) if the graph of \(f\) switches concavity at \(x = a.\)
EXAMPLE 7
Let \(f(x) = x^4 - 2x^3 - 12x^2 - 17x + 4.\) Find the \(x\)-coordinates of all the inflection points of \(f.\)
At an inflection point, the graph of \(f\) changes concavity, so we seek sign changes in \(f''.\) Because \(f''\) is continuous, we find the solutions to \(f''(x) = 0.\) At these values of interest, we analyze whether \(f''(x)\) changes sign. Differentiating gives \[f'(x) = 4x^3 - 6x^2 - 24x - 17 \and f''(x) = 12x^2 - 12x - 24 \pd\] We then solve \(f''(x) = 0 \col\) \[ \ba 12x^2 - 12x - 24 &= 0 \nl x^2 - x - 2 &= 0 \nl (x - 2)(x + 1) &= 0 \nl \implies x &= -1 \cma x = 2 \pd \ea \] Because \(f''(x) \gt 0\) for \(x \lt -1\) and \(f''(x) \lt 0\) for \(-1 \lt x \lt 2,\) we confirm that \(x = -1\) is an inflection point because at that point \(f\) changes from concave up to concave down. Likewise, \(f''(x) \gt 0\) for \(x \gt 2;\) accordingly, \(f\) has an inflection point at \(x = 2\) because \(f\) switches from concave down to concave up at that point. (See Figure 12.)

Second-Derivative Test

Another application of second derivatives is the Second-Derivative Test, shown below. This test is an alternative to the First-Derivative Test.

SECOND-DERIVATIVE TEST
Let \(f''(x)\) be continuous near \(x = c.\)
  1. If \(f'(c) = 0\) and \(f''(c) \gt 0,\) then \(f\) has a relative minimum at \(x = c.\)
  2. If \(f'(c) = 0\) and \(f''(c) \lt 0,\) then \(f\) has a relative maximum at \(x = c.\)
  3. If \(f'(c) = 0\) and \(f''(c) = 0\) or \(f''(c)\) does not exist, then the test is inconclusive.
The Second-Derivative Test has its limitations: The test fails if \(f''(c)\) equals \(0\) or does not exist. In either case, we must instead use the First-Derivative Test.

In part (a), because \(f''(c) \gt 0,\) \(f\) is concave up at \(c.\) Accordingly, the graph of \(f\) must be above the tangent at \(c,\) so \(f\) has a relative minimum at \(c.\) (See Figure 13A.) Conversely, in part (b) \(f''(c) \lt 0,\) meaning \(f\) is concave down at \(c.\) So the graph of \(f\) lies below the tangent at \(c;\) accordingly, \(c\) must be a relative maximum of \(f,\) as in Figure 13B. Part (c) states that \(f\) can have either a relative minimum, relative maximum, or neither at the critical number \(c\) if \(f''(c)\) equals \(0\) or does not exist.

EXAMPLE 8
Find and classify the relative extrema of \(f(x) = 2x^3 - 6x^2 - 18x + 5\) using the Second-Derivative Test.
We locate the critical numbers of \(f\) by letting \(f'(x) = 0.\) Differentiating gives \[f'(x) = 6x^2 - 12x - 18 = 6(x - 3)(x + 1) \cma\] whose zeros are \(x = -1\) and \(x = 3.\) Thus, these are the critical numbers of \(f.\) We will use the Second-Derivative Test to determine whether each critical number corresponds to a relative minimum, relative maximum, or neither of \(f.\) The second derivative of \(f\) is \[f''(x) = 12x - 12 \pd \] We have \[f''(-1) = -24 \and f''(3) = 24 \pd\] Because \(f''(-1) \lt 0,\) \(f\) has a relative maximum at \(x = -1.\) Since \(f''(3) \gt 0,\) \(f\) has a relative minimum at \(x = 3.\) (See Figure 14.)

Increasing and Decreasing The function \(f',\) the first derivative of \(f,\) signifies whether \(f\) is increasing or decreasing over an interval.

  1. If \(f' \gt 0\) on an interval, then \(f\) is increasing on that interval.
  2. If \(f' \lt 0\) on an interval, then \(f\) is decreasing on that interval.

First-Derivative Test The First-Derivative Test enables us to determine whether a function has a relative maximum, a relative minimum, or neither at a critical number. If \(c\) is a critical number of \(f,\) then

  1. \(f\) has a relative minimum at \(c\) if \(f'\) changes sign from negative to positive at \(c.\)
  2. \(f\) has a relative maximum at \(c\) if \(f'\) changes sign from positive to negative at \(c.\)
  3. \(f\) has no relative extremum at \(c\) if \(f'\) does not change sign at \(c.\)

Concavity We use the second derivative \(f''\) to determine the concavity of the graph of \(f.\) The graph of \(f\) is concave up over an interval if the graph lies above its tangents on that interval. Conversely, the graph is concave down if the graph lies below its tangents.

  1. If \(f''(x) \gt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave up on that interval.
  2. If \(f''(x) \lt 0\) for all \(x\) in an interval, then the graph of \(f\) is concave down on that interval.

The function \(f\) has an inflection point at \(x = a\) if the graph of \(f\) switches concavity at \(x = a.\)

Second-Derivative Test The Second-Derivative Test enables us to classify critical points of \(f\) as corresponding to relative minima or relative maxima. It is an alternative to the First-Derivative Test.

  1. If \(f'(c) = 0\) and \(f''(c) \gt 0,\) then \(f\) has a relative minimum at \(x = c.\)
  2. If \(f'(c) = 0\) and \(f''(c) \lt 0,\) then \(f\) has a relative maximum at \(x = c.\)
  3. If \(f'(c) = 0\) and \(f''(c) = 0\) or \(f''(c)\) does not exist, then the test is inconclusive.
