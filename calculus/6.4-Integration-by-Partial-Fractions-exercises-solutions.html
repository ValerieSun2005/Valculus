6.4 Exercise Solutions: Integration by Partial Fractions

EXERCISE 1
For
\[\ds \frac{2}{(x + 3)(x - 2)}\]
perform the partial fraction decomposition.
SOLUTION The rational expression contains two linear factors: \((x + 3)\) and \((x - 2).\) The partial fraction decomposition then takes the shape \[\frac{2}{(x + 3)(x - 2)} = \frac{A}{x + 3} + \frac{B}{x - 2} \pd\] Manipulating these fractions, we have \[ \ba \frac{2}{(x + 3)(x - 2)} &= \frac{A}{x + 3} \par{\frac{x - 2}{x - 2}} + \frac{B}{x - 2} \par{\frac{x + 3}{x + 3}} \nl 2 &= A(x - 2) + B(x + 3) \pd \ea \] Our objective is now to solve for \(A\) and \(B.\) Letting \(x = 2\) yields \[2 = 0 + 5B \implies B = \frac{2}{5} \pd\] Similarly, for \(x = -3\) we get \[2 = -5A + 0 \implies A = -\frac{2}{5} \pd\] We have, therefore, \[\frac{2}{(x + 3)(x - 2)} = \boxed{\frac{-2/5}{x + 3} + \frac{2/5}{x - 2} }\]
EXERCISE 2
For
\[\ds \frac{3x - 1}{x^2 - 8x - 9}\]
perform the partial fraction decomposition.
SOLUTION We factor the denominator as \[\frac{3x - 1}{(x - 9)(x + 1)} \cma\] from which our partial fraction decomposition takes the form \[ \frac{3x - 1}{(x - 9)(x + 1)} = \frac{A}{x - 9} + \frac{B}{x + 1} \, \] Algebraic manipulation gives \[ \ba \frac{3x - 1}{(x - 9)(x + 1)} &= \frac{A}{x - 9} \par{\frac{x + 1}{x + 1}} + \frac{B}{x + 1} \par{\frac{x - 9}{x - 9}} \nl 3x - 1 &= A(x + 1) + B(x - 9) \pd \ea \] We now perform substitutions to solve for \(A\) and \(B.\) Letting \(x = -1\) gives \[-4 = 0 - 10B \implies B = \frac{2}{5} \pd\] Also, assigning \(x = 9\) produces \[26 = 10A + 0 \implies A = \frac{13}{5} \pd\] We now have, for our final answer, \[\frac{3x - 1}{(x - 9)(x + 1)} = \boxed{\frac{13/5}{x - 9} + \frac{2/5}{x + 1}} \]
EXERCISE 3
For
\[\ds \frac{2x + 3}{9x^2 - x} \]
perform the partial fraction decomposition.
SOLUTION Factoring the denominator, we have \[\frac{2x + 3}{x(9x - 1)} \cma\] so our decomposition shape is \[\frac{2x + 3}{x(9x - 1)} = \frac{A}{x} + \frac{B}{9x - 1} \pd\] Adding these partial fractions results in \[2x + 3 = A(9x - 1) + Bx \pd\] We now let \(x = 0\) to get \[3 = -A + 0 \implies A = -3 \pd\] Furthermore, letting \(x = 1/9\) gives \[\frac{2}{9} + 3 = 0 + \frac{1}{9} B \implies B = 29 \pd\] Thus, we obtain \[\frac{2x + 3}{9x^2 - x} = \boxed{\frac{-3}{x} + \frac{29}{9x - 1} }\]
EXERCISE 4
For
\[\ds \frac{t^2}{8t^3 - 1}\]
perform the partial fraction decomposition.
SOLUTION The denominator is a difference of cubes that can be factored to obtain \[\frac{t^2}{8t^3 - 1} = \frac{t^2}{(2t - 1)(4t^2 + 2t + 1)} \pd\] The quadratic \((4t^2 + 2t + 1)\) is irreducible, so our decomposition is in the form \[\frac{t^2}{(2t - 1)(4t^2 + 2t + 1)} = \frac{A}{2t - 1} + \frac{Bt + C}{4t^2 + 2t + 1} \pd\] Adding the partial fractions gives \[t^2 = A(4t^2 + 2t + 1) + Bt(2t - 1) + C(2t - 1) \pd\] We now solve for \(A,\) \(B,\) and \(C.\) Two of the terms contain the factor \((2t - 1),\) so letting \(t = 1/2\) zeroes them. We attain \[\frac{1}{4} = 3A + 0 \implies A = \frac{1}{12} \pd\] We now have two unknowns: \[t^2 = \frac{1}{12} (4t^2 + 2t + 1) + Bt(2t - 1) + C(2t - 1) \pd\] Letting \(t = 0\) provides \[0 = \frac{1}{12} + 0 - C \implies C = \frac{1}{12} \pd\] At this point, our remaining unknown is \(B;\) we have \[t^2 = \frac{1}{12}(4t^2 + 2t + 1) + Bt(2t - 1) + \frac{1}{12}(2t - 1) \pd\] We assign \(t\) any value to directly solve for \(B.\) The simplest value is \(t = 1,\) which gives \[1 = \frac{7}{12} + B + \frac{1}{12} \implies B = \frac{1}{3} \pd\] Our final answer is \[ \frac{t^2}{8t^3 - 1} = \boxed{\frac{\frac{1}{12}}{2t - 1} + \frac{\frac{1}{3}t + \frac{1}{12}t}{4t^2 + 2t + 1} }\]
EXERCISE 5
For
\[\ds \frac{x^3 + 1}{9 - x^2}\]
perform the partial fraction decomposition.
SOLUTION We cannot perform partial fraction decomposition immediately, since the degree of the numerator is greater than the degree of the denominator. So our first step is to perform polynomial long division. Doing so gives the simplified expression \[-x + \frac{9x + 1}{9 - x^2} \pd\] Factoring the denominator of \(\frac{9x + 1}{9 - x^2}\) gives \[\frac{9x + 1}{(3 + x)(3 - x)} \pd\] Our partial fraction decomposition is then in the form \[\frac{9x + 1}{(3 + x)(3 - x)} = \frac{A}{3 + x} + \frac{B}{3 - x} \pd\] Adding the partial fractions, we obtain \[9x + 1 = A(3 - x) + B(3 + x) \pd\] Now we let \(x = 3 \! :\) \[28 = 0 + 6B \implies B = \frac{14}{3} \pd\] Then letting \(x = -3\) gives \[-26 = 6A + 0 \implies A = -\frac{13}{3} \pd\] Therefore, our final answer is \[\frac{x^3 + 1}{9 - x^2} = \boxed{-x - \frac{13/3}{3 + x} + \frac{14/3}{3 - x} }\]
EXERCISE 6
For
\[\ds \frac{3 - 10x^3}{5x^3 + 8x^2 - 11x} \]
perform the partial fraction decomposition.
SOLUTION We begin by performing polynomial division because the degrees of the numerator and denominator are equal. Doing so gives \[\frac{3 - 10x^3}{5x^3 + 8x^2 - 11x} = -2 + \frac{16x^2 - 22x + 3}{5x^3 + 8x^2 - 11x} \pd \] Our objective is now to find the partial fraction decomposition of \(\frac{16x^2 - 22x + 3}{5x^3 + 8x^2 - 11x} \col\) \[ \ba \frac{16x^2 - 22x + 3}{5x^3 + 8x^2 - 11x} &= \frac{16x^2 - 22x + 3}{x(5x^2 + 8x - 11)} \nl &= \frac{A}{x} + \frac{Bx + C}{5x^2 + 8x - 11} \pd \ea \] We then have \[16x^2 - 22x + 3 = A(5x^2 + 8x - 11) + Bx^2 + Cx \pd\] To solve for \(A,\) we let \(x = 0\) to get \[3 = -11A + 0 + 0 \implies A = -\frac{3}{11} \pd \] There is no convenient substitution to solve for \(B\) or \(C.\) Thus, we should arrange a system of equations to solve for these unknowns. Letting \(x = 1,\) for simplicity, gives \[ -3 = -\frac{6}{11} + B + C \implies B + C = -\frac{27}{11} \pd \] Also, letting \(x = -1\) provides the simple equation \[41 = -14 \par{-\frac{3}{11}} + B - C \implies B - C = \frac{409}{11} \pd\] Our system is then \[ \ba B + C &= -\frac{27}{11} \nl B - C &= \frac{409}{11} \cma \ea \] for which the solutions are \[B = \frac{191}{11} \and C = -\frac{218}{11} \pd \] Our final answer is then \[ \ba \frac{3 - 10x^3}{5x^3 + 8x^2 - 11x} &= \boxed{\frac{-3/11}{x} + \frac{191x - 218}{11(5x^2 + 8x^2 - 11)} } \ea \]
EXERCISE 7
For
\[\ds \int \frac{4 - 3x}{(7x + 3)(4 - x)} \di x\]
evaluate the integral.
SOLUTION The partial fraction decomposition of the integrand takes the shape \[\frac{4 - 3x}{(7x + 3)(4 - x)} = \frac{A}{7x + 3} + \frac{B}{4 - x} \pd\] We then get \[4 - 3x = A(4 - x) + B(7x + 3) \pd\] We solve for \(B\) by letting \(x = 4 \col\) \[-8 = 0 + 31B \implies B = -\frac{8}{31} \pd\] Letting \(x = -3/7\) yields \[4 + \frac{9}{7} = A \par{4 + \frac{3}{7} } \implies A = \frac{37}{31} \pd\] We have, therefore, \[ \ba \int \frac{4 - 3x}{(7x + 3)(4 - x)} \di x &= \int \frac{37/31}{7x + 3} - \frac{8/31}{4 - x} \di x \nl &= \boxed{\frac{37}{217} \ln|7x + 3| + \frac{8}{31} \ln|4 - x| + C } \ea \]
EXERCISE 8
For
\[\ds \int \frac{x}{2x^2 - 5x - 3} \di x\]
evaluate the integral.
SOLUTION Factoring the denominator gives \[\int \frac{x}{(x - 3)(2x + 1)} \di x \cma\] whose integrand has the partial fraction decomposition \[\frac{x}{(x - 3)(2x + 1)} = \frac{A}{x - 3} + \frac{B}{2x + 1} \pd\] We then get \[x = A(2x + 1) + B(x - 3) \pd\] To solve for \(A,\) we let \(x = 3\) to get \[3 = 7A + 0 \implies A = \frac{3}{7} \pd\] Letting \(x = -1/2\) gives \[-\frac{1}{2} = 0 - \frac{7}{2} B \implies B = \frac{1}{7} \pd\] Thus, \[ \ba \int \frac{x}{2x^2 - 5x - 3} \di x &= \int \frac{3/7}{x - 3} + \frac{1/7}{2x + 1} \di x \nl &= \boxed{\frac{3}{7} \ln|x - 3| + \frac{1}{14} \ln|2x + 1| + C } \ea \]
EXERCISE 9
For
\[\ds \int \frac{3t^2 + 2}{5t^2 - 25t} \di t \]
evaluate the integral.
SOLUTION The degree of the numerator isn't less than the degree of the denominator, so we first perform polynomial division to get \[ \ba \frac{3t^2 + 2}{5t^2 - 25t} &= \frac{3}{5} + \frac{15t + 2}{5t^2 - 25t} \nl &= \frac{3}{5} + \frac{15t + 2}{5t(t - 5)} \pd \ea \] The partial fraction decomposition of \(\frac{15t + 2}{5t(t - 5)} \) takes the form \[\frac{15t + 2}{5t(t - 5)} = \frac{1}{5} \par{\frac{A}{t} + \frac{B}{t - 5} } \pd\] We then get \[15t + 2 = A(t - 5) + Bt \pd\] Solving for \(A,\) we let \(t = 0\) to get \[2 = -5A + 0 \implies A = -\frac{2}{5} \pd\] Letting \(t = 5\) gives \[77 = 0 + 5B \implies B = \frac{77}{5} \pd\] Thus, \[ \ba \int \frac{3t^2 + 2}{5t^2 - 25t} \di t &= \int \frac{3}{5} + \frac{1}{5} \par{ \frac{-2/5}{t} + \frac{77/5}{t - 5} } \di t \nl &= \boxed{\frac{3}{5} t - \frac{2}{25} \ln|t| + \frac{77}{25} \ln|t - 5| + C } \ea \]
EXERCISE 10
For
\[\ds \int \frac{3}{x(x + 3)(x^2-4)} \di x\]
evaluate the integral.
SOLUTION The partial fraction decomposition for the integrand is \[\frac{A}{x} + \frac{B}{x + 3} + \frac{C}{x + 2} + \frac{D}{x - 2} \pd\] We then have \[3 = A(x + 3)(x + 2)(x - 2) + Bx(x + 2)(x - 2) + Cx(x + 3)(x - 2) + Dx(x + 3)(x + 2) \pd\] To solve for \(A,\) we let \(x = 0\) to obtain \[3 = -12A \implies A = -\frac{1}{4} \pd\] Similarly, assigning \(x = -3\) gives \[3 = 0 -15B + 0 + 0 \implies B = -\frac{1}{5} \pd\] Solving for \(C\) is just as simple—by letting \(x = -2 \col\) \[3 = 0 + 0 + 8C + 0 \implies C = \frac{3}{8} \pd\] Lastly, we let \(x = 2\) to find \(D \col\) \[3 = 0 + 0 + 0 + 40D \implies D = \frac{3}{40} \pd\] Substituting these coefficients and integrating yield \[\small \int \frac{-1/4}{x} - \frac{1/5}{x + 3} + \frac{3/8}{x + 2} + \frac{3/40}{x - 2} \di x \nl = \boxed{-\frac{1}{4} \ln\abs x - \frac{1}{5} \ln|x + 3| + \frac{3}{8} \ln|x + 2| + \frac{3}{40} \ln|x - 2| + C } \]
EXERCISE 11
For
\[\ds \int \frac{5}{t(t - 3)^2} \di t\]
evaluate the integral.
SOLUTION We rewrite the integrand as \[\frac{5}{t(t - 3)(t - 3)} \cma\] whose partial fraction decomposition takes the form \[\frac{5}{t(t - 3)(t - 3)} = \frac{A}{t} + \frac{B}{t - 3} + \frac{C}{(t - 3)^2} \pd\] We then get, after multiplying both sides by \(t(t - 3)^2,\) \[5 = A(t - 3)^2 + Bt(t - 3) + Ct \pd\] We immediately solve for \(A\) by letting \(t = 0 \col\) \[5 = 9A + 0 + 0 \implies A = \frac{5}{9} \pd \] Likewise, letting \(t = 3\) yields \[5 = 0 + 0 = 3C \implies C = \frac{5}{3} \pd\] Plugging in these coefficients, our equation is now \[5 = \frac{5}{9}(t - 3)^2 + Bt(t - 3) + \frac{5}{3} t \pd\] We now let \(t\) equal any value to solve for \(B.\) A simple value is \(t = 4,\) from which we obtain \[5 = \frac{5}{9} + 4B + \frac{20}{3} \implies B = -\frac{5}{9} \pd\] We have, therefore, \[ \ba \int \frac{5}{t(t - 3)^2} \di t &= \int \frac{5/9}{t} - \frac{5/9}{t - 3} + \frac{5/3}{(t - 3)^2} \di t \nl &= \boxed{\frac{5}{9} \ln|t| - \frac{5}{9} \ln|t - 3| - \frac{5/3}{t - 3} + C } \ea \]
EXERCISE 12
For
\[\ds \int \frac{6 - 2t}{(t + 1)(27 - 3t^2)} \di t\]
evaluate the integral.
SOLUTION We rewrite the integrand as \[ \frac{-2(t - 3)}{3(t + 1)(3 + t)(3 - t)} \pd\] The partial fraction decomposition then takes the shape \[\frac{-2(t - 3)}{3(t + 1)(3 + t)(3 - t)} = \frac{1}{3} \par{ \frac{A}{t + 1} + \frac{B}{3 + t} + \frac{C}{3 - t}} \pd\] We then have \[-2(t - 3) = A(3 + t)(3 - t) + B(t + 1)(3 - t) + C(t + 1)(3 + t) \pd\] To solve for \(C,\) we let \(t = 3\) to get \[0 = 0 + 0 + 36C \implies C = 0 \pd\] Likewise, letting \(t = -1\) gives \[8 = 8A + 0 \implies A = 1 \pd\] Finally, we let \(t = -3\) to obtain \[12 = 0 - 12B + 0 \implies B = -1 \pd\] We have, for our integral, \[\frac{1}{3} \int \frac{1}{t + 1} - \frac{1}{3 + t} \di t = \boxed{\frac{1}{3} \ln|t + 1| - \frac{1}{3} \ln|3 + t| + C }\]
EXERCISE 13
For
\[\ds \int \frac{9 + 5q^2}{q^3 - q^2 - 12q} \di q\]
evaluate the integral.
SOLUTION We re-express the integrand as \[\frac{9 + 5q^2}{q(q - 4)(q + 3)} \cma\] whose partial fraction decomposition takes the shape \[\frac{A}{q} + \frac{B}{q - 4} + \frac{C}{q + 3} \pd\] We then get \[9 + 5q^2 = A(q - 4)(q + 3) + Bq(q + 3) + Cq(q - 4) \pd\] To solve for \(A,\) we let \(q = 0\) to get \[9 + 0 = -12A + 0 + 0 \implies A = \frac{-3}{4} \pd \] Assigning \(t = 4\) provides \[89 = 0 + 28B + 0 \implies B = \frac{89}{28} \pd\] Lastly, we solve for \(C\) by letting \(t = -3 \col\) \[54 = 0 + 0 + 21C \implies C = \frac{18}{7} \pd\] We have, therefore, \[ \ba \int \frac{9 + 5q^2}{q^3 - q^2 - 12q} \di q &= \int \frac{-3/4}{q} + \frac{89/28}{q - 4} + \frac{18/7}{q + 3} \di q \nl &= \boxed{\frac{89}{28} \ln|q - 4| + \frac{18}{7} \ln|q + 3| - \frac{3}{4} \ln|q| + C } \ea \]
EXERCISE 14
For
\[\ds \int \frac{x^4 + 1}{16x - 4x^3} \di x\]
evaluate the integral.
SOLUTION The degree of the numerator is greater than the degree of the denominator, so we must first simplify the expression using polynomial division. Doing so gives \[\frac{x^4 + 1}{16 - 4x^2} = -\frac{x}{4} + \frac{4x^2 + 1}{16x - 4x^3} \pd\] Factoring \(\frac{4x^2 + 1}{16x - 4x^3}\) yields \[\frac{4x^2 + 1}{4x(2 + x)(2 - x)} \cma\] whose partial fraction decompositions take the form \[ \frac{4x^2 + 1}{4x(2 + x)(2 - x)} = \frac{A}{4x} + \frac{B}{2 + x} + \frac{C}{2 - x} \pd \] Adding these partial fractions, we attain \[4x^2 + 1 = A(2 + x)(2 - x) + B(4x)(2 - x) + C(4x)(2 + x) \pd\] Two of the terms possess the factor \((4x),\) so letting \(x = 0\) cancels them out. We get \[1 = 4A + 0 + 0 \implies A = \frac{1}{4} \pd\] In addition, letting \(x = 2\) cancels out two more factors, enabling us to solve for \(C \! :\) \[17 = 0 + 0 + 32 C \implies C = \frac{17}{32} \pd\] Finally, letting \(x = -2\) allows us to obtain the value of \(B \! :\) \[17 = 0 - 32 B \implies B = -\frac{17}{32} \pd\] Our partial fraction decomposition is then \[ \frac{x^4 + 1}{16x - 4x^3} = -\frac{x}{4} + \frac{1}{16x} - \frac{17/32}{2 + x} + \frac{17/32}{2 - x} \pd \] Integrating this expression, we get \[ \ba \int \frac{x^4 + 1}{16x - 4x^3} \di x &= \int -\frac{x}{4} + \frac{1}{16x} - \frac{17/32}{2 + x} + \frac{17/32}{2 - x} \di x \nl &= \boxed{-\frac{1}{8}x^2 + \frac{1}{16} \ln\abs x - \frac{17}{32} \ln|2 + x| - \frac{17}{32} \ln|2 - x| + C } \ea \]
EXERCISE 15
For
\[\ds \int \frac{x}{(2x + 1)(x^2 + 4x + 1)} \di x\]
evaluate the integral.
SOLUTION The partial fraction composition of the integrand is expressed by \[\frac{x}{(2x + 1)(x^2 + 4x + 1)} = \frac{A}{2x + 1} + \frac{Bx + C}{x^2 + 4x + 1} \pd\] We then have \[x = A(x^2 + 4x + 1) + Bx(2x + 1) + C(2x + 1) \pd\] We solve for \(A\) by letting \(x = -1/2 \col\) \[-\frac{1}{2} = -\frac{3}{4}A + 0 + 0 \implies A = \frac{2}{3} \pd \] Another simple substitution is letting \(x = 0,\) from which we obtain \[0 = \frac{2}{3} + 0 + C \implies C = -\frac{2}{3} \pd\] Our equation is now \[x = \frac{2}{3}(x^2 + 4x + 1) + Bx(2x + 1) - \frac{2}{3} (2x + 1) \pd\] To solve for \(B,\) we let \(x\) equal any value; an easy choice is letting \(x = 1\) to attain \[1 = 4 + 3B - 2 \implies B = -\frac{1}{3} \pd \] Our new integral is now \[\int \frac{2/3}{2x + 1} - \frac{x + 2}{3(x^2 + 4x + 1)} \di x \pd\] The integral \(\int \frac{x + 2}{3(x^2 + 4x + 1)} \di x\) can be evaluated using the substitution \(u = x^2 + 4x + 1.\) We then acquire, for our final answer, \[\boxed{\frac{1}{3} \ln|2x + 1| - \frac{1}{6} \ln|x^2 + 4x + 1| + C }\]
EXERCISE 16
For
\[\ds \int \frac{7w^3 - 9}{(w + 2)(4 - w^2)} \di w \]
evaluate the integral.
SOLUTION The degree of the numerator is equal to the degree of the denominator. Thus, we perform polynomial division first to obtain \[-7 + \frac{-14w^2 + 28w + 47}{(w + 2)(4 - w^2)} \pd\] The partial fraction decomposition of \(\frac{-14w^2 + 28w + 47}{(w + 2)(4 - w^2)}\) takes the shape \[ \ba \frac{-14w^2 + 28w + 47}{(w + 2)(4 - w^2)} &= \frac{-14w^2 + 28w + 47}{(w + 2)^2(2 - w)} \nl &= \frac{A}{w + 2} + \frac{B}{(w + 2)^2} + \frac{C}{2 - w} \pd \ea \] Multiplying both sides by \((w + 2)^2 (2 - w),\) we obtain \[-14w^2 + 28w + 47 = A(w + 2) (2 - w) + B(2 - w) + C(w + 2)^2 \pd\] To solve for \(C,\) we let \(w = 2\) to get \[47 = 0 + 0 + 16C \implies C = \frac{47}{16} \pd\] Similarly, letting \(w = -2\) provides \[-65 = 0 + 4B + 0 \implies B = -\frac{65}{4} \pd\] Our resulting equation is \[-14w^2 + 28w + 47 = A(w + 2) (2 - w) - \frac{65}{4}(2 - w) + \frac{47}{16} (w + 2)^2 \pd\] To solve for \(A,\) we assign \(w\) any value. A simple choice is \(w = 0,\) which gives \[47 = 4A - \frac{65}{2} + \frac{47}{4} \implies A = \frac{271}{16} \pd \] We have, therefore, \[ \ba \int \frac{7w^3 - 9}{(w + 2)(4 - w^2)} \di w &= \int -7 + \par{\frac{271/16}{w + 2} - \frac{65/4}{(w + 2)^2} + \frac{47/16}{2 - w} } \di w \nl &= \boxed{-7w + \frac{271}{16} \ln|w + 2| + \frac{65/4}{w + 2} - \frac{47}{16} \ln|2 - w| + C } \ea \]
EXERCISE 17
For
\[\ds \int \frac{5}{2x^3 + 3x} \di x\]
evaluate the integral.
SOLUTION We factor the denominator to get \[\frac{5}{2x^3 + 3x} = \frac{5}{x(2x^2 + 3)} \cma\] whose partial fraction decomposition is \[\frac{5}{x(2x^2 + 3)} = \frac{A}{x} + \frac{Bx + C}{2x^2 + 3} \pd\] We then get \[5 = A(2x^2 + 3) + Bx^2 + Cx \pd \] To solve for \(A,\) we let \(x = 0\) and get \[5 = 3A + 0 + 0 \implies A = \frac{5}{3} \pd\] We now have \[5 = \frac{5}{3} (2x^2 + 3) + Bx^2 + Cx \pd\] At this point, there aren't any more convenient substitutions. We can, however, establish a system of equations to find \(B\) and \(C.\) This procedure involves attaining equations by assigning \(x\) any value. Letting, say, \(x = 1\) gives \[ 5 = \frac{25}{3} + B + C \implies 3B + 3C = -10 \pd \] Another simple value is \(x = -1,\) from which we get \[5 = \frac{25}{3} + B - C \implies 3B - 3C = -10 \pd\] Thus, our system is \[ \ba 3B + 3C &= -10 \nl 3B - 3C &= -10 \cma \ea \] which has solutions \(B = -10/3\) and \(C = 0.\) We have, therefore, \[ \ba \int \frac{5}{2x^3 + 3x} \di x &= \int \frac{5/3}{x} + \frac{-10x/3}{2x^2 + 3} \di x \nl &= \int \frac{5}{3x} - \frac{10x}{6x^2 + 9} \di x \pd \ea \] To evaluate \(\int \frac{10x}{6x^2 + 9} \di x,\) we let \(u = 6x^2 + 9.\) Then \(\dd u = 12x \di x,\) and we get \[ \ba \int \frac{10x}{6x^2 + 9} \di x = \frac{10}{12} \int \frac{1}{u} \di u &= \frac{5}{6} \ln|u| + C \nl &= \frac{5}{6} \ln(6x^2 + 9) + C \pd \ea \] Therefore, \[ \int \frac{5/3}{x} + \frac{-10x/3}{2x^2 + 3} \di x = \boxed{\frac{5}{3} \ln\abs x - \frac{5}{6} \ln(6x^2 + 9) + C } \]
EXERCISE 18
The region bounded by the curve \(y = 7/x,\) the \(x\)-axis, and the lines \(x = 1\) and \(x = 2\) is the base of a solid whose cross sections perpendicular to the \(x\)-axis are rectangles of height \(1/(x + 4)^2.\) Calculate the solid's volume.
SOLUTION At any \(x\) the rectangle has a cross-sectional area of \[A(x) = \par{\frac{7}{x}} \par{\frac{1}{(x + 4)^2}} = \frac{7}{x (x + 4)^2} \pd\] The solid's volume is therefore given by \(V = \int_1^2 A(x) \di x.\) (See Section 5.2 to review calculating solids with known cross sections.) The rational function \(A(x)\) can be decomposed into partial fractions as follows: \[\frac{7}{x (x + 4)^2} = \frac{A}{x} + \frac{B}{x + 4} + \frac{C}{(x + 4)^2} \pd\] Multiplying both sides by \(x(x + 4)^2\) to clear the denominators, we get \[7 = A(x + 4)^2 + Bx(x + 4) + Cx \pd\] Letting \(x = 0\) shows \[7 = 16A + 0 + 0 \implies A = \frac{7}{16} \pd\] Similarly, we let \(x = -4\) to see \[7 = 0 + 0 - 4C \implies C = -\frac{7}{4} \pd\] To find \(B\) we let \(x\) be any other number—say, \(x = 1 \col\) \[7 = 25 \par{\frac{7}{16}} + 5B - \frac{7}{4} \implies B = -\frac{7}{16} \pd\] We therefore find the volume to be \[ \ba V &= \int_1^2 \par{\frac{7/16}{x} - \frac{7/16}{x + 4} - \frac{7/4}{(x + 4)^2}} \di x \nl &= \par{\tfrac{7}{16} \ln \abs x - \tfrac{7}{16} \ln \abs{x + 4} + \frac{7/4}{x + 4}} \intEval_1^2 \nl &= \boxed{\tfrac{7}{16} \ln \tfrac{5}{3} - \tfrac{7}{120}} \approx 0.165 \pd \ea \]
EXERCISE 19
On \(1 \leq x \leq 4,\) find the average value of \[f(x) = \frac{2x + 5}{x^2 (x + 4)} \pd\]
SOLUTION The average value is \[\avg f = \frac{1}{4 - 1} \int_1^4 \frac{2x + 5}{x^2 (x + 4)} \di x = \tfrac{1}{3} \int_1^4 \frac{2x + 5}{x^2 (x + 4)} \di x \pd\] (See Section 5.6 to review average values of functions.) We split the rational expression into partial fractions as follows: \[\frac{2x + 5}{x^2 (x + 4)} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x + 4} \pd\] Multiplying both sides by \(x^2 (x + 4)\) clears the denominators, giving us \[2x + 5 = Ax(x + 4) + B(x + 4) + Cx^2 \pd\] Letting \(x = 0\) shows \[0 + 5 = 0 + 4B + 0 \implies B = \frac{5}{4} \pd\] Likewise, letting \(x = -4\) gives \[-3 = 0 + 0 + 16C \implies C = -\frac{3}{16} \pd\] To find \(A\) we let \(x\) be any other value—say, \(x = 1 \col\) \[7 = 5A + 5 \par{\frac{5}{4}} - \frac{3}{16}(1) \implies A = \frac{3}{16} \pd\] We therefore have \[ \ba \avg f &= \tfrac{1}{3} \int_1^4 \par{\frac{3/16}{x} + \frac{5/4}{x^2} - \frac{3/16}{x + 4}} \di x \nl &= \tfrac{1}{3} \par{\tfrac{3}{16} \ln \abs x - \frac{5}{4x} - \tfrac{3}{16} \ln \abs{x + 4}} \intEval_1^4 \nl &= \boxed{\tfrac{5}{16} + \tfrac{1}{16} \ln \tfrac{5}{2}} \ea \]
EXERCISE 20
An object placed on the \(x\)-axis experiences a force given by \[F(x) = \frac{x^2 + 1}{x^3 + 5x^2 + 6x} \cma\] where \(F(x)\) is measured in newtons and \(x\) is measured in meters. How much work is done on the object as it is moved from \(x = 1\) to \(x = 5 \ques\)
SOLUTION The work done is \(W = \int_1^5 F(x) \di x.\) Since \[x^3 + 5x^2 + 6x = x (x^2 + 5x + 6) = x(x + 2)(x + 3) \cma\] the rational function \(F\) can be split into partial fractions as follows: \[\frac{x^2 + 1}{x^3 + 5x^2 + 6x} = \frac{A}{x} + \frac{B}{x + 2} + \frac{C}{x + 3} \pd\] Multiplying both sides by \(x(x + 2)(x + 3)\) gives \[x^2 + 1 = A(x + 2)(x + 3) + Bx(x + 3) + Cx(x + 2) \pd\] Letting \(x = 0,\) we get \[0 + 1 = 6A + 0 + 0 \implies A = \frac{1}{6} \pd\] Likewise, letting \(x = -3\) shows \[10 = 0 + 0 + 3C \implies C = \frac{10}{3} \pd\] To solve for \(B\) we let \(x\) be any other value—say, \(x = 1 \col\) \[2 = \frac{1}{6} (12) + 4B + \frac{10}{3} (3) \implies B = -\frac{5}{2} \pd\] So the work, in joules, is \[ \ba W &= \int_1^5 \par{\frac{1/6}{x} - \frac{5/2}{x + 2} + \frac{10/3}{x + 3}} \di x \nl &= \par{\tfrac{1}{6} \ln \abs x - \tfrac{5}{2} \ln \abs{x + 2} + \tfrac{10}{3} \ln \abs{x + 3}} \intEval_1^5 \nl &= \boxed{\tfrac{1}{6} \ln 5 + \tfrac{5}{2} \ln \tfrac{3}{7} + \tfrac{10}{3} \ln 2} \approx 0.460 \un{J} \pd \ea \]

By factoring the denominator as \(x (x^{2023} - 1)\) and splitting the integrand into two fractions, evaluate \[\int \frac{1}{x^{2024} - x} \di x \pd\]
SOLUTION Observe that \[\frac{1}{x^{2024} - x} = \frac{-1}{x} + \frac{x^{2022}}{x^{2023} - 1} \pd\] By substituting \(u = x^{2023} - 1,\) we get \(\dd u = 2023 x^{2022} \di x;\) thus, we see \[ \ba \int \frac{x^{2022}}{x^{2023} - 1} \di x &= \int \frac{1}{2023 u} \di u \nl &= \tfrac{1}{2023} \ln \abs u + C \nl &= \tfrac{1}{2023} \ln \abs{x^{2023} - 1} + C \pd \ea \] Hence, we have \[ \int \frac{1}{x^{2024} - x} \di x = \boxed{-\ln \abs x + \tfrac{1}{2023} \ln \abs{x^{2023} - 1} + C} \]
EXERCISE 22
For
\[\ds \int \frac{3 \cos x}{2 \sin^2 x + 5 \sin x} \di x\]
evaluate the integral by first performing a substitution.
SOLUTION Letting \(u = \sin x,\) we get \(\dd u = \cos x \di x.\) Then the integral becomes \[\int \frac{3}{2u^2 + 5u} \di u \pd\] Since the denominator is factorable as \(u(2u + 5),\) we attain the partial fraction shape \[\frac{3}{u(2u + 5)} = \frac{A}{u} + \frac{B}{2u + 5} \pd\] Multiplying across by \(u(2u + 5)\) to clear the denominators, we attain \[3 = A(2u + 5) + Bu \pd\] Substituting \(u = 0,\) we see \[3 = 5A + 0 \implies A = \frac{3}{5} \pd\] Then letting \(u = -5/2\) shows \[3 = 0 - \frac{5}{2} B \implies B = -\frac{6}{5} \pd\] Accordingly, we see \[ \ba \int \frac{3}{2u^2 + 5u} \di u &= \int \par{\frac{3/5}{u} - \frac{6/5}{2u + 5}} \di u \nl &= \tfrac{3}{5} \ln \abs u - \tfrac{3}{5} \ln \abs{2u + 5} + C \nl &= \boxed{\tfrac{3}{5} \ln \abs{\sin x} - \tfrac{3}{5} \ln \abs{2 \sin x + 5} + C} \ea \]
EXERCISE 23
For
\[\ds \int \frac{e^x}{e^{2x} - 5e^x + 6} \di x\]
evaluate the integral by first performing a substitution.
SOLUTION Substituting \(u = e^x,\) we have \(\dd u = e^x \di x.\) Then the integral becomes \[ \int \frac{1}{u^2 - 5u + 6} \di u \pd \] We factor the denominator as \((u - 3)(u - 2),\) so we split the rational expression into partial fractions as follows: \[\frac{1}{(u - 3)(u - 2)} = \frac{A}{u - 3} + \frac{B}{u - 2} \pd\] Multiplying across by \((u - 3)(u - 2)\) to clear the denominators, we get \[1 = A(u - 2) + B(u - 3) \pd\] Substituting \(u = 2\) shows \[1 = 0 - B \implies B = -1 \pd\] Similarly, letting \(u = 3\) gives \[1 = A + 0 \implies A = 1 \pd\] The integral therefore becomes \[ \ba \int \par{\frac{1}{u - 3} - \frac{1}{u - 2}} \di u &= \ln \abs{u - 3} - \ln \abs{u - 2} + C \nl &= \ln \abs{\frac{u - 3}{u - 2}} + C \nl &= \boxed{\ln \abs{\frac{e^x - 3}{e^x - 2}} + C} \ea \]
EXERCISE 24
For

evaluate the integral by first performing a substitution.
SOLUTION We first perform the trigonometric substitution \(x = \tan \theta.\) Then \(\dd x = \sec^2 \theta \di \theta\) and \[ \ba \sqrt{x^2 + 1} &= \sqrt{\tan^2 \theta + 1} = \abs{\sec \theta} \cma \nl \sqrt{16x^2 + 16} &= \sqrt{16 \tan^2 \theta + 16} = 4 \abs{\sec \theta} \pd \ea \] Since we work with an indefinite integral, we drop the absolute value bars for simplicity. The integral therefore becomes \[ \int \frac{\tan \theta}{(\sec \theta + 5)(4 \sec \theta - 1)} \, \sec^2 \theta \di \theta \pd \] Next we substitute \(u = \sec \theta,\) from which \(\dd u = \sec \theta \tan \theta \di \theta.\) Then the integral becomes \[\int \frac{u}{(u + 5)(4u - 1)} \di u \cma\] where the integrand can be decomposed as follows: \[\frac{u}{(u + 5)(4u - 1)} = \frac{A}{u + 5} + \frac{B}{4u - 1} \pd\] Multiplying both sides by \((u + 5)(4u - 1)\) enables us to clear the denominators and so get \[u = A(4u - 1) + B(u + 5) \pd\] Letting \(u = -5,\) we find \[-5 = -21A + 0 \implies A = \frac{5}{21} \pd\] Also, substituting \(u = 1/4\) shows \[\frac{1}{4} = 0 + \frac{21}{4} B \implies B = \frac{1}{21} \pd\] Thus, we have \[ \ba \int \frac{u}{(u + 5)(4u - 1)} \di u &= \int \par{\frac{5/21}{u + 5} + \frac{1/21}{4u - 1}} \di u \nl &= \tfrac{5}{21} \ln \abs{u + 5} + \tfrac{1}{84} \ln \abs{4u - 1} + C \nl &= \tfrac{5}{21} \ln \abs{\sec \theta + 5} + \tfrac{1}{84} \ln \abs{4 \sec \theta - 1} + C \pd \ea \] Because \(x = \tan \theta,\) it follows that \(\sec \theta = \sqrt{x^2 + 1}.\) The result therefore becomes \[\boxed{\tfrac{5}{21} \ln \par{\sqrt{x^2 + 1} + 5} + \tfrac{1}{84} \ln \par{4 \sqrt{x^2 + 1} - 1} + C}\] (The absolute value bars are unnecessary, so we drop them.)
