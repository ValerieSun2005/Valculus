2.6 Exercise Solutions: Logarithmic Differentiation

EXERCISE 1
For
\[\ds y = e^x \cos x\]
find the derivative using logarithmic differentiation.
SOLUTION We take the natural logarithm of each side: \[\ln y = \ln[e^x \cos x]\pd\] Since \(\ln[e^x \cos x] = \ln(e^x) + \ln(\cos x) = x + \ln(\cos x),\) the equation becomes \[\ln y = x + \ln(\cos x) \pd\] We now differentiate both sides; on the left \(y\) is a function of \(x,\) so we use the Chain Rule: \[\tfrac{1}{y} \deriv{y}{x} = 1 - \frac{\sin x}{\cos x} = 1 - \tan x \pd\] We solve for \(\textDeriv{y}{x}\) by multiplying both sides by \(y = e^x \, \cos x \col\) \[ \ba \deriv{y}{x} &= e^x \cos x (1 - \tan x) \nl &= \boxed{e^x(\cos x - \sin x)} \ea \]
EXERCISE 2
For
\[\ds y = (\sin 3x)^{x + 7}\]
find the derivative using logarithmic differentiation.
SOLUTION Taking the natural logarithm of both sides gives \[\ln y = \ln \left[(\sin 3x)^{x + 7}\right] \pd\] Using logarithm properties then gives \[\ln y = (x + 7) \ln(\sin 3x) \pd\] We then perform Implicit Differentiation to obtain \(\textDeriv{y}{x} \col\) \[ \ba \tfrac{1}{y} \deriv{y}{x} &= \ln(\sin 3x) + (x + 7) \frac{3 \cos 3x}{\sin 3x} \nl &= \ln(\sin 3x) + 3(x + 7) \cot 3x \nl \deriv{y}{x} &= y \left[\ln(\sin 3x) + 3(x + 7) \cot 3x\right] \nl &= \boxed{(\sin 3x)^{x + 7} \left[\ln(\sin 3x) + 3(x + 7) \cot 3x\right]} \ea \]
EXERCISE 3
For
\[\ds y = x^4 \, 9^x\]
find the derivative using logarithmic differentiation.
SOLUTION We take the natural logarithm of each side: \[\ln y = \ln \left[x^4 \, 9^x \right] \pd\] Using logarithm properties then gives \[\ln y = 4 \ln x + x \ln 9 \pd\] Differentiating both sides then gives \[\tfrac{1}{y} \deriv{y}{x} = \frac{4}{x} + \ln 9 \pd\] Solving for \(\textDeriv{y}{x},\) we obtain \[ \ba \deriv{y}{x} &= y \left(\frac{4}{x} + \ln 9\right) \nl &= \boxed{x^4 \, 9^x \left(\frac{4}{x} + \ln 9\right)} \ea \]
EXERCISE 4
For
\[\ds y = \frac{x \sin x}{\sqrt{x^3 + 1}}\]
find the derivative using logarithmic differentiation.
SOLUTION We take the natural logarithm of both sides, getting \[ \ba \ln y &= \ln \left[\frac{x \sin x}{\sqrt{x^3 + 1}} \right] \nl &= \ln[x \sin x] - \ln \left[\sqrt{x^3 + 1} \right] \nl &= \ln x + \ln(\sin x) - \tfrac{1}{2} \ln[x^3 + 1] \pd \ea \] Differentiating the last expression gives \[\tfrac{1}{y} \deriv{y}{x} = \frac{1}{x} + \frac{\cos x}{\sin x} - \frac{1}{2} \left(\frac{3x^2}{x^3 + 1}\right) \pd\] Solving for \(\textDeriv{y}{x}\) then provides \[ \ba \deriv{y}{x} &= y \left[\frac{1}{x} + \cot x - \frac{1}{2} \left(\frac{3x^2}{x^3 + 1}\right)\right] \nl &= \boxed{\frac{x \sin x}{\sqrt{x^3 + 1}} \left[\frac{1}{x} + \cot x - \frac{1}{2} \left(\frac{3x^2}{x^3 + 1}\right)\right]} \ea \]
EXERCISE 5
For
\[\ds y = \frac{x^2 - 6}{2^{\cos x}}\]
find the derivative using logarithmic differentiation.
SOLUTION Taking the natural logarithm of both sides produces \begin{align*} \ln y &= \ln \frac{x^2 - 6}{2^{\cos x}} \nonumber \nl &= \ln(x^2 - 6) - \cos(x) \ln 2 \period \label{eq:ex-2-lny} \end{align*} Differentiating both sides and solving for \(\textDeriv{y}{x}\) yield \[ \ba \tfrac{1}{y} \deriv{y}{x} &= \frac{2x}{x^2 - 6} + \sin(x) \ln 2 \nl \deriv{y}{x} &= \boxed{\frac{x^2 - 6}{2^{\cos x}} \left(\frac{2x}{x^2 - 6} + \sin(x) \ln 2\right)} \ea \]
EXERCISE 6
For
\[\ds y = t^{2t}\]
find the derivative using logarithmic differentiation.
SOLUTION Inserting the natural logarithm on both sides yields \[\ln y = \ln \left[t^{2t}\right] = 2t \ln t \pd\] Then differentiating gives \[\tfrac{1}{y} \deriv{y}{t} = 2 \ln t + 2 \pd\] Solving for \(\textDeriv{y}{t},\) we find \[ \ba \deriv{y}{t} &= y[2 \ln t + 2] \nl &= \boxed{t^{2t} \, [2 \ln t + 2]} \ea \]
EXERCISE 7
For
\[\ds y = \frac{3 - x}{5^x \tan x} \]
find the derivative using logarithmic differentiation.
SOLUTION Taking the natural logarithm of both sides gives \[ \ba \ln y &= \ln \left[\frac{3 - x}{5^x \tan x}\right] \nl &= \ln[3 - x] - \ln \left[5^x \tan x \right] \nl &= \ln[3 - x] - \ln(\tan x) - x \ln 5 \pd \ea \] Differentiating, we get \[\tfrac{1}{y} \deriv{y}{x} = \frac{1}{x - 3} - \frac{\sec^2 x}{\tan x} - \ln 5 \pd\] Now we solve for \(\textDeriv{y}{x} \col\) \[ \ba \deriv{y}{x} &= y \left[ \frac{1}{x - 3} - \frac{\sec^2 x}{\tan x} - \ln 5\right] \nl &= \boxed{\frac{3 - x}{5^x \tan x} \left[ \frac{1}{x - 3} - \frac{\sec^2 x}{\tan x} - \ln 5\right]} \ea \]
EXERCISE 8
For
\[\ds y = \frac{\sec^2 x}{x^3 + \ln x}\]
find the derivative using logarithmic differentiation.
SOLUTION We use logarithms to rewrite the equation as \[ \ba \ln y &= \ln \left[\frac{\sec^2 x}{x^3 + \ln x}\right] \nl &= \ln[\sec^2 x] - \ln \left[x^3 + \ln x\right] \nl &= 2 \ln(\sec x) - \ln[x^3 + \ln x] \pd \ea \] We then take the derivative of each side: \[ \ba \tfrac{1}{y} \deriv{y}{x} &= 2 \frac{\sec x \tan x}{\sec x} - \frac{3x^2 + \frac{1}{x}}{x^3 + \ln x} \nl &= 2 \tan x - \frac{3x^2 + \frac{1}{x}}{x^3 + \ln x} \pd \ea \] Solving for \(\textDeriv{y}{x}\) then yields \[ \ba \deriv{y}{x} &= y \left[2 \tan x - \frac{3x^2 + \frac{1}{x}}{x^3 + \ln x}\right] \nl &= \boxed{\frac{\sec^2 x}{x^3 + \ln x} \left[2 \tan x - \frac{3x^2 + \frac{1}{x}}{x^3 + \ln x}\right]} \ea \]
EXERCISE 9
For
\[\ds y = \sqrt{\frac{x^2 - x}{x^3 + 2}}\]
find the derivative using logarithmic differentiation.
SOLUTION Taking the natural logarithm of both sides and using logarithm properties give \[ \ba \ln y &= \ln \sqrt{\frac{x^2 - x}{x^3 + 2}} \nl &= \tfrac{1}{2} \ln \left[\frac{x^2 - x}{x^3 + 2}\right] \nl &= \tfrac{1}{2} \ln \left[x^2 - x \right] - \tfrac{1}{2} \ln \left[x^3 + 2 \right] \pd \ea \] Differentiating, we obtain \[\tfrac{1}{y} \deriv{y}{x} = \frac{2x - 1}{2(x^2 - x)} - \frac{3x^2}{2(x^3 + 2)} \pd\] Multiplying both sides by \(y = \sqrt{(x^2 - x)/(x^3 + 2)}\) then gives \[ \deriv{y}{x} = \boxed{\sqrt{\frac{x^2 - x}{x^3 + 2}} \left[\frac{2x - 1}{2(x^2 - x)} - \frac{3x^2}{2(x^3 + 2)}\right]} \]
EXERCISE 10
For
\[\ds y = \left(\frac{x^2 \sin x}{x - 1}\right)^2\]
find the derivative using logarithmic differentiation.
SOLUTION We take the natural logarithm of both sides and use logarithm properties: \[ \ba \ln y &= \ln \left[ \left(\frac{x^2 \sin x}{x - 1}\right)^2 \right] \nl &= 2 \ln \left[\frac{x^2 \sin x}{x - 1}\right] \nl &= 2 \ln x^2 + 2 \ln(\sin x) - 2 \ln(x - 1) \nl &= 4 \ln x + 2 \ln(\sin x) - 2 \ln(x - 1) \pd \ea \] Differentiating then gives \[\tfrac{1}{y} \deriv{y}{x} = \frac{4}{x} + 2 \frac{\cos x}{\sin x} - \frac{2}{x - 1} \pd\] Multiplying both sides by \(y = \left[(x^2 \sin x)/(x - 1)\right]^2\) gives \[\deriv{y}{x} = \boxed{\left(\frac{x^2 \sin x}{x - 1}\right)^2 \left[\frac{4}{x} + 2 \cot x - \frac{2}{x - 1}\right]}\]
EXERCISE 11
For
\[\ds y = \frac{e^x}{x \cos(x^2 - 1)}\]
find the derivative using logarithmic differentiation.
SOLUTION We utilize logarithms: \[ \ba \ln y &= \ln \left[\frac{e^x}{x \cos(x^2 - 1)} \right] \nl &= \ln(e^x) - \ln \left[x \cos(x^2 - 1)\right] \nl &= x - \ln x - \ln \left[\cos(x^2 - 1)\right] \pd \ea \] Differentiating then produces \[ \ba \tfrac{1}{y} \deriv{y}{x} &= 1 - \frac{1}{x} + \frac{2x \sin(x^2 - 1)}{\cos(x^2 - 1)} \nl &= 1 - \frac{1}{x} + 2x \tan(x^2 - 1) \pd \ea \] Solving for \(\textDeriv{y}{x}\) then gives \[ \ba \deriv{y}{x} &= y \left[1 - \frac{1}{x} + 2x \tan(x^2 - 1) \right] \nl &= \boxed{\frac{e^x}{x \cos(x^2 - 1)} \left[1 - \frac{1}{x} + 2x \tan(x^2 - 1) \right]} \ea \]
EXERCISE 12
For
\[\ds y = \sqrt[3]{\frac{e^t \sin t}{t^2}}\]
find the derivative using logarithmic differentiation.
SOLUTION We take the natural logarithm of both sides and use logarithm properties: \[ \ba \ln y &= \ln \sqrt[3]{\frac{e^t \sin t}{t^2}} \nl &= \tfrac{1}{3} \ln \frac{e^t \sin t}{t^2} \nl &= \tfrac{1}{3} \ln(e^t) + \tfrac{1}{3} \ln(\sin t) - \tfrac{1}{3} \ln(t^2) \nl &= \tfrac{1}{3} t + \tfrac{1}{3} \ln(\sin t) - \tfrac{2}{3} \ln|t| \pd \ea \] Differentiating, we obtain \[\tfrac{1}{y} \deriv{y}{t} = \frac{1}{3} + \frac{\cos t}{3 \sin t} - \frac{2}{3t} \pd\] Multiplying both sides by \(y = \sqrt[3]{(e^t \sin t)/t^2}\) then gives \[\deriv{y}{t} = \boxed{\sqrt[3]{\frac{e^t \sin t}{t^2}} \left[\frac{1}{3} + \frac{1}{3} \cot t - \frac{2}{3t}\right]}\]
EXERCISE 13
For
\[\ds y = \frac{u^2 e^u}{\sin(u) \ln u} \]
find the derivative using logarithmic differentiation.
SOLUTION Taking the natural logarithm of each side and using logarithm properties give \[ \ba \ln y &= \ln \left[\frac{u^2 e^u}{\sin(u) \ln u} \right] \nl &= \ln \left[u^2 e^u\right] - \ln[\sin(u) \ln u] \nl &= \ln(u^2) + \ln(e^u) - \ln(\sin u) - \ln(\ln u) \nl &= 2 \ln u + u - \ln(\sin u) - \ln(\ln u) \pd \ea \] Differentiating then gives \[ \ba \tfrac{1}{y} \deriv{y}{u} &= \frac{2}{u} + 1 - \frac{\cos u}{\sin u} - \frac{1/u}{\ln u} \nl &= \frac{2}{u} + 1 - \cot u - \frac{1}{u \ln u} \pd \ea \] We now solve for \(\textDeriv{y}{u} \col\) \[ \ba \deriv{y}{u} &= y \left[\frac{2}{u} + 1 - \cot u - \frac{1}{u \ln u}\right] \nl &= \boxed{\frac{u^2 e^u}{\sin(u) \ln u} \left[\frac{2}{u} + 1 - \cot u - \frac{1}{u \ln u}\right]} \ea \]
