1.1 Exercise Solutions: Defining a Limit

EXERCISE 1
Interpret the limit statement \[\lim_{x \to a} f(x) = L \pd\]
SOLUTION This limit statement asserts that the values of \(f(x)\) become and closer and closer to \(L\) as we make \(x\) closer and closer to \(a\) but not equal to \(a.\) In arrow notation, we can write \(f(x) \to L\) as \(x \to a.\)
EXERCISE 2
Let \(f\) be a function that satisfies \[\lim_{x \to 3^-} f(x) = 5 \and \lim_{x \to 3^+} f(x) = -8 \pd\]
  1. Does \(\lim_{x \to 3} f(x)\) exist?
  2. What do these one-sided limits tell you about \(f(3) \ques\)
SOLUTION
  1. No, this limit does not exist. Since the one-sided limits of \(f\) at \(3\) do not equal each other, \(\lim_{x \to 3} f(x)\) does not exist.
  2. Nothing. The values of one-sided limits provide no information about the value of \(f(3)\) or whether it exists. Limits only tell us what number a function approaches, not what value the function outputs.
EXERCISE 3
If \(\lim_{x \to -1} f(x) = 3,\) then evaluate the following expressions if possible.
  1. \(\ds \lim_{x \to -1^-} f(x)\)
  2. \(\ds \lim_{x \to -1^+} f(x)\)
  3. \(f(-1)\)
SOLUTION
  1. The statement \(\lim_{x \to -1} f(x) = 3\) is true if and only if the one-sided limits to \(f\) at \(-1\) also equal \(3.\) Thus, \[\lim_{x \to -1^-} f(x) = \boxed 3\]
  2. Since \(\lim_{x \to -1} f(x) = 3,\) the one-sided limits to \(f\) at \(-1\) must also equal \(3.\) So \[\lim_{x \to -1^+} f(x) = \boxed 3\]
  3. Impossible to evaluate. The value of \(\lim_{x \to -1} f(x)\) is unrelated to the value of \(f(-1).\) We only know that \(f(x)\) approaches \(3\) as \(x\) approaches \(-1,\) but we don't know what value \(f\) takes on at \(3\) or whether \(f(3)\) is defined.
EXERCISE 4
For the graph of \(y = f(x)\) in Figure 10, evaluate the following limits if they exist.
  1. \(\ds \lim_{x \to 6^-} f(x)\)
  2. \(\ds \lim_{x \to -1^+} f(x)\)
  3. \(\ds \lim_{x \to 1} f(x)\)
  4. \(\ds \lim_{x \to 3} f(x)\)
SOLUTION
  1. We trace the graph of \(f\) from the left of \(x = 6.\) For \(x\) near \(6\) and less than \(6,\) the value of \(f(x)\) is constantly \(-1.\) Accordingly, \(f(x) \to -1\) as \(x \to 6^-,\) or \[\lim_{x \to 6^-} f(x) = \boxed{-1}\]
  2. Tracing the graph of \(f\) from the right of \(x = -1,\) we see that \(f(x)\) approaches the value \(2.\) So \[\lim_{x \to -1^+} f(x) = \boxed 2\]
  3. As we trace the graph of \(f\) from the left of \(x = 1,\) we see \(f(x)\) approaches \(3.\) Conversely, as we trace the graph of \(f\) from the right of \(x = 1,\) we observe that \(f(x)\) tends to \(5.\) So \[\lim_{x \to 1^-} f(x) = 3 \and \lim_{x \to 1^+} f(x) = 5 \pd\] Since these one-sided limits do not equal each other, \(\lim_{x \to 1} f(x)\) does not exist.
  4. Tracing \(f\) from the left of \(x = 3,\) we notice the value of \(f(x)\) approaching \(1.\) But when we trace \(f\) from the right of \(x = 3,\) we see \(f(x)\) approaching \(-1.\) Accordingly, \[\lim_{x \to 3^-} f(x) = 1 \and \lim_{x \to 3^+} f(x) = -1 \pd\] These one-sided limits don't agree with each other, meaning \(\lim_{x \to 3} f(x)\) does not exist.
EXERCISE 5
For the graph of \(y = f(x)\) in Figure 11, evaluate the following limits if they exist.

  2. We trace the graph of \(f\) from the left of \(x = 6,\) seeing that \(f(x)\) approaches \(-5.\) Thus, \[\lim_{x \to 6^-} f(x) = \boxed{-5}\]
  3. Tracing the graph of \(f\) from the left of \(x = -3,\) we see \(f(x)\) approaches \(4.\) Yet tracing the graph from the right of \(x = -3\) shows \(f(x)\) approaching \(3.\) Thus, \[\lim_{x \to -3^-} f(x) = 4 \and \lim_{x \to -3^+} f(x) = 3 \pd\] Since these one-sided limits are not equivalent, \(\lim_{x \to -3} f(x)\) does not exist.

From the left of \(x = 2,\) \(f(x)\) approaches (roughly) \(5.5.\) But from the right of \(x = 2,\) \(f(x)\) approaches \(1.\) Accordingly, \[\lim_{x \to 2^-} f(x) = 5.5 \and \lim_{x \to 2^+} f(x) = 1 \pd\] Because these limits don't agree with each other, \(\lim_{x \to 2} f(x)\) does not exist.
EXERCISE 6
Evaluate the following limits, if they exist, for the graph of \(y = f(x)\) in Figure 12.
  1. \(\ds \lim_{x \to 4} f(x)\)
  2. \(\ds \lim_{x \to 2} f(x)\)
  3. \(\ds \lim_{x \to -1} f(x)\)
SOLUTION
  1. From both sides of \(x = 4,\) \(f(x)\) approaches \(-2.\) Thus, \[\lim_{x \to 4} f(x) = \boxed{-2}\]
  2. Because \(f(x)\) doesn't approach the same value as we trace the graph from the left of \(x = 2\) or as we trace from the right of \(x = 2,\) \(\lim_{x \to 2} f(x)\) does not exist.
  3. The graph of \(f(x)\) oscillates as \(x\) approaches \(-1.\) Through the endless waves, \(f\) does not settle to a final value as we make \(x\) close to \(-1.\) So \(\lim_{x \to -1} f(x)\) does not exist.
EXERCISE 7
The function \(g\) is a piecewise function defined by \[g(x) = \bc 6 &x \leq 0 \nl 5 &x \gt 0 \pd \ec \] For what value of \(a\) does \(\lim_{x \to a} g(x)\) fail to exist?
SOLUTION The limit \(\lim_{x \to a} g(x)\) fails to exist if the one-sided limits of \(g\) at \(a\) aren't equal—that is, if \[\lim_{x \to a^-} g(x) \ne \lim_{x \to a^+} g(x) \pd\] We investigate \(x = 0,\) the point of the piecewise split. As \(x\) approaches \(0\) from the left of \(x = 0,\) \(g(x)\) approaches \(6.\) So \[\lim_{x \to 0^-} g(x) = 6 \pd\] But as \(x\) approaches \(0\) from the right of \(x = 0,\) \(g(x)\) tends to \(5.\) Accordingly, \[\lim_{x \to 0^+} g(x) = 5 \pd\] Since these one-sided limits do not equal each other, \(\lim_{x \to 0} g(x)\) does not exist, meaning \(\boxed{a = 0}.\)

EXERCISE 21
For
\[\ds \lim_{x \to 4^-} \frac{\abs{x - 4}}{x - 4}\]
determine the exact value of the limit or justify why it does not exist.
SOLUTION For \(x \lt 4,\) the quantity \((x - 4)\) is negative. Accordingly, over this interval \(\abs{x - 4}\) \(= -(x - 4),\) meaning \[\frac{\abs{x - 4}}{x - 4} = \frac{-(x - 4)}{x - 4} = -1 \pd\] Thus, as we trace the graph from the left of \(x = 4,\) we simply trace the flat line \(y = -1,\) so \[\lim_{x \to 4^-} \frac{\abs{x - 4}}{x - 4} = \boxed{-1}\] If you substitute any \(x \lt 4\) (but \(x \ne 4\)) into the fraction—for example, \(x = 3.5\) or \(x = 3.9\)—then you see that \(\abs{x - 4}/(x - 4) = -1.\)
EXERCISE 22
For
\[\ds \lim_{x \to 6^+} \frac{\abs{x - 6}}{x - 6}\]
determine the exact value of the limit or justify why it does not exist.
SOLUTION For \(x \gt 6,\) the quantity \((x - 6)\) is positive. Thus, over this interval \(\abs{x - 6}\) \(= x - 6,\) meaning \[\frac{\abs{x - 6}}{x - 6} = \frac{x - 6}{x - 6} = 1 \pd\] If we trace the graph of \(\abs{x - 6}/(x - 6)\) from the right of \(x = 6,\) then we are simply tracing the flat line \(y = 1.\) So \[\lim_{x \to 6^+} \frac{\abs{x - 6}}{x - 6} = \boxed{1}\] If you substitute any \(x \gt 6\) (but \(x \ne 6\)) into the fraction—for example, \(x = 6.5\) or \(x = 6.1\)—then you see that \(\abs{x - 6}/(x - 6) = 1.\)

EXERCISE 25
For
\[\ds \lim_{x \to 3^-} \frac{\abs{3x - 9}}{3 - x}\]
determine the exact value of the limit or justify why it does not exist.
SOLUTION For \(x \lt 3,\) the quantity \((3x - 9)\) is negative. To eliminate the absolute value bars, we write \(\abs{3x - 9}\) \(= -(3x - 9)\) for \(x \lt 3,\) so \[ \ba \frac{\abs{3x - 9}}{3 - x} &= \frac{-(3x - 9)}{3 - x} \nl &= \frac{9 - 3x}{3 - x} \nl &= 3 \cdot \frac{3 - x}{3 - x} \nl &= 3 \pd \ea \] Thus, the function \(\abs{3x - 9}/(3 - x)\) constantly evaluates to \(3\) for \(x \lt 3.\) Therefore, \[\lim_{x \to 3^-} \frac{\abs{3x - 9}}{3 - x} = \boxed 3\]
