1.5 Exercise Solutions: Formal Definition of a Limit

EXERCISE 1
Because \(\lim_{x \to 1} \par{x^3 - 2x + 7}\) \(= 6,\) a number \(\delta \gt 0\) exists such that for any \(\varepsilon,\) \[\abs{\par{x^3 - 2x + 7} - 6} \lt \varepsilon \if \abs{x - 1} \lt \delta \pd\] Using a graphing calculator, find \(\delta\) if \(\varepsilon = 0.05.\)
SOLUTION Because \(\varepsilon = 0.05,\) the curve must lie between the lines \[y = 6 - 0.05 = 5.95 \and y = 6 + 0.05 = 6.05\] when \(\abs{x - 1} \lt \delta.\) The curve intersects the line \(y = 5.95\) when \(x = 0.939\) and intersects the line \(y = 6.05\) when \(x = 1.044.\) So for \(0.939 \lt x \lt 1.044,\) the curve is between the two lines. We need \(\delta\) to be the smaller difference in \(x\) from \(1\)â€”namely, \(\delta = \abs{1 - 1.044} = \boxed{0.044}.\)

EXERCISE 4
For
\[\ds \lim_{x \to 2} \par{6x - 4} = 8\]
prove the limit statement.
SOLUTION

Let \(\varepsilon \gt 0\) be any number. We want to find a number \(\delta \gt 0\) such that \[ \abs{(6x - 4) - 8} \lt \varepsilon \if \abs{x - 2} \lt \delta \pd \] The first inequality becomes \[\abs{6x - 12} = 6 \abs{x - 2} \lt \varepsilon \iffArrow \abs{x - 2} \lt \frac{\varepsilon}{6} \pd\] Comparing this inequality to \(\abs{x - 2} \lt \delta,\) we can equate \(\delta\) to the bound \(\varepsilon/6.\) So we choose \(\delta = \varepsilon/6.\)

Constructing the Proof With \(\delta = \varepsilon/6\) we see, for \(\abs{x - 2} \lt \delta,\) \[\abs{(6x - 4) - 8} = \abs{6x - 12} = 6 \abs{x - 2} \lt 6 \delta = 6 \par{\frac{\varepsilon}{6}} = \varepsilon \pd\] Accordingly, \[\abs{(6x - 4) - 8} \lt \varepsilon \if \abs{x - 2} \lt \frac{\varepsilon}{6} \pd\] Thus, \(\lim_{x \to 2} (6x - 4) = 8.\)

EXERCISE 6

\[\ds \lim_{x \to 2} (x - 2)^2 = 0\]

SOLUTION

Let \(\varepsilon \gt 0\) be any number. We want to find a number \(\delta \gt 0\) such that \[ \abs{(x - 2)^2 - 0} \lt \varepsilon \if \abs{x - 2} \lt \delta \cma \] that is, \[(x - 2)^2 \lt \varepsilon \if \abs{x - 2} \lt \delta \pd\] In the first inequality, taking the square root of both sides gives \(\abs{x - 2} \lt \sqrt \varepsilon.\) Comparing this inequality to \(\abs{x - 2} \lt \delta,\) we select \(\delta = \sqrt \varepsilon.\)

Constructing the Proof With \(\delta = \sqrt \varepsilon\) we see, for \(\abs{x - 2} \lt \delta,\) \[\abs{(x - 2)^2 - 0} = (x - 2)^2 \lt \delta^2 = \par{\sqrt \varepsilon}^2 = \varepsilon \pd\] Accordingly, \[ \abs{(x - 2)^2 - 0} \lt \varepsilon \if \abs{x - 2} \lt \sqrt \varepsilon \cma \] which proves that \(\lim_{x \to 2} (x - 2)^2 = 0.\)

EXERCISE 8
For
\[\ds \lim_{x \to 9^-} \sqrt{18 - 2x} = 0\]
prove the limit statement.
SOLUTION

Let \(\varepsilon \gt 0\) be any number. We want to find a number \(\delta \gt 0\) such that \[ \abs{\sqrt{18 - 2x} - 0} \lt \varepsilon \if 9 - \delta \lt x \lt 9 \pd \] The first inequality becomes simply \(\sqrt{18 - 2x} \lt \varepsilon.\) Squaring both sides, we see \[18 - 2x \lt \varepsilon^2 \iffArrow 9 - x \lt \frac{\varepsilon^2}{2} \pd\] Since \(9 - \delta \lt x \lt 9,\) we have \(9 - x \lt \delta.\) By comparison of inequalities, we select \(\delta = \varepsilon^2/2.\) (After you define \(\delta\) and \(\varepsilon,\) it is a game of algebra to attain a bound for \(9 - x\) from the inequality \(\abs{\sqrt{18 - 2x} - 0} \lt \varepsilon.\) This process is the most difficult in constructing a proof.)

Constructing the Proof With \(\delta = \varepsilon^2/2\) we see, for \(9 - \delta \lt x \lt 9\) (that is, for \(9 - x \lt \delta\)), \[\abs{\sqrt{18 - 2x} - 0} = \sqrt{18 - 2x} = \sqrt{2(9 - x)} \lt \sqrt{2 \delta} = \sqrt{2 \par{\frac{\varepsilon^2}{2}}} = \varepsilon\] (since \(\varepsilon \gt 0\)). Accordingly, \[ \abs{\sqrt{18 - 2x} - 0} \lt \varepsilon \if 9 - \frac{\varepsilon^2}{2} \lt x \lt 9 \pd \] This proves that \(\lim_{x \to 9^-} \sqrt{18 - 2x} = 0.\)

EXERCISE 9
For
\[\ds \lim_{x \to 4^+} \sqrt[4]{3x - 12} = 0\]
prove the limit statement.
SOLUTION

Let \(\varepsilon \gt 0\) be any number. We want to find a number \(\delta \gt 0\) such that \[ \abs{\sqrt[4]{3x - 12} - 0} \lt \varepsilon \if 4 \lt x \lt 4 + \delta \pd \] The first inequality becomes simply \(\sqrt[4]{3x - 12} \lt \varepsilon.\) Raising both sides to the power of \(4\) shows \[3x - 12 \lt \varepsilon^4 \iffArrow 3(x - 4) \lt \varepsilon^4 \iffArrow x - 4 \lt \frac{\varepsilon^4}{3} \pd\] From \(4 \lt x \lt 4 + \delta\) we have \(x - 4 \lt \delta.\) By comparison of inequalities, we select \(\delta = \varepsilon^4/3.\) This concludes the game of algebra in bounding the factor \(x - 4.\)

Constructing the Proof With \(\delta = \varepsilon^4/3\) we see, for \(4 \lt x \lt 4 + \delta\) (that is, for \(x - 4 \lt \delta\)), \[\abs{\sqrt[4]{3x - 12} - 0} = \sqrt[4]{3x - 12} = \sqrt[4]{3(x - 4)} \lt \sqrt[4]{3 \delta} = \sqrt[4]{3 \par{\frac{\varepsilon^4}{3}}} = \varepsilon\] (since \(\varepsilon \gt 0\)). So \[ \abs{\sqrt[4]{3x - 12} - 0} \lt \varepsilon \if 4 \lt x \lt 4 + \frac{\varepsilon^4}{3} \cma \] which proves that \(\lim_{x \to 4^+} \sqrt[4]{3x - 12} = 0.\)

EXERCISE 10
For
\[\ds \lim_{x \to 0^+} \frac{1}{2x} = \infty\]
prove the limit statement.
SOLUTION

Let \(M \gt 0\) be any number. We want to find a number \(\delta \gt 0\) such that \[ \frac{1}{2x} \gt M \if 0 \lt x \lt 0 + \delta \cma \] that is, \[ \frac{1}{2x} \gt M \if 0 \lt x \lt \delta \pd \] Let's reciprocate both sides of the first inequality (remembering to also flip the inequality sign); doing so yields \[2x \lt \frac{1}{M} \implies x \lt \frac{1}{2M} \pd\] Comparing this inequality to \(x \lt \delta,\) we should choose \(\delta = 1/2M.\)

Constructing the Proof With \(\delta = 1/2M\) we see, for \(0 \lt x \lt \delta,\) \[\frac{1}{2x} \gt \frac{1}{2 \delta} = \frac{1}{2 \par{\dfrac{1}{2M}}} = M \pd\] It follows that \[ \frac{1}{2x} \gt M \if 0 \lt x \lt 0 + \frac{1}{2M} \cma \] therefore proving that \(\lim_{x \to 0^+} \par{1/2x} = 0.\)

EXERCISE 11
For
\[\ds \lim_{x \to 6^+} \frac{1}{\sqrt{x - 6}} = \infty\]
prove the limit statement.
SOLUTION

Let \(M \gt 0\) be any number. We want to find a number \(\delta \gt 0\) such that \[ \frac{1}{\sqrt{x - 6}} \gt M \if 6 \lt x \lt 6 + \delta \cma \] from which we get \(x - 6 \lt \delta.\) By reciprocating and then squaring both sides, we convert the first inequality into \[\sqrt{x - 6} \lt \frac{1}{M} \or x - 6 \lt \frac{1}{M^2} \pd\] This concludes our mission of bounding \(x - 6\) in terms of \(M.\) Comparing this inequality to \(x - 6 \lt \delta,\) we should choose \(\delta = 1/M^2.\)

Constructing the Proof With \(\delta = 1/M^2\) we see, for \(6 \lt x \lt 6 + \delta,\) \[\frac{1}{\sqrt{x - 6}} \gt \frac{1}{\sqrt \delta} = \frac{1}{\sqrt{1/M^2}} = M \pd\] So \[ \frac{1}{\sqrt{x - 6}} \gt M \if 6 \lt x \lt 6 + \frac{1}{M^2} \cma \] therefore proving that \(\lim_{x \to 6^+} \par{1/\sqrt{x - 6}} = \infty.\)

EXERCISE 12
For
\[\ds \lim_{x \to 7^+} \frac{8}{7 - x} = -\infty\]
prove the limit statement.
SOLUTION

Let \(M\) be some negative number that bounds \(8/(7 - x)\) from above. We want to find a number \(\delta \gt 0\) such that \[ \frac{8}{7 - x} \lt M \if 7 \lt x \lt 7 + \delta \pd \] Because \(7 \lt x \lt 7 + \delta,\) we have \(x - 7 \lt \delta;\) in the first inequality, let's try to bound \(x - 7.\) We reciprocate both sides of the first inequality (remembering to also flip the inequality sign) to get \[\frac{7 - x}{8} \gt \frac{1}{M} \iffArrow 7 - x \gt \frac{8}{M} \iffArrow x - 7 \lt -\frac{8}{M}\pd\] We therefore select \(\delta = -8/M.\)

Constructing the Proof With \(\delta = -8/M\) we see, for \(7 \lt x \lt 7 + \delta\) (or \(x - 7 \lt \delta\)), \[\frac{8}{7 - x} = -\frac{8}{x - 7} \lt -\frac{8}{\delta} = -\frac{8}{-8/M} = M \pd\] So \[ \frac{8}{7 - x} \lt M \if 7 \lt x \lt 7 - \frac{8}{M} \pd \] Therefore, we have proved that \(\lim_{x \to 7^+} [8/(7 - x)] = -\infty.\)

We want to find a number \(\delta \gt 0\) such that \[ \ln x \lt M \if 0 \lt x \lt \delta \pd \] In the first inequality, we perform algebraic manipulation as follows: \[\ln x \lt M \iffArrow x \lt e^M \pd\] Comparing this form to \(x \lt \delta,\) we use \(\delta = e^M.\)

EXERCISE 15
For
\[\ds \lim_{x \to 3} \sqrt{5x + 1} = 4\]
prove the limit statement.
SOLUTION

The limit statement is true if and only if, for positive \(\varepsilon\) and \(\delta,\) \[ \abs{\sqrt{5x + 1} - 4} \lt \varepsilon \if \abs{x - 3} \lt \delta \pd \] In the first inequality, multiplying by the conjugate \(\sqrt{5x + 1} - 4\) shows \[ \ba \abs{\sqrt{5x + 1} - 4} &= \abs{\par{\sqrt{5x + 1} - 4} \cdot \frac{\sqrt{5x + 1} + 4}{\sqrt{5x + 1} + 4}} \nl &= \abs{\frac{5x - 15}{\sqrt{5x + 1} + 4}} = 5 \abs{\frac{x - 3}{\sqrt{5x + 1} + 4}} \lt \varepsilon \pd \ea \] We want to express \(\abs{x - 3}\) as an inequality with \(\varepsilon.\) But this task isn't easy, so let's compromise by bounding the factor using a larger number. Observe that \(1/\par{\sqrt{5x + 1} + 4}\) \(\lt 1/4;\) hence, \[5 \abs{\frac{x - 3}{\sqrt{5x + 1} + 4}} \lt 5 \abs{\frac{x - 3}{4}} \implies 5 \abs{\frac{x - 3}{4}} \lt \varepsilon \pd\] Accordingly, \(\abs{x - 3} \lt 4 \varepsilon/5,\) so we choose \(\delta = 4 \varepsilon/5.\)

Constructing the Proof For \(0 \lt \abs{x - 3} \lt \delta,\) we see \[\abs{\sqrt{5x + 1} - 4} = 5 \abs{\frac{x - 3}{\sqrt{5x + 1} + 4}} \lt 5 \abs{\frac{x - 3}{4}} \lt \frac{5}{4} \delta = \frac{5}{4} \par{\frac{4}{5} \varepsilon} = \varepsilon \pd\] So we have \[ \abs{\sqrt{5x + 1} - 4} \lt \varepsilon \if \abs{x - 3} \lt \frac{4}{5} \varepsilon \pd \] This statement proves that \(\lim_{x \to 3} \sqrt{5x + 1} = 4.\)

EXERCISE 21
For any \(k,\) prove that \(\lim_{x \to k^-} \sqrt{k - x} = 0.\)
SOLUTION Given \(\varepsilon \gt 0,\) we want to find a number \(\delta \gt 0\) for which \[\abs{\sqrt{k - x} - 0} \lt \varepsilon \if k - \delta \lt x \lt k \pd\] The first inequality becomes \[\sqrt{k - x} \lt \varepsilon \iffArrow k - x \lt \varepsilon^2 \iffArrow k - \varepsilon^2 \lt x \pd\] Comparing this inequality to \(k - \delta \lt x \lt k,\) we select \(\delta = \varepsilon^2.\) From \(k - \delta \lt x \lt k,\) we observe that \(k - x \lt \delta.\) Then for \(k - \delta \lt x \lt k\) we see \[\abs{\sqrt{k - x} - 0} = \sqrt{k - x} \lt \sqrt{\delta} = \sqrt{\varepsilon^2} = \varepsilon \pd\] So \(\lim_{x \to k^-} \sqrt{k - x} = 0\) is true because \[\abs{\sqrt{k - x} - 0} \lt \varepsilon \if k - \varepsilon^2 \lt x \lt k \pd\]
EXERCISE 22
For \(n \gt 0,\) prove that \[\lim_{x \to (-b/a)^+} \sqrt[n]{ax + b} = 0 \pd\]
SOLUTION Given \(\varepsilon \gt 0,\) we want to find a number \(\delta \gt 0\) such that \[\abs{\sqrt[n]{ax + b} - 0} \lt \varepsilon \if -\frac{b}{a} \lt x \lt -\frac{b}{a} + \delta \pd\] The first inequality becomes \[\sqrt[n]{ax + b} \lt \varepsilon \iffArrow ax + b \lt \varepsilon^n \iffArrow x + \frac{b}{a} \lt \frac{\varepsilon^n}{a} \pd\] From \(-b/a \lt x \lt -b/a + \delta\) we identify \(x + b/a \lt \delta.\) Thus, by comparison of inequalities we select \(\delta = \varepsilon^n/a.\) We therefore see, for \(-b/a \lt x \lt -b/a + \delta,\) \[\sqrt[n]{ax + b} = \sqrt[n]{a \par{x + \frac{b}{a}}} \lt \sqrt[n]{a \delta} = \sqrt[n]{a \par{\frac{\varepsilon^n}{a}}} = \varepsilon \pd\] Accordingly, \(\lim_{x \to (-b/a)^+} \sqrt[n]{ax + b} = 0\) because \[\abs{\sqrt[n]{ax + b} - 0} \lt \varepsilon \if -\frac{b}{a} \lt x \lt -\frac{b}{a} + \frac{\varepsilon^n}{a} \pd\]
