3.4: Particle Motion

Isaac Newton invented calculus to model motion in physics. In Section 2.1 we used calculus to connect position, velocity and acceleration. Yet properties of first and second derivatives enable us to further analyze these contexts. In this section we study properties of particle motion: calculating when an object changes direction, how much distance it covers, and whether it is speeding up or slowing down.

Let's first review the fundamental terminology in particle motion. Suppose that a particle travels along a line with time \(t.\) Then its position (from a reference point) is given by some function of time, namely, \(s(t).\) If \(s_1\) is the particle's position at \(t = t_1\) and \(s_2\) is its position at some later time \(t = t_2,\) then the particle's displacement is \(\Delta s\) \(= s_2 - s_1.\) In words, displacement is the difference in a particle's position. (If you walk \(5\) units north and then \(4\) units south, then your displacement is \(1\) unit north.) Recall, from Section 2.1, that the velocity function is the time derivative of position: \[v(t) = \deriv{s}{t} \pd\] The acceleration function is the time derivative of velocity—namely, \[a(t) = \deriv{v}{t} = \derivOrder{s}{t}{2} \pd\] The axis along which the particle travels contains a positive direction and a negative direction. For example, if a particle moving to the right is defined to have a positive velocity, then its velocity becomes negative upon turning around and moving to the left. Speed is the absolute value of velocity, so we define a particle's speed function to be \[\abs{v(t)} = \abs{s'(t)} \pd\] Hence, velocity is the speed in a certain direction.

EXAMPLE 1
An object travels in a straight line with time as modeled by the position function \(s(t) = t^3 + 3t - 4\) for \(t \geq 0.\) Calculate the object's initial velocity and acceleration.
The word initial means at \(t = 0.\) Taking derivatives, we see \[ \baat{2} v(t) &= s'(t) &&= 3t^2 + 3 \cma \nl a(t) &= v'(t) &&= 6t \pd \eaat \] So the initial velocity and acceleration are, respectively, \[ \ba v(0) &= 3(0)^2 + 3 = \boxed{3} \nl a(0) &= 6(0) = \boxed 0 \ea \]

An object changes direction when its velocity function swaps sign. The First-Derivative Test (from Section 3.1) asserts that a function has relative extrema when its derivative function changes sign. Since \(s'(t)\) \(= v(t),\) the times at which a particle changes direction correspond to the relative extrema of \(s(t).\)

EXAMPLE 2
A particle travels along a line with time \(t \geq 0\) as represented by the position function \[s(t) = \tfrac{1}{3} t^3 - 5t^2 + 16t + 4 \pd\] When does the particle change direction?
The particle changes direction when the velocity function \(v(t)\) \(= s'(t)\) switches sign. Differentiating and factoring, we see \[ \ba v(t) &= t^2 - 10t + 16 \nl &= (t - 8)(t - 2) \pd \ea \] The zeros of \(v(t)\) are \(t = 2\) and \(t = 8;\) the following table gives the function's sign across the intervals bounded by these zeros.
Interval of \(t\) \(v(t) = (t - 8)(t - 2)\)
\((0, 2)\) \(+\)
\((2, 8)\) \(-\)
\((8, \infty)\) \(+\)
Hence, the particle turns around at \(\boxed{t = 2}\) and \(\boxed{t = 8}.\) The graph of \(s(t)\) is shown in Figure 1, and a (figurative) schematic diagram of the particle's path is shown in Figure 2.
EXAMPLE 3
A bicyclist rides along the \(x\)-axis with time \(t \geq 0\) as modeled by the position function \[x(t) = 2t^4 - 8t^3 + 8t^2 + 16 \pd\] Calculate the total distance the bicyclist travels from \(t = 0\) to \(t = 3.\)
Do not confuse distance with displacement. The bicyclist's displacement over \(0 \leq t \leq 3\) is \[x(3) - x(0) = 18 \pd\]
By differentiating, the velocity function is \[ \ba v(t) &= 8t^3 - 24t^2 + 16t \nl &= 8t (t^2 - 3t + 2) \nl &= 8t (t - 2)(t - 1) \pd \ea \] The zeros of \(v(t)\) are \(t = 0,\) \(t = 1,\) and \(t = 2.\) The following table shows the function's sign across the intervals bounded by these zeros.
Interval of \(t\) \(v(t) = 8t (t - 2)(t - 1)\) Displacement
\((0, 1)\) \(+\) \(x(1) - x(0) = 2\)
\((1, 2)\) \(-\) \(x(2) - x(1) = -2\)
\((2, 3)\) \(+\) \(x(3) - x(2) = 18\)
The bicycle's turning points are at \(t = 1\) and \(t = 2.\) The total distance traveled over \(0 \leq t \leq 3\) is the sum of the absolute values of the displacements—namely, \[d = \abs{2} + \abs{-2} + \abs{18} = \boxed{22}\] Geometrically, these absolute value expressions are the lengths of the horizontal segments of the schematic diagram in Figure 3.

When you step on the gas pedal, the car is propelled in the same direction it's facing, so you speed up. In contrast, by pressing the brake pedal, the car is pushed in the opposite direction and so you slow down. Thus, a particle is speeding up if the acceleration is in the direction of motion, and slowing down if the acceleration is opposite the direction of motion. For example, a particle with negative velocity and negative acceleration is speeding up. But the particle is slowing down if the acceleration is positive, since it is being pushed in the positive direction (opposing its motion in the negative direction). We summarize this logic as follows.

SPEEDING UP AND SLOWING DOWN
A particle moving along a straight line is
  1. speeding up if its velocity and acceleration have the same sign.
  2. slowing down if its velocity and acceleration have opposite signs.
EXAMPLE 4
An object moves along the \(y\)-axis with time \(t \geq 0\) according to the position function \(y(t) = \sin^2 4t.\) When \(t = \pi/12,\) is the object speeding up or slowing down?
Differentiating \(y(t)\) gives the velocity function \[ \ba v(t) &= 8 \sin 4t \cos 4t \nl &= 4 \sin 8t \pd \ea \] (The last step is true by the double-angle identity for sine.) Then the acceleration function is \(v'(t),\) that is, \[a(t) = 32 \cos 8t \pd\] We see \(v(\pi/12) = 2 \sqrt 3\) and \(a(\pi/12) = -16.\) Because these signs are different, the object is slowing down.

If a particle travels along a line with time \(t,\) as modeled by the position function \(s(t),\) then its velocity function is \(v(t) = \textderiv{s}{t}\) and its acceleration function is \(a(t)\) \(= \textderiv{v}{t}\) \(= \textderivOrder{s}{t}{2}.\) If \(s(t_1) = s_1\) and \(s(t_2) = s_2,\) where \(t_2 \gt t_1,\) then the particle's displacement on \([t_1, t_2]\) is \(\Delta s =\) \(s_2 - s_1.\) Since speed is the absolute value of velocity, we define a particle's speed function to be \(\abs{v(t)}.\) The times at which a particle changes direction correspond to the relative extrema of \(s(t);\) the object switches direction when the velocity function changes sign. In addition, a particle is

  1. speeding up if its velocity and acceleration have the same sign.
  2. slowing down if its velocity and acceleration have opposite signs.
