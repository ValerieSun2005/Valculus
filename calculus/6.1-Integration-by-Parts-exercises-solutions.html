6.1 Exercise Solutions: Integration by Parts

EXERCISE 1
For
\[\ds \int (2x + 1) \cos x \di x\]
evaluate the integral.
SOLUTION The factor \((2x + 1)\) becomes simpler when differentiated. So let's try Integration by Parts with \[ \baat{2} u &= 2x + 1 \lspace &\dd v &= \cos x \di x \nl \dd u &= 2 \di x \lspace &v &= \sin x \pd \eaat \] Hence, we find \[ \ba \int (2x + 1) \cos x \di x &= (2x + 1) \sin x - \int (\sin x)(2) \di x \nl &= \boxed{2x \sin x + \sin x + 2 \cos x + C} \ea \]

For
\[\ds \int x e^{7x} \di x\]
evaluate the integral.
SOLUTION The factor \(x\) becomes simpler when differentiated. Let's therefore perform Integration by Parts with \[ \baat{2} u &= x \lspace &\dd v &= e^{7x} \di x \nl \dd u &= 1 \di x = \dd x \lspace &v &= \tfrac{1}{7} e^{7x} \pd \eaat \] We then see \[ \ba \int x e^{7x} \di x &= x \par{\tfrac{1}{7} e^{7x}} - \int \par{\tfrac{1}{7} e^{7x}} (1) \di x \nl &= \boxed{\tfrac{1}{7} x e^{7x} - \tfrac{1}{49} e^{7x} + C} \ea \]

For
\[\ds \int_0^{\pi/2} x \cos x \di x\]
evaluate the integral.
SOLUTION The function \(x\) becomes simpler when differentiated. Let's therefore perform Integration by Parts with \[ \baat{2} u &= x \lspace &\dd v &= \cos x \di x \nl \dd u &= 1 \di x = \dd x \lspace &v &= \sin x \pd \eaat \] We therefore see \[ \ba \int_0^{\pi/2} x \cos x \di x &= \par{x \sin x} \intEval_0^{\pi/2} - \int_0^{\pi/2} \sin x \di x \nl &= \par{x \sin x + \cos x} \intEval_0^{\pi/2} \nl &= \par{\frac{\pi}{2} \sin \frac{\pi}{2} + \cos \frac{\pi}{2}} - \par{0 + \cos 0} \nl &= \boxed{\frac{\pi}{2} - 1} \ea \]
EXERCISE 4
For
\[\ds \int_1^e x \ln x \di x\]
evaluate the integral.
SOLUTION In using Integration by Parts, we generally reserve \(u\) to be a logarithmic function or an inverse trigonometric function. The function \(\ln x\) becomes simpler when integrated, so let's choose \[ \baat{2} u &= \ln x \lspace &\dd v &= x \di x \nl \dd u &= \frac{1}{x} \di x \lspace &v &= \frac{x^2}{2} \pd \eaat \] We then have \[ \ba \int_1^e x \ln x \di x &= \par{\frac{x^2}{2} \ln x} \intEval_1^e - \int_1^e \par{\frac{x^2}{2}} \par{\frac{1}{x}} \di x \nl &= \par{\tfrac{1}{2} x^2 \ln x} \intEval_1^e - \tfrac{1}{2} \int_1^e x \di x \nl &= \par{\tfrac{1}{2} x^2 \ln x - \tfrac{1}{4} x^2} \intEval_1^e \nl &= \boxed{\tfrac{1}{4} \par{e^2 + 1}} \ea \]

EXERCISE 13
For
\[\ds \int_0^1 \atan x \di x\]
evaluate the integral.
SOLUTION To maintain organization, it is a good idea to first determine an antiderivative and then use the Fundamental Theorem of Calculus.

Antidifferentiation As we perform Integration by Parts, our only option is \[ \baat{2} u &= \atan x \lspace &\dd v &= \dd x \nl \dd u &= \frac{1}{x^2 + 1} \di x \lspace &v &= x \pd \eaat \] Doing so, we see \[ \int \atan x \di x = x \atan x - \int \frac{x}{x^2 + 1} \di x \pd \] To evaluate the integral on the right side, we substitute \(w = x^2 + 1.\) Then \(\dd w = 2x \di x\) and so we see \[ \ba \int \frac{x}{x^2 + 1} \di x &= \int \frac{1}{2w} \di w \nl &= \tfrac{1}{2} \ln \abs w + C \nl &= \tfrac{1}{2} \ln \par{x^2 + 1} + C \pd \ea \] (We exclude the absolute value bars around the natural logarithm because \(x^2 + 1\) is always positive.) Hence, we get \[ \int \atan x \di x = x \atan x - \tfrac{1}{2} \ln \par{x^2 + 1} + C \pd \]

Definite Integral By Part II of the Fundamental Theorem of Calculus, we have \[ \ba \int_0^1 \atan x \di x &= \parbr{x \atan x - \tfrac{1}{2} \ln \par{x^2 + 1}} \intEval_0^1 \nl &= \par{1 \atan 1 - \tfrac{1}{2} \ln 2} - \par{0} \nl &= \boxed{\frac{\pi}{4} - \ln \sqrt 2} \approx 0.439 \pd \ea \]
