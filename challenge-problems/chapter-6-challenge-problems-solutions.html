Chapter 6 Challenge Problems Solutions

EXERCISE 1
For
\[\ds \int \frac{\tan^3 x + \tan x}{6 \tan^2 x - 2} \di x\]
evaluate the integral.
SOLUTION By factoring the numerator, the integral becomes \[ \int \frac{\tan x (\tan^2 x + 1)}{6 \tan^2 x - 2} \di x = \frac{\tan x \sec^2 x}{6 \tan^2 x - 2} \di x \pd \] Substituting \(u = 6 \tan^2 x - 2,\) we attain \(\dd u = 12 \tan x \sec^2 x \di x\) and so the integral becomes \[ \ba \int \frac{1}{12 u} \di u &= \tfrac{1}{12} \ln \abs u + C \nl &= \boxed{\tfrac{1}{2} \ln \par{6 \tan^2 x - 2} + C} \ea \]
EXERCISE 2
For
\[\ds \int \frac{1}{\sin^6 x + \cos^6 x} \di x\]
evaluate the integral.
SOLUTION Multiplying the numerator and denominator each by \(\sec^6 x\) yields \[\int \frac{\sec^6 x}{\tan^6 x + 1} \di x \pd\] Since the power of secant is even, we can conserve an extra \(\sec^2 x\) factor, similar to the strategy in Section 6.2. Because \(\sec^2 x\) \(= \tan^2 x + 1,\) we have \[\int \frac{\sec^4 x \sec^2 x}{\tan^6 x + 1} \di x = \int \frac{(\tan^2 x + 1)^2 \sec^2 x}{\tan^6 x + 1} \di x \pd\] Substituting \(u = \tan x,\) we attain \(\dd u = \sec^2 x \di x\) and so the integral becomes \[ \ba \int \frac{(u^2 + 1)^2}{u^6 + 1} \di u &= \int \frac{(u^2 + 1)^2}{(u^2 + 1)(u^4 - u^2 + 1)} \di u \nl &= \int \frac{u^2 + 1}{u^4 - u^2 + 1} \di u \pd \ea \] Dividing the numerator and denominator each by \(u^2\) yields \[ \int \frac{1 + \dfrac{1}{\strut u^2}}{u^2 - 1 + \dfrac{\strut 1}{u^2}} \di u = \int \frac{1 + \dfrac{1}{\strut u^2}}{\par{u - \dfrac{1}{u}}^2 + 1} \di u \pd \] Letting \(v = u - 1/u,\) we get \(\dd v = (1 + 1/u^2) \di u\) and thus \[ \ba \int \frac{1}{v^2 + 1} \di v &= \atan v + C \nl &= \atan \par{u - \frac{1}{u}} + C \nl &= \atan \par{\tan x - \frac{1}{\tan x}} + C \nl &= \boxed{\atan \par{\tan x - \cot x} + C} \ea \]
EXERCISE 3
For
\[\ds \int \par{\cos^4 x - \sin^4 x} e^{\sqrt x} \sec 2x \sqrt x \di x\]
evaluate the integral.
SOLUTION While it appears menacing, the integrand can be drastically simplified. Note that \[ \ba \cos^4 x - \sin^4 x &= (\cos^2 x + \sin^2 x)(\cos^2 x - \sin^2 x) \nl &= \cos^2 x - \sin^2 x \nl &= \cos 2x \pd \ea \] Thus, the integral becomes \[\int \cos 2x e^{\sqrt x} \sec 2x \sqrt x \di x = \int \sqrt x \, e^{\sqrt x} \di x \pd\] Substituting \(t = \sqrt x,\) we get \(\dd t = 1/(2 \sqrt x) \di x\) \(= 1/2t \di x.\) Accordingly, \(\dd x = 2t \di t\) and so the integral becomes \[\int t e^t (2t) \di t = \int 2t^2 e^t \di t \pd\] Performing Integration by Parts (Section 6.1) with \[ \baat{2} u &= 2t^2 \lspace &\dd v &= e^t \di t \nl \dd u &= 4t \di t \lspace &v &= e^t \eaat \] yields \[2 t^2 e^t - \int 4t e^t \di t \pd\] We use Integration by Parts again with \[ \baat{2} u &= 4t \lspace &\dd v &= e^t \di t \nl \dd u &= 4 \di t \lspace &v &= e^t \cma \eaat \] attaining \[ \ba 2t^2 e^t - \par{4t e^t - \int 4 e^t \di t} &= 2t^2 e^t - \par{4t e^t - 4 e^t} + C \nl &= 2t^2 e^t - 4t e^t + 4e^t + C \nl &= \boxed{2x e^{\sqrt x} - 4 \sqrt x e^{\sqrt x} + 4 e^{\sqrt x} + C} \ea \]
EXERCISE 4
For what values of \(c\) does \(\int_c^{2c} \frac{1}{x^2 - 8x + 16} \di x\) diverge?
SOLUTION By factoring the denominator, the integral becomes \[\int_c^{2c} \frac{1}{(x - 4)^2} \di x \pd\] The graph of \(y = 1/(x - 4)^2\) has a vertical asymptote at \(x = 4.\) For any \(a \lt 4 \lt b,\) observe that \[ \ba \int_4^b \frac{1}{(x - 4)^2} \di x &= \lim_{k \to 4^+} \int_k^b \frac{1}{(x - 4)^2} \di x \nl &= \lim_{k \to 4^+} \par{-\frac{1}{x - 4}} \intEval_k^b \nl &= \lim_{k \to 4^+} \par{\frac{1}{k - 4} - \frac{1}{b - 4}} \nl &= \infty \pd \ea \] Also, \[ \ba \int_a^4 \frac{1}{(x - 4)^2} \di x &= \lim_{k \to 4^-} \int_a^k \frac{1}{(x - 4)^2} \di x \nl &= \lim_{k \to 4^+} \par{-\frac{1}{x - 4}} \intEval_a^k \nl &= \lim_{k \to 4^+} \par{\frac{1}{a - 4} - \frac{1}{k - 4}} \nl &= -\infty \pd \ea \] Thus, any integral whose bounds run through the vertical asymptote is divergent. Hence, the values of \(c\) must cross \(x = 4,\) meaning \(c \lt 4 \lt 2c.\) With \(4 \lt 2c,\) we have \(c \gt 2.\) We have \[ \ba \{c \mid c \gt 2 \text{ and } c \lt 4\} = \boxed{(2, 4)} \ea \]
EXERCISE 5
For any integer \(n \geq 1,\) develop a formula for \(\int x^n e^x \di x.\)
SOLUTION Let's perform Integration by Parts with \[ \baat{2} u &= x^n \lspace &\dd v &= e^x \di t \nl \dd u &= n x^{n - 1} \di x \lspace &v &= e^x \pd \eaat \] Doing so gives \[\int x^n e^x \di x = x^n e^x - \int n x^{n - 1} e^x \di x \pd\] Performing Integration by Parts again with \[ \baat{2} u &= n x^{n - 1} \lspace &\dd v &= e^x \di x \nl \dd u &= n (n - 1) x^{n - 2} \di x \lspace &v &= e^x \eaat \] shows \[\int x^n e^x \di x = x^n e^x - n x^{n - 1} e^x + \int n (n - 1) x^{n - 2} e^x \di x \pd\] Continuing this pattern, we have \[ \ba \int x^n e^x \di x &= \small x^n e^x - n x^{n - 1} e^x + n (n - 1) (n - 2) x^{n - 2} e^x - n (n - 1)(n - 2)(n - 3) x^{n - 3} e^x + \cdots \nl &= \boxed{\sum_{i = 0}^n x^{n - 1} \, e^x \frac{n!}{(n - i)!}} \ea \] Note that \(0! = 1.\)
